# 查看我的 IP 地址

可以快速查看运行代理程序的机器的公共 IP 地址。这对于调试网络问题、验证 VPN 连接、确认服务器身份或设置防火墙规则非常有用。

## 使用方法

```bash
bash scripts/get-ip.sh
```

## 输出结果

通过查询 `ifconfig.me`，程序会返回公共的 **IPv4** 地址（如果支持 IPv6 的系统也会返回 IPv6 地址）。

示例输出：
```
=== Public IP ===
IPv4: 203.0.113.42
IPv6: 2001:db8::1
```

## 适用场景

- 当用户询问“我的 IP 地址是多少？”或“显示我的公共 IP”时
- 验证用于允许列表的出站 IP 地址
- 检查 VPN 或代理是否处于活动状态
- 确认服务器的网络身份

## 所需条件

- 需要安装 `curl`（大多数系统都已预装）
- 必须能够访问互联网
