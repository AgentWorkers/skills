# Twitter 线程生成器

这是一个由人工智能驱动的 Twitter 线程创建工具，可以将任何主题转化为引人入胜、具有传播力的 Twitter 线程。

## 功能介绍
- 输入主题 → 生成对应的 Twitter 线程
- 优化每个推文的互动效果
- 支持自定义推文的互动元素
- 线程格式化功能
- 支持自动发布

## 设置
```bash
pip install openai
export OPENAI_API_KEY="your_key"
```

## 使用方法
```bash
python thread_gen.py --topic "5 tips for fitness" --length 10
```

## 主要特性
- 丰富的互动元素库（用于提升帖子的传播力）
- 多种线程结构模板可供选择
- 优化推文的互动效果
- 提供详细的线程分析数据
