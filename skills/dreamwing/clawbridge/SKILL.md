# ClawBridge 仪表板

**您的代理，随时都在您的掌中。**

ClawBridge 是一个专为 OpenClaw 设计的轻量级移动 Web 界面。它作为一个辅助进程运行，提供以下功能：

*   🧠 **实时活动 feed**：实时查看代理的运行状态和思考过程（即代理的执行逻辑）。
*   💰 **代币经济**：追踪 340 多种模型的代币消耗情况。
*   🚀 **任务控制**：手动触发定时任务。
*   ⚡ **零配置**：自动检测 VPN 或快速创建隧道。

## 安装

```bash
curl -sL https://clawbridge.app/install.sh | bash
```

请参阅 [README.md](https://github.com/dreamwing/clawbridge/blob/master/README.md) 以获取完整安装说明。
