# 自动验证码破解工具

使用此工具可以通过浏览器自动化的方式解决简单的验证码图片。

## 支持的验证码类型

- 4到6个字符的文本验证码
- 被扭曲的字母数字验证码
- 数字验证码
- 简单旋转的字符验证码
- 算术验证码（例如：`3+8`）

请勿将此工具用于 reCAPTCHA、hCaptcha、滑块验证码或需要点击目标的验证码类型。

## 工作流程

1. 从页面中检测到验证码图片元素。
2. 截取验证码的屏幕截图。
3. 对图片进行预处理（灰度化、对比度调整、缩放、去噪）。
4. 运行光学字符识别（OCR）并清理识别结果。
5. 检测验证码中的算术运算内容（如有需要）。
6. 填写验证码内容，并可选择性地提交。

## 拍摄指南

- 仅截取验证码元素，而非整个页面。
- 在读取验证码图片来源时，仅接受来自 `http` 或 `https` 协议的图片链接。
- 拒绝使用 `javascript:` 或 `file:` 等可疑的图片来源方式。
- 在进行 OCR 处理之前，确保图片符合尺寸和像素限制。

## 返回格式

返回一个结果对象，包含以下信息：

- `solved`：布尔值，表示验证码是否被成功破解
- `value`：破解后的验证码文本
- `type`：验证码类型（字母数字、数字、算术或未知）
- `confidence`：OCR 识别的置信度分数
- `hash`：验证码的 SHA1 哈希值（用于缓存）
- `fromCache`：布尔值，表示是否使用了缓存的结果

## 模块结构

- `solve.js`：处理图片数据的主要模块
- `preprocess.js`：图片预处理模块
- `ocr.js`：OCR 识别及文本清理模块
- `cache.js`：验证码缓存模块
- `browser.js`：为 Playwright、Puppeteer 和 Selenium 提供的自动化辅助模块
