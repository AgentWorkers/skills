# 简单数学技能

能够安全地执行基本的算术运算。

## 功能特点

- 加法、减法、乘法、除法
- 不使用 `eval()` 或 `exec()` 函数
- 输入验证
- 安全的实现方式

## 使用方法

当用户请求计算时，使用标准的 Python 运算符即可。

## 示例

用户：“25 加 17 等于多少？”
机器人：*使用简单数学技能* 结果是 42。
