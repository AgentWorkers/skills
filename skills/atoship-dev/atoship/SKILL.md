# atoship — 您的智能物流管理助手

想象一下，有一个永远不会犯错、永远不会输入错误数据、并且能立即找到从A地到B地运输包裹的最便宜方式的物流管理助手。atoship正是为您的AI助手提供的这样的功能。

安装了这个技能后，您的AI就能成为一个功能齐全的物流管理专家。只需用简单的语言告诉它您的需求，它就能自动完成承运商的选择、价格比较、标签购买以及物流追踪更新等工作。以前需要花费10分钟在各个承运商的网站上操作的任务，现在只需一句话就能完成。

**使用atoship之前的流程：**  
- 打开承运商网站  
- 输入收发地址  
- 在多个标签页中比较价格  
- 复制并粘贴追踪号码  
- 手动更新订单状态  

**使用atoship之后的流程：**  
*“将这个包裹寄给奥斯汀的约翰，选择3天内的最便宜方案。”* 完成！

## 主要功能：  
- **比较运费**：  
  - 在几秒钟内同时获取USPS、FedEx和UPS的实时折扣运费信息。  

- **购买物流标签**：  
  - 在结账时立即购买物流标签，支持PDF、PNG或ZPL格式。  

- **追踪包裹**：  
  - 实时追踪包裹的运输情况，并提供完整的事件记录（适用于所有承运商）。  

- **管理标签**：  
  - 查看、撤销或重新打印之前的物流标签。  

- **查看钱包余额**：  
  - 监控您的邮资信用额度和物流费用支出。  

- **验证地址**：  
  - 在购买前验证收货地址，以避免额外费用。  

## 开始使用方法：  
这个技能无需任何命令行工具（CLI）或额外的软件。您的AI助手会直接使用您的API密钥调用atoship的API。  

**步骤1：创建一个免费的atoship账户**  
在https://atoship.com注册（免费，无需信用卡）。  

**步骤2：获取API密钥**  
进入“仪表盘”→“设置”→“API密钥”，然后创建一个新的密钥。  

**步骤3：设置API密钥**  
（具体代码请参考```bash
export ATOSHIP_API_KEY=ak_live_your_key_here
```）  
或者您也可以在AI助手的环境设置中配置API密钥。  

**步骤4：为钱包充值**  
进入“仪表盘”→“账单”，为钱包充值邮资。标签费用将从您的钱包余额中扣除——您只需支付实际使用的费用。  

> **关于权限的说明：**  
您的API密钥用于授权标签购买和钱包扣费。我们建议：  
- 在测试阶段使用较小的钱包余额（例如20美元）。  
- 开发时使用测试密钥（如`ak_test_...`），测试标签是免费的且不会实际发货。  
- 在“仪表盘”→“账单”→“通知”中设置费用提醒。  
- 随时可以通过“仪表盘”→“设置”→“API密钥”来撤销或更换密钥。

## 物流工作流程：  
### 第1步：比较运费  
我会调用atoship API来获取所有承运商的实时运费信息：  
（具体代码请参考```
From: ZIP code or "City, State"
To:   ZIP code or "City, State"
Weight: e.g. 2oz, 1lb, 500g
Dimensions (optional): length × width × height in inches
```）  
结果会显示每个承运商的服务、价格和预计送达时间（包括USPS、FedEx Ground、FedEx Express、UPS Ground、UPS 2-Day等）。  

### 第2步：购买标签  
选择服务后，我会收集完整的收发地址并购买标签：  
（具体代码请参考```
Sender:    Name, Street, City, State, ZIP
Recipient: Name, Street, City, State, ZIP
```）  
您将获得：  
- ✅ 追踪号码  
- ✅ 标签下载链接（PDF或PNG格式）  
- ✅ 费用将从您的钱包中扣除。  
如果标签在承运商规定的有效期内未被使用，您可以申请退款。  

### 第3步：追踪包裹  
提供追踪号码后，我会显示完整的物流事件记录：  
（具体代码请参考```
Status:    IN TRANSIT
Location:  Memphis, TN
ETA:       Feb 19, 2026
Events:    Feb 17 10:42 — Departed facility, Memphis TN
           Feb 17 06:15 — Arrived at USPS facility
           Feb 16 18:30 — Accepted at origin post office
```）

## 常见用途：  
- **电子商务订单处理**：无需切换页面，即可自动为Shopify、eBay、Etsy或Amazon的订单选择最便宜的承运商。  
- **小型企业物流**：比较不同承运商（如USPS First Class、Priority Mail、FedEx Ground、UPS Ground）的费用，节省每笔订单的成本。  
- **代发货与第三方物流**：将atoship的API集成到您的物流系统中，批量生成和追踪包裹。  
- **国际物流**：支持通过USPS International、FedEx International和UPS Worldwide进行跨境运输（覆盖加拿大、英国、澳大利亚等200多个国家）。  
- **退货管理**：通过一个命令即可生成预付的退货标签。  
- **批量物流**：使用https://atoship.com的仪表盘进行CSV文件导入和批量标签生成。  

## 使用技巧：  
- **查询最便宜的运输方式**：输入“寄X件包裹最便宜的方式是什么？”  
- **支持的单位**：盎司（oz）、磅（lb）、克（g）、千克（kg）。  
- **标签格式**：默认为PDF，支持PNG和ZPL（适用于热敏打印机）。  
- **是否需要签名**：购买标签时可以选择“需要签名确认”。  
- **是否需要保险**：可以选择“添加100美元的保险”。  
- **参考编号**：可以在标签上标注“参考编号：ORDER-123”。  

## 为什么选择atoship？  
物流是运营在线业务时最耗时的环节之一。每个订单都需要登录承运商网站、手动比较价格、复制地址、下载标签并逐一更新追踪信息。对于每天处理数十到数百个包裹的团队来说，这是个巨大的工作负担。  

atoship彻底解决了这些问题。通过将您的AI助手与atoship平台连接起来，您将获得一个高效的物流管理助手：  
- **响应迅速**：瞬间完成所有承运商的价格比较。  
- **避免地址错误**：数据从对话到标签的传输过程都是结构化的，无需手动重新输入。  
- **记住上下文**：AI会记住您要运输的物品、目的地和用途。  
- **灵活扩展**：无论每天发送1件还是1000件包裹，工作流程都是一样的。  
- **节省成本**：提供折扣运费，无最低订单量要求，也无需支付月费。  

atoship专为电子商务卖家、小型企业主和物流协调员设计，帮助他们无需复杂的集成即可自动化物流流程。

## 安全性与API密钥保护：  
此技能会代表您调用atoship的REST API（https://atoship.com/api/v1）。它不会在磁盘上写入任何文件，也不会访问您的系统数据，仅使用您提供的`ATOSHIP_API_KEY`。  

**影响钱包的API操作：**  
- `purchase_label`：从您的钱包余额中扣除标签费用。  
- `void_label`：在承运商规定的有效期内办理退款。  

**仅读的API操作：**  
- `get_shipping_rates`、`track_package`、`list_labels`、`get_label`、`get_account`、`list_carrier_accounts`、`validate_address`。  

为降低风险，您可以在“仪表盘”→“设置”→“API密钥”中创建一个仅读的API密钥，并限制其访问的IP地址或使用次数。
