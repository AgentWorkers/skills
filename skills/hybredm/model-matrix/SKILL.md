# 模型矩阵技能

使用此技能，根据综合评分和政策约束为每个类别选择最佳模型。

## 核心政策
1. 选择质量达标且成本最低的模型。
2. 如果 Anthropic 不可用，则自动选择第二名模型。
3. 仅当模型评分差异显著且置信度较高时，才进行模型切换。

## 权重分配
- 实际任务评估：45%
- 基准测试：30%
- 情感分析（X/Reddit）：20%
- 成本：5%

## 当前模型分配规则
- 研究/规划：Gemini 3.1 Pro
- 复杂编码/复杂任务/企业讨论：GPT-5.3 Codex
- 常规编码/重复性任务：GPT-5-mini
- 公民情感分析（X）：Grok (x_search)
- 图片/图像处理：Gemini 图像处理工具
- 视频智能分析：Grok 生态系统
