# network-scan

## 描述

该技能使用 `nmap` 工具扫描网络中的端口和设备。它支持多种目标格式，包括 CIDR 表示法、IP 范围以及逗号分隔的列表。默认情况下，对于 CIDR 地址，会使用快速扫描模式。

## 输入参数

*   `target`：要扫描的目标网络或 IP 地址（例如：192.168.1.0/24、192.168.1.1-192.168.1.10、192.168.1.1,192.168.1.2）。
*   `ports`：要扫描的端口（例如：80、443）。
*   `--quick`：（可选）仅扫描前 10 个端口。
*   `--fast`：（可选）使用高级扫描设置（`nmap -T4`）。
*   `--timeout`：（可选）设置扫描的超时时间（单位：秒，默认值：30）。
*   `--top-ports`：（可选）扫描使用频率最高的 N 个端口（例如：`--top-ports 100`）。
*   `--hosts-limit`：（可选）限制扫描的主机数量（例如：`--hosts-limit 50`）。
*   `--exclude`：（可选）从扫描中排除特定的 IP 地址（例如：`--exclude 192.168.1.5`）。

## 输出结果

输出是一个 JSON 对象，其中包含 `nmap` 的扫描结果、使用的命令以及扫描的相关信息。

## 使用方法

要使用该技能，请传入 `target` 和 `ports` 参数：

```
network-scan target=192.168.1.0/24 ports=80,443 --fast --timeout 60 --hosts-limit 50 --exclude 192.168.1.100
```
