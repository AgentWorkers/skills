# claw-todolist 技能

该技能实现了 **Todo List Management Protocol V3.2**，通过对话式命令支持结构化的任务管理。

## 核心协议（命令）
该技能支持以下别名：**a**（添加任务）、**x**（标记任务为已完成）、**ls**（显示任务列表）、**e**（编辑任务）。
同时，它还支持 **REVIEW** 命令，用于自动执行结构化分析。

## 功能与输出格式
- **任务结构：** 任务会根据预定义的规则（V3.2）被本地保存。
- **可视化展示：** 任务列表的输出默认采用 **文本格式**，其中会包含 **优先级分组标题**，以及任务文本后的 **权重标记（⭐）**。
- **自动分析：** 使用 **REVIEW** 命令可以根据预设的阈值（如 P1 比例、任务逾期情况、任务老化程度或战略性忽略项）自动执行任务分析。

## 数据持久化
所有状态信息、规则以及显示配置都保存在技能对应的文件夹中，确保安装的稳定性。
