# Appsmith

一个用于构建管理面板、内部工具和仪表盘的平台，可与25种以上的数据库及任何API集成。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 显示信息
- `status` - 显示当前状态

## 特性

- 采用 appsmithorg/appsmith 提供的核心功能

## 使用方法

运行任意命令：`appsmith <命令> [参数]`
