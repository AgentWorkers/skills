# 备忘录（Memos）——快速笔记管理工具

## 为什么需要这个技能？

- 受到热门开源项目的启发（在 GitHub 上获得了数千个星标）
- 无需安装——可直接使用标准系统工具
- 具有实际功能——可以执行真实的命令并生成相应的输出结果

## 命令

运行 `scripts/memos.sh <命令>` 即可使用以下功能：

- `add` — <文本>        添加一条备忘录
- `list` — [n]          列出最近的备忘录（默认显示10条）
- `search` — <查询>    搜索备忘录
- `tag` — <标签>         按标签筛选备忘录（例如：#标签）
- `tags` —              显示所有标签
- `edit` — <id>         编辑备忘录
- `delete` — <id>       删除备忘录
- `export` — [格式]   导出所有备忘录（格式：md/json）
- `stats` —             备忘录统计信息
- `info` —              版本信息

## 快速入门

```bash
memos.sh help
```
