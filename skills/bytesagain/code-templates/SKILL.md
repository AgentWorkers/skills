# Claude Code Templates

这是一个用于配置和监控 Claude Code 的命令行工具（CLI）。

## 命令

- `help` - 显示帮助信息
- `run` - 运行相关任务
- `info` - 查看详细信息
- `status` - 获取当前状态

## 功能

- 该工具的核心功能源自 davila7/code-templates。

## 使用方法

运行任意命令的格式为：`code-templates <命令> [参数]`
