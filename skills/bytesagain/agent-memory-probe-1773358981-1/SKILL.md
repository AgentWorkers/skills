# Agent Memory Pro

- **名称**: Agent Memory Pro  
- **描述**: 专为AI代理设计的量产级结构化存储解决方案，具备基于类别的存储功能、MD5哈希去重机制、数据整合功能以及数据导出功能。该系统可将学习内容按类型分类存储，通过MD5哈希检测重复项，并以Markdown或JSON格式导出，以实现跨代理之间的数据共享。支持的命令包括：`init`（初始化）、`save`（保存数据）、`recall`（检索数据）、`review`（查看数据）、`consolidate`（整合数据）、`export`（导出数据）、`stats`（查看统计信息）和`forget`（选择性删除数据）。

## 常见问题解答（FAQ）

**问：这与基本的代理存储系统有何不同？**  
**答：** Agent Memory Pro提供了结构化的数据存储方式，支持MD5哈希去重，并支持数据的完整导出，而非简单的平面文件存储。

**问：数据存储在哪里？**  
**答：** 数据存储在`~/agent-memory/`目录下，该目录下包含多个子目录，如`learnings`（学习数据）、`facts`（事实信息）、`decisions`（决策记录）和`archives`（存档文件）等。

**问：可以在不同代理之间迁移数据吗？**  
**答：** 可以。使用`export`命令可以将数据导出为通用的Markdown格式，任何代理都可以导入这些数据。

**问：数据整合是如何实现的？**  
**答：** 通过MD5哈希算法检测数据中的重复项，并将重复项标记出来以便后续清理。

## 命令列表  

| 命令        | 功能                |
|-------------|-------------------|
| `init`       | 初始化结构化存储目录          |
| `save`       | 保存分类后的学习数据         |
| `recall`     | 根据关键词检索数据，并提供排序结果   |
| `review`     | 显示过去72小时内的数据         |
| `consolidate`   | 基于哈希算法去重数据         |
| `export`      | 将数据导出为Markdown格式       |
| `stats`      | 提供各类别的数据统计信息       |
| `forget`     | 选择性地删除数据           |
