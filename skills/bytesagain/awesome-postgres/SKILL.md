# 精选PostgreSQL资源

## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 显示命令详情
- `status` - 显示当前状态

## 功能

- 包含了dhamaniasad/awesome-postgres中的核心功能

## 使用方法

运行任意命令：`awesome-postgres <命令> [参数]`
