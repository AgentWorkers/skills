# SimianArmy

这是一套用于维护云服务稳定运行的工具集。Chaos Monkey 是一款具备高弹性的工具，能够帮助应用程序应对随机出现的实例故障。

## 命令

- `help` - 显示帮助信息
- `run` - 执行命令
- `info` - 查看信息
- `status` - 查看当前状态

## 功能特点

- 采用 Netflix/SimianArmy 的核心功能架构

## 使用方法

执行任意命令的格式为：`chaos-testing <命令> [参数]`
