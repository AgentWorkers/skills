# 我的 AWS 安全工具库

这是一个包含各种开源 AWS 安全工具的列表，涵盖了防御性、攻击性、审计、数据泄露与事件响应（DFIR）等方面的工具。

## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 查看信息
- `status` - 查看状态

## 功能特点

- 基于 `toniblyx/aws-security` 的核心功能开发

## 使用方法

运行任意命令的格式为：`aws-security <命令> [参数]`
