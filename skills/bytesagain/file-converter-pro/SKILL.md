# File Converter Pro

💾 一个自托管的在线文件转换工具，支持1000多种文件格式 ⚙️

灵感来源于 [C4illin/ConvertX](https://github.com/C4illin/ConvertX)（在GitHub上获得了16,173个星标）。

## 命令

- `help` - 显示帮助信息
- `run` - 运行转换
- `info` - 查看软件信息
- `status` - 查看转换状态

## 功能特点

- 兼备了C4illin/ConvertX的核心功能

## 使用方法

运行任意命令：`file-converter-pro <命令> [参数]`
