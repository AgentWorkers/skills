# Paint

这是一个数字绘画工具包，支持颜色调色板的生成、渐变效果的创建、纹理的生成、图案的设计以及颜色空间的转换。

## 命令

| 命令 | 描述 |
|---------|-------------|
| `paint run` | 执行主功能 |
| `paint list` | 列出所有可用项目 |
| `paint add <项目>` | 添加新项目 |
| `paint status` | 显示当前状态 |
| `paint export <格式>` | 导出数据 |
| `paint help` | 显示帮助信息 |

## 使用方法

```bash
# Show help
paint help

# Quick start
paint run
```

## 示例

```bash
# Run with defaults
paint run

# Check status
paint status

# Export results
paint export json
```

## 工作原理

该工具包使用内置逻辑处理输入数据，并输出结构化的结果。所有数据均保留在本地。
