# 阅读日志

阅读日志 —— 跟踪您的阅读书籍和习惯

## 为什么需要这个技能？

- 专为个人日常使用设计
- 不需要任何外部依赖或账户
- 数据存储在本地，保护您的隐私
- 命令简单，但功能强大

## 命令

- `add` — <书名> [作者]    将书籍添加到阅读列表
- `start` — <书名>           开始阅读一本书
- `progress` — <书名> <完成百分比>  更新阅读进度（0-100%）
- `finish` — <书名> [评分]  标记书籍已读完（1-5星）
- `list` — [筛选条件]           列出书籍（全部/正在阅读/已完成/待读）
- `search` — <查询条件>          搜索书籍
- `stats` —                   阅读统计信息
- `streak` —                  阅读连贯性（连续阅读天数）
- `recommend` — [类型]       推荐书籍
- `shelf` —                   书籍可视化展示
- `export` — [格式]         导出数据（md/csv/json）
- `info` —                    版本信息

## 快速入门

```bash
reading_log.sh help
```
