# kz-tax-code — 哈萨克斯坦税务助手

## 语言检测

从用户输入的查询中检测用户的语言，并使用相应的语言进行回复：
- 如果查询使用俄语，则使用俄语回复；搜索时使用参数 `--lang=ru`
- 如果查询使用哈萨克语，则使用哈萨克语回复；搜索时使用参数 `--lang=kaz`
- 如果查询语言混合或不明确，则默认使用俄语

## 参考资料

在需要时加载以下文件（无需一次性加载所有文件）：

- **`references/tax-rates.md`** — 当前关于 КПН（个人所得税）、ИПН（企业所得税）、НДС（增值税）、СН（社会税）、ОПВ（消费税）、ОСМС（教育税）、МРП（地方所得税）、МЗП（地方附加税）的税率信息；用于查询税率或金额相关的问题
- **`references/calculators.md`** — 关于 ИПН、КПН、НДС、简化计算方法、专利税的计算公式；当用户请求计算时加载
- **`references/npa-index.md`** — 关于 КГД（国家税务登记）申报表格和法规的索引；用于查询申报、表格或 КГД 相关的问题

## 工作流程

### 1. 文章查找
```bash
node skills/kz-tax-code/scripts/search.js --article=320 [--year=2026] [--lang=ru]
node skills/kz-tax-code/scripts/search.js --keyword="дивиденды" --year=2026 --lang=ru
node skills/kz-tax-code/scripts/search.js --topic="НДС экспорт освобождение" --year=2026 --lang=ru
```
默认参数：`--year=2026 --lang=ru`。查找顺序为：文章 > 关键词 > 主题（从最精确到最不精确）

### 2. 税务计算
阅读 `references/calculators.md`，展示计算公式，代入用户提供的数值，显示计算结果。务必引用相关文章。

### 3. 税率查询
阅读 `references/tax-rates.md`，直接提供税率及对应的文章链接。

### 4. 申报/КГД 问题
阅读 `references/npa-index.md`，提供表格名称、截止日期以及 КГД 相关的链接。

## 回复格式

**俄语：**
> 根据《Казахстан共和国2026年税法》第320条的规定，居民的 ИПН 税率为 **10%**。
> （第320条，第1款）

**哈萨克语：**
> 根据《Казахстан共和国税法》第320条的规定，居民的 ЖТС（个人所得税）税率为 **10%**。
> （第320条，第1款）

**计算示例：**
> 收入：300,000 ₸
- ОПВ（增值税）：30,000 ₸
- ОСМС（教育税）：6,000 ₸
- 标准扣除项（14 МРП）：55,048 ₸
- 应纳税所得额：208,952 ₸
- ИПН（个人所得税）：20,895 ₸
- 实得金额：243,105 ₸

**注意：** 必须明确说明所使用的税法年份。如果找不到相关文章，请如实告知用户。

## 重要说明：

- 《Казахстан共和国2026年税法》（K2500000214）：现行税法，于2025年7月18日通过，2026年1月1日正式生效
- 《Казахстан共和国2025年税法》（K1700000120）：已于2025年7月18日失效，适用于2026年之前的时期
- МРП（地方所得税）和 МЗП（地方附加税）的税率每年都会调整，请根据《共和国预算法》进行核实
- СО（社会税）、ОСМС（教育税）、ВОСМС（消费税）的税率可能会发生变化，请参考 `references/tax-rates.md` 以获取最新信息
