# kz-tax-code — 哈萨克斯坦税务助手

## 法律依据

所有计算结果均基于 `data/` 目录中的税法典文本。  
系统中未内置任何税率信息，所有税率均来源于原始法律文件；如果找不到相关条文，会如实告知用户。

## 哈萨克斯坦税法中未包含的内容  

以下内容由单独的法律规定，因此不包含在哈萨克斯坦税法（НК РК）中：  

| 中文术语 | 哈萨克斯坦术语 | 相关法律 |
|----------------|------------------|-------|
| МРП | АЕК（айлық есептік көрсеткіш） | 《共和国预算法》 |
| МЗП | ЕАМ（еңбекақыныҳ ең төмен мөлшері） | 《共和国预算法》 |
| ОПВ | МЖЖ（міндетті жинақтаушы зейнетақ жарналары） | 《养老保险法》 |
| ОПВР | МЖЖЖ（міндетті жұмыс берушы зейнетақ жарналары） | 《养老保险法》 |
| СО | ӘА（әлеуметтік аударымы） | 《强制性社会保险法》 |
| ВОСМС / ОСМС | МӘМС（міндетті әлеуметтік медициналыҳ сақтандыру） | 《医疗保险法》 |

相关文件的代码、URL 及最新数据存储在 `data/laws.json` 中（每年更新）。  

请注意：这些数据每年都会发生变化，请务必从官方来源获取最新信息，切勿凭记忆填写。  

### 获取最新数据  

1. 查看 `data/laws.json`，获取文件代码、URL 及最新数据。  
2. 根据用户需求下载相应法律文件（语言取决于用户请求）。  
3. 通过搜索功能查找所需信息。  

如果文件已下载过，可直接使用，无需重复下载。  
如果下载失败，请告知用户需通过 egov.kz 或 adilet.zan.kz 确认数据，并提供 `laws.json` 中的 `knownValues` 作为参考（标注为“需核实”）。  

## 个性化计算  

对于针对个人的查询（如“我需要缴纳多少税款”、“帮我计算税款”等），如用户未提供年龄信息，请询问其年龄：  
> “请提供您的年龄，这会影响计算结果。”  

### 年龄为何重要？  

> ⚠️ 税率（如 ОПВ、ВОСМС 等）及退休年龄均受法律调整。以下数据为编写时的参考值，请始终从 `data/laws.json`（`knownValues`）中获取最新税率；退休年龄信息请参考《养老保险法》（通过 fetch.js 获取）。  

| 年龄 | 特殊规定 |  
|---------|-------------|  
| 男性 63 岁以下 / 女性 61.5 岁以下 ⚠️ | ОПV 税率按标准征收（税率详见 laws.json） |  
| 男性 63 岁及以上 / 女性 61.5 岁及以上（退休人员）⚠️ | ОПV 税率免征（《养老保险法》第 24 条） |  
| 1975 年及以后出生的人员 | ОПВR 税率免征（仅适用于 1975 年 1 月 1 日以后出生的人） |  
| 任何年龄 | ВОСМС、ОСМС、СО 税率均无年龄限制（详见 laws.json） |  

> **补充说明**：所有签订劳动合同的雇员（无论年龄）均需缴纳 ОПV 税；未签订劳动合同的学生无需缴纳 ОПV 税，但这并非基于年龄因素，而是由于不存在劳动关系。  

若未提供年龄信息，ОПVR 的计算结果可能不准确（ОПVR 仅适用于 1975 年以后出生的人）。  

## 税法版本  

当前版本及旧版本的元数据存储在 `data/versions.json` 中：  

| 版本 | 说明 | 适用场景 |  
|--------|-------|-------------------|  
| `current`（默认） | 现行税法 | 除非用户特别指定旧版本 |  
| `outdated` | 旧版本 | 仅当用户询问过去时期的法律时使用 |  

脚本不会在命令中明确说明版本年份，仅使用 `--version=current` 或 `--version=outdated` 来区分版本。  
当前生效的税法版本请通过 `versions.json` 查阅，并在回复中告知用户（脚本返回 `versionName`）。  

## 语言自动识别  

系统会根据请求内容自动判断语言，无需手动设置 `--lang` 参数：  
- 如果请求中包含哈萨克字母（如 ә、ғ 等），则自动识别为哈萨克语；  
- 如果仅包含西里尔字母（无哈萨克字母），则自动识别为俄语；  
- 如果明确指定 `--lang=ru\|kaz`，则优先使用哈萨克语。  

**注意**：  
- 哈萨克斯坦税法相关术语包括：ЖТС（ИПН）、КТС（КПН）、ҚҚС（НДС）、ӘС（СН）、АЕК（МРП）、ЕАМ（МЗП）、МЖЖ（ОПВ）、МЖЖЖ（ОПВР）、ӘА（СО）、МӘМС（ВОСМС/ОСМС）。  

## 工作流程  

1. **确定版本**：  
   - 默认使用当前版本（`--version=current`）；  
   - 若用户明确要求旧版本，使用 `--version=outdated`。  

2. **选择搜索模式**：  
   - 搜索顺序：`--article` > `--keyword` > `--topic`  

3. **显示结果并回答**：  
   - 从 JSON 结果中提取：  
     - `versionName`（用于在回复中注明税法版本）；  
     - `results[].article` + `results[].title`（用于引用文章标题）；  
     - `results[].text`（全文，仅限 `article` 和 `title_match` 模式）；  
     - `results[].context`（匹配内容的高亮显示部分，仅限 `keyword_exact`、`keyword_fallback`、`topic` 模式）。  
   - 规则：优先显示 `text`，若仅有 `context` 也可提供相关内容。  

**务必说明**：  
- 明确指出使用的税法版本（`versionName`）、文章标题及具体条款。  
- 计算税款时，请根据税法条文准确查找相关条款（税率、减免项目、计算方法），切勿凭主观猜测。  

## 回复格式  

回复中必须包含税法版本（`versionName`），以免混淆年份。  

**示例回复（俄文）：**  
> 根据《Казахстан税法》第 363 条（{{versionName}}，居民收入低于 8,500 АЕК 时，ЖТС 税率为 10%；超过部分税率为 15%。**  
> （第 363 条第 1 款）  

**示例回复（哈萨克文）：**  
> 根据《Казахстан税法》第 363 条（{{versionName}}，居民收入在 8,500 АЕК 以下的部分，ЖТС 税率为 10%；超过部分税率为 15%。**  
> （第 363 条第 1 款）  

**注意事项**：  
- 若回复中包含 `externalHints`，说明该术语不在税法典中，建议用户通过 fetch.js 获取相关信息。  
- 如果未找到相关条文，提示用户确认文章编号或关键词。  

## 税法更新  

新税法发布时：  
**推荐安全操作方式**：  
1. 在浏览器中访问 adilet.zan.kz，下载新税法文件；  
2. 将页面内容保存为 HTML 格式（包含俄文和哈萨克文版本）；  
3. 运行以下命令（仅限于安全网络环境，需使用 `--insecure` 参数）：  
```bash
node skills/kz-tax-code/scripts/update.js \
  --doc=K2600000XXX --name="НК РК 2027" --effective=2027-01-01 \
  --html-ru=./nk-ru.html --html-kaz=./nk-kaz.html
```  
> ⚠️ `--insecure` 选项会忽略 TLS 证书验证（adilet.zan.kz 使用的证书不在 Node.js 的受信任证书库中，切勿在不可信网络中使用）。  

脚本流程：  
1. 将 HTML 文件解析为 Markdown 格式；  
2. 将当前版本（`current`）替换为旧版本（`outdated`）；  
3. 将新版本保存为当前版本（`current`）；  
4. 更新 `data/versions.json` 文件。  
文档编号（`--doc`）来自 adilet.zan.kz 页面的 URL。
