# 会话清理器

用于清理 `~/.openclaw/agents/main/sessions/` 目录中过时的会话记录文件。

## 规则

- **必须保留**：当前的主会话（即具有 `agent:main:main` 键且处于活动状态的会话）
- **必须保留**：所有群组聊天会话（目标地址包含 `oc_`）
- **移至备份目录**：其他所有文件（旧的主会话、已完成的子代理会话、API 会话、定时任务/心跳检测记录）
- **严禁删除**：所有被移动的文件都会被保存到 `sessions/backup/` 目录中，以便后续恢复

## 使用方法

```bash
bash scripts/clean-sessions.sh [--dry-run]
```

- **默认行为**：将过时的文件移至备份目录
- `--dry-run`：仅报告需要移动的文件，不执行实际移动操作
