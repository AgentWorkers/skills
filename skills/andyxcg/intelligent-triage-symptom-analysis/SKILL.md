# 智能分诊与症状分析

> **版本**: 1.0.2  
> **类别**: 医疗保健  
> **计费方式**: SkillPay（每次呼叫1个代币，约0.001 USDT）

该工具为医疗保健提供者、远程医疗平台及患者提供基于人工智能的分诊辅助服务，能够进行准确的初步症状评估和紧急程度建议。

## 主要功能

1. **全面的症状覆盖**：涵盖11个身体系统的650多种症状  
2. **标准化的分诊体系**：分为5个等级（从需要立即抢救到非紧急情况）  
3. **危险信号检测**：对危及生命的状况的检测准确率≥95%  
4. **自然语言处理（NLP）分析**：能够从用户描述中提取症状信息  
5. **鉴别诊断**：利用机器学习辅助对疾病进行排序  
6. **计费方式**：每次分析收取1个代币（约0.001 USDT）

## 快速入门

### 分析症状：

```python
from scripts.triage import analyze_symptoms
import os

# Set environment variables
os.environ["SKILL_BILLING_API_KEY"] = "your-api-key"
os.environ["SKILL_ID"] = "your-skill-id"

# Analyze patient symptoms
result = analyze_symptoms(
    symptoms="胸痛，呼吸困难，持续30分钟",
    age=65,
    gender="male",
    vital_signs={"bp": "160/95", "hr": 110, "temp": 37.2},
    user_id="user_123"
)

# Check result
if result["success"]:
    print("分诊等级:", result["triage"]["level"])
    print("紧急程度:", result["triage"]["urgency"])
    print("建议措施:", result["recommendations"])
else:
    print("错误:", result["error"])
    if "paymentUrl" in result:
        print("充值链接:", result["paymentUrl"])
```

## 分诊等级

| 等级 | 名称 | 响应时间 | 描述 | 举例 |
|-------|------|---------------|-------------|----------|
| 1   | 需要立即抢救 | 立即行动 | 需要立即干预的危及生命状况 | 心脏骤停、严重创伤、呼吸衰竭 |
| 2   | 紧急情况 | <15分钟 | 需要快速评估的高风险状况 | 胸痛、严重出血、意识改变 |
| 3   | 紧急状况 | <30分钟 | 需要及时医疗处理的严重状况 | 腹痛、高烧、中度创伤 |
| 4   | 半紧急状况 | <60分钟 | 需要在几小时内评估的较轻微状况 | 轻微伤害、慢性症状、病情稳定 |
| 5   | 非紧急状况 | >60分钟 | 可以等待几天到几周处理的轻微状况 | 需要复诊、配药续期、行政事务 |

## 风险分层因素

- **人口统计风险**：年龄、性别、病史  
- **生命体征异常**：关键参数的异常值  
- **合并症影响**：现有疾病对紧急程度的影响  
- **药物相互作用**：可能出现的药物相关并发症  
- **社会决定因素**：获得医疗服务的便利性、支持系统  
- **时间敏感性**：不治疗情况下病情恶化的风险
