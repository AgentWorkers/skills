# 农业产量预测

> **版本**: 1.0.1  
> **类别**: 农业 / 分析  
> **计费方式**: SkillPay（每次调用消耗1个令牌，约0.001美元）  
> **免费试用**: 每位用户可享受10次免费调用  

这是一个基于大数据的农业产品产量预测工具，可帮助农民、农学家和农业企业预测作物产量和产出。

## 主要功能  

1. **产量预测**：根据历史数据和当前情况预测作物产量。  
2. **天气影响分析**：考虑天气模式和气候数据。  
3. **市场趋势整合**：结合市场价格和需求趋势进行分析。  
4. **多作物支持**：支持多种农产品（谷物、蔬菜、水果等）。  
5. **SkillPay计费**：按使用次数计费（每次调用1个令牌，约0.001美元）。  
6. **免费试用**: 每位新用户可享受10次免费调用。

## 环境变量  

本技能需要以下环境变量：  

### 必需环境变量（试用结束后设置）  

| 变量 | 说明 | 是否必需 | 示例值 |  
|---------|---------|---------|---------|  
| `SKILL_BILLING_API_KEY` | SkillPay计费API密钥 | 是 | `skp_abc123...` |  
| `SKILL_ID` | SkillPay仪表板中的技能ID | 是 | `skill_def456...` |  

### 可选环境变量  

| 变量 | 说明 | 默认值 |  
|---------|---------|---------|  
| `OPENWEATHER_API_KEY` | OpenWeatherMap天气数据API密钥 | 可选 | `your_api_key` |  
| `WEATHERAPI_KEY` | 其他天气数据API密钥 | 可选 | `your_api_key` |  
| `USDA_API_KEY` | USDA农业数据API密钥 | 可选 | `your_api_key` |  
| `OPENAI_API_KEY` | OpenAI增强预测API密钥 | 可选 | `your_api_key` |  
| `CACHE_DURATION_MINUTES` | 天气/市场数据缓存时长（分钟） | `60` |  
| `MAX_FORECAST_AREA` | 每次请求的最大预测面积（公顷） | `10000` |  

请参阅`.env.example`文件以获取完整的环境变量列表。

## 支持的作物类型  

- 谷物：小麦、水稻、玉米、大麦、高粱

- 水果：苹果、橙子、葡萄、桃子  
- 其他：大豆、棉花、甘蔗

## 安全注意事项  

### 数据隐私  
- 农业数据被视为机密商业信息，不会收集任何个人身份信息（PII）。  
- 天气和市场数据会被缓存以减少API调用次数。  

### API密钥安全  
- 请勿将API密钥提交到版本控制系统中。  
- 所有敏感配置信息应通过环境变量传递。  
- 定期更换API密钥。
