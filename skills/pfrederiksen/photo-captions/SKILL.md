# 照片说明

当用户分享一张带有背景信息（位置、使用的相机、镜头、胶片类型、拍摄对象、拍摄氛围）的照片时，需要为以下所有平台生成统一的说明文字。每个平台都有其独特的表达风格和格式要求。

如果用户指定了具体的摄影设备（相机机身、镜头、胶片类型或数码设置），请在说明中适当提及；但不要编造用户未提供的设备细节。

## 平台

### 📸 Instagram
- **风格**：简洁、富有诗意或幽默感，让图片本身说话。
- **格式**：1-2行说明 → 空行 → 设备信息（如果有的话） → 空行 → 标签。
- **标签**：最多使用5个标签（Instagram的限制）。优先选择最具代表性的标签：类型（如#filmphotography）、地点、胶片/设备类型以及一个与氛围或风格相关的标签。质量胜过数量。

### 📷 Flickr
- **风格**：描述性更强，带有思考色彩。Flickr的用户重视故事性和摄影技巧。
- **格式**：使用斜体标题，后跟1-3句背景描述或故事。最后注明设备信息。
- **内容要求**：包括拍摄地点以及吸引摄影师拍摄的元素，尽量写出类似“照片随笔”的风格。

### 🐦 X (Twitter)
- **风格**：简洁有力，字数控制在280个字符以内。避免滥用标签。
- **格式**：用一句话概括照片的主题。如果合适，可以在结尾处提及设备信息。
- **目标**：让读者停下滚动浏览的步伐，认真阅读你的说明。

### 🪟 Glass
- **风格**：摄影师之间的交流方式，低调真诚，不使用标签，也不追求互动。
- **格式**：1-3句话，简要介绍拍摄地点和观察感受。设备信息用点（·）分隔。
- **氛围**：就像在摄影散步时与朋友聊天一样自然。

### 🟦 Tumblr
- **风格**：更具文学性，表达力强，篇幅稍长。Tumblr用户喜欢故事性和情感表达。
- **格式**：使用加粗的地点作为标题，接着是2-4句叙述或反思。最后加上设备信息。
- **标签**：标签之间使用空格，例如：#filmphotography（而非#filmphotography）。
- **标签数量**：8-12个。

### 🔵 Bluesky
- **风格**：亲切、温暖，注重社区互动。类似于早期的Twitter风格。
- **格式**：1-2句，轻松自然，但富有深度。设备信息可选。
- **标签**：除非能增加真实价值，否则不要使用标签（Bluesky社区反对滥用标签）。

### 🧵 Threads
- **风格**：轻松随意，类似于Instagram的风格，但更注重对话感。
- **格式**：1-2句，语气自然。如果设备信息有趣，可以提及。标签数量控制在3-5个以内。

### 🔢 500px
- **风格**：注重技术和摄影技巧。500px是一个以摄影为核心的社区，重视技术细节。
- **格式**：先写标题，再写1-3句描述照片的拍摄技巧、环境以及成功的原因。务必包含完整的设备信息。
- **内容要求**：如果可能，可以提及相机设置、光线条件或拍摄技巧。

### 🟠 Reddit
- **风格**：真实、略带自嘲，注重社区互动。避免过度自我宣传。
- **格式**：帖子标题（简洁明了）+ 评论区（包含背景信息和设备信息）。
- **标题格式**：地点或拍摄对象 + 设备信息（例如：Bombay Beach [Canon EOS 1V, Tri-X 400]）。
- **评论区**：2-3句背景描述或故事。根据照片类型，可以提及相关子版块（如r/analog、r/photography、r/streetphotography等）。

### 👤 Facebook
- **风格**：个人化、亲切，像和朋友家人分享一样自然。
- **格式**：2-3句简短的内容，讲述拍摄时的情境和感受。只有当设备信息对故事有帮助时才提及。
- **标签**：尽量少用标签（1-2个），Facebook用户更关注故事本身。

### 🎞️ VSCO
- **风格**：简洁、富有诗意，低调自然。VSCO更注重图片本身的美感。
- **格式**：最多1行说明，有时只是一个词或短语。避免使用标签。
- **氛围**：像低声细语一样，不追求过度解释。VSCO的说明更像标题而非详细的描述。
- **设备信息**：除非是关于胶片类型的，否则尽量避免提及。

### 📝 Substack
- **风格**：叙事性强，带有作者的个人风格。Substack读者期待的是一篇完整的文章，而不仅仅是简单的说明文字。
- **格式**：2-4句的说明文字，作为新闻邮件中图片下方的文字说明。内容应包含丰富的背景信息（拍摄地点、观察细节等），读起来像杂志中的照片说明加上个人感悟。
- **内容要求**：如果设备信息能增加故事感，可以提及。同时提供地点和拍摄环境，以及照片背后的情感体验。
- **氛围**：结合了《纽约客》杂志的风格和旅行日记的质感，让读者感受到真实的故事。

### 📌 Pinterest
- **风格**：描述性强，便于搜索。Pinterest是一个基于关键词的发现平台。
- **格式**：标题清晰且包含关键词（例如：Desert Film Photography 或 Route 66 Aesthetic），后跟2-3句描述场景、氛围和风格的文字。自然地使用相关关键词。
- **内容要求**：确保关键词有助于用户找到你的内容（如“desert film photography”或“Route 66 aesthetic”）。
- **标签**：Pinterest使用关键词来帮助用户发现内容，而非强制使用标签。

## 编写指南

- 根据照片的具体内容、地点和氛围调整说明文字。
- 避免在不同平台重复相同的表述，每个平台的说明都应符合其独特的特点。
- 适当的幽默和机智是可接受的，但应与照片的整体氛围相协调。
- 如果照片是黑白的，可以在说明中添加相关的黑白摄影标签。
- 每条说明都应针对特定照片量身定制。
- 对于胶片照片，强调其模拟摄影的美感；对于数码照片，则重点描述拍摄的瞬间和技巧。
- 如果用户只要求生成某些平台的说明，只需生成相应的内容。
- 以自然的语言写作，避免使用复杂的标点符号（如破折号或分号），保持句子的流畅性。只有当你能直接用语言表达时，才写下来。
