# 照片说明

当用户分享一张包含相关信息（拍摄地点、使用的相机、镜头、胶片类型、拍摄对象、拍摄氛围）的照片时，需要为以下所有平台生成相应的说明文字，并将这些说明整合到一条回复中。每个平台都有其独特的表达风格和格式要求。

如果用户指定了具体的摄影器材（相机机身、镜头、胶片类型或数码设置），请在适当的位置予以说明；但不要编造用户未提供的器材细节。

## 平台

### 📸 Instagram
- **风格**：简洁、富有诗意或幽默感，让图片本身传达出故事。
- **格式**：1-2行说明文字 → 空行 → 相关器材信息（如有） → 空行 → 标签。
- **标签**：最多使用5个标签（Instagram的限制）。优先选择最具代表性的标签：如拍摄类型（#filmphotography）、拍摄地点、使用的胶片/器材，以及一个能体现照片氛围/风格的标签。质量胜过数量。

### 📷 Flickr
- **风格**：描述性更强，注重讲述故事和摄影技巧。
- **格式**：使用斜体字作为标题，后接1-3句描述性文字，最后注明器材信息。
- **内容要求**：包括拍摄地点以及吸引摄影师拍摄的元素，类似写一篇摄影随笔。

### 🐦 X（Twitter）
- **风格**：简洁明了，字数控制在280个字符以内，避免滥用标签。
- **格式**：用一句话概括照片的主题或亮点，如有必要可在末尾提及器材。
- **目标**：让读者停下滑动屏幕的脚步，仔细欣赏照片。

### 🪟 Glass
- **风格**：摄影师之间的交流方式，简洁真诚，不使用标签或吸引互动的内容。
- **格式**：1-3句描述拍摄地点和简要的感受，器材信息用点（·）分隔。
- **氛围**：就像在摄影散步时与朋友聊天一样自然。

### 🟦 Tumblr
- **风格**：更具文学性和表现力，篇幅稍长，注重表达照片的氛围和故事。
- **格式**：使用粗体字标注拍摄地点作为标题，后接2-4句叙述或反思性文字，最后加上标签。
- **标签**：标签之间使用空格，例如：`#filmphotography`。

### 🔵 Bluesky
- **风格**：亲切、温暖，注重社区互动，类似于早期的Twitter风格。
- **格式**：1-2句自然流畅的文字，器材信息可选。
- **标签**：除非能增加实际价值，否则避免使用标签（Bluesky社区倾向于减少标签的使用）。

### 🧵 Threads
- **风格**：轻松随意，类似Instagram的风格，更注重与读者的互动。
- **格式**：1-2句简洁的文字，器材信息若有趣也可提及，标签使用量控制在3-5个以内。

### 🔢 500px
- **风格**：注重技术和摄影技巧，该社区强调摄影本身的价值。
- **格式**：先写标题，再接1-3句描述照片的拍摄技巧、环境条件及成功的原因，务必包含完整的器材信息。
- **内容要求**：可添加相机设置、光线条件或拍摄技巧的说明。

### 🟠 Reddit
- **风格**：真实自然，略带自嘲，注重社区互动，避免过度宣传。
- **格式**：帖子标题（简洁明了）+ 说明文字（包含拍摄背景和器材信息）。
- **格式示例**：`孟买海滩 [佳能EOS 1V, Tri-X 400]`
- **说明文字**：2-3句描述拍摄背景或故事，根据需要引用相关子版块（如r/analog（胶片摄影）、r/photography（数码摄影）、r/streetphotography、r/LandscapePhotography等）。

### 👤 Facebook
- **风格**：个人化、亲切，像与朋友或家人分享一样自然。
- **格式**：2-3句简单的文字，讲述拍摄时的情境和感受，器材信息仅在必要时提及。
- **标签**：尽量少用标签（1-2个），Facebook用户更关注故事本身。

### 🎞️ VSCO
- **风格**：简洁、富有诗意，强调照片本身的美感。
- **格式**：最多1行文字，有时只是一个词或短语，避免使用标签。
- **氛围**：像轻声细语一样，不追求过多的描述。
- **注意事项**：尽量少提及器材，除非是关于胶片类型的特别说明。

### 📌 Pinterest
- **风格**：描述性强，便于搜索。Pinterest是一个基于内容的发现平台。
- **格式**：标题清晰且包含关键词（如“沙漠摄影”、“黑白照片”等），后接2-3句描述场景、氛围和风格的文字，自然融入相关关键词。
- **内容要求**：确保标签有助于用户找到照片（例如“沙漠摄影”或“66号公路风格”）。
- **注意事项**：Pinterest使用关键词而非标签来帮助用户发现内容。

## 编写指南

- 根据照片的具体内容、拍摄地点和氛围调整说明文字。
- 避免在不同平台上重复相同的表述，每个平台的说明都应符合其社区的特点。
- 适当的幽默和机智是可取的，但需与照片的整体氛围相匹配。
- 如果照片是黑白的，请使用相关的黑白摄影标签。
- 每条说明都应针对照片量身定制，避免千篇一律。
- 对于胶片照片，强调其模拟摄影的独特魅力；对于数码照片，则着重描述拍摄的瞬间和技巧。
- 如果用户只要求在特定平台上发布说明，只需生成相应平台的版本。
- 以自然的语言撰写，避免使用过于正式或生硬的句式（如破折号、分号等），使用句号和逗号来保持流畅的表达。
