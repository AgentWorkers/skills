**名称：local-life**  
**描述：** 提供实时的本地信息可视化仪表板。当用户询问天气、温度、货币汇率或请求特定城市的当日概况时，激活此技能。常见的触发语包括：“戈亚尼亚今天的天气如何？”，“当日总结”，“本地状况”，“local-life”，“今天的天气预报是什么？”。  

**技能：L**  

**local-life**  
此技能将代理转变为一个本地情境助手，从多个免费数据源收集信息，并将其整理成易于阅读的格式化仪表板。  

**📋 代理使用说明：**  

1. **定位**  
   - 确定用户所在的城市。如果未提供城市名称，使用戈亚尼亚（Goiânia）作为默认值。  
   - 将城市名称存储在一个变量中（例如：`CIDADE`），以便在API调用中使用。  

2. **数据收集**  
   - **天气、天文和月相**：  
     - 使用 `curl "https://wttr.in/{CIDADE}?format=j1"` 获取以下数据：  
       - `current_condition`（当前天气状况）、`FeelsLikeC`（体感温度）、`windspeedKmph`（风速，单位：公里/小时）、`humidity`（湿度）、`uvIndex`（紫外线指数）。  
     - `weather -> hourly`：`chanceofrain`（降雨概率）。  
     - `weather -> astronomy`：`sunrise`（日出时间）、`sunset`（日落时间）、`moon_phase`（月相）。  

   - **货币汇率**：  
     - 使用 `curl "https://economia.awesomeapi.com.br/last/USD-BRL,EUR-BRL,BTC-BRL"` 获取货币汇率。  

   - **国家节假日**：  
     - 使用 `curl "https://date.nager.at/api/v3/PublicHolidays/2026/BR"` 检查当前日期是否有国家节假日。  

3. **额外数据查询（备用方案）**  
   - **空气质量（AQI）**：`wttr.in` API不提供此数据。可以使用Brave API等工具查询“{CIDADE}今天的空气质量”，并提取状态（例如：“良好”、“中等”、“较差”）。  

4. **情境提示逻辑**  
   - 根据收集到的数据生成相应的提示：  
     - 如果降雨概率大于30%：“外出时请携带雨伞！”  
     - 如果紫外线指数大于6：“今天请使用防晒霜。”  
     - 如果空气质量为“较差”或“非常差”：“今天避免户外运动。”  
     - 如果美元汇率波动幅度大于1%：“今天市场波动较大。”  
     - 如果上述条件均不满足，则不显示任何提示。  

5. **格式化规则**  
   - **代码块**：最终输出必须使用代码块（`````）生成，以保持对齐和ASCII格式。  
   - 将月相名称（如“New Moon”）和天气描述翻译成葡萄牙语。  
   - **输出模板**：严格遵循以下输出模板，并用收集到的数据填充变量。  

**💻 输出模板：**
