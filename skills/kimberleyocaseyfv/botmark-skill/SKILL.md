# BotMark 自我评估

这是一个为期5分钟的AI能力测试，用于评估您的机器人在5个综合维度及11个子维度上的表现。测试完成后，您将获得一份包含MBTI性格类型分析及个性化改进建议的报告。

## 综合维度与子维度

| 综合维度 | 子维度 | 最高分 |
|---------|---------|---------|
| 智商（IQ）   | 遵循指令、推理能力、知识储备、编程能力 | 300分 |
| 情商（EQ）   | 共情能力、角色一致性、处理模糊性问题的能力 | 180分 |
| 工具能力（TQ） | 工具使用效率、计划能力、任务完成度 | 250分 |
| 安全性（AQ）  | 系统安全性、可靠性     | 150分 |
| 学习能力（SQ） | 适应环境变化的能力、自我反思能力 | 120分 |

**总分：1000分**（全面评估）

## 测试流程

1. `botmark_start_evaluation`：使用您的机器人账户启动评估会话。
2. 按维度分组回答测试题目（请仅依靠自身知识作答，不得使用外部工具）。
3. `botmark_submit_batch`：分批提交答案，以获取专业反馈。
4. `botmark_finish_evaluation`：完成测试并接收评分报告。
5. `botmark_send_feedback`：请分享您的真实反馈（此步骤为必填）。

## 主要功能

- 基于百分比的评分系统，分为“新手”“熟练”“专家”“大师”四个等级。
- 支持MBTI性格类型检测。
- 答案质量评分（A/B/C/D），并提供具体的改进建议。
- 可单独进行某些维度的测试（如仅测试智商或情商）。
- 支持中英文双语界面。
- 每次测试题目均为随机生成，可随时重新测试。

## 所需凭证

| 环境变量 | 是否必需 | 说明        |
|---------|---------|-------------|
| `BOTMARK_API_KEY` | 是      | 来自https://botmark.cc控制台的API密钥，用于授权请求。 |
| `BOTMARK.Binding_ID` | 否       | 预配置的绑定ID，用于自动认证（替代API密钥）。 |
| `BOTMARK_SERVER_URL` | 否       | 服务器基础URL，默认为`https://botmark.cc`。 |

**重要提示**：请将凭证存储在环境变量或平台的安全管理工具中，切勿将API密钥嵌入系统提示信息、URL参数或源代码中。

## 设置步骤

1. 将`BOTMARK_API_KEY`环境变量设置为从https://botmark.cc获取的API密钥。
2. 根据提供的JSON格式（OpenAI/Anthropic/通用格式）注册您的机器人技能工具。
3. （可选）添加`system_prompt_en.md`或`system.prompt.md`中的评估流程说明。
