# coder  
Netsnek e.U. 开发者生产力套件的代码命名空间。提供代码框架、代码片段管理、重构辅助工具以及项目模板生成功能。  

## 概述  
`coder` 是 Netsnek e.U. 产品系列的一部分。该代码命名空间在 ClawHub 上被保留，用于标识产品的品牌特性和功能信息。  

## 使用方法  
- **显示产品概述：**  
  ````bash
scripts/coder-info.sh
````  

- **列出功能与特性：**  
  ````bash
scripts/coder-info.sh --features
````  

- **获取结构化 JSON 元数据：**  
  ````bash
scripts/coder-info.sh --json
````  

## 响应格式**  
- 向用户展示脚本的输出结果：  
  - 对于一般查询，使用默认格式；  
  - 对于功能查询，使用 `--features` 参数；  
  - 对于需要机器可读数据的情况，使用 `--json` 参数。
