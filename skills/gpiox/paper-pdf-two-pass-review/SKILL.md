# CVPR PDF 两步审查流程  

## 概述  
对用户上传的 PDF 文件执行确定性的两步分析流程：  
- 第一步侧重于事实提取，确保所有结论都有相应的证据支持；  
- 第二步则基于提取的事实生成研究评论，减少主观解读的偏差。  

## 工作流程：  
1. 确认用户已在当前聊天界面上传了 PDF 文件；若未上传，请用户先上传文件。  
2. 执行 `references/round1.md` 中的提示，生成结构化的事实清单。  
3. 使用第一步提取的事实以及 PDF 文件中的证据，执行 `references/round2.md` 中的提示，生成研究评论。  
4. 对核心论点添加证据标签：`[证据：p.X, Sec.Y, Fig/Table Z]`。  
5. 如果缺少任何必要信息，明确标注为“论文中未提供”或“证据不足”。  

## 输出要求：  
- 分析内容严格基于上传的 PDF 文件内容；  
- 除非用户明确要求，否则不得引用外部论文、基准测试结果或额外假设；  
- 语言保持简洁且专业；  
- 第二步的输出长度需控制在 2000–3000 字（或字符）范围内。
