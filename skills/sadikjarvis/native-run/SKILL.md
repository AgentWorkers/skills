# **Native Run**

## **概述**
Native Run 是 OpenClaw 的一项功能，它允许在运行 OpenClaw 网关的本地机器上执行原生命令。  
该功能适用于自动化、测试以及本地工具的开发与使用。

## **功能说明**  
- 执行原生命令  
- 将执行结果返回给 OpenClaw  
- 仅在运行 OpenClaw 网关的本地机器上执行命令  

## **使用方法**  
发送符合预设模式的消息即可触发该功能。  
**示例：**  
`Run native: whoami`  

## **支持平台**  
Windows  

## **安全提示**  
该功能会执行本地命令，请仅在受信任的环境中使用。
