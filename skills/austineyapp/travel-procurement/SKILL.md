# 旅行采购流程

## 工作流程
1. 收集需求：日期/时间范围、路线、乘客人数、行李情况、预算以及舒适度要求。
2. 按优先顺序筛选出行方式：首先考虑公共交通工具，其次是共享班车服务，最后是私人接送服务或出租车。
3. 将每种出行方式的信息统一转换为标准格式（包括价格、耗时、取消政策以及适用性/风险）。
4. 提出推荐方案：综合考虑性价比、便利性以及备用方案。
5. 征求明确的决策（选择A、B或C），并准备下一步的执行计划。

## 输出标准
- 保持内容简洁，便于决策。
- 每种出行方式都应包含总价及实际注意事项。
- 必须提供至少一种备用方案。
