# 费用与索赔操作

## 工作流程
1. 收集输入信息：收据集、时间范围、货币、政策限制（如有）以及索赔提交方式（公司内部工具/电子邮件/手动提交）。
2. 提取各项费用明细：日期、商户名称、金额、货币、费用类别、支付方式以及收据的验证状态。
3. 验证索赔是否满足提交条件：
   - 缺少收据
   - 缺少费用使用的业务目的说明
   - 重复的索赔记录
   - 不符合政策规定的情况（如政策中有所规定）
4. 生成可供决策使用的报告：
   - 符合索赔条件的费用项目
   - 被拒绝的费用项目及需要解决的具体问题
   - 按费用类别和货币分类的总额
5. 生成提交所需的文件：
   - 索赔汇总信息
   - 各项费用的详细说明
   - 异常情况或需要审批的后续处理建议

## 输出标准
- 保持内容简洁明了，便于操作。
- 报告分为三个部分：已处理费用、被拒绝的费用、下一步需要执行的操作。
- 必须包含各项费用的总额以及缺少文件的数量。
- 如果政策信息不明确，将相关费用标记为“政策信息未提供”。
