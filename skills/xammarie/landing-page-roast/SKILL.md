# 页面诊断与优化（Landing Page Analysis and Optimization）

使用此技能来诊断页面转化率低的原因，并提出具体的优化方案。

## 输入信息：
- 页面URL或页面内容的副本
- 目标受众
- 主要的转化目标（例如：购买产品、填写表单等）
- 价格信息/优惠详情

## 评估维度：
1. **页面首屏的清晰度**：用户是否能在5秒内理解页面的核心信息？
2. **受众匹配度**：页面内容是否明确针对目标受众？
3. **产品优势**：产品或服务的优势是否具体、可衡量且可信？
4. **信任度**：页面是否提供了足够的信任背书（如用户评价、数据支持等）？
5. **用户操作流程的便捷性**：页面设计是否过于复杂（例如：字段过多、点击按钮的层级不明确、存在干扰因素）？
6. **疑虑处理**：是否解决了用户可能存在的价格、时间或风险方面的疑虑？

## 输出结果：
- 总体评分（0–10分）
- 最主要的5个阻碍转化的因素（按影响程度排序）
- 可快速实施的优化措施（修改时间小于30分钟）
- 需要修改的页面结构部分（如标题、副标题、点击按钮区域）
- 3种不同的点击按钮（CTA）设计方案
- 可选的A/B测试方案（优先测试哪些内容）

## 优化标准：
- 使用简单明了的语言
- 强调具体的成果而非空洞的口号
- 减少不切实际的承诺，增加实际的证据支持
- 保持与目标受众相符的语气风格

## 示例请求：
“针对中小型企业主（SMB）优化这个AI自动化服务页面，重新设计页面的标题和第一个点击按钮（CTA），以提高电话预约的数量。”
