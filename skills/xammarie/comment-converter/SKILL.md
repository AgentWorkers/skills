# 评论转换器

该工具能够生成与主题帖内容自然融合的评论，即使没有明确的行动号召（CTA），这些评论依然具有实用性。

## 输入内容：
- 原帖文本
- 热门评论（2–10条）
- 你的观点或经验
- 转换目标（可选）

## 可生成的评论样式：
1. **直接回答**：简洁明了、实用性强
2. **检查清单式回答**：提供逐步的操作指南
3. **案例分析式回答**：通过具体示例说明问题
4. **细致的反对意见**：以尊重的方式提出不同观点并说明理由
5. **轻量级行动号召式回答**：邀请用户进一步交流（不含强制性的销售内容）

## 规则：
- 开篇务必提供有价值的内容，切勿以链接开头
- 表达要具体：使用数字、例子来支持观点
- 保持与子版块的整体氛围一致（无论是技术类还是非正式的讨论）
- 避免使用人工智能生成的陈词滥调或企业式的语言
- 除非主题帖需要深入讨论，否则保持评论的简洁性

## 转换方法（遵循伦理原则）：
- 第一条评论：仅提供有价值的内容
- 如果用户提出请求，提供相应的模板或资源
- 第三条评论：仅在相关情况下邀请用户私信或提供链接

## 输出格式：
对于每种评论样式，需包含以下内容：
- 转换目标
- 评论草稿
- 说明为什么这种评论方式适用于当前的主题帖

## 示例：
输入：“创始人询问为什么冷启动式的推广方式没有收到任何回复”
输出示例：
**目标：** 提供一种有效的评论方式，帮助创始人解决这个问题。

**评论草稿：**
“大家好！我认为冷启动式推广可能效果不佳，因为缺乏吸引用户的元素。以下是一些建议：

1. **测试内容**：先确保你的推广信息具有吸引力。
2. **保持内容相关性**：确保你的信息与用户的需求和兴趣相关。
3. **简洁明了**：避免使用复杂的术语或长篇大论。
4. **提供帮助**：如果可能的话，提供一些实用的建议或资源。

**为什么这种评论方式适用于这个主题帖？**
- 这种评论方式直接针对主题帖的核心问题，提供了具体的解决方案。
- 通过简洁明了的表述，帮助用户快速理解问题所在。
- 提供了实用的建议和资源，有助于解决创始人的问题。”

（注：在实际应用中，可以根据需要调整评论内容和格式。）
