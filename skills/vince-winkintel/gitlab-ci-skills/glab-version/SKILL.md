# glab 版本

## 概述

```

  Show version information for glab.                                                                                    
         
  USAGE  
         
    glab version [--flags]  
         
  FLAGS  
         
    -h --help  Show help for this command.
```

## 快速入门

```bash
glab version --help
```

## 子命令

该命令没有子命令。
