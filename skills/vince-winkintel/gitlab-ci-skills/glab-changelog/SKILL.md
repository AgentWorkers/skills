# glab 更新日志

## 概述

```

  Interact with the changelog API.                                                                                      
         
  USAGE  
         
    glab changelog <command> [command] [--flags]  
            
  COMMANDS  
            
    generate [--flags]  Generate a changelog for the repository or project.
         
  FLAGS  
         
    -h --help           Show help for this command.
```

## 快速入门

```bash
glab changelog --help
```
