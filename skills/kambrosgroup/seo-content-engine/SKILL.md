# SEO内容引擎

本技能将指导您完成整个SEO内容创作流程——从关键词研究到内容发布，每一步都基于数据做出决策。

## 功能介绍

- **关键词研究**：寻找竞争较小、用户搜索意图明确的关键词。
- **SERP分析**：了解Google对目标关键词的偏好。
- **内容结构规划**：构建能够满足用户搜索意图的文章结构。
- **SEO优化草稿**：使用正确的标题层级、内部链接和元标签来撰写内容。
- **内容更新**：更新旧文章以恢复其排名。

## SEO内容创作流程

### 第一阶段：关键词分析

**输入**：主题或竞争对手的URL
**输出**：按难度排序的关键词列表

需要考虑的关键指标：
- **搜索量**：每月的搜索次数（数量多并不一定更好）
- **关键词难度**：竞争程度（新网站建议选择难度低于30的关键词）
- **搜索意图**：信息型、导航型、交易型或商业型
- **CPC**：点击成本（反映内容的商业价值）

### 第二阶段：SERP分析

在开始写作之前，分析排名前10的结果：
1. **内容类型**：博客文章？产品页面？列表文章？指南？
2. **内容深度**：字数、涵盖的章节、细节程度
3. **内容缺失的部分**：有哪些内容是你可以补充的？
4. **共同元素**：所有排名靠前的内容是否都包含视频、表格或常见问题解答？
5. **域名权威性**：你的内容是在与维基百科还是小型博客竞争？

### 第三阶段：内容结构规划

为了提高SEO效果，需要制定以下结构：
```
H1: Target Keyword (exact or close variant)
  H2: Introduction (hook + promise + proof)
  H2: What is [Topic]? (definition for informational intent)
  H2: Why [Topic] Matters (benefits, statistics)
  H2: [Number] Best [Options] (for listicles/comparisons)
    H3: Option 1
    H3: Option 2
    H3: Option 3
  H2: How to [Achieve Result] (tutorial section)
  H2: Common Mistakes to Avoid
  H2: FAQ (target "People Also Ask" questions)
  H2: Conclusion + CTA
```

### 第四阶段：草稿撰写

**页面内SEO检查清单**：
- 关键词出现在前100个单词中
- 至少一个H2标题中
- URL路径中包含关键词
- 元标题中包含关键词
- 元描述中包含关键词
- 每1000字添加3-5个内部链接
- 添加指向权威来源的外部链接
- 图片alt文本中包含关键词
- 长篇文章添加目录
- 建议使用Schema标记

**内容质量检查清单**：
- 提供原创见解或数据
- 提供可操作的建议（而不仅仅是理论）
- 使用视觉元素（如图片、表格、图表）
- 段落简短（2-3句话）
- 使用项目符号提高可读性
- 包含示例和案例研究
- 使用最新的统计数据（不超过2年）

## 高级技巧

### Skyscraper Technique 2.0

1. 找到目标关键词的相关内容排名
2. 找出内容的弱点（过时、浅显或枯燥）
3. 创作出质量更高的内容
4. 使新内容有显著区别（不要简单复制）
5. 向曾经引用过原内容的人推广新内容

### 搜索意图匹配

| 搜索意图类型 | 内容格式 | 行动号召策略 |
|-------------|----------------|--------------|
| 信息型 | 教程、解释性文章 | 注册新闻通讯、推荐相关内容 |
| 商业型 | 对比文章、评论 | 提供免费试用、请求演示 |
| 交易型 | 产品页面、价格信息 | 立即购买、限时优惠 |
| 导航型 | 品牌页面、登录帮助 | 直接引导用户到达目标页面 |

### 内容集群

使用“枢纽-辐射”模型建立主题权威性：

**核心页面**（广泛主题，3000字以上）：
- “项目管理完全指南”

**集群内容**（具体子主题，1000-1500字）：
- “敏捷开发与瀑布模型：哪种更适合你？”
- “10款项目管理工具对比”
- “如何召开站立会议”
- “项目管理认证指南”

所有集群内容都链接到核心页面；核心页面也链接到所有集群内容。

## 内容更新策略

### 何时更新内容

- 内容排名下降5位以上
- 内容发布超过2年
- 竞争对手发布了更好的内容
- 出现新的信息或产品
- 流量逐月下降

### 更新检查清单

1. 更新统计数据和示例（使用最新数据）
2. 扩展内容深度（在竞争对手不足的地方补充内容）
3. 提高可读性（缩短段落、改进格式）
4. 添加新章节（如行业趋势、新工具、更新的方法）
5. 优化新关键词（相关词汇、长尾关键词）
6. 更新元标签（使用更吸引人的标题和描述）
7. 添加内部链接（指向网站内的新内容）
8. 更新发布日期（标记为“已更新”）

## 常见错误及避免方法

1. **关键词堆砌**——自然语言的内容更受欢迎
2. **忽略搜索意图**——针对错误意图排名第一的内容无法带来转化
3. **忽视内部链接**——大多数人都忽略了内部链接的重要性
4. **忽略移动设备**——超过60%的搜索来自移动设备
5. **不推广内容**——优秀的内容如果没有推广就难以被发现
6. **发布后忽略维护**——内容需要定期更新以保持相关性
