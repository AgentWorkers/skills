# 内容再利用工具

该工具能够将一篇内容转化为多种形式，从长篇内容中提取关键信息，并根据不同的平台和受众进行重新格式化。

## 功能介绍

- **博客文章 → Twitter/X 话题帖**：将长篇文章拆分为280个字符的短句，并添加吸引人的开头语。
- **文章 → LinkedIn 帖子**：采用专业的语气，采用故事化的表达方式，并设置互动环节。
- **文字记录 → 通讯**：提取核心要点，添加章节标题和行动号召。
- **视频 → 社交媒体片段**：生成带有时间戳的引文、文字转录内容以及轮播式的视觉素材。
- **播客 → 播客笔记**：提供内容摘要、时间戳、关键引文和参考资源。

## 快速入门

### 1. 重新利用博客文章

粘贴你的博客文章，并指定目标格式：

```
"Repurpose this blog post into a Twitter thread and LinkedIn post:
[paste content]

Target audience: SaaS founders
Tone: Educational but conversational
Include: 2-3 engagement questions"
```

### 2. 从URL批量转换内容

```
"Repurpose the content at https://example.com/blog-post into:
1. A 5-tweet Twitter thread
2. A LinkedIn post with bullet points
3. An email newsletter subject line + preview

Style: Thought leadership
CTA: Subscribe to newsletter"
```

### 3. 播客/视频文字记录

```
"Turn this transcript into show notes and 3 social snippets:
[paste transcript]

Host: [name]
Guest: [name]
Episode topic: [topic]"
```

## 输出格式

### Twitter/X 话题帖
- 吸引注意力的开头语
- 3-7条包含关键信息的推文
- 结尾处包含行动号召
- 可选：设置互动问题（如提问或投票）

### LinkedIn 帖子
- 以个人故事或引人注目的观点作为开头
- 3-5个要点或简短段落
- 内容总结
- 提出互动问题
- 使用3-5个相关标签

### 通讯
- 3种不同的主题行选项
- 预览文本
- 吸引人的开头语
- 3-5个带有标题的章节
- 提供资源链接
- 包含行动号召

### 社交媒体片段
- 用于图片配文的引文卡片
- 5-7张轮播幻灯片
- 15-30秒的短视频脚本

## 高级功能

- Week 1: Twitter thread (Monday), LinkedIn (Wednesday), Newsletter snippet (Friday)
- Week 2: Quote graphics (Tuesday), Carousel post (Thursday), Engagement question (Saturday)

Include posting times and platform-specific formatting."

```
"Repurpose this content for [language] audience:
- Translate the core message
- Adapt cultural references
- Adjust tone for [region] business culture
- Output: LinkedIn post + Twitter thread"
```

### 语气调整

可选的语气风格：
- **专业**：正式、权威、数据驱动
- **对话式**：友好、平易近人、以故事为主
- **尖锐/大胆**：具有争议性、引人思考
- **教育性**：以教程形式呈现，步骤清晰、实用
- **励志**：鼓舞人心、激发目标

## 最佳实践

1. **使用高质量的内容源** — 糟糕的输入会导致糟糕的输出。
2. **明确目标受众** — 目标越具体，输出效果越好。
3. **添加背景信息** — 展示你的专业视角、独特见解和个人经验。
4. **审核并个性化内容** — 加入你的观点、示例和个人故事。
5. **测试不同的开头语** — 第一句话至关重要；生成3-5个选项进行测试。

## 平台特定建议

### Twitter/X
- 在第一条推文中提供核心内容
- 使用换行符提高可读性
- 结尾处设置互动问题
- 避免在第一条推文中使用外部链接（符合算法偏好）

### LinkedIn
- 以个人经历或不同观点作为开头
- 使用简短的段落（1-2句话）
- 为移动设备优化排版
- 适量添加相关标签

### 通讯
- 主题行至关重要（占80%的效果）——多尝试几个不同的版本
- 预览文本应与主题行相辅相成，而非重复主题内容
- 每封邮件中只包含一个明确的行动号召
- 优化移动设备阅读体验（大部分打开是在手机上）

## 常见应用模式

### “一个内容，五种格式”的工作流程

1. **原始内容**：2000字的博客文章
2. **Twitter**：7条包含关键信息的推文
3. **LinkedIn**：个人故事 + 3个学习要点
4. **通讯**：精选内容汇总，原始文章作为亮点
5. **Instagram**：5张轮播幻灯片
6. **TikTok/Reels**：3段15秒的视频

### “永恒内容”的工作流程

1. 分析表现最好的内容
2. 使用新数据或示例进行更新
3. 结合当前趋势重新制作内容
4. 安排最佳发布时间
5. 在多个平台上进行交叉推广
