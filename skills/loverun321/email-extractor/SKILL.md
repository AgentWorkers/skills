# 电子邮件提取器

从任何网页或文本内容中提取电子邮件地址。

## 功能

- 从 URL 中提取电子邮件
- 从纯文本中提取电子邮件
- 去除重复结果
- 仅返回前 20 个结果

## 使用方法

```
"Extract emails from https://example.com"
"Get all email addresses from this page"
"Find emails in [text content]"
```

## 示例响应

```json
{
  "emails": ["john@example.com", "jane@company.org"],
  "count": 2,
  "source": "https://example.com"
```
