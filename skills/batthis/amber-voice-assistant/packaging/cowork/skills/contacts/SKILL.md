# 联系人信息

将联系人姓名解析为电话号码，以便进行外拨电话。

## 工作流程

1. 用户输入“call [姓名]”或“/amber:call [姓名] [目标]”。
2. 通过 `crm(action: "lookup", identifier: "姓名")` 在客户关系管理系统（CRM）中查找该姓名。
3. 如果找到该姓名，则使用存储的电话号码进行拨打电话。
4. 如果找到多个匹配项，向用户显示选项并让其进行选择。
5. 如果未找到该姓名，则询问用户具体的电话号码。

## 指导原则

- 姓名匹配时可以有一定的模糊性（例如，“John”应与“John Smith”匹配）。
- 在拨打电话之前，务必进行确认（例如：“我找到了John Smith，电话号码是+1-416-555-1234，对吗？”）。
