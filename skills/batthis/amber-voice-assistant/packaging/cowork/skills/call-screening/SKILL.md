# 呼叫筛选

Amber 充当人工智能接待员，负责处理来电。她会专业地接听电话，收集相关信息，并提供结构化的摘要，这样您就可以只接听重要的电话。

## 筛选流程

1. **问候** — Amber 会以可自定义的问候语回应来电者。
2. **身份确认** — 询问来电者的姓名以及来电目的。
3. **信息收集** — 收集来电者的姓名、回电号码以及留言内容。
4. **CRM 查询** — 检查来电者是否为已知联系人（自动补充相关信息）。
5. **日程安排** — 如果来电者希望预约时间，会检查您的可用性。
6. **摘要发送** — 向您发送包含所有收集到的详细信息的结构化摘要。

## MCP 工具

### start_screening
在配置好的 Twilio 号码上启用来电筛选功能。

### stop_screening
禁用筛选功能（电话将正常接通）。

### get_screening_status
检查当前的筛选状态是否处于激活状态。

## 自定义设置

筛选的“个性”、问候语以及行为设置都保存在 AGENT.md 文件中。用户可以编辑该文件来：
- 更改助手的名称和“个性”。
- 自定义问候语。
- 设置工作时间和非工作时间的行为规则。
- 规定哪些来电者可以直接接通，哪些需要经过筛选。
- 添加与组织相关的信息（如公司名称、服务内容等）。
