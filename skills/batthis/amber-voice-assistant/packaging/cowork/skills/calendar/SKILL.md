# 日历

在通话期间或通话之外，可以查询和管理操作员的日历。

## MCP 工具

### calendar_query
用于查找或创建日历事件。
- `action` (字符串，必填): "lookup" 或 "create"
- `query` (字符串，查找时必填): 自然语言查询（例如："我周二下午3点有空吗？")
- `title` (字符串，创建时必填): 事件标题
- `start` (字符串，创建时必填): 开始时间（ISO 8601 格式）
- `end` (字符串，创建时必填): 结束时间（ISO 8601 格式）
- `calendar` (字符串): 使用的日历（默认为操作员的主要日历）
- `location` (字符串): 事件地点
- `notes` (字符串): 事件备注

## 使用指南

- 在通话中查询可用时间时，要自然地提供选择项（例如："我看到下午2点和4点都有空——哪个时间更合适？")
- 在创建事件之前，务必确认最终的日期、时间和详细信息
- 将通话中的相关背景信息添加到事件备注中（谁提出了请求、事件目的等）
