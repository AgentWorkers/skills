# 电话通话

Amber可以通过Twilio发起和接听真实的电话通话。此功能涵盖了基本的电话通信能力。

## MCP工具

### make_call
发起一个出站电话通话。
- `to` (字符串，必填)：电话号码（E.164格式，例如：+14165551234）
- `objective` (字符串，必填)：通话的目的
- `mode` (字符串)：`conversation`（正常通话）或`message`（单向信息发送）

### get_call_status
检查当前正在进行或最近完成的通话状态。
- `callId` (字符串，必填)：由`make_call`返回的通话ID

### end_call
终止一个正在进行的通话。
- `callId` (字符串，必填)：要终止的通话ID

### get_call_history
检索最近的通话记录及通话内容。
- `filter` (字符串)：`all`、`inbound`（来电）、`outbound`（去电）、`missed`（未接来电）
- `limit` (数字)：返回的通话记录数量（默认：10条）

## 使用指南

- 在拨打电话之前，务必与用户确认对方的电话号码和通话目的。
- 如果通话涉及支付、存款或财务承诺，请先获得用户的明确同意。
- 每次通话结束后，提供总结信息，包括：通话对象、通话结果、交换的关键信息以及来电者的情绪。
- 对于出站通话，Amber会自主完成通话目标——她不会机械地按照脚本执行操作。
- 电话通话具有实际影响，请始终以专业的方式代表用户进行通话。
