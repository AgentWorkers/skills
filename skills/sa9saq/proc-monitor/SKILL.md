# 进程监控器

用于监控系统进程及资源使用情况。

## 功能

- **热门进程**：按CPU或内存使用量显示进程
- **进程搜索**：通过名称或PID查找特定进程
- **资源概览**：显示总CPU使用量、内存使用量、交换空间使用量以及负载平均值
- **警报**：识别超出阈值的进程

## 使用方法

您可以要求代理执行以下操作：
- “显示内存使用量最高的10个进程”
- “是否有进程的CPU使用率超过了80%？”
- “查找所有节点上的进程”
- “提供系统资源使用情况的概览”

## 工作原理

该工具使用标准的系统工具来实现监控功能：

```bash
ps aux --sort=-%mem | head -20
top -bn1 | head -20
free -h
uptime
```

## 系统要求

- 需要标准的Unix工具（`ps`、`top`、`free`、`uptime`）
- 不需要API密钥
