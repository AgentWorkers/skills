# 着陆页转换技能

使用 Jovan Industries 转换框架中经过验证的模板，将任何产品或服务信息转换为高转化率的着陆页或产品页面。

### 各部分详细说明

**1. 导航栏（顶部菜单）**
- 水平导航栏
- 左侧显示品牌标志
- 中间显示菜单链接（商店、关于我们、博客、联系我们）
- 右侧显示购物车图标及商品数量

**2. 购物框（主打产品模块）**
- 左侧展示产品图片（大尺寸、清晰可见）
- 产品名称醒目显示
- 产品名称上方显示社交认可信息（星级评分 + 评论数量）
- 价格清晰可见
- 选择产品规格的选项（颜色、尺寸、数量）
- 主要呼叫行动（CTA）：“加入购物车”
- 次要呼叫行动（CTA）：“立即购买”

**3. 第一种社交认可方式（客户评价）**
- 信任标题（“受到 X+ 位客户的喜爱”）
- 三行客户评价
- 每条评价包括：引用语、星级评分、客户姓名、头像
- 评价上方显示分享按钮

**4. 第二种社交认可方式（用户生成内容）**
- 用户生成的内容区域
- 以网格形式展示客户照片
- 显示图片来源（Instagram/Twitter）
- 显示更多评价或社交提及

**5. 产品优势（图标列表）**
- 4-6 项产品优势，每项优势配图标
- 每项优势下方附简短描述
- 图标需与品牌风格一致

**6. 成分/材料（规格表格）**
- 关键成分/材料的可视化表格
- 每项成分的图片
- 标签/复选框显示包含的内容
- 下方提供完整成分列表或规格表

**7. 第三种社交认可方式（视频/案例研究）**
- 视频弹出按钮（“观看使用效果”）
- 显示使用前后的对比案例（包含客户姓名和地点）
- 带有社交媒体账号的影响者引用语

**8. 对比表**
- 产品与竞品的对比
- 逐项功能对比（用复选框或 X 标记）
- 价格对比栏
- 质量/评分对比

**9. 常见问题解答（折叠式菜单）**
- 可展开/可折叠的问题列表
- 解答常见疑问
- 4-8 对问答，附清晰答案

**10. 最终呼叫行动（CTA）**
- 大型、醒目的“加入购物车”按钮
- 强调紧迫性的标题
- 显示信任标志（保修、配送、安全信息）
- 底部导航栏（与顶部菜单相同，但简化）

## 如何使用此技能

### 第一步：收集内容
在构建页面之前，收集以下信息：
- 产品/服务名称
- 目标受众
- 你想要解决的主要问题
- 3-5 项主要优势
- 产品特性/规格
- 价格和优惠信息
- 客户评价（或占位内容）
- 品牌颜色/字体

### 第二步：选择模板
- **着陆页**：用于吸引潜在客户、收集电子邮件地址、注册网络研讨会、提供免费试用
- **产品页面**：用于直接销售产品、电子商务

### 第三步：将内容对应到模板中

**对于着陆页：**
1. 主打产品部分 → 标题、子标题、呼叫行动（CTA）、产品图片
2. 社交认可部分 → 客户数量、品牌标志
3. 问题描述部分 → 用客户的语言描述问题
4. 问题加剧部分 → 问题会如何变得更严重
5. 解决方案部分 → 产品介绍及优势
6. 使用方法部分 → 使用步骤（3 步）
7. 客户评价部分 → 社交认可信息
8. 产品详情部分 → 包含的内容及额外福利
9. 风险规避部分 → 保修政策
10. 最终呼叫行动部分 → 强调紧迫性及按钮

**对于产品页面（10 部分结构）：**
1. 导航栏 → 顶部水平导航栏，包含菜单链接、品牌标志、购物车图标
2. 购物框 → 中心位置的产品购买模块：左侧显示产品图片，右侧显示产品名称和价格，下方有“加入购物车”按钮；确保产品名称清晰可见，上方有社交认可信息
3. 第一种社交认可方式 → 第一行客户评价：引用语、星级评分、头像；评价上方显示分享按钮
4. 第二种社交认可方式 → 更多客户评价或用户生成的内容
5. 产品优势部分 → 图标列表，每项优势下方附简短描述
6. 成分/材料部分 → 产品成分或规格表格：图片、标签、复选框
7. 第三种社交认可方式 → 视频弹出按钮、影响者引用或案例研究
8. 对比表 → 产品与竞品的对比表：列出功能、价格等信息
9. 常见问题解答部分 → 可展开的问题列表
10. 最终呼叫行动部分 → 大型、醒目的“加入购物车”按钮；下方显示信任标志及简化后的底部导航栏

### 第四步：输出格式
输出格式包括：
- **Figma 可用的设计规范**：包含准确的文本内容、所需图片和颜色
- **HTML 结构**：语义化的 HTML 代码，包含占位内容
- **文本文档**：仅提供设计师使用的文本内容和标题

## 转换原则

### 标题编写公式
- [情感表达] + [具体结果] + [时间框架]
- 例如：“7 天内拥有光彩照人的肌肤”

### 呼叫行动（CTA）按钮规则
- 使用与品牌颜色对比明显的颜色
- 使用动词+优势的表述：“获取我的免费指南”
- 既放在页面上方，也放在页面底部
- 按钮尺寸至少为 44 像素

### 社交认可信息的放置位置
1. 紧接在主打产品部分之后（品牌标志下方）
2. 优惠信息部分之前（客户评价部分）
3. 页面上方（星级评分下方）

### 必须包含的风险规避信息
- 退款保证（期限）
- “无任何疑问”
- “免费配送”（如适用）
- “安全支付”
