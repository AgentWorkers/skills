## 使用方法

### 1. 创建钱包

```typescript
import { privateKeyToAccount, generatePrivateKey } from 'viem/accounts';
import { createWalletClient, http } from 'viem';
import { mainnet } from 'viem/chains';

const privateKey = generatePrivateKey();
const account = privateKeyToAccount(privateKey);
console.log('Address:', account.address);
// Store privateKey securely!

const walletClient = createWalletClient({
  account,
  chain: mainnet,
  transport: http('https://eth.llamarpc.com'),
});
```

### 2. 铸造一个ClawPhunk角色

**端点：** `POST https://clawphunks.vercel.app/mint`

**支付方式：** 使用x402协议支付1.99美元（Base币种）

**请求内容：**
```json
{
  "recipient": "0xYourWalletAddress"
}
```

**响应内容：**
```json
{
  "success": true,
  "tokenId": 1234,
  "txHash": "0x...",
  "ethscriptionId": "0x...",
  "gasStipendWei": "13333333333333",
  "viewerUrl": "https://ethscriptions.com/ethscriptions/0x...",
  "nextSteps": {
    "trade": "Use escrow contract on L1",
    "list": "depositAndList(ethscriptionId, priceWei)",
    "buy": "buy(ethscriptionId) with msg.value = price"
  }
}
```

你将获得：
- 一个随机生成的、尚未铸造的ClawPhunk角色（存储在以太坊L1上）
- 约0.03美元的ETH燃气费用（用于交易）

### 在L1上进行交易

**托管合约：** `0x3e67d49716e50a8b1c71b8dEa0e31755305733fd`

该托管合约使用ESIP-2协议实现无信任交易。

#### 出售流程

1. 将你的ClawPhunk角色的以太坊地址转移到托管合约中
2. 调用 `depositAndList(bytes32 ethscriptionId, uint256 priceWei)` 

```typescript
import { encodeFunctionData } from 'viem';

// First, send ethscription to escrow (send tx to contract with ethscription data)
// Then list it:
const data = encodeFunctionData({
  abi: ESCROW_ABI,
  functionName: 'depositAndList',
  args: [ethscriptionId, priceWei],
});

await walletClient.sendTransaction({
  to: '0x3e67d49716e50a8b1c71b8dEa0e31755305733fd',
  data,
});
```

#### 购买列表中的角色

```typescript
// Check listing
const [active, seller, price] = await publicClient.readContract({
  address: '0x3e67d49716e50a8b1c71b8dEa0e31755305733fd',
  abi: ESCROW_ABI,
  functionName: 'getListing',
  args: [ethscriptionId],
});

// Buy it
await walletClient.writeContract({
  address: '0x3e67d49716e50a8b1c71b8dEa0e31755305733fd',
  abi: ESCROW_ABI,
  functionName: 'buy',
  args: [ethscriptionId],
  value: price,
});
```

#### 取消列表

```typescript
await walletClient.writeContract({
  address: '0x3e67d49716e50a8b1c71b8dEa0e31755305733fd',
  abi: ESCROW_ABI,
  functionName: 'cancelAndWithdraw',
  args: [ethscriptionId],
});
```

## x402支付协议

ClawPhunks使用x402协议进行支付：

1. 如果尝试铸造角色但未支付，系统会返回“402 Payment Required”并提示支付详情
2. 支付1.99美元（Base币种）
3. 重新尝试时需要在请求头中包含支付证明
4. 支付成功后即可获得角色

如果你使用Coinbase AgentKit，x402协议的支付过程将自动完成。
