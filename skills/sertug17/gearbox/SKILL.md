# 风力涡轮机齿轮箱智能诊断系统

该系统通过五个输入参数来评估齿轮箱的运行状态，并生成结构化的维护报告。

## 使用场景

当用户需要以下操作时，请使用此功能：
- 根据现场检查或传感器数据评估齿轮箱的健康状况
- 将损坏程度分为1-5级
- 判断是否需要关闭涡轮机，或在监测下继续运行
- 生成详细的维护或升级计划

## 快速参考

| 输入参数            | 需要收集的信息                                      |
|-------------------|--------------------------------------------|
| 视觉检查            | 表面裂纹、凹陷、剥落、变色、异物                         |
| 油中铁含量（Fe ppm）       | 齿轮油中的铁颗粒浓度（ppm）                         |
| 温度（℃）            | 轴承/齿轮温度（相对于基线的变化值）                      |
| 振动              | 均方根（RMS）或峰峰值加速度（g），频率异常                 |
| 声学/声音            | 噪音类型：研磨声、敲击声、啸叫声、咔嗒声                     |

## 故障阈值（参考值）

| 参数                | 正常值        | 警告值        | 危险值        |
|-------------------|---------------|----------------|----------------|
| 油中铁含量（ppm）        | < 100         | 100 - 300      | > 300          |
| 温度相对于基线        | < 5℃         | 5 - 15℃       | > 15℃         |
| 振动（RMS，g）         | < 0.5         | 0.5 - 1.5      | > 1.5          |
| 声学异常            | 无异常        | 偶发性       | 持续性       |
| 视觉检查            | 表面干净       | 轻微凹陷       | 剥落/裂纹        |

## 常见故障模式

| 故障模式            | 典型迹象                                        |
|-------------------|---------------------------------------------------------------|
| 微小凹陷            | 高铁含量、轻微振动增加、无明显裂纹                     |
| 剥落                | 高铁含量、振动加剧、表面可见损伤                     |
| 疲劳裂纹            | 敲击声、齿轮啮合频率处的振动峰值                     |
| 轴承磨损            | 啜叫声、温度升高、宽频振动增加                         |
| 油液污染            | 铁含量极高、油液变色、可能起泡                        |

## 操作步骤

1. 收集所有五个参数的数据。如果某个参数无法获取，标记为“未测量”并继续下一步。
2. 根据阈值表评估每个参数的状态，标记出警告或危险区域。
3. 分析各参数之间的关联：
   - 铁含量增加 + 振动加剧 → 磨损/剥落进程
   - 敲击声 + 振动峰值 → 疲劳裂纹
   - 温度升高 + 啜叫声 → 轴承故障
   | 多个危险标志 → 故障等级为5级
4. 判断故障等级：
   - 1级（健康）：所有参数正常，无需采取行动。
   - 2级（早期磨损）：1-2个参数处于警告范围内，增加监测频率。
   - 3级（中度损坏）：2-3个参数处于警告/危险范围内，需在2周内进行检查。
   - 4级（严重损坏）：多个参数处于危险范围内，需在48-72小时内计划停机。
   - 5级（危险）：存在立即故障风险，需立即停机。
5. 根据故障模式表确定根本原因。
6. 使用以下格式生成输出报告。

## 输出格式

=== 齿轮箱健康报告 ===

**根本原因**：[例如：中间轴齿轮的逐渐剥落现象]
**故障等级**：[1-5级] - [健康 / 早期磨损 / 中度损坏 / 严重损坏 / 危险]
**是否需要停机**：[是 / 否 / 条件性]
**监测策略**：
  - [例如：72小时后重新采样油样]
  - [例如：连续一周监测每日振动趋势]
**升级触发条件**：
  - [例如：铁含量超过400 ppm - 立即停机]
  - [例如：振动均方根超过2.0 g - 立即停机]

## 注意事项

- 严禁仅根据一个参数判断故障等级，需至少结合两个数据来源进行验证。
- 在极端环境条件下，温度读数可能具有误导性，请使用相对于基线的值。
- 声学描述具有主观性，需明确噪音类型及是否为持续性噪音。
- 如果传感器数据缺失，应以视觉检查和油液状况作为主要依据。
- 不要将换油周期与油液健康状况混淆，新油也可能含有高浓度的铁颗粒。

## 验证步骤

生成报告后，与用户确认：
- 故障等级是否与现场观察结果一致？
- 所设定的升级阈值是否适用于当前的监测环境？
- 是否有额外的数据（如CMS趋势数据、历史铁含量数据）可以进一步细化评估？
