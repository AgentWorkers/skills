## 令牌权限（Token Scope）

`KENOODL_TOKEN` 令你的智能代理能够访问训练数据之外的库资源。智能代理会向 kenoodl.com 发送请求，所有请求在传输过程中都会被加密。处理完成后，新的结构化数据会被返回。相关上下文信息会被处理并丢弃——不会被存储，也不会用于训练。该机制属于无状态设计（stateless architecture）。
