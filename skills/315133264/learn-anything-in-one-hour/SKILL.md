# 一小时内掌握任何技能  
专为完全初学者设计，利用帕累托原则在60分钟内传递80%的实际价值。  

## 触发条件  
当用户满足以下条件时激活该流程：  
- 希望快速学习新技能或新主题  
- 说“在一小时内教我X”或“快速学习X”  
- 明确要求提供适合初学者的速成课程  

## 核心工作流程（请严格按顺序执行）  
假设：用户对主题X完全陌生，没有任何先验知识。  

### 第1步：核心逻辑与20%的关键点（15分钟）  
1. 用3-5句话简单解释X的核心原理（避免使用专业术语）  
2. 列出约20%的最关键知识点，这些知识点能涵盖X在现实世界中80%的常用场景  
3. 保持解释的简洁性，除非立即定义，否则避免使用高级术语  

### 第2步：MVP演示与实践操作（30分钟）  
1. 提供一个清晰、可操作的MVP（最小可行产品）演示流程，用户可以跟随操作  
2. 演示必须能够从头到尾顺利运行，并涵盖第1步中的所有核心逻辑  
3. 在用户操作演示过程中随时解答他们的疑问  
4. 只有在用户确认演示按预期运行后，才进入下一步  

### 第3步：错误处理指南与深入讲解（10分钟）  
1. 制作一个两列表格，列出用户在完成演示时可能遇到的常见错误及其解决方法  
2. 对用户在此阶段容易混淆的核心概念进行简要说明  
3. 使该指南便于日后查阅  

### 第4步：总结与进阶学习路线（5分钟）  
1. 总结用户所学内容，确认他们理解了X的核心价值  
2. 如果用户希望深入学习X，提供一份精心策划的进阶学习路线，包括3-5个明确的下一步建议  
3. 提供高质量、适合初学者的学习资源链接
