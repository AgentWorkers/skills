# 社交媒体自动化工具 📱

将您的长篇内容自动转换为吸引人的社交媒体帖子。

## 功能介绍

- 📄 **文章分析**：从任何文章中提取关键点和引用
- 🐦 **Twitter 线程**：生成 3-10 条带有吸引人的引言的推文
- 💼 **LinkedIn 帖子**：创建专业的長篇帖子
- 📸 **Instagram 说明文**：编写带有标签的吸引人的说明文
- 📅 **调度**：为 Buffer/Hootsuite 准备帖子
- 🔄 **多版本生成**：生成多个版本以进行 A/B 测试

## 命令

### 生成所有帖子
```
Create social media posts from "[article URL or text]"
```

### 仅生成 Twitter 线程
```
Generate Twitter thread from "[article]" - 5 tweets
```

### 仅生成 LinkedIn 帖子
```
Create LinkedIn post from "[article]"
```

### 带有调度功能
```
Generate posts from "[article]" and schedule for next week
```

## 输出格式

### Twitter 线程（3-10 条推文）
- 引言推文（吸引注意力）
- 每条推文包含一个关键点
- 突出显示数据和统计信息
- 最后一条推文包含行动号召
- 每条推文包含 2-3 个相关标签

### LinkedIn 帖子
- 专业风格
- 1000-1300 字
- 通过换行提高可读性
- 包含 3-5 个行业相关标签
- 文末包含互动问题

### Instagram 说明文
- 休闲、吸引人的风格
- 150-200 字
- 包含 10-15 个相关标签
- 使用表情符号

## 配置

可选的环境变量：
- `DEFAULTPLATFORM`：默认平台（twitter|linkedin|instagram|all）
- `THREAD_LENGTH`：默认 Twitter 线程长度（默认：5）
- `INCLUDE_EMOJIS`：是否在帖子中添加表情符号（true|false，默认：true）
- `HASHTAG_COUNT`：标签数量（默认：3）
