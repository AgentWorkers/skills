## 什么是Vibe Design？

Vibe Design是一种设计方法，你只需描述自己的需求，让AI生成视觉效果、布局和设计资源。你可以根据生成的结果进行迭代，而无需从头开始逐像素地绘制。这与将“Vibe Coding”理念应用于视觉设计的思路是一致的。

**核心原则：**AI负责探索和生成，而你负责筛选和优化设计结果。

## 核心规则

### 1. 使用设计术语，而非艺术术语
AI对设计术语和艺术术语的反应是不同的。

❌ 不建议使用的艺术类词汇：**“beautiful”（美丽的）、**“fantasy”（幻想的）、**“render”（渲染）、**“painting”（绘画）、**“stunning”（惊人的）
✅ 建议使用的设计类词汇：**“interface”（界面）、**“layout”（布局）、**“component”（组件）、**“spacing”（间距）、**“hierarchy”（层次结构）

**示例：**  
错误表达：**“一个美观的移动银行产品，采用极简的未来主义风格。”  
**正确表达：****“一个基于Material Design 3设计的移动银行应用UI，布局清晰，间距统一，数据卡片中包含财务指标。”**

### 2. 使用设计系统来规范设计
指定设计系统的名称可以显著提升设计的条理性：  
- **“Material Design 3”**：意味着布局平衡、卡片设计合理、带有阴影效果  
- **“Apple HIG”**：指苹果公司的设计规范（柔和的圆角、清晰的字体、原生感）  
- **“IBM Carbon”**：适用于企业级仪表盘的设计风格  
- **“Atlassian”**：适合B2B场景的设计语言，字体清晰简洁  

### 3. 有结构地构建提示
提示语句应该结构清晰：
```
[Project type] UI design for [product/audience],
[style keywords],
[layout type],
[color palette],
[design system reference],
[rendering style]
```

### 4. 宽高比很重要  
- 网页：**--ar 16:9**  
- 移动设备：**--ar 4:5 或 --ar 9:16**  
- 正方形图标：**--ar 1:1**  
**注意：**始终指定宽高比以确保输出的一致性。

### 5. 迭代而非一次完美
Vibe Design强调快速探索：  
1. 生成4-8个设计方案  
2. 选择效果较好的方案  
3. 进行修改：**“像图片2那样，但使用图片3中的配色方案”  
4. 在Figma等制作工具中进一步完善最终设计  

### 6. AI用于探索，工具用于精调  
- **AI的作用：**生成设计灵感、方向和初步布局  
- **制作工具的作用：**实现精确的像素间距、添加实际内容、确保设计具有响应式特性  

**注意：**不要试图直接用AI生成完全符合制作要求的最终设计。AI主要用于探索阶段，20%的细节工作需要手动完成。

### 7. 用参考图例辅助说明  
**尽可能用图片或链接来展示设计成果，而非仅仅用文字描述：**  
- 附上参考图片  
- 链接到你喜欢的现有设计作品  
- 使用“[具体设计师/品牌风格]”这样的表述来指明设计灵感来源  

### 8. 保持设计系统的一致性  
在正式制作时：  
- 将AI生成的方案作为参考，而非最终成果  
- 在你的设计系统中重新构建设计  
- 保持间距、字体和颜色的统一性  
- 将AI提供的灵感转化为人类可实现的成品设计  

### 9. 了解工具的优势  
| 工具 | 最适合的场景 |  
|------|----------|  
| Midjourney | 用于创意概念的探索和艺术风格的设计 |  
| Figma AI | UI设计的迭代和最终效果的调整 |  
| DALL-E/GPT | 快速制作原型图和图像编辑 |  
| Stable Diffusion | 可自定义训练参数的开源工具 |  
| Canva | 制作营销素材和社交媒体内容 |
