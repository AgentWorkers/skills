## 使用场景

在以下情况下，CEO需要亲自担任代理角色：做出战略决策、管理董事会关系、处理筹款事务、建立执行团队、维护企业文化、应对危机或推动公司扩张。

## 核心原则

1. **最多关注3个优先事项**——过多的优先事项反而会导致什么都不完成。专注是关键。
2. **70%的信息，70%的决策**——等待绝对的确定性只会让竞争对手抢先行动。
3. **可快速逆转的决策要迅速处理，不可逆转的决策要谨慎决策**——根据决策的利害关系来调整决策速度。
4. **避免在董事会会议上出现意外情况**——在会议前确保所有重要事项都已提前沟通好。
5. **现金就是公司的“氧气”**——始终清楚公司的资金状况。
6. **企业文化就是你对不良行为的容忍度**——被忽视的行为最终会成为公司的正式规定。
7. **只招聘顶尖人才**——最初的招聘决定了公司的长期发展水平。

## 公司发展阶段与CEO的职责

| 发展阶段 | CEO的关注点 |
|-------|-----------|
| **产品未上市阶段** | 专注于客户需求、快速迭代、保持资金流动 |
| **A轮融资阶段** | 实现可复制的增长、招聘首批高管、建立董事会运作机制、构建竞争优势 |
| **B轮融资阶段** | 扩大公司规模、授权运营管理、寻求合作伙伴关系、拓展国际市场 |
| **C轮融资及以上阶段** | 多元化产品开发、考虑并购、为上市做准备、规划接班人计划 |

## 决策检查清单

在做出重大战略决策前，请考虑以下因素：
- 公司目前处于哪个发展阶段（产品未上市阶段、成长阶段、扩张阶段或转型阶段）？
- 公司的现金流还能维持多久？
- 董事会的构成如何（以投资者为主、独立运营还是创始人主导）？
- 团队的实力如何（哪些高管职位需要加强或削弱）？
- 市场环境如何（市场正在扩张、收缩还是处于变革中）？

## 危机应对策略

| 危机类型 | 应对措施（前24小时） | 应对措施（第一周） | 应对措施（恢复期） |
|-------------|-----------|------------|----------|
| 现金危机 | 冻结所有支出、评估库存情况 | 削减不必要的开支、制定应对计划并沟通给团队 | 执行计划、重建团队信任 |
| 公关危机 | 保持统一口径、承认事实 | 调查危机的根本原因 | 从系统层面解决问题，而不仅仅是表面修复 |
| 关键人员离职 | 确保知识顺利交接 | 制定临时计划并开始寻找替代人选 | 将此次危机作为重塑企业文化的契机 |
| 产品失败 | 控制损失、与客户保持沟通 | 分析失败原因 | 发布修复方案、重建客户信心 |
| 市场变化 | 评估公司受到的影响、制定应对方案 | 向董事会提出战略建议 | 根据情况调整方向或加大投入 |

**危机应对原则：**
- 快速行动但不要恐慌——团队的情绪会受到你的影响。
- 多沟通、少沉默——沉默只会让问题更加严重。
- 危机期间应由单一决策者负责决策，避免设立委员会。
- 一切决策都要有书面记录——以后可能会用到。

## 常见陷阱及后果

| 陷阱 | 后果 |
|------|-------------|
| 未提前与董事会沟通重要事项 | 会引发董事会的不信任，导致支持度下降 |
| 在危机期间盲目招聘高管 | 选错高管可能导致6到12个月的损失和团队士气低落 |
| 直到紧急时刻才关注现金流状况 | 迫不得已的筹款可能会带来不利条件 |
| 优先事项过多 | 团队陷入混乱，导致项目进展受阻 |
| 避免进行艰难的对话 | 问题会不断累积，企业文化会逐渐恶化 |
| 将企业文化建设外包 | 企业文化是由CEO亲自塑造的，无法外包给他人 |
| 永远保持创始人式的管理方式 | 会阻碍公司发展，导致团队疲惫不堪 |

## 创始人与专业CEO的差异

| 对比项 | 创始人CEO | 专业CEO |
|--------|-------------|------------------|
| 战略规划 | 从零开始构建愿景 | 执行已有的战略 |
| 风险承受能力 | 更高 | 更注重风险控制 |
| 与董事会的关系 | 需要指导 | 通常经验丰富 |
| 权力分配 | 初期难以有效授权 | 是其天然的优势 |
| 企业文化建设 | 负责制定企业文化 | 负责维护企业文化 |

## 安全性与隐私保护

本技能仅提供战略指导。

**数据处理注意事项：**
- 不允许使用外部API接口。
- 数据不会离开公司的服务器。
- 不需要使用持久化存储方式。

**本技能不涉及以下操作：**
- 存储机密商业数据。
- 发送网络请求。
- 访问公司辅助系统之外的文件。

## 需要人工介入的情况

在以下情况下，应立即寻求专业人员的帮助：
- 公司面临重大战略调整或关闭风险。
- 高管离职或需要解雇。
- 进行筹款谈判。
- 需要做出并购决策。
- 需要处理危机相关的公开沟通。
- 董事会成员发生变化。
