## 代理的功能

| 用户请求 | 代理操作 |
|--------------|--------------|
| “在$Y价格范围内找到最好的X产品” | 搜索、比较产品评价/评分/价格历史，提供推荐 |
| “跟踪[产品]的价格” | 监控价格变化，发送价格下降提醒，建议购买时机 |
| “重新订购我的[消耗品]” | 查找之前的订单，核对当前价格与上次购买时的价格，重新下单 |
| “这个优惠是真的吗？” | 检查价格历史，判断价格是否虚假上涨后打折 |
| “比较[A]和[B]” | 逐项对比产品规格、用户评价和价值分析 |
| “帮我销售[产品]” | 优化商品列表、进行关键词研究、制定定价策略 |
| “生成联盟链接” | 创建带有标签的联盟链接，并跟踪链接效果 |

## 买家模式 — 核心功能

**产品研究：**
- 使用过滤器搜索（价格、评分、是否支持Prime会员、卖家类型）
- 汇总产品评价，总结优缺点，识别虚假评价 |
- 通过功能矩阵比较不同产品 |
- 检查卖家信誉（第三方风险评估）

**价格监控：**
- 跟踪历史价格变化（识别虚假折扣）
- 在价格下降时发送提醒 |
- 确定最佳购买时机（如Prime Day、Black Friday等促销活动期间）
- 在适用的情况下跨亚马逊地区比较价格

**购物流程：**
- 将商品加入购物车，应用优惠券或订阅服务 |
- 为重复购买的商品重新下单，并验证价格 |
- 购买礼品商品时协调配送事宜 |
- 发起退货/退款流程

详细操作流程请参阅`buying.md`。

## 卖家模式 — 核心功能

**商品列表管理：**
- 创建或优化商品列表 |
- 进行关键词研究以提高商品在搜索结果中的可见性 |
- 提供A+级别的商品描述建议 |
- 确保图片符合亚马逊的发布要求

**运营管理：**
- 监控库存并接收补货提醒 |
- 规划FBA发货 |
- 监控竞争对手的价格策略 |
- 监控用户评价并准备回复评论

**数据分析：**
- 分析销售速度和转化率 |
- 评估广告效果（如PPC广告） |
- 计算利润率（包括费用、运费、退货等因素）

卖家相关操作流程请参阅`selling.md`。

## 联盟模式 — 核心功能

**链接管理：**
- 生成带有正确标签的联盟链接 |
- 根据平台需求缩短或自定义链接长度 |
- 跟踪链接的点击率和转化情况

**内容优化：**
- 选择高佣金率的商品类别 |
- 关注季节性热门产品 |
- 提供产品对比类内容建议

联盟计划相关策略请参阅`affiliates.md`。

## 重要的安全规则

**账户凭证管理：**
- **严禁** 以明文形式存储亚马逊密码 |
- **禁止** 在不同设备之间共享会话cookie |
- **禁止** 绕过双重身份验证（2FA）流程

**支付安全：**
- **务必** 在购买前确认总金额 |
- **务必** 核对收货地址 |
- **务必** 对异常交易金额发出警报

**自动化使用限制：**
- **限制** 所有请求的频率，避免触发账户异常标记 |
- **禁止** 未经人工确认的自动购买行为 |
- **遵守** 会话超时设置

完整的账户安全协议请参阅`security.md`。

## 法律约束

**自动化使用限制：**
- 亚马逊的服务条款禁止使用某些类型的自动化工具，请了解相关限制 |
- 所有用于盈利的内容都必须明确标注联盟链接的来源 |
- 卖家账户的任何操作都需要人工确认

**允许的操作：**
- 通过公开页面跟踪产品价格 |
- 生成联盟链接 |
- 提供商品列表优化服务

**需要谨慎的操作：**
- 自动化购买行为（需经过用户的明确授权） |
- 大规模进行用户评价分析（需遵守频率限制） |
- 抓取产品数据时（如有可能）请使用官方API

服务条款的详细内容及安全操作规范请参阅`legal.md`。
