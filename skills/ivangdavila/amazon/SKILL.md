## 代理的功能

| 用户请求 | 代理操作 |
|--------------|--------------|
| “在$Y价格范围内找到最好的X产品” | 搜索、比较产品评价/评分/价格历史，提供推荐 |
| “跟踪[产品]的价格” | 监控价格变化，价格下降时发送提醒，建议购买时机 |
| “重新订购我的[消耗品]” | 查找之前的订单，核对当前价格与上次购买时的价格，然后重新下单 |
| “这个优惠是真的吗？” | 检查价格历史，识别虚假优惠 |
| “比较[A]和[B]” | 详细对比产品规格、用户评价和性价比 |
| “帮我销售[产品]” | 优化商品列表、进行关键词研究、制定定价策略 |
| “生成联盟链接” | 创建带有标签的联盟链接，并跟踪链接的点击和转化情况 |

## 买家模式 — 核心功能

**产品研究：**
- 使用过滤器（价格、评分、Prime会员资格、卖家类型）进行搜索 |
- 汇总产品评价，总结优缺点，识别虚假评价 |
- 通过功能矩阵比较不同产品 |
- 检查卖家信誉（第三方风险评估）

**价格分析：**
- 跟踪历史价格变化（识别虚假折扣） |
- 在价格下降时向用户发送提醒 |
- 确定最佳购买时机（如Prime Day、黑色星期五等促销活动期间） |
- 在适用的情况下，比较不同亚马逊地区的商品价格 |

**购物流程：**
- 将商品加入购物车 |
- 使用优惠券或订阅服务以节省费用 |
- 为定期购买的商品重新下单，并验证价格 |
- 处理礼品购买及相关配送事宜 |
- 提交退货/退款申请 |

详细购物流程请参阅 `buying.md`。

## 卖家模式 — 核心功能

**商品列表管理：**
- 创建或优化商品列表 |
- 进行关键词研究以提高商品在搜索结果中的可见性 |
- 提供A+级别的商品描述建议 |
- 确保商品图片符合亚马逊的要求

**运营管理：**
- 监控库存并及时补货 |
- 规划FBA发货 |
- 监控竞争对手的价格策略 |
- 监控用户评价并准备回复评论

**数据分析：**
- 分析销售速度和转化率 |
- 评估广告效果（如PPC广告） |
- 计算利润率（包括费用、运费、退货等因素）

卖家相关操作流程请参阅 `selling.md`。

## 联盟模式 — 核心功能

**链接管理：**
- 生成带有正确标签的联盟链接 |
- 根据平台需求缩短或自定义链接长度 |
- 跟踪链接的点击量和转化情况

**内容优化：**
- 选择高佣金率的商品类别 |
- 关注季节性热门商品 |
- 提供产品对比分析内容

联盟计划相关策略请参阅 `affiliates.md`。

## 重要的安全规则

**账户凭证管理：**
- **严禁** 以明文形式存储亚马逊密码 |
- **严禁** 在不同设备之间共享会话cookie |
- **严禁** 绕过两步验证（2FA）流程

**支付安全：**
- **务必** 在购买前确认总金额 |
- **务必** 核对收货地址 |
- **务必** 对异常交易金额发出警报

**自动化使用限制：**
- **限制** 所有请求的频率，以避免账户被标记为可疑 |
- **严禁** 在未经用户明确授权的情况下进行自动购买 |
- **务必** 遵守会话超时设置

完整的账户安全协议请参阅 `security.md`。

## 法律约束

**自动化使用限制：**
- 亚马逊的服务条款禁止使用某些类型的自动化工具，请了解相关限制 |
- 所有涉及盈利的内容都必须明确标注联盟链接的来源 |
- 卖家账户的相关操作需要人工确认

**允许的操作：**
- 通过公开页面跟踪商品价格 |
- 生成联盟链接 |
- 提供商品列表优化服务

**需要谨慎的操作：**
- 自动购买功能需要用户的明确授权 |
- 大规模分析用户评价时需遵守相关频率限制 |
- 如需抓取产品数据，请使用官方提供的API

服务条款（ToS）的详细规定及安全操作指南请参阅 `legal.md`。
