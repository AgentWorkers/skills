## 关键词与出价
### 匹配类型
| 类型 | 行为 | 使用场景 |
|------|----------|----------|
| **精确匹配** | 查询内容与关键词完全一致 | 品牌相关词汇，高转化率 |
| **广泛匹配** | 同义词、相关词汇 | 发现新用户，扩大覆盖范围 |
| **搜索匹配** | 由Apple自动匹配 | 新应用，关键词研究 |

### 关键词对象

### 出价策略规则
1. **品牌关键词** → 设置较高出价（保护品牌）
2. **竞争对手关键词** → 小心测试，监控成本转化率（CPA）
3. **通用关键词** → 从低出价开始，根据效果调整
4. **发现新用户（搜索匹配）** → 低出价，挖掘潜在关键词
5. **否定关键词** | 必须设置，避免浪费预算

## 归因集成
### AdServices框架（iOS 14.3及以上版本）
支持无需用户跟踪的现代归因功能，可直接在iOS应用中集成。

### SKAdNetwork 4.0
注重用户隐私的归因方式。Apple汇总数据，不进行用户级跟踪。

### SKAdNetwork转化价值策略
| 值 | 含义 | 例子 |
|-------|---------|---------|
| 0 | 仅安装 | 应用被打开 |
| 1-10 | 用户互动 | 用户在应用中的停留时间 |
| 11-30 | 功能使用 | 关键功能被使用 |
| 31-50 | 营收信号 | 试用开始，内容被查看 |
| 51-63 | 收入产生 | 购买完成 |

### MMP集成（如AppsFlyer、Adjust等）
如果使用MMP（广告管理系统），请按照其SDK文档进行集成。关键步骤包括：
1. 在任何跟踪操作前初始化MMP SDK
2. 在MMP控制台中配置SKAdNetwork的转化值
3. 将Apple Search Ads账户关联到MMP以获取成本数据
4. 使用MMP的深度链接处理功能进行归因

## 报告与分析
### 广告活动报告请求

### 关键指标
| 指标 | 描述 | 优秀范围 |
|--------|-------------|------------|
| **TTR** | 点击转化率 | 5-10%以上 |
| **CVR** | 转化率（安装次数/点击次数） | 30-60% |
| **CPA** | 每次获取的成本 | < 用户生命周期价值（LTV）的1/3 |
| **CPT** | 每次点击的成本 | 0.50-3.00美元（因应用而异） |
| **ROAS** | 广告支出回报率 | > 100% |
| **展示量分享** | 符合条件的展示量占比 | 跟踪变化趋势 |

### 搜索词报告
该报告对优化至关重要，可显示触发广告的实际搜索词。

**每周推荐操作：**
1. 获取搜索词报告
2. 如果展示量高且转化率高，将这些搜索词添加为精确匹配关键词
3. 如果展示量高但转化率低，将这些搜索词添加为否定关键词
4. 对于无关的搜索词，立即将其设为否定关键词

### 多国家扩展
1. **初始阶段：** 美国、英国、加拿大、澳大利亚（使用英语）
2. **后续扩展：** 德国、法国、日本、韩国（进行本地化）
3. **测试阶段：** 巴西、墨西哥、印度（增加投放量，降低每次点击成本）

**本地化检查清单：**
- [ ] 应用商店页面已翻译
- [ ] 为每个国家定制产品页面
- [ ] 为每种语言研究关键词

- 为每个主要关键词主题创建定制产品页面
- 进行测试：比较默认页面与定制页面的效果
- 根据季节（如节假日、活动）调整页面内容

## 核心规则
1. **按意图划分广告活动**
将品牌关键词、类别关键词、竞争对手关键词和发现新用户的关键词分别放入不同的广告活动中。混合使用这些关键词会降低优化效果。
2. **先使用精确匹配关键词，再扩展到广泛匹配关键词**
先使用你确信有效的精确匹配关键词，对于发现新用户的广告活动，再使用广泛匹配关键词并设置较低出价。
3. **每周分析搜索词**
搜索词报告非常宝贵。每周分析数据，将表现良好的关键词添加为精确匹配关键词，表现不佳的关键词设为否定关键词。
4. **保护品牌**
竞争对手可能会竞标你的品牌名称。务必为品牌创建高出价的广告活动。
5. **正确处理归因**
对于iOS 14.3及以上版本的应用，必须使用AdServices进行归因处理。没有归因数据，优化将失去依据。
6. **每个国家创建独立的广告活动**
混合不同国家的广告活动会严重影响出价优化效果。请为每个国家/地区创建独立的广告活动。
7. **根据成本转化率（CPA）分配预算**
设定CPA目标，而不仅仅是预算总额。如果CPA低于预期目标的两倍，请暂停并进一步分析后再继续投放。

## 常见陷阱
- **在同一广告组中混合使用不同匹配类型**：难以判断哪些策略有效。请将精确匹配、广泛匹配和搜索匹配分开使用。
- **未设置否定关键词**：会导致预算浪费在无关搜索上。每周检查搜索词。
- **忽略搜索标签页/今日标签页**：这些位置的点击意向较低，但成本较低，适合发现新用户。
- **所有关键词使用相同的出价**：品牌关键词通常比通用关键词更有价值，应相应调整出价。
- **未进行归因集成**：会导致优化效果不佳。请启用AdServices或MMP。
- **在未优化应用商店的情况下投放广告**：转化率会很低。请先优化应用商店内容。
- **未经测试就竞标竞争对手的关键词**：往往无法盈利。先从小范围测试开始。
- **报告中未考虑时区**：可能导致数据不一致。请始终使用UTC或明确的时区。
- **未使用定制产品页面**：会错过潜在的转化机会。请为不同受众创建定制页面。
- **扩展速度过快**：扩展过快可能导致CPA突然上升。每次扩展预算时增加20-30%。

**数据安全与隐私**
- **发送到Apple Ads API的数据**：广告活动配置信息
- **发送到Apple的数据**：来自iOS应用的归因令牌
- **本地存储的数据**：
  - `~/apple-search-ads/credentials.md`中的凭证
  - 报告和分析结果
  - 策略相关笔记

**注意事项：**
- **不要以明文形式存储API密钥**：使用环境变量进行管理
- **不访问用户级数据**：归因数据是汇总后的信息
- **不要向未指定的API端点发送请求**
