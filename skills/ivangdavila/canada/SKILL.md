## 核心规则（Core Rules）

### 1. 具体化而非泛化
- 不要笼统地说“游览多伦多的各个社区”。例如：建议“在上午10:30之前先去圣劳伦斯市场，午餐后等人群散去后再步行前往酿酒区；如果未提前预订时间，日落时可以不参观CN塔”。

### 2. 以当地人的视角出发
- 提供的是当地人的实际建议，而非宣传资料上的内容：
  - 多伦多主要地标附近的美食广场价格昂贵且质量一般；更好的餐饮选择通常在附近街区。
  - 温哥华的卡皮拉诺悬索桥虽然外观漂亮但价格较高；林恩峡谷则免费且更加宁静。
  - 蒙特利尔市中心的连锁早午餐店人流量较大；社区内的面包店性价比更高且服务更快。
  - 班夫在夏季高峰期可能会显得拥挤；日出时间与班夫班车的时间安排非常重要。

### 3. 地区差异
| 地区 | 主要差异 |
|--------|----------------|
| 安大略省 | 大都市的节奏、丰富的博物馆和美食资源、便捷的公路旅行条件 |
| 魁北克省 | 以法语为主的文化、强烈的地方特色、独特的美食文化 |
| 不列颠哥伦比亚省 | 拥有海洋和山脉景观、注重户外活动、住宿费用较高 |
| 阿尔伯塔省落基山区 | 以自然景观为主、天气变化迅速、班车服务至关重要 |
| 大西洋沿岸省份 | 海岸小镇、以海鲜为主、旅行节奏较慢、天气多变 |
| 北部地区（育空/西北地区/努纳武特） | 距离遥远、有可能看到极光、费用较高 |

### 4. 时间安排至关重要
- 在主要公园游玩时：务必提前数月预订住宿和班车服务。
- 秋季树叶变色：许多地区的最佳观赏期为9月底至10月中旬，具体日期因省份而异。
- 冬季城市旅行：价格较为便宜、人流量较少，但白昼时间较短。
- 长周末期间：高速公路和机场可能会出现交通拥堵。
- “淡季”通常是结合城市观光和自然景观旅行的最佳时机。

### 5. 避免常见的旅行陷阱
- 避免以下情况：
  - 那些靠近主要景点、等待时间过长且主要依靠社交媒体宣传的餐厅。
- 7月和8月尝试在班夫停车（这些时段停车非常困难）。
- 在不了解铁路或巴士替代方案的情况下，选择昂贵的机场接送服务。
- 由于地图显示的距离较短而低估山区驾驶时间。
- 忽视魁北克地区服务中的双语使用情况。
- 在不先了解eSIM卡或预付费数据套餐的情况下，期望数据费用较低。
- 在信号覆盖不稳定的偏远地区依赖网约车服务。
- 忽略包含户外活动的旅行需要购买旅行保险。
