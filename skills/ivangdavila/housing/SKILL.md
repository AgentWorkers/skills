## 使用场景

当用户需要关于房地产决策的帮助时（例如购买首套房、租赁公寓、投资房产或作为房东管理出租物业），代理人将负责进行研究、分析、比较以及合规性检查。

## 核心规则

### 1. 首先核实当地情况
- 房地产市场具有很强的地域性——价格、法律和税收因城市/国家而异
- 在提供任何市场建议之前，务必了解用户所在地的具体情况
- 切勿假设以美国为中心的术语（如HOA、交易手续费）在其他地方同样适用
- 仔细研究当地法规：租客权利、租金管制、转让税费等

### 2. 计算总拥有成本
| 类别 | 需包含的费用 |
|----------|---------|
| 购买 | 首付、交易手续费、转让税费、公证费 |
| 月度费用 | 抵押贷款、保险费、房产税、业主协会/社区费用、水电费 |
| 隐性费用 | 维护费用（每年1-2%）、空置期费用、资本支出储备金 |

**切勿仅引用房源挂牌价格**——务必估算实际的月度/年度成本。

### 3. 不同角色的关注重点不同
| 角色 | 主要关注点 |
|------|-----------------|
| 首次购房者 | 价格合理性、抵押贷款审批、房屋检查中的风险因素 |
| 投资者 | 现金流、资本回报率、房产增值潜力、租客需求 |
| 租客 | 租约条款、租客权利、隐性费用、周边环境安全性 |
| 房东 | 租客筛选、租金定价、法律合规性、税收减免 |

在提供建议前，务必确认用户的角色。

### 4. 谨慎使用最新数据
- 房地产市场每月都在变化——数据可能存在滞后
- 提供分析框架，而非具体的价格预测
- 建议使用当地可靠的信息来源：房地产信息管理系统（MLS）、政府登记机构、当地房地产经纪人
- 当信息可能过时时，务必予以说明

### 5. 财务分析标准
对于投资性房产：
- **资本回报率（Cap Rate）** = 净营业收入 / 购买价格
- **现金回报率（Cash-on-Cash）** = 年度现金流 / 总投资额
- **1%规则**（快速判断标准）：月租金 ≥ 购买价格的1%
- 始终进行压力测试：如果空置期翻倍或租金上涨2%会如何？

### 6. 一切都要记录在案
房地产交易需要大量的文件：
- 事先审批函、定金、房屋检查报告
- 房产所有权查询、保险文件、交易披露文件
- 租约协议、入住前房屋检查报告、押金收据

根据用户的具体交易类型，制定相应的检查清单。

### 7. 需警惕的警示信号
| 信号 | 风险 |
|--------|------|
| 房价远低于市场水平 | 房屋存在隐蔽问题、法律纠纷或欺诈行为 |
| 卖方急于完成交易 | 未披露房屋问题 |
| 不允许进行房屋检查 | 房屋存在重大结构问题 |
| 房东拒绝签订正式租约 | 租客权益无法得到法律保护 |
| 业主协会财务信息缺失 | 即将面临额外费用或评估

**始终建议进行专业房屋检查和法律审查。**

## 房地产交易中的常见陷阱
- **错误比较不同市场的数据**——东京的租金走势与奥斯汀或柏林不同；切勿将一个市场的规则套用到另一个市场 |
- **忽视机会成本**——首付资金可以用于其他投资；需比较不同投资方案 |
- **情感锚定效应**——对“理想房屋”的过度执着可能导致过高定价；应参考类似房产的交易数据 |
- **低估交易成本**——在大多数市场中，买卖费用通常占房价的5-10% |
- **盲目假设房产会增值**——房产价值可能停滞不前或下跌；不要过分依赖增值预期 |
- **忽视租客筛选**——一个不良租客造成的损失可能超过数个月的空置费用 |
- **租约条款不完善**——违约罚款、续租条款、维护责任等细节常常被忽视
