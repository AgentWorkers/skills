## 设置

首次使用时，请阅读 `setup.md` 以获取集成指南。在创建 `~/genomics/` 工作区之前，请征得用户同意。

## 使用场景

用户已经处理了基因组数据（VCF 文件），并需要临床解读。该工具负责变异分类、药物基因组学建议以及注释查询。**不适用于原始数据处理**——请使用 `bioinformatics` 工具进行比对和变异检测。

## 核心规则

### 1. 根据 ACMG 指南对变异进行分类
所有变异都需要进行系统化的分类：

| 分类 | 判断标准 |
|----------|----------|
| 致病性 | PVS1、PS1-4、PM1-6、PP1-5 的评分结果 |
| 可能致病性 | 有强烈或中等程度的证据 |
| VUS（不确定意义） | 证据不足或存在矛盾 |
| 可能良性 | BS1-4、BP1-7 的评分结果 |
| 良性 | 有强烈的良性证据 |

**在没有证据的情况下，切勿进行分类。** 如有必要，应注明“数据不足”。

### 2. 首先检查变异在人群中的频率
在临床解读之前，需验证变异的频率：

| 数据源 | 适用场景 |
|--------|---------|
| gnomAD v4 | 全球人群频率 |
| gnomAD 非癌症数据 | 体细胞分析 |
| 特定人群数据 | 根据祖先背景进行筛选 |

**在任何人群中，如果 MAF（变异频率）>1%，则该变异对于罕见疾病而言很可能是良性的。**

### 3. 跨数据库交叉验证
| 数据库 | 提供的信息 |
|----------|-------------|
| ClinVar | 临床分类及提交者的证据 |
| OMIM | 基因-疾病关联 |
| HGMD | 文献中报道的突变 |
| UniProt | 蛋白质功能及结构域 |

**仅依赖单一数据库的解读是不够的**——需要综合多个来源的证据。

### 4. 提供可操作的药物治疗建议
对于药物-基因相互作用的情况，需提供以下信息：
- 基因型（例如：CYP2D6 *1/*4）
- 预测的表型（代谢缓慢/中等/正常/超快速）
- 受影响的药物列表
- 剂量指导（如有 CPIC/DPWG 数据）

### 5. 区分生殖细胞和体细胞变异
| 变异类型 | 关键区别 |
|---------|-----------------|
| 生殖细胞变异 | 与家族遗传相关，适用于携带者检测 |
| 体细胞变异 | 与肿瘤类型相关，影响治疗选择，不具有遗传性 |

**务必明确说明所解读的变异类型。**

### 6. 承认不确定性
- 新发现的变异往往缺乏足够的证据支持
- VUS（不确定意义）并不等同于良性——需要持续监测
- 变异分类可能会更新（ClinVar 每月更新一次）
- 计算预测结果仅具有参考价值，而非最终定论

## 药物基因组学参考

### 高优先级药物-基因组合（CPIC A 级）

| 基因 | 相关药物 | 临床建议 |
|------|-------|-----------------|
| CYP2D6 | 可待因、曲马多、他莫昔芬、SSRIs | 剂量调整或更换药物 |
| CYP2C19 | 曲匹氯雷定、PPIs、伏立康唑 | 剂量调整或更换药物 |
| CYP2C9 + VKORC1 | 华法林 | 剂量计算算法 |
| DPYD | 氟尿嘧啶、卡培他滨 | 减少剂量或避免使用 |
| TPMT + NUDT15 | 甲氨蝶呤、巯嘌呤 | 减少剂量 |
| HLA-B*57:01 | 阿巴卡韦 | 禁忌使用 |
| HLA-B*15:02 | 卡马西平 | （亚洲人群）禁忌使用 |
| SLCO1B1 | 辛伐他汀 | 剂量限制或更换他汀类药物 |
| G6PD | 拉斯布里卡塞、伯氨喹 | 禁忌使用 |
| CYP3A5 | 塔克洛利姆 | 剂量调整 |

### 表型解读

| 代谢状态 | 含义 | 常见处理方式 |
|--------------------|---------|----------------|
| 代谢缓慢（PM） | 酶活性极低 | 更换药物或降低剂量 |
| 代谢中等（IM） | 酶活性降低 | 考虑降低剂量 |
| 代谢正常（NM） | 酶活性正常 | 使用标准剂量 |
| 代谢快速/超快速（UM） | 酶活性过高 | 增加剂量或更换药物 |

## 常见解读误区
- **忽略人群特异性**：在非洲人群中常见的变异，在以欧洲人群为基础的数据库中可能显示为罕见变异
- **过分依赖 ClinVar 的提交结果**：检查提交者的数量和审核状态（建议至少有 2 个提交者，且无矛盾信息）
- **将计算预测结果与临床证据混为一谈**：CADD/REVEL 等工具仅提供参考信息，不能作为诊断依据
- **忽略化合物的杂合性**：两个 VUS 变异可能共同导致致病性
- **使用过时的数据库版本**：gnomAD v4 收集了 800 万个外显子组数据，而 v2 版本仅收集了 12.5 万个
- **忽视基因水平的限制**：pLI/LOEUF 分数可反映基因对功能丧失的耐受性

## 安全性与隐私

**数据存储方式**：
- 所有解读工作均在本地完成
- 该工具不会将变异数据发送到外部
- 不会自动调用任何数据库的 API

**该工具不执行以下操作**：
- 自动发起网络请求
- 将患者变异数据上传到外部
- 未经用户明确授权，不会连接数据库
- 不会将可识别的基因组信息存储在 `~/genomics/` 目录之外
