## 使用场景

Agent 工具适用于 B2B 销售场景，包括潜在客户开发、外联沟通、商机评估、销售流程分析、销售预测审核以及客户策略制定。适用于企业创始人、销售代表或销售经理。

## 核心规则

### 1. 在外联前进行充分调研
- 对每个潜在客户至少收集 3 项关键信息：最新动态、招聘情况、技术栈、融资状况
- 确定决策者和关键影响者（绘制组织结构图）
- 察觉潜在的购买信号（如领导层变动、业务扩张或竞争对手问题）
- 绝不要发送类似“我看到您在 [公司] 工作”的泛泛信息

### 2. 严格筛选潜在客户（MEDDIC 原则）
| 评估要素 | 问题 |
|---------|----------|
| **Metrics**（指标） | 他们需要什么可衡量的结果？ |
| **Economic Buyer**（经济决策者） | 谁是最终付款人？ |
| **Decision Criteria**（决策标准） | 他们会根据什么标准进行评估？ |
| **Decision Process**（决策流程） | 购买前需要经过哪些步骤？ |
| **Identify Pain**（识别痛点） | 他们面临的具体问题是什么？ |
| **Champion**（内部销售推动者） | 谁会在公司内部推动销售？ |

### 3. 对每个商机采取多线程处理方式
- 绝不要只依赖单一联系人（单一联系人意味着商机稳定性较低）
- 明确识别内部销售推动者（Champion）、经济决策者（Economic Buyer）、技术评估者（Technical Evaluator）和最终用户（End Users）
- 询问：“还有哪些人应该参与这个决策过程？”
- 如果内部销售推动者离职，商机很可能会失败

### 4. 确保每次沟通都有明确的下一步行动和截止日期
- 每次交流都必须明确下一步行动及具体日期
- “我会跟进”不算真正的下一步行动
- 对于复杂的商机，应使用“共同行动计划”（Mutual Action Plans）
- 如果没有明确的下一步行动，说明商机处于停滞状态

### 5. 不要完全依赖 CRM 系统的阶段划分
- 即使商机处于“谈判”阶段，也可能已经失败
- 通过实际活动、安排的下一步行动以及内部销售推动者的参与情况来验证进展
- 销售阶段持续时间过长（如超过 60 天）是一个危险信号
- 仅根据实际概率来评估销售流程的进展

### 6. 培养内部销售推动者，而不仅仅是建立联系人
- 内部销售推动者是指在您不在时也能推动销售的人
- 为他们提供 ROI 数据、产品竞争优势以及内部销售演示材料
- 如果他们无法独立阐述产品的价值，说明他们还不是真正的推动者
- 保护并培养他们的积极性——他们的成功就是您的成功

## 常见陷阱

| 陷阱 | 实际情况 |
|------|---------|
| 表面化的个性化问候 | “嗨 [姓名]，我看到您在 [公司] 工作” 并不是真正的个性化沟通 |
| 先谈论产品功能再讨论问题 | 应先了解他们的实际问题，再演示所需的功能 |
| 忽视采购团队 | 销售成功需要团队共识，而不仅仅是某个人的支持 |
| 仅得到积极回应但缺乏实际行动 | 销售代表可能只是表示“通话很愉快”，但并未承诺下一步行动 |
| 追求销售数量而非潜在客户的实际需求 | 10 个有潜力的客户比 100 个随机联系人的价值更高 |
- 过早发送报价 | 在未确认预算或授权之前就发送报价 |
| 销售完成后忽视内部销售推动者 | 这会导致客户流失 |
| 对 CEO 和普通用户发送相同的信息 | 不同的利益相关者需要不同的信息和支持内容 |

## 销售预测的警示信号

- 商机进展时间超过平均周期长度的三倍
- 超过 14 天没有任何进展
- 仅依赖单一联系人处理整个商机
- 只得到口头承诺但未签署任何文件
- 内部销售推动者表示“太忙”而无法参加下一次会议
- 采购或法律部门在后期未参与决策过程
- 承诺的截止日期多次推迟
