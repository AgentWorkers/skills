## 使用场景

用户希望在eBay上购买、出售或转售商品。代理会协助用户完成商品信息的发布、定价策略的制定、物流安排，并避免常见的陷阱。

## 核心规则

### 1. 先研究已售商品的价格
在定价之前，请先搜索eBay上已售出的同类商品：
- 使用产品的确切型号，避免使用通用术语
- 注意商品的状态、附带配件以及运输方式
- 定价时考虑价格范围，而不仅仅是单个价格点

### 2. 标题优化（最多80个字符）
```
[Brand] [Model] [Specs] [Condition] [Key Features] [Accessories]
```
- 将关键词放在标题的开头
- 避免使用填充词（如“WOW”、“LOOK”、“L@@K”等）
- 必须包含产品的型号，而不仅仅是名称
- 如果适用，还需注明颜色和尺寸

### 3. 费用计算
eBay的费用结构如下：
| 类别 | 最终费用比例 |
|----------|-----------------|
| 大多数类别 | 13.25% |
| 服装 | 12% |
| 吉他 | 6% |
| 重型设备 | 3% |

另外：每笔订单还需支付0.30美元的费用。使用PayPal或托管支付方式时，总费用约为2.9% + 0.30美元。
**快速估算方法：** 在售价的基础上扣除15-17%作为净利润。

### 4. 拍卖与“立即购买”（Buy It Now）的选择
| 适合拍卖的情况 | 适合“立即购买”的情况 |
|------------------|--------------|
| 稀有/收藏品 | 市场价格明确的普通商品 |
| 市场价值不确定的商品 | 需要保证售价的商品 |
| 时间敏感的商品（如节假日） | 高价值商品（>500美元） |
| 需要建立销售记录的商品 | 库存稳定的商品 |

### 5. 物流策略
- 提供“免费配送”服务可以提升商品的吸引力，但需在商品信息中明确标注价格
- 对于重型或体积较大的商品，应计算实际运费
- 始终预留至少1天的处理时间
- 在发布商品信息前先测量商品的准确尺寸

### 6. 诈骗检测（卖家侧）
买家可能出现的可疑行为：
- 要求将商品寄送到与PayPal不同的地址
- 提议“以X美元的价格提前结束商品拍卖”
- 过度付款并要求对方汇款差额
- 在付款完成前要求提供物流追踪信息
- 使用新账户并通过货运代理发货

### 7. 诈骗检测（买家侧）
卖家可能出现的可疑行为：
- 价格低得离谱（低于市场价格的50%以上）
- 只提供库存照片
- 发货地址与商品描述不符
- 不提供退货政策或描述模糊
- 要求买家通过非eBay渠道付款

### 8. 退货政策的影响
| 退货政策 | 影响 |
|--------|--------|
| 30天免费退货 | 提高商品在搜索结果中的排名，增加销量 |
| 不允许退货 | 降低商品曝光率，买家会更加谨慎 |
| 重新上架费用 | 合法规定，但可能降低转化率 |

建议：对于大多数商品，应设置30天的免费退货政策（但实际上很少有人使用）。

## 常见陷阱
- **低估运费** → 会导致重型商品的成本过高；在发布商品信息前务必称重和测量
- **忽略商品的具体信息** → 没有详细信息的商品在搜索结果中容易被忽略
- **未经验证就接受PayPal付款** → 可能导致高价值商品被拒付；超过750美元的交易需要买家签名确认
- **根据询问价格定价而非实际售价** → 未售出的商品价格可能无法反映市场行情；建议使用“已售商品”作为筛选条件
- **在付款完成前发货** → 可能导致买家投诉“未收到商品”；请等待付款到账后再发货
