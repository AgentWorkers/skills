## 核心规则

### 1. 首先明确用户的需求
- 了解用户是希望游览旧城区景点、体验赌场度假区、家庭搬迁、跨境通勤，还是将澳门作为大湾区的商业基地。
- 澳门虽然面积较小，但半岛、大澳、科塔伊和高隆各区域的实际需求有很大差异。

### 2. 将边境政策作为核心规划要素
- 大多数麻烦都源于入境规定、渡轮或桥梁的使用时间，以及一日游或通勤的安排。
- 一定要询问用户从何处前来：香港、珠海、中国大陆还是直飞。
- 在做出任何决定前，请务必再次查阅 `visas.md` 中的最新入境规定。

### 3. 澳门不仅仅是赌场
- 虽然博彩和旅游业是澳门的经济支柱，但还应提供关于历史遗迹、葡萄牙文化、美食、家庭生活便利性以及澳门在大湾区中的位置的详细信息。
- 除非用户特别要求，否则不要只提供以赌场为主的行程建议。

### 4. 小面积并不意味着无需考虑交通问题
- 虽然半岛部分地区适合步行，但高温、湿度、人群密集以及桥梁的通行限制会影响实际出行体验。
- 科塔伊的度假区在地图上看起来很近，但实际上仍需要规划好步行路线或使用班车。
- 在估算旅行时间前，请务必阅读 `transport.md` 及相关区域文件。

### 5. 2026年3月的最新数据

| 项目 | 范围 |
|------|-------|
| 半岛地区1居室租金 | 每月8,000至14,000澳门元 |
| 大澳/科塔伊地区1居室租金 | 每月12,000至20,000澳门元 |
| 月平均工资 | 约17,000澳门元 |

| 出租车起步价 | 21澳门元 |
| 一顿简单餐食 | 60至120澳门元 |

### 6. 现金、货币与支付方式
- 澳门官方货币为澳门元（MOP）。
- 游客区普遍接受港元（1港元=1澳门元），虽然方便但汇率不利。
- 小商店、面包店、出租车及部分老城区场所更倾向于接受现金或本地电子钱包。

### 7. 根据用户需求选择合适的区域
| 用户类型 | 最适合的区域 |
|---------|------------|
| 首次游客 | 塞纳多（Senado）/圣保罗（St. Paul’s）/大澳（Barra）或大澳（Taipa） |
| 以度假区为主的周末旅行 | 科塔伊（Cotai） |
| 历史文化与美食爱好者 | 历史半岛地区 + 大澳村（Taipa Village） |
| 家庭搬迁者 | 大澳（Taipa）或半岛南部地区 |
| 预算有限的游客 | 半岛内部及简单设施的酒店 |
| 喜欢安静环境的游客 | 高隆（Coloane） |

## 澳门特有的注意事项

- 不要误以为澳门和香港的入境规则或货币制度相同。
- 仅在大澳地区停留却认为澳门没有本地特色。
- 不要把赌场度假区当作普通城区来对待。
- 不要假设所有商家都接受信用卡支付，也不是所有价格都以港元标价。
- 切勿低估周末或节假日期间港口和桥梁的拥堵情况。
- 7月或8月午后避免安排长时间的户外活动。
- 请注意许多当地餐馆在午餐和晚餐时间会暂停营业。
- 不要以为葡萄牙文化意味着慢节奏的生活；澳门的实际生活节奏更快、更紧凑。

## 法律须知

- 仅允许在持牌场所进行赌博活动。
- 携带毒品是严重违法行为，请严格遵守规定。
- 街道上一般可以拍照，但赌场楼层及某些博物馆或寺庙可能有限制。
- 在港口、赌场和交通区域，公共秩序、吸烟及行为规范都有严格规定。
- 在提供任何关于移民的信息前，请务必查阅 `visas.md` 中的最新官方通知。

## 大湾区背景

在正确的大湾区框架下，澳门能发挥最大优势：
- 作为旅游和酒店业的枢纽，吸引大量来自内地的游客；
- 与香港有紧密的航空和金融联系；
- 与珠海有便捷的陆地连接，有助于降低成本和便利往来内地；
- 作为纯粹的科技基地，其优势不如深圳或香港明显。
