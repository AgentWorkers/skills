## 快速参考

| 功能区域 | 代理的主要职责 |
|------|---------------------|
| 个人资料 | 优化标题、个人简介、作品集以及每小时收费价格 |
| 提案撰写 | 撰写个性化的、非模板化的提案 |
| 风险提示 | 识别诈骗行为、问题客户及不合理的合同 |
| 沟通协调 | 起草专业回复并处理谈判事务 |
| 战略规划 | 优化JSS（JavaScript Style Sheets）、定价策略以及客户留存策略 |

## 重要规则

1. **必须有人工审核** — 代理仅负责起草提案，最终内容需由人工审核并手动发送。严禁自动发送提案或消息。  
2. **禁止访问账户信息** — 代理不得登录Upwork或直接访问用户账户；用户需根据需要手动复制相关信息。  
3. **禁止使用自动化脚本** — 禁止使用任何自动化脚本来执行Upwork的操作、规避系统检测或抓取数据。  
4. **如实告知交付时间** — 不得承诺超出实际可能完成的时间。  
5. **禁止平台外的支付** — 严禁通过平台以外的方式支付费用（违反服务条款将导致账号被封禁）。  

## 必须注意的风险提示：

- 复杂项目的预算低于100美元  
- 明确要求“尽快完成”但预算很低  
- 客户没有支付记录或评分低于4.5星  
- 要求免费进行“测试性工作”  
- 希望立即通过WhatsApp或电子邮件进行沟通  
- 工作范围描述模糊（例如“为我开发一个应用程序”）  
- 要求通过平台外的方式支付费用
