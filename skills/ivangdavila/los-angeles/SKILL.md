## 核心规则

### 1. 首先明确用户需求
- **用户角色**：游客、居民、科技工作者、学生或从事创意/娱乐行业的人
- **时间安排**：短期访问、计划搬家、已经居住在洛杉矶
- 根据用户需求加载相应的辅助文件以获取详细信息

### 2. 了解洛杉矶的交通现实
洛杉矶是一个以汽车为主的城市。几乎所有生活活动都离不开汽车：
- 地铁系统正在改善，但覆盖范围有限
- 高峰时段的交通状况非常糟糕（尤其是405、101、10号高速公路）
- 每月的汽车相关费用约为400-600美元
详情请参阅 `transport.md`。

### 3. 天气与微气候
- **常见误解** | **实际情况** |
| “洛杉矶总是阳光明媚” | 沿海地区在5月至7月会有雾 |
| “所有地方的天气都一样” | 山谷地区的温度比沿海地区高10-15°F |
| “洛杉矶从不下雨” | 雨季为12月至3月 |
- **最佳天气**：9月至11月（最温暖、空气最清新）

### 4. 最新数据
| 项目 | 数据范围 |
|------|-------|
| 单间公寓租金 | 2,000-3,000美元（地区差异较大） |
| 高级软件工程师的薪资 | 总收入为18万至35万美元 |
| 汽车保险费用 | 每月150-300美元（较高） |
| 墨西哥卷饼价格 | 2-4美元一个 |

### 5. 游客需要注意的陷阱
- **可跳过的景点**：好莱坞星光大道（环境较差，体验不佳）、威尼斯海滩步道（治安不佳）
- **必去的景点**：格里菲斯天文台（免费，视野绝佳）、盖蒂中心（免费，世界级博物馆）
- **提前预订**：环球影城和热门餐厅

### 6. 不同社区的居住体验
洛杉矶面积庞大，居住地点会直接影响生活方式：
- **适合年轻科技工作者的社区**：圣莫尼卡、卡尔弗城、普拉亚维斯塔
- **适合娱乐产业从业者的社区**：西好莱坞、洛斯费利兹、银湖
- **适合家庭的社区**：帕萨迪纳、南湾、斯图迪오城
- **预算有限的游客**：山谷地区（谢尔曼奥克斯、伯班克）、韩国城
- **适合享受海滩生活的社区**：圣莫尼卡、曼哈顿海滩、威尼斯

### 7. 安全问题需特别注意
- 整体来说洛杉矶比较安全，但安全状况因社区而异
- 需特别关注的区域：洛杉矶市中心的部分地区、夜间的好莱坞地区
- 汽车盗窃事件：虽然比旧金山少，但在旅游区仍时有发生
详情请参阅 `safety.md`。

## 洛杉矶特有的注意事项

- **“我不需要汽车”** — 其实你需要汽车。地铁系统虽然在改善，但还不够完善。
- **好莱坞的“魅力”** — 好莱坞大道的环境并不如想象中那么好。
- **“交通并不糟糕”** — 实际上交通非常拥堵，出行时间需要比预计多1.5到2倍。
- **“山谷地区不适合居住”** — 实际上山谷地区性价比高，空间宽敞，适合家庭居住。
- **“海滩地区是最好的居住地”** — 沿海地区在5月至7月会有雾，山谷地区阳光更充足。
- **“所有地方都很近”** — 洛杉矶横跨50多英里，生活安排需根据具体区域来规划。
- **环球影城 vs 迪士尼乐园** — 迪士尼乐园更有趣，但位于阿纳海姆（需要1小时以上车程）。
