# 色情小说写作指南

## 核心原则

1. **文学质量至上** — 这是创意写作，而非低俗内容。应注重文笔、氛围和情感深度。
2. **逐步营造紧张感** — 故事需要情节发展和节奏控制。读者的期待往往比情节本身更具吸引力。
3. **调动所有感官** — 视觉、听觉、触觉、嗅觉、味觉。让读者完全沉浸在故事中。
4. **掌握暗示的艺术** — 未说出口的部分往往比直白的描述更具感染力。

## 写作要素

### 氛围营造
- **环境描写**：光线、温度、气味、环境声音
- **心理描写**：紧张、期待、犹豫、欲望
- **节奏控制**：缓慢升温 → 情感紧张 → 高潮 → 余韵

### 角色塑造
- 角色应具有鲜明的个性，而不仅仅是外表
- 对话应自然流畅，符合角色的性格特点
- 情感表达必须真实可信 — 展现角色之间的化学反应

### 场景类型（参考）
- 🌙 **浪漫场景**：初次相遇、分离后的重逢、烛光晚餐
- 🔥 **激情场景**：办公室里的紧张氛围、电梯中的邂逅、暴风雨之夜
- 💕 **温柔场景**：慵懒的早晨、一起洗澡、午后的亲密时刻
- ⛓️ **禁忌场景**：秘密恋情、权力关系、禁忌的吸引力

## 文体选择

根据用户的偏好选择：
- **含蓄/诗意**：通过暗示来激发读者的想象
- **感官描写**：丰富的细节，营造沉浸式的阅读体验
- **文学艺术风格**：精炼的文笔、大量的隐喻，充满浪漫色彩

## 写作流程

1. **确定写作背景**：角色设定、关系模式、场景设定、文体偏好、语言（英语/中文）
2. **控制写作节奏**：逐步推进情节发展 → 达到高潮 → 解决问题
3. **建议篇幅**：短篇作品约1,000至3,000字

## 成品保存

```

```

如果目录不存在，请先创建该目录。

文件格式：
```markdown

**Characters**: {names}
**Setting**: {brief description}
**Style**: {subtle/sensual/literary}

{story content}
```

## 注意事项

- 所有角色都必须是自愿参与的成年人
- 人物之间的关系必须基于双方同意
- 禁止包含任何非法、有害或违背他人意愿的内容
- 尊重用户的界限和偏好
