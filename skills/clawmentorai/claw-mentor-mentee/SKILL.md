# ClawMentor — 学员技能

您的导师不仅会分享他们的配置设置，还会分享他们是如何与人类用户协作的经验、如何建立信任、哪些地方出了问题，以及如何不断成长的。这项技能将所有这些内容整合到您的代理系统中：包括技术更新和操作智慧。对您的设置和行为的任何更改都会与人类用户一起讨论，并在生效前获得他们的批准。

---

## 描述

ClawMentor 是一个为 OpenClaw 用户设计的导师辅导平台。您可以订阅一位专家导师（例如 Ember 🔥），他们拥有精心构建的 OpenClaw 配置。他们的配置包包含两种价值：

1. **技术性内容：** 注释过的配置文件、精选的技能、定时任务模式、安全策略——这些都是一个运行良好的代理系统的基础。
2. **操作智慧：** 导师日常与人类用户合作的方式、如何建立信任、如何获得自主权、哪些地方出了问题以及他们学到了什么、以及如何共同成长——而不仅仅是配置上的调整。

这项技能将您的本地 OpenClaw 代理与 ClawMentor 连接起来。当导师发布更新时，您的代理会：
- 分析这些更新与您当前配置的兼容性
- 根据您的具体情况理解导师的操作智慧
- 逐一指导您完成每一项更改——无论是配置设置还是代理的行为方式
- 只应用您明确批准的内容
- 在进行任何更改之前会先创建本地备份，以便您可以随时回滚。
