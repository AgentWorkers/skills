# SF Express

## 概述

使用此技能可帮助用户完成常见的 SF Express 相关任务，例如追踪货物运输情况、了解服务等级、估算费用以及准备发送包裹。

## 本地数据持久化

当使用本地 CLI/运行时代码时，此技能可能会在以下路径创建并持久化本地数据：

- `~/.openclaw/data/sf-express/sfexpress.db`  
  - 存储查询历史记录  
  - 存储货物订阅信息  
  - 如果使用了相关命令，还会存储保存的地址信息  

- `~/.openclaw/data/sf-express/secure/`  
  - 存储隐私/存储辅助工具使用的加密本地文件  

- `~/.openclaw/data/sf-express/secure/.key`  
  - 存储一个加密密钥文件（权限设置为 `600`）  

- `~/.openclaw/data/sf-express/privacy_export.json`  
  - 在用户执行隐私数据导出操作时可能会生成此文件  

仅在用户的工作流程确实需要时才使用数据持久化功能。如果用户询问隐私相关问题，请明确告知这些数据文件的存储路径。

## 隐私控制

本地 CLI 支持以下隐私相关操作：  
- `privacy info` — 显示本地存储路径及存储的文件信息  
- `privacy clear` — 清除本地 SQLite 历史记录、订阅数据及加密文件  
- `privacy export` — 将本地存储的元数据导出到 `privacy_export.json` 文件中  

## 工作流程：  
1. 确定用户的目标：  
   - 追踪现有货物的运输情况  
   - 估算费用或交货时间  
   - 比较不同的服务类型  
   - 查找附近的取货点或储物柜  
   - 准备包裹发送的相关信息  
   - 查看或清除本地历史记录、订阅信息及隐私数据  

2. 仅询问用户真正需要的必要信息（如追踪号码、运输路线、包裹尺寸或紧急程度）。  

3. 首先提供最实用的答案。  
4. 如果无法确定确切的费用或交货时间，应给出谨慎的估算，并说明相关假设。  
5. 如果任务涉及使用会持久化数据的本地运行时功能，请告知用户相关数据文件可能存储在 `~/.openclaw/data/sf-express/` 目录下。  
6. 除非可以使用实时工具并得到确认，否则不要声称已经完成了实际的运输操作。
