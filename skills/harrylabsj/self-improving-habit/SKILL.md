# 自我提升习惯系统

这是一个基于行为科学的习惯管理系统，通过分阶段推进、环境设计以及意志力管理来帮助人们养成持久的习惯。

## 快速参考

| 情况 | 应对措施 |
|-----------|--------|
| 开始新习惯 | 定义习惯参数，设计执行环境，设定一个简单的初始目标 |
| 习惯养成失败（错过几天） | 分析失败原因，减少执行障碍，调整触发条件 |
| 习惯执行感到困难 | 使习惯自动化，与其他习惯结合，增加奖励 |
| 多次尝试均失败 | 检视个人价值观与习惯的匹配度，从小目标开始，调整方法 |
| 长期维持习惯 | 定期回顾习惯，逐步增加挑战难度，防止厌倦 |

## 习惯形成的四个阶段

### 第一阶段：启动阶段（第1-7天） - “开始行动”
- **目标**：至少完成一次习惯动作 |
- **重点**：消除执行障碍，建立触发条件 |
- **成功标准**：至少完成一次，不必追求完美 |
- **心理学原理**：降低行动所需的启动能量（即减少阻力）

### 第二阶段：持续阶段（第8-21天） - “养成规律”
- **目标**：形成固定的行为模式 |
- **重点**：保持习惯的连续性，维持连续的完成记录 |
- **成功标准**：完成率超过70% |
- **心理学原理**：重复练习会强化“提示-习惯-奖励”之间的循环

### 第三阶段：整合阶段（第22-90天） - “自动化”
- **目标**：让习惯成为自然的行为 |
- **重点**：将习惯融入日常生活环境，调整个人身份认同 |
- **成功标准**：执行习惯时几乎不需要刻意努力 |
- **心理学原理**：习惯与个人身份相一致

### 第四阶段：优化阶段（90天后） - “提升质量”
- **目标**：提高习惯的质量和效率 |
- **重点**：追求习惯的熟练度，尝试多种执行方式 |
- **成功标准**：习惯对生活其他方面产生积极影响 |
- **心理学原理**：增强自我效能感，提升个人能力

## 习惯参数设置

### 核心参数
- **习惯名称**：晨间冥想、每日写作、晚间反思等 |
- **类别**：健康、学习、生产力、人际关系、正念等 |
- **难度等级**：简单（1）、较难（2）、中等（3）、困难（4）、非常困难（5） |
- **执行频率**：每天、工作日、每周3次、每周一次、根据触发条件执行 |
- **预计持续时间**：5分钟、15分钟、30分钟、60分钟、120分钟

### 成功策略设计
- **触发条件**：时间、地点、前一个动作、情绪状态等 |
- **执行步骤**：明确的具体步骤 |
- **奖励方式**：内在奖励、外在奖励、社交认可、通过记录获得奖励 |
- **执行障碍评分**：1-10分（分数越低，障碍越小）

## 日志记录格式

- **习惯定义**：记录在 `.learnings/habits/HABIT_NAME.md` 文件中 |
- **每日习惯记录**：记录在 `.learnings/habits/HABIT_NAME_LOGS.md` 文件中 |
- **习惯回顾**：记录在 `.learnings/habits/HABIT_NAME_REVIEWS.md` 文件中 |

## 行为科学原理

### Fogg 行为模型
- **动机**：会波动，不可完全依赖 |
- **能力**：简化执行流程（减少障碍） |
- **提示设计**：使习惯触发条件显而易见 |

### Duhigg 的习惯循环理论
- **提示设计**：确保触发条件明显且不可避免 |
- **执行流程设计**：使习惯执行过程简单且令人愉悦 |
- **奖励设计**：奖励要即时且具有强化效果 |
- **建立渴望**：将习惯触发条件与奖励预期联系起来

### 意志力管理
- **决策疲劳**：选择越少，可用于习惯的意志力就越多 |
- **意志力有限**：优先用于重要的习惯 |
- **自动化目标**：通过重复练习，使习惯执行几乎不需要意志力

## 高级技巧

- **习惯叠加**：例如：喝完咖啡后，立即冥想1分钟。
- **环境设计**：让习惯触发条件更加显眼且易于触发 |
- **基于个人身份的习惯设定**：例如：“我是一个每天冥想的人” |
- **诱惑管理**：例如：锻炼时只听喜欢的播客。

## 习惯养成失败的分析

### 常见失败原因及解决方法

- **原因1：目标过高**：一开始就设定过于复杂的目标，难以坚持 |
- **解决方法**：从小目标开始，逐步提高难度。
- **原因2：触发条件不明确**：习惯触发条件不明显或不可靠 |
- **解决方法**：将新习惯与现有习惯结合，使用视觉提醒。
- **原因3：执行障碍过多**：习惯步骤太多或准备过程繁琐 |
- **解决方法**：简化习惯流程，提前做好准备。
- **原因4：奖励延迟**：长期收益不明显 |
- **解决方法**：设置即时奖励（如记录进度、奖励小零食）。
- **原因5：习惯与自我形象冲突**：习惯与个人价值观不符 |
- **解决方法**：重新定义个人身份认同，从小习惯开始。

## 成功指标

- **阶段特定指标**：
  - **启动阶段**：完成次数、触发条件的可靠性、执行障碍的减少。
  - **持续阶段**：习惯的连续性、执行时间的延长、意志力的降低。
  - **整合阶段**：习惯的自动化程度、习惯与个人身份的融合度。
  - **优化阶段**：习惯质量的提升、习惯之间的协同效应。

### 入门指南

- **步骤1：选择习惯**：选择1-2个关键习惯。
- **步骤2：第一周的准备工作**：准备执行环境，设定简单的初始版本。
- **步骤3：逐步改进**：完成启动阶段后，评估并调整习惯的执行方式。
