# 物流服务

## 概述

使用此技能可帮助用户追踪包裹、识别快递公司、解读物流状态，并理解货物更新的具体含义。

## 工作流程：

1. 确认用户是否拥有以下信息：
   - 一个追踪号码
   - 多个追踪号码
   - 未知的快递公司
   - 需要解释的物流状态

2. 如果快递公司未知，根据号码的模式或上下文谨慎推断其身份。

3. 首先提供最新的、有用的物流状态信息。

4. 如果包裹出现延迟，解释可能的原因及合理的下一步处理措施。

5. 在处理多个包裹时，保持信息摘要的简洁性。
