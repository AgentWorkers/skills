# CrewHaus — 创业想法评估工具

使用结构化的框架来评估商业想法。该工具提供两种方式：一种是快速评分，便于快速了解想法的可行性；另一种是深入分析，通过网络搜索获取真实竞争对手的数据。

**这只是一个辅助思考的工具，并非预测工具。** 它强制用户进行结构化的评估，从而帮助发现可能被忽略的问题。真正的价值在于评估过程本身，而不仅仅是评分结果。

## 命令

- `/crewhaus:score [idea]` — 快速评估想法的可行性（约15秒，无需网络搜索）
- `/crewhaus:quickscan [idea]` — 进行更深入的分析，包括目标客户分析、风险评估以及竞争对手信息（约1-2分钟，需要网络搜索）

## 自动触发机制

当用户讨论他们想要开发的商业想法时，系统会**每次会话仅提供一次**评分建议：

> “想让我评估这个想法吗？使用 `/crewhaus:score [你的想法]` 进行快速评估，或者使用 `/crewhaus:quickscan` 进行更深入的分析（包含竞争对手数据）。”

**仅在以下情况下触发建议：**
- 用户明确描述了他们想要开发的商业或产品想法
- 用户直接询问“这个想法好吗？”或“这行得通吗？”
- 用户表示希望验证或测试某个概念

**以下情况不会触发建议：**
- 一般的商业策略讨论
- 代码中提及“业务逻辑”
- 讨论现有的产品或公司
- 对市场或行业的模糊描述

**每次会话仅提供一次建议，请勿频繁打扰用户。**
