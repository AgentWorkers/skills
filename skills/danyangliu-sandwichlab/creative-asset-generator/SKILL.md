# 广告素材生成器

## 目的
核心任务：
- 生成广告素材简介、调整素材格式、制定多样化策略

该技能专为广告工作流程设计，旨在提供可执行的计划，而非泛泛而谈的建议。

## 适用场景
当用户提出以下需求时，请使用该技能：
- 需要与业务目标相关的广告执行指导
- 需要涉及收入、投资回报率（ROAS）、每点击成本（CPA）或预算效率的增长决策
- 需要针对以下平台的广告投放策略：Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads、Shopify Ads
- 需要执行以下具体操作：生成广告素材简介、调整素材格式、制定多样化策略

**高频关键词**：
- 广告、广告活动、增长、收入、利润
- 投资回报率（ROAS）、每点击成本（CPA）、投资回报率（ROI）、预算、出价、流量、转化率、销售漏斗
- Meta、Google Ads、TikTok Ads、YouTube Ads、Amazon Ads、Shopify Ads

## 输入参数
**必填项**：
- 产品/服务（product_or_offer）
- 目标受众（target_audience）
- 广告投放平台（placement_scope）

**可选项**：
- 现有素材（existing_assets）
- 品牌风格（brand_tone）
- 禁止使用的表述（prohibited_claims）
- 创意制作限制（creative_constraints）

## 输出内容
1. 创意目标（Creative Objective）
2. 创意方向与吸引点（Angle and Hook Set）
3. 素材规格（Asset Specification）
4. 测试方案（Test Variant Plan）
5. 创意质量检查清单（Creative QA Notes）

## 工作流程
1. 将创意目标与销售漏斗阶段及关键绩效指标（KPI）对齐。
2. 生成多种创意方向及吸引点。
3. 将每种创意方向适配到相应的广告平台格式要求。
4. 定义创意变体矩阵及测试顺序。
5. 设置质量与合规性检查点。

## 决策规则
- 如果目标受众较为陌生，优先选择能够引发问题的创意方向或能够证明有效性的吸引点。
- 在再营销阶段，优先考虑信息的清晰度及紧迫性。
- 如果平台对素材格式有严格限制，将信息层次简化为单一的呼叫行动（CTA）。

## 平台注意事项
**主要适用平台**：
- Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads、Shopify Ads

**平台使用建议**：
- 建议针对不同平台制定个性化的投放策略，避免将所有平台的策略合并为一个通用方案。
- 对于Meta和TikTok Ads，优先考虑创意的测试频率。
- 对于Google Ads和Amazon Ads，重点关注需求捕捉及用户查询/列表意图。
- 对于DSP/程序化广告平台，重点关注受众控制及投放频率管理。

## 限制与注意事项
- 严禁伪造数据或预测结果。
- 将观察到的事实与假设区分开来。
- 对每个建议的操作使用可衡量的语言进行描述。
- 在存在支出风险的情况下，必须包含至少一种回退或止损措施。

## 故障处理与升级流程
- 如果关键输入信息缺失，仅要求提供最低限度的必要字段。
- 如果平台限制相互冲突，需展示权衡方案及安全默认方案。
- 如果信心度较低，需明确标注并提供验证清单。
- 如果出现高风险问题（如政策变更、计费问题、跟踪系统故障），需通过结构化的方式上报问题。

## 代码示例
### 创意简介示例
```
creative_id: CR-001
angle: 从问题到解决方案（pain_to_outcome）
hook: “在第一周内停止浪费广告预算”
formats: [9:16视频、1:1图片]
```

### 创意变体矩阵
```
V1: 吸引点调整（hook_change）
V2: 呼叫行动调整（CTA_change）
V3: 视觉元素调整（visual_proof_change）
```
