# 广告创意评分

## 目的
核心任务：
- 制定评分标准、提供修改建议，并做出是否发布的决策

该技能专为广告工作流程设计，应生成可执行的计划，而非泛泛而谈的建议。

## 适用场景
当用户提出以下需求时，可使用该技能：
- 需要与业务成果相关的广告执行指导
- 需要涉及收入、投资回报率（ROAS）、每点击成本（CPA）或预算效率的增长决策
- 需要针对以下平台的广告策略：Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads
- 需要具体的评分标准、修改建议或发布决策

**高相关关键词**：
- 广告、广告活动、增长、收入、利润
- 投资回报率（ROAS）、每点击成本（CPA）、投资回报率（ROI）、预算、出价、流量、转化率、营销漏斗
- Meta、Google Ads、TikTok Ads、YouTube Ads、Amazon Ads、Shopify Ads、DSP（数字广告平台）

## 输入参数
**必填项**：
- 产品或服务（product_or_offer）
- 目标受众（target_audience）
- 广告投放范围（placement_scope）

**可选项**：
- 现有创意素材（existing_assets）
- 品牌风格（brand_tone）
- 禁止使用的宣传内容（prohibited_claims）
- 创意制作限制（creative_constraints）

## 输出内容
1. 创意目标（Creative Objective）
2. 创意方向与吸引点（Angle and Hook Set）
3. 创意素材规格（Asset Specification）
4. 测试方案（Test Variant Plan）
5. 创意质量检查表（Creative QA Notes）

## 工作流程
1. 将创意目标与营销漏斗阶段及关键绩效指标（KPI）对齐。
2. 生成多种创意方向及吸引点。
3. 将每种创意方向与相应的广告投放格式要求相匹配。
4. 定义创意变体矩阵及测试顺序。
5. 设置质量与合规性检查点。

## 决策规则
- 如果目标受众较为冷淡（对广告不感兴趣），优先选择能够引起受众注意的创意方向。
- 在再营销阶段，优先考虑信息的清晰度及紧迫感。
- 如果广告格式有严格要求，简化信息结构，确保只有一个明确的行动号召（CTA）。

## 平台注意事项
**主要适用平台**：
- Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads

**平台使用建议**：
- 建议针对不同平台制定具体的策略；不要将所有平台的策略合并为一个通用方案。
- 对于Meta和TikTok Ads，优先考虑创意的测试频率。
- 对于Google Ads和Amazon Ads，重点关注需求捕捉及用户查询/列表意图。
- 对于DSP/程序化广告平台，优先考虑受众控制及投放频率的管理。

## 限制与注意事项
- 绝不允许伪造数据或预测结果。
- 区分观察到的事实与假设。
- 对每个建议的行动使用可量化的表述。
- 当存在支出风险时，必须包含至少一种回退或止损策略。

## 故障处理与升级流程
- 如果关键输入信息缺失，仅要求提供最低限度的必要字段。
- 如果平台限制存在冲突，需展示权衡方案及安全默认选项。
- 如果信心度较低，需明确标注并提供验证清单。
- 如果出现高风险问题（如政策变更、计费问题、数据追踪异常），需通过结构化的流程进行上报。

## 示例代码
### 创意简报示例
```json
{
  "creative_id": "CR-001",
  "angle": "从问题到解决方案",
  "hook": "在第一周内停止浪费广告预算",
  "formats": ["9:16视频", "1:1图片"]
}
```

### 创意变体矩阵示例
```json
{
  "V1": {
    "hook": "改变广告内容",
    "cta": "立即尝试新方案"
  },
  "V2": {
    "hook": "优化广告效果",
    "cta": "提升转化率"
  },
  "V3": {
    "hook": "提供视觉证据",
    "cta": "查看效果对比"
  }
}
```

## 实际应用示例
### 示例1：生成新的创意方向
**输入**：
- 现有的创意素材使用频率过高，受众疲劳
**输出**：
- 新的创意方向建议
- 创意方向库
- 测试优先级

### 示例2：根据平台调整创意素材
**输入**：
- 一个核心创意概念，需适配多个平台
**输出**：
- 各平台的格式要求
- 文本长度的调整建议
- 创意质量检查清单

### 示例3：发布前的创意评分
**输入**：
- 有12个创意素材待发布，但审查时间有限
**输出**：
- 创意质量评分
- 应拒绝或修改的创意列表
- 可发布的创意候选名单
