# 广告创意思路

## 目的
核心任务：
- 通过挖掘用户痛点、分析广告角度以及收集创意灵感来提升广告效果。

该技能专为广告工作流程设计，旨在提供可执行的计划，而非泛泛而谈的建议。

## 适用场景
当用户需要以下内容时，请使用此技能：
- 与业务目标相关的广告执行指导
- 涉及收入、投资回报率（ROAS）、每点击成本（CPA）或预算效率的增长决策
- 针对以下平台的广告策略：Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads
- 具体功能需求：挖掘用户痛点、分析广告角度、收集创意灵感

**高频关键词**：
- 广告、广告活动、增长、收入、利润
- 投资回报率（ROAS）、每点击成本（CPA）、投资回报率（ROI）、预算、出价、流量、转化率、销售漏斗
- Meta、Google Ads、TikTok Ads、YouTube Ads、Amazon Ads、Shopify Ads、DSP

## 输入要求
**必填项**：
- 产品或服务内容
- 目标受众
- 广告投放范围

**可选项**：
- 现有创意素材
- 品牌调性
- 禁止使用的宣传内容
- 创意制作限制

## 输出内容
1. 创意目标
2. 广告角度与创意方向
3. 创意素材规格
4. 测试方案
5. 创意质量检查清单

## 工作流程
1. 将创意目标与销售漏斗阶段及关键绩效指标（KPI）对齐。
2. 生成多种广告角度和创意方向。
3. 根据不同平台的要求调整每个创意方向。
4. 定义创意变体矩阵及测试顺序。
5. 设置质量与合规性检查点。

## 决策规则
- 如果目标受众对广告不敏感，优先选择能够引发用户共鸣的创意。
- 在再营销阶段，重点关注广告内容的清晰度和紧迫性。
- 如果平台有严格的格式限制，简化信息结构，确保只有一个明确的呼吁行动（CTA）。

## 平台注意事项
**主要适用平台**：
- Meta（Facebook/Instagram）、TikTok Ads、YouTube Ads、Google Ads、Amazon Ads

**平台使用建议**：
- 根据不同平台的特点制定针对性的建议，避免将所有平台的策略合并为统一方案。
- 对于Meta和TikTok Ads，重点关注创意的测试频率。
- 对于Google Ads和Amazon Ads，重点关注吸引用户的需求和展示意图。
- 对于DSP/程序化广告平台，重点关注受众控制及投放频率的管理。

## 限制与注意事项
- 不得伪造任何数据或预测结果。
- 区分观察到的事实和假设。
- 对每个建议的行动使用可衡量的语言进行描述。
- 在存在支出风险的情况下，必须包含至少一种回退或止损策略。

## 故障处理与升级流程
- 如果缺少关键输入信息，仅要求提供最低限度的必要字段。
- 如果平台限制存在冲突，需展示权衡方案及安全默认选项。
- 如果对创意质量的信心较低，需明确标注并提供验证清单。
- 如出现高风险问题（如政策变更、计费问题、数据追踪异常），需通过结构化的流程进行上报和处理。

## 示例代码
### 创意简报示例
```
creative_id: CR-001
angle: “解决用户痛点”
hook: “在第一周内停止浪费广告预算”
formats: [9:16视频、1:1图片]
```

### 创意变体矩阵
```
V1: 广告角度调整
V2: 呼吁行动（CTA）调整
V3: 视觉展示内容调整
```
