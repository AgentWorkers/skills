# 广告投资回报率（ROAS）预测

## 目的
核心任务：
- 进行预测场景建模、分析归因敏感性，并提供预算建议

该技能专为广告工作流程设计，旨在生成可执行的计划，而不仅仅是通用性的建议。

## 适用场景
当用户需要以下内容时，请使用此技能：
- 与业务成果相关的广告执行指导
- 涉及收入、ROAS、CPA（每点击成本）或预算效率的增长决策
- 针对以下平台的操作：Meta（Facebook/Instagram）、Google Ads、TikTok Ads、YouTube Ads、Amazon Ads、Shopify Ads、DSP（程序化购买）
- 具体功能：预测场景建模、归因敏感性分析、预算建议

**高频关键词**：
- 广告、广告活动、增长、收入、利润
- ROAS、CPA、ROI（投资回报率）、预算、出价、流量、转化率、销售漏斗
- Meta、GoogleAds、TikTokAds、YouTubeAds、AmazonAds、ShopifyAds、DSP

## 输入参数
**必填项**：
- 预测目标：ROAS、CPA 或收入
- 规划周期
- 基本假设

**可选项**：
- 归因窗口选项
- 预算方案
- 季节性因素
- 风险容忍度

## 输出内容
1. 模型输入数据
2. 不同预测场景的结果
3. 敏感性分析
4. 归因影响说明
5. 预算建议

## 工作流程
1. 对基准指标和假设进行标准化处理。
2. 构建基础情景、乐观情景和悲观情景。
3. 对转化率和CPC（每次点击成本）的假设进行敏感性分析。
4. 比较不同归因窗口下的预期变化。
5. 提出带有置信区间的预算分配方案。

## 决策规则
- 如果假设存在不确定性，应扩大预测区间并降低投资力度。
- 如果预测结果差异较大，建议分阶段投入预算。
- 如果归因窗口对结果影响显著，应提供双方案供选择。

## 平台注意事项
**主要适用平台**：
- Meta（Facebook/Instagram）、Google Ads、TikTok Ads、YouTube Ads、Amazon Ads、Shopify Ads、DSP（程序化购买）

**平台操作建议**：
- 建议根据不同平台的特点制定具体策略；不要将所有平台的策略合并为一个通用方案。
- 对于Meta和TikTok Ads，优先考虑创意内容的测试频率。
- 对于Google Ads和Amazon Ads，重点关注需求捕获和用户意图分析。
- 对于DSP/程序化购买，重点控制受众范围和投放频率。

## 约束与限制
- 禁止伪造任何指标或政策结果。
- 将观察到的数据与假设区分开来。
- 对每个建议措施使用可量化的表述。
- 在存在支出风险的情况下，必须包含至少一种回退或止损策略。

## 故障处理与升级流程
- 如果关键输入信息缺失，仅要求提供最低限度的必要信息。
- 如果平台限制相互冲突，需展示权衡方案及安全默认值。
- 如果预测置信度较低，应明确标注并提供验证清单。
- 如果出现高风险问题（如政策变更、计费问题、数据跟踪异常），需通过结构化流程进行上报。

## 代码示例
### 预测输入
```
支出：50000
CPC：1.2
转化率：0.035
平均每次点击花费：68
```

### 预测结果
```
基础ROAS：2.6
乐观预测ROAS：3.1
悲观预测ROAS：2.1
```

## 示例
### 示例1：存在不确定性的预算规划
**输入**：
- 下个月支出翻倍
- 基准转化率不稳定

**输出重点**：
- 基础情景、乐观情景、悲观情景
- 敏感性影响因素
- 安全的预算分配方案

### 示例2：归因敏感性分析
**输入**：
- 1天和7天的归因分析得出不同的ROAS结果
- 需要制定分配策略

**输出重点**：
- 归因差异模型
- 决策阈值
- 不同渠道的影响分析

### 示例3：季节性预测
**输入**：
- 计划进行节日促销
- 历史CPC波动较大

**输出重点**：
- 季节性因素的调整
- 经过风险调整后的预测范围
- 最终预算建议
