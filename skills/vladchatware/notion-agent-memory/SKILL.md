# 代理记忆工具包（Agent Memory Kit）

这款工具包适用于人类用户和AI代理，使用相同的文件和格式。

## 问题所在

每次会话开始时，所有内容都会被清空，用户或代理无法记住之前的信息或上下文，因此需要不断重复学习相同的内容。

## 解决方案

我们提供了两个模板来帮助保持知识的连续性：

1. **MEMORY.md**：用于存储持续的上下文信息（如模式、项目、学习心得等）。
2. **AGENTS.md**：包含操作指南（说明如何在系统中进行工作）。

这两个模板通过Notion平台提供，您可以将其复制到自己的工作空间并立即开始使用。无论是用于记录个人成长，还是帮助AI代理保持记忆，这个工具都非常实用。

## 快速入门

1. **在 [shop.vlad.chat](https://shop.vlad.chat) 获取模板**。
2. 将Notion模板复制到您的工作空间。
3. （对于AI代理）：设置Notion API访问权限（使用说明已提供）。
4. 每次开始会话时，先阅读 `MEMORY.md`。
5. 随着工作的进行不断记录内容——这样的持续记录机制能帮助您保持学习的连贯性。

## 记忆结构

| 层级 | 文件名 | 用途 |
|-------|------|---------|
| **每日记录** | `memory/YYYY-MM-DD.md` | 纯粹的事件记录、决策、笔记 |
| **长期存储** | `MEMORY.md` | 经过整理的模式、学习心得、正在进行的项目 |
| **结构化工具** | ACT Scrolls（可选） | 用于深度自我反思的工具 |

## 持续性循环（Continuity Cycle）

**两个关键步骤：** 在完成任何事情之前，先问自己：“明天在没有上下文信息的情况下，我还能继续完成这项工作吗？”

## 适用范围

**对人类用户来说：** 可以记录自己的成长过程、学习到的模式和经验教训。未来的自己会因此受益。
**对AI代理来说：** 可以在不同会话之间保持知识的连贯性，避免重复学习相同的内容。

**相同的文件格式**：无论谁使用这个工具，都能遵循相同的方法论。
