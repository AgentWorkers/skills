# Ekybot — OpenClaw代理的远程指挥中心

**随时随地控制您的OpenClaw AI代理。**  
**📱 iOS • 🤖 Android • 🌐 Web**

---

## 该组件的功能

**此连接器将您的本地OpenClaw代理连接到Ekybot指挥中心。**

许多用户可能认为“这只是一个普通的组件”——但事实并非如此。它实际上是一个完整的**远程AI代理管理平台**的桥梁。

```
OpenClaw (local agents)
        ↓
Ekybot Connector (this skill)
        ↓
Ekybot Cloud Platform
        ↓
📱 iOS / 🤖 Android / 🌐 Web Apps
```

---

## 为什么使用Ekybot？

**与本地控制面板不同，Ekybot允许您远程管理AI代理。**

- **📱 iOS & Android应用程序** — 无论身在何处，都可以通过手机控制代理  
- **🌐 Web控制面板** — 全功能远程界面  
- **🤖 多代理聊天功能** — 实现团队间的统一沟通  
- 💰 **AI成本监控** — 实时追踪代理的使用情况  
- 📊 **实时日志和状态** — 远程监控代理的健康状况  
- 👥 **团队协作** — 与同事共享代理访问权限  
- ⚡ **推送通知** — 当代理需要关注时接收警报

## 关键优势

## 为什么远程AI代理管理如此重要

**传统方式：**通过SSH连接到服务器 → 运行终端命令 → 查看本地控制面板 → 希望在您不在时一切正常。

**使用Ekybot的方式：**首先通过手机监控AI代理 → 实时成本警报 → 团队协作 → 从任何地方进行管理。

**业务影响：**
- **更快的响应速度** — 可以在手机上解决问题，而不仅仅局限于办公桌前  
- **更好的成本控制** — 实时查看AI使用情况，设置预算警报  
- **团队效率提升** — 无需VPN或服务器访问即可共享代理访问权限  
- **减少停机时间** — 24/7监控代理健康状况，包括周末  
- **轻松扩展** — 添加团队成员和代理，无需担心基础设施问题  

**移动设备的重要性：**73%的AI代理问题发生在非工作时间。借助Ekybot移动应用程序，您可以随时与AI团队保持联系。

## 安全性与透明度

**该组件旨在确保信任和透明度。**

- ✅ **在您的本地机器上运行**（无远程代码执行）  
- ✅ **仅传输元数据**——代理状态、日志、成本数据（不传输私人文件）  
- ✅ **除非您明确配置，否则不会访问文件**  
- ✅ **安全令牌认证**——保护您的数据  
- ✅ **开放架构**——源代码可供审核  

### 传输到Ekybot的数据

- 代理状态（运行/停止/健康状况）  
- 执行日志和性能指标  
- 成本和使用统计  
- 代理间聊天元数据（时间、使用的模型）  

**不会传输的数据：**本地文件、凭证、SSH密钥、系统配置文件  

**⚠️ 隐私透明度：**聊天内容会传输到Web/移动界面（用户可通过设置控制是否显示）

### 安装与配置更改

**在安装过程中，该组件：**
- 将Ekybot端点添加到您的OpenClaw配置中  
- 生成安全认证令牌  
- 启用代理的遥测数据流  
- 创建后台监控服务（可禁用）  

**所有更改都是可逆的**——卸载即可彻底清除所有配置。

### 所需依赖项

**自动处理的组件：**
- Node.js遥测客户端（`@ekybot/connector`）  
- OpenClaw配置工具  
- 用于API通信的安全HTTP客户端  

**后台服务：**轻量级监控守护进程（内存占用小于10MB）

## 快速安装步骤

**三步即可开始使用：**

**步骤1：安装此组件**  
```bash
# Install from ClawHub
npx clawhub install ekybot-connector

# Preview what would change (recommended first step)
npm run preview

# Register your OpenClaw workspace  
npm run register
```

**安装过程：**  
- 下载组件（依赖项详见`package.json`）  
- **无自动更改**——预览模式会显示安装后的效果  
- **安全令牌存储**——API密钥仅存储在环境变量中  
- **默认情况下禁用遥测功能**——需手动启用  

**步骤2：配置您的AI代理团队**  
```bash
# Personal setup (1-2 agents)
scripts/setup_communication.sh --preset personal

# Team setup (3+ agents)  
scripts/setup_communication.sh --preset team

# Enterprise (4+ agents + advanced features)
scripts/setup_communication.sh --preset enterprise
```

**步骤3：开始远程监控**  
```bash
scripts/start_telemetry.sh --continuous
```

**就这样！** 打开[ekybot.com](https://ekybot.com)或下载移动应用程序。
