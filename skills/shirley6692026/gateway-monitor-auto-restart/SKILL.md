# Gateway Monitor 自动重启技能

该技能可自动监控 OpenClaw 网关的状态，并在网关无响应时自动重启它。具备 3 小时一次的检查频率、智能重启逻辑、问题诊断功能以及 7 天的日志轮换机制。

## 描述

此技能为 OpenClaw 网关提供全面的监控服务，并具备自动重启功能，具体包括：

- 每 3 小时进行一次健康检查
- 当网关故障时自动重启
- 在启动失败时诊断问题
- 实现 7 天的日志轮换
- 优先考虑快速重启网关，以快速恢复服务

## 功能特点

- **自动监控**：每 3 小时检查一次网关状态
- **智能重启**：在网关无响应时自动重启
- **问题诊断**：识别并报告启动过程中出现的问题
- **快速恢复**：优先执行网关重启操作
- **日志管理**：日志保留 7 天，并定期轮换
- **错误处理**：优雅地处理“网关已运行中”的错误情况

## 使用方法

该技能会自动设置一个 cron 作业，每 3 小时运行一次监控脚本。监控系统将执行以下操作：

1. 检查网关是否正常响应
2. 如果网关无响应，尝试重启它
3. 如果重启失败，诊断问题原因
4. 记录所有操作及其时间戳
5. 删除超过 7 天的旧日志

## 需求条件

- 已安装并配置好 OpenClaw 网关
- 具备管理网关服务的相应权限
- 具备调度监控任务的 cron 访问权限

## 配置说明

无需额外配置。该技能会自动安装监控系统，并使用最优设置进行运行。
