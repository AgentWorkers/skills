## 💡 快速入门（5 分钟）

### 第 1 步：获取 API 密钥（2 分钟）
1. 访问 [anvevoice.com/developer](https://anvevoice.com/developer)
2. 注册或登录您的 AnveVoice 账户
3. 点击 “生成 API 密钥”
4. 复制您的 `anvk_...` 密钥

### 第 2 步：配置（1 分钟）
```bash
openclaw config set ANVEVOICE_API_KEY anvk_your_key_here
```

### 第 3 步：创建您的第一个机器人（2 分钟）
```bash
openclaw skills test anvevoice --input "create_bot with { name: 'My Support Bot', system_prompt: 'You are a helpful assistant for my website.' }"
```

### 第 4 步：部署（复制粘贴）
获取嵌入代码并将其添加到网站的 HTML 中：
```bash
openclaw skills test anvevoice --input "get_embed_code with { bot_id: 'YOUR_BOT_ID' }"
```

✅ **完成！** 您的网站现在已拥有一个语音 AI 助手。

## 错误处理

- **401**：API 密钥无效或缺失。请确保 `ANVEVOICE_API_KEY` 设置正确。
- **403**：权限错误 —— 你只能访问自己的机器人和数据。
- **429**：使用次数限制。技能会自动重试，采用指数级退避策略。
- **5xx**：服务器错误。技能会自动重试最多 4 次。

所有工具的响应格式为 `{ success: true, data: {...} }` 或 `{ success: false, error: "message" }`。
