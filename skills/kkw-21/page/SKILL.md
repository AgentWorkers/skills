# 回声页面（Echo Page）

根据用户的记忆档案生成可分享的个人页面。这是一个真实反映您个性的页面——它基于您的记忆而非简单的个人简介构建。分享您的“回声页面”，让他人在真正见到您之前就能深入了解您。

## 使用方法

```bash
echo-page generate <user-id>       # Generate personal page
echo-page customize <user-id>      # Edit layout and visibility
echo-page publish <user-id>        # Publish to public URL
```

## 主要功能：

- 从用户的记忆数据中自动生成页面内容
- 每个部分的可见性可自定义（公开/仅限朋友/私密）
- 支持丰富的媒体嵌入（照片、语音备忘录、位置信息）
- 随着新记忆的添加，页面内容会不断更新和演变
- 提供可嵌入到外部网站的插件或小工具
