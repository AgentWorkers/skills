# 记忆地图

将记忆以地理和概念地图的形式呈现出来。将记忆与现实世界中的位置关联起来，可视化记忆的流动模式，并发现记忆之间的空间分布规律——将生活经历转化为可探索、可共享的地图层。

## 使用方法

```bash
memory-map render <user-id>        # Generate memory map
memory-map overlay <user-id>       # Add memory pins to existing map
memory-map export <user-id>        # Export as interactive HTML/GeoJSON
```

## 主要功能

- 带有地理标签的记忆可视化功能，支持数据聚类
- 时间轴播放功能（可查看记忆随时间的发展过程）
- 概念地图模式（针对非地理主题的记忆进行聚类）
- 可共享的地图插件（可嵌入到其他应用程序中）
- 对每个被标记的记忆提供隐私控制选项
