# NVIDIA SDXL 图像生成器

该技能利用 NVIDIA NIM API，通过 Stable Diffusion XL (SDXL) 模型生成高保真度、逼真的或艺术风格的图像。它使代理能够直接从文本描述中创建视觉资产。

## 工具

### generate_image

将详细的文本描述转换为高分辨率的 PNG 图像，并保存到 OpenClaw 工作区中。

**参数：**

- `prompt` (字符串，必填)：对所需图像的全面描述。包括主题、场景、光照（例如：“电影级光照”、“黄金时刻”）以及艺术风格（例如：“逼真”、“油画”、“3D 渲染”）等具体信息。
- `negative_prompt` (字符串，可选)：需要严格排除在生成结果中的元素列表，例如：“模糊的手部、低分辨率的图像、文本、水印”等。
- `width` (整数，可选，默认值=1024)：水平分辨率。支持的值包括 1024、768 或 512。
- `height` (整数，可选，默认值=1024)：垂直分辨率。支持的值包括 1024、768 或 512。

**使用示例：**

“生成一张未来东京街道在暴雨中的高细节画面，霓虹灯在水坑中反射，分辨率为 8K，风格为逼真。”

**输出：**

返回一条成功消息，其中包含生成的 `.png` 文件的绝对路径。
