# DocuClaw 技能

DocuClaw 提供了一套独立的数据基础设施，用于处理和归档文档。它利用多模态大语言模型（LLM）从扫描件、照片和电子邮件中提取结构化信息，并将所有数据存储在人类可读且版本可控的 Markdown 文件中。

## 使用场景

- **费用管理**：从收据中提取总额、税费和日期，以便进行税务申报。
- **合同分析**：监控法律文件中的到期日期和续签条款。
- **独立归档**：维护符合 GDPR/GoBD 标准的本地文档归档系统，涵盖所有纸质和数字邮件。
- **统一查询**：无需依赖云服务即可查询文档历史记录。

## 主要特性

- **100% 本地化**：完全不依赖云服务。您的私人数据不会离开您的设备。
- **即插即用解析器**：支持针对不同国家/地区的文档格式的扩展架构。
- **人工智能驱动**：支持 Ollama、OpenAI Vision 或任何多模态模型，实现智能信息提取。
- **Markdown 架构**：所有文档均被统一转换为基于 YAML 元数据的通用格式。

## 工作流程示例

1. **输入**：上传 PDF 发票或收据的照片。
2. **处理**：运行 `docuclaw process` 命令以触发人工智能提取。
3. **归档**：文档会被保存为 `YYYY/MM/filename.md` 格式的文件。
4. **应用**：提取的数据会同步到您的日历或会计工具中。

## 集成

DocuClaw 与 OpenClaw 生态系统无缝集成，允许 AI 代理在您的本地文档归档系统中执行检索增强生成（RAG, Retrieval-Augmented Generation）操作。
