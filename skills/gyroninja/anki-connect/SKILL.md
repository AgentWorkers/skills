# AnkiConnect 技能

此技能提供了与 [AnkiConnect](https://foosoft.net/projects/anki-connect/) 的集成功能。AnkiConnect 是 Anki 的一个插件，它提供了一个 REST API，允许外部应用程序与 Anki 进行交互。
