# Cyber OWASP Review

## 概述

将应用程序安全检测结果归类到 OWASP 标准中，并生成相应的修复措施。

## 工作流程

1. 从扫描工具、测试或审查中获取原始检测结果。
2. 使用关键词和上下文匹配将检测结果映射到 OWASP 分类中。
3. 按类别和严重程度对检测结果进行汇总。
4. 生成针对每个类别的修复检查清单。

## 使用捆绑资源

- 运行 `scripts/map_findings_to_owasp.py` 以进行精确的映射。
- 阅读 `references/owasp-mapping-guide.md` 以了解分类规则和启发式方法。

## 注意事项

- 确保提供的指导内容仅侧重于修复措施。
- 请勿提供攻击载荷或攻击策略。
