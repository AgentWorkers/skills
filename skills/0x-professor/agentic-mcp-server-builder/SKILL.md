# Agentic MCP Server Builder

## 概述

该工具用于根据结构化的工具列表，生成一个最小化的MCP服务器框架及相应的合约摘要。

## 工作流程：

1. 定义服务器名称及工具列表，并为每个工具添加描述。
2. 生成服务器框架文件的结构图及工具合约的摘要。
3. （非测试模式时）可选地生成初始配置文件。
4. 在添加业务逻辑之前，先审查生成的合约内容以确保其正确性。

## 使用捆绑资源：

- 运行 `scripts/scaffold_mcp_server.py` 命令以生成初始配置文件。
- 阅读 `references/mcp-scaffold-guide.md` 以了解文件结构和合约检查的详细要求。

## 注意事项：

- 明确界定各工具的职责范围，并确保其功能尽可能简洁。
- 确保所有工具的输出结果都是可预测的，并提供清晰的输入/输出规范。
