# 机器学习实验跟踪器

## 概述

本工具用于生成结构化的实验计划，以便在实验跟踪系统中进行统一记录。

## 工作流程

1. 定义数据集、目标任务、模型系列以及参数搜索范围。
2. 在训练之前确定评估指标和接受标准。
3. 生成包含版本信息及预期结果的实验执行计划。
4. 将实验执行计划导出，以便在跟踪工具中执行。

## 使用捆绑资源

- 运行 `scripts/build_experiment_plan.py` 命令以生成统一的实验执行计划。
- 阅读 `references/tracking-guide.md` 以获取可复现性检查清单。

## 规范与注意事项

- 确保输入信息明确且易于机器读取。
- 必须包含评估指标和基线标准。
