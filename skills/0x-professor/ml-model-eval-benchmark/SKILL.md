# 机器学习模型评估基准测试

## 概述

该基准测试旨在根据基于指标加权的评估结果，生成一致的模型排名。

## 工作流程

1. 定义指标权重及可接受的指标范围。
2. 获取每个候选模型的评估指标数据。
3. 计算加权得分和模型排名。
4. 导出排行榜及模型提升建议。

## 使用捆绑资源

- 运行 `scripts/benchmark_models.py` 以生成评估结果。
- 阅读 `references/benchmarking-guide.md` 以获取关于权重计算和排序规则的详细指导。

## 规范要求

- 确保所有候选模型的指标名称和评分尺度保持一致。
- 将权重计算的相关信息记录在输出结果中。
