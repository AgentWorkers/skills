# 智能文件整理器

这是一项功能，允许Claw扫描目录，根据文件类型和内容对文件进行分类，并将它们重新组织成清晰、逻辑清晰的文件夹结构。

## 该功能的作用

该功能提供了一套智能的文件整理工作流程：

1. **目录扫描** — 递归扫描目标目录，记录所有文件的名称、大小、类型和修改日期。
2. **文件分类** — 根据文件扩展名和内容类型（如文档、图片、代码、数据、压缩文件、媒体文件等）对文件进行分类。
3. **重复文件检测** — 通过比较文件的校验和来识别重复文件。
4. **文件夹结构规划** — 根据文件分类结果提出新的文件夹结构方案。
5. **安全重组** — 将文件移动到新的文件夹结构中，并生成详细的变更日志，以便在需要时进行回滚操作。

## 使用方法

将Claw指向目标目录，并说明您的整理需求：

- “按文件类型整理我的下载文件夹”
- “扫描这个项目目录并找出所有重复文件”
- “根据文件的修改日期将这些文件按年份分类到相应的文件夹中”
- “清理这个目录——将文档、图片和代码文件分别存放”

## 安全特性

- **预览模式** — 在移动任何文件之前，可以预览所提出的变更（默认启用）
- **变更日志** — 每次文件移动操作都会被记录到`reorganization_log.txt`文件中，以确保可追溯性。
- **禁止删除文件** — 该功能不会自动删除文件；重复文件会被移动到一个 `_duplicates` 文件夹中，供用户手动处理。

## 输出结果

- 一份总结报告，显示文件清单以及建议的/实际执行的整理方案。
- 一个 `reorganization_log.txt` 文件，记录了所有的操作细节。
- 重新组织后的目录结构。
