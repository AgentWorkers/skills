# Markdown 报告生成器

这是一项功能，它允许 Claw 从多个来源收集信息，并将其编译成格式规范、结构清晰的 Markdown 报告。该报告支持使用表格、章节，并可选择性导出为 PDF 格式。

## 该功能的作用

该功能提供了完整的报告生成流程：

1. **内容收集**：从提供的文本、文件、URL 或之前的对话记录中收集信息。
2. **结构规划**：将内容组织成逻辑清晰的章节，并生成目录。
3. **格式化**：应用统一的 Markdown 格式，包括标题、列表、表格和重点内容的显示方式。
4. **表格生成**：将原始数据转换为格式规范的 Markdown 表格。
5. **导出**：将报告保存为 `.md` 文件，并可根据需要转换为 PDF 或 HTML 格式。

## 使用方法

你可以要求 Claw 生成关于任何主题或数据的报告：

- “根据这些会议记录生成项目状态报告”
- “生成这三个产品的对比报告，并附上功能对照表”
- “为这个代码库编写技术文档报告”
- “根据这些数据点生成每周总结报告”

## 报告结构

生成的报告遵循以下模板：

```
# Report Title
## Table of Contents
## Executive Summary
## Section 1: [Topic]
### Subsection with details
## Section 2: [Topic]
## Data Tables
## Conclusions & Recommendations
## Appendix (if applicable)
```

## 自定义选项

- **语气**：正式、技术性、执行摘要或非正式
- **长度**：简短（1-2 页）、标准（3-5 页）或详细（5 页以上）
- **章节**：指定需要包含或排除的章节
- **品牌设计**：添加自定义的页眉或页脚文本

## 输出结果

- 报告以 `.md` 文件的形式保存在指定位置
- 可根据需求转换为 HTML 或 PDF 格式
- 格式清晰、一致，适合分享或发布
