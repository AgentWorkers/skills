# CSV 数据分析器

这是一项使 Claw 能够加载、探索、分析并可视化 CSV 数据集的技能，能够提供数据统计分析结果，并帮助用户解答关于数据的问题。

## 该技能的功能

该技能为 Claw 提供了一种结构化的 CSV 数据分析方法：

1. **数据加载与检查** — 读取 CSV 文件，检测列类型，并显示数据的基本结构（数据形状、列数、样本行）。
2. **数据清洗** — 识别并处理缺失值、重复值以及数据类型不一致的问题。
3. **统计汇总** — 计算数值列的描述性统计量（均值、中位数、众数、标准差、百分位数）。
4. **过滤与分组** — 根据条件筛选数据，并按类别进行聚合。
5. **相关性分析** — 查找数值变量之间的关系。
6. **可视化** — 生成图表（条形图、折线图、散点图、直方图）以展示数据模式。

## 使用方法

提供 CSV 文件并请求 Claw 进行分析：

- “分析这些销售数据，告诉我哪个产品类别的收入最高。”
- “找出该数据集中温度列中的异常值。”
- “从这个 CSV 文件中生成一个显示月度趋势的图表。”
- “比较实验数据中 A 组和 B 组的表现。”

## 使用要求

- 输入必须是有效的 CSV 文件（默认使用逗号作为分隔符；也可以指定其他分隔符）。
- 环境中必须安装 Python 及 pandas 和 matplotlib 库。

## 输出结果

- 统计汇总表。
- 以通俗易懂的语言呈现的关键分析结果。
- 如果请求可视化，生成的图表会保存为 PNG 文件。
- 如果进行了数据清洗，清洗后的数据集会以新的 CSV 文件形式导出。
