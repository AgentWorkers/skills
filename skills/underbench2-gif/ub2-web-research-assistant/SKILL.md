# 网络研究助手

这是一项让Claw能够针对任何主题进行结构化网络研究的技能，它可以从多个来源汇总信息，并生成一份包含引用信息的条理清晰的摘要。

## 该技能的功能

该技能引导Claw完成以下系统的网络研究流程：

1. **问题构建** — 将一个广泛的研究问题分解为具体、有针对性的搜索查询。
2. **多源信息收集** — 在多个来源中搜索，以收集不同的观点和数据点。
3. **来源评估** — 评估每个来源的可信度和相关性。
4. **信息整合** — 将收集到的信息整合成一个连贯、结构清晰的摘要。
5. **引用标注** — 为所有引用的材料提供正确的出处和URL。

## 使用方法

你可以要求Claw研究任何主题，例如：

- “研究量子计算的现状，并总结过去一年的主要突破。”
- “查找并比较前五大云服务提供商的定价模式。”
- “调查电动汽车与传统汽车对环境的影响。”

## 输出格式

该技能会生成一份结构化的研究报告，内容包括：

- **执行摘要** — 用2-3句话概括主要研究结果。
- **详细发现** — 按子主题分类，并附有支持证据。
- **来源列表** — 所有引用的URL和出版日期。
- **可信度评估** — 每项发现在不同来源中的支持程度。

## 最佳实践

- 请明确你的研究问题，以获得更精确的结果。
- 如果需要最新信息，请指定时间范围（例如：“过去6个月内”）。
- 如果需要特定角度的见解（技术、商业、消费者等），请予以说明。
- 如果默认的输出格式不符合你的需求，请提出具体要求。
