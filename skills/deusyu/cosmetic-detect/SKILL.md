# 美容手术检测

## 核心原理

美容手术会以偏离自然发育规律的方式改变人体组织。检测的目标是识别那些“反自然”的信号——即那些在解剖结构、比例、质地或动态方面不符合未经过手术改变的人脸/身体统计特征的迹象。

这种检测具有对抗性：最精妙的手术往往设计得难以被察觉。**切勿断言结果的绝对性——应使用概率性的语言进行描述**（如“与……一致”、“可能表明……”等）。

## 分析流程

在开始分析之前，请先阅读 `references/analysis-framework.md` 文件，以获取详细的区域检查清单。

### 第一步：初步评估

- **图像质量**：分辨率、光线、拍摄角度、化妆程度。图像质量差或使用了过多滤镜会显著降低检测的准确性，请在报告中予以说明。
- **明显的种族/血统**：有助于确定基本的解剖特征。例如，高鼻梁对欧洲人来说很常见，但对东亚人来说则较为罕见。
- **外观年龄**：有助于判断皮肤质量、皮肤体积以及老化迹象。
- **图像处理**：检查是否存在数字化处理（如平滑处理、变形、面部调整等）——将这些情况标记为非美容手术，以避免误判。

### 第二步：逐区域分析

使用参考文件中的指标对每个面部区域进行独立分析。针对每个区域，需要评估以下几点：
1. 该区域的特征是否符合该人种族和年龄的正常范围？
2. 是否存在手术或非手术干预的特定迹象？
3. 信任度：低 / 中等 / 高

### 第三步：跨区域一致性检查

这是最关键的检测步骤。自然的人脸具有内在的一致性。需要关注以下几点：
- **种族一致性**：所有面部特征是否与同一遗传背景相符？（例如，东亚人的骨骼结构与高鼻梁的组合是不匹配的）
- **年龄一致性**：不同区域的衰老程度是否一致？（额头光滑但手部有老化迹象可能意味着注射了肉毒杆菌）
- **对称性**：自然的人脸存在一定程度的不对称性。过度对称可能表明进行了整形手术。
- **比例协调性**：各面部特征之间的比例是否在自然范围内？

### 第四步：输出结果

```

- 年龄估计 / Estimated age

### 区域分析 / Regional Analysis
For each region with findings:
```

## 特殊模式

### 术前/术后对比

当提供同一人在不同时间的多张照片时：
- 在脑海中将面部特征进行对齐。
- 将**骨骼/结构变化**视为最可靠的判断依据（骨骼和软骨不会自然发生变化）。
- 体积的变化可能是由于衰老、体重变化或填充物造成的。
- 皮肤/质地的变化可能是由于衰老、护肤或手术引起的。

### 名人/公众人物

- 如果有该人物的外貌历史记录，请参考这些信息。
- 需要注意的是，顶级医生的手术效果最难被检测出来。
- 在判断信任度时需格外谨慎。

### 批量分析

当分析多个人（如合影或一组照片）时：
- 对每个人进行独立分析。
- 以整个群体作为对比的基准。

## 注意事项：

- **切勿断言结果的绝对性**。即使是经验丰富的外科医生也无法仅凭照片做出绝对判断。
- **承认检测的局限性**：光线、拍摄角度、化妆、基因因素以及图像质量都会影响检测结果。
- **区分手术与非手术干预**：隆鼻手术和肉毒杆菌注射的视觉特征有所不同，务必明确区分。
- **保持中立**：不要对某人是否应该接受手术发表评论。
- **尊重文化差异**：双眼皮手术在东亚非常普遍，而隆鼻手术在全球范围内也很常见。在分析时应保持客观的态度。
