# Minecraft 构建服务器技能包

> 默认语言：英语。

## 范围

本技能涵盖了在运行中的 Minecraft Java 服务器上进行的所有与构建相关的内容，包括使用 FAWE 进行编辑、机器人辅助的建造、设计图工作流程、材料供应链管理、红石生产线搭建、村民交易以及大型建筑项目的执行方法。  
**不包括** 服务器基础设施的维护工作，例如服务器的启动/停止、JVM 调优、数据备份、插件安装/更新或服务器健康状况监控等，这些属于专门的服务器运维技能范畴。  
可以这样理解：本技能帮助用户决定如何在运行中的服务器上**构建和操作构建工作流程**；而运维技能则帮助用户**管理服务器本身**。

---

## 1. 激活规则

### 触发条件  
当请求涉及以下与构建相关的内容时，激活本技能：  
- 构建服务器的概念、创意工作流程或大型建筑项目的执行方法  
- FAWE/WorldEdit 的使用或大规模编辑工具  
- 设计图工作流程、Litematica 工具或像素艺术创作  
- 机器人辅助的建造（如 Mineflayer、Baritone、FAWE 脚本）  
- 材料供应链管理、村民交易或用于辅助建造的红石自动化系统  
- 专为构建任务设计的工具（如 Paper、Purpur、Spigot）  

### 非触发条件  
**不适用于以下场景**：  
- 服务器生命周期管理、数据备份、恢复计划、插件安装/更新或服务器健康状况监控  
- 服务器运行时间检查、JVM 调优、Paper 工具的使用故障排除或操作手册  
- 玩家行为管理（白名单/封禁/踢出玩家）、通用 RCON（远程控制）操作  
- Bedrock Edition 版本的设置  
- Forge/Fabric 模组的源代码开发  
- PvP（玩家对战）或迷你游戏服务器的运营  
- 与构建无关的生存指南  
- 反作弊手段、漏洞利用、盗版行为或账户盗窃

## 2. 响应流程  
1. 确定用户询问的是**自托管服务器**还是**公共服务器**。如果不清楚，请先询问后再提供涉及权限或自动化的建议。  
2. 将请求分类到以下几类：  
   - 构建服务器的基础知识/软件选择  
   - FAWE 设计图工作流程  
   - 材料管理/资源农场/生产流程  
   - 机器人辅助或客户端辅助的建造方式  
3. 仅从 `references/` 目录中加载所需的参考文件。  
4. 以清晰的结构回答问题，并在必要时明确说明自托管服务器与公共服务器之间的差异。  
5. 在提供执行建议前，务必遵守以下安全规则。  

---

## 3. 安全规则  
**绝对禁止**：  
- 不提供任何反作弊手段  
- 不协助在第三方公共服务器上未经授权地部署机器人  
- 不提供破坏性行为或恶意行为的指导  
- 不协助用户进行破解、盗版行为或账户盗窃  
- 未经明确授权，不得帮助用户在公共服务器上获得不公平的自动化优势  

### 必需标注的内容：  
- **需要管理员权限的操作** → 明确标注为仅限自托管服务器或获得管理员权限的用户使用  
- **机器人自动化** → 明确标注为仅允许自托管服务器使用，公共服务器需要管理员权限  
- **RCON（远程控制）相关内容** → 仅适用于用户自己控制的服务器  
- **Litematica 打印机模式** → 提醒某些服务器禁止使用该功能  

### 模糊的请求场景  
如果无法确定用户是否控制服务器，请在提供自动化建议或权限许可之前先确认用户对服务器的实际控制权。
