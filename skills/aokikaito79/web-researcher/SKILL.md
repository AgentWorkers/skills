# Web Researcher Skill

## 使用场景
- 当用户请求关于某个主题的“最新信息”、“新闻”或“研究结果”时，使用此技能。
- 当你需要验证本地训练数据中不存在的事实时，也使用此技能。

## 研究流程
1. **多次查询搜索**：不要只搜索一次，而是进行2-3次有针对性的搜索（例如：“OpenClaw 2026.3.2 的功能” AND “OpenClaw 2026.3.2 的漏洞”）。
2. **深入分析**：对排名前2的相关URL使用 `web_fetch` 功能获取完整内容；仅依赖片段信息无法进行深入研究。
3. **总结分析**：将搜索结果归纳为“关键事实”、“时间线”以及“矛盾点”（如果有的话）。
4. **引用来源**：务必在报告末尾列出你实际阅读的所有URL。

## 输出格式
- 以 🪐 **Jupiter 研究简报** 作为标题开头。
- 使用项目符号提高可读性。
- 用粗体标出任何“突发新闻”或“重要警告”。
