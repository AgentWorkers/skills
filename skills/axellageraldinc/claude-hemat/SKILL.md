# ClaudeHemat

# 介绍
1. **Haiku 4.5** 是默认模型。只有在任务确实需要 **Sonnet 4.5** 或 **Opus 4.6** 时，才使用这些模型。
2. 使用 `sessions_spawn` 命令可以调用更高级的模型。

```
sessions_spawn(
  message: "<the full task description>",
  model: "anthropic/claude-sonnet-4-5",
  label: "<short task label>"
)
```
```
sessions_spawn(
  message: "<the full task description>",
  model: "anthropic/claude-opus-4-6",
  label: "<short task label>"
)
```

# 模型
## Haiku 4.5
1. 用于简单的问答：询问什么、何时发生、谁参与了、在哪里发生。
2. 适用于非正式的聊天场景，无需进行推理。
3. 提供快速的信息查询服务。
4. 可用于文件查找。
5. 适用于简单的任务（包括但不限于重复性任务）。
6. 适用于 Cron 作业；如果需要推理，则需要升级到更高级的模型。
5. 适用于所有不需要思考的简单任务。
6. 提供简洁的答案，无需解释。
7. **禁止** 使用 Haiku 4.5 进行编程或分析。
8. **禁止** 使用 Haiku 4.5 进行任何形式的推理。
9. 如果认为请求不属于上述 1-6 类别，应升级到更高级的模型。
10. 如果认为使用 Haiku 4.5 会违反上述 8-10 条规定，也应升级到更高级的模型。

## Sonnet 4.5
1. 用于分析数据、编写代码、制定计划以及进行推理。
2. 适用于需要复杂逻辑处理的场景。
3. 可用于比较不同选项或生成报告。
4. 如果用户认为某个请求非常重要，应升级到更高级的模型。

## Opus 4.6
1. 适用于深度研究、关键决策以及极其复杂的推理和规划任务。
2. 可提供详细的解释和结果。

# 其他注意事项
1. 如果用户明确要求使用某个特定模型，请按照其要求执行。
2. 在所有输出结果中必须明确说明使用了哪个模型。
3. 在使用更高级的模型（Sonnet 4.5 或 Opus 4.6）之后，应将其设置为默认模型。
