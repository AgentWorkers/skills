# ClaudeHemat

# 介绍
1. **Haiku 4.5** 是默认模型。只有在任务确实需要 **Sonnet 4.5** 或 **Opus 4.6** 时，才使用这些模型。
2. 请查阅 **模型规则** 部分以确定应使用哪个模型！
3. 可以使用 `sessions_spawn` 来使用更高级的模型。
```
sessions_spawn(
  message: "<the full task description>",
  model: "anthropic/claude-sonnet-4-5",
  label: "<short task label>"
)
```
```
sessions_spawn(
  message: "<the full task description>",
  model: "anthropic/claude-opus-4-6",
  label: "<short task label>"
)
```

# 模型规则
## Haiku 4.5
1. 适用于简单的问答场景（问题涉及“什么、何时、谁、哪里”）。
2. 适用于非正式的聊天交流（无需进行推理）。
3. 适用于快速查询信息。
4. 适用于简单的文件查找任务。
5. 适用于重复性任务。
6. 如果任务需要推理，应立即升级到更高级的模型（Sonnet 4.5 或 Opus 4.6）。
7. 适用于所有不需要思考的简单任务。
8. **禁止** 使用 Haiku 4.5 进行编程或数据分析。
9. **禁止** 使用 Haiku 4.5 进行任何形式的推理。
10. 如果认为某个请求不属于上述适用范围，应立即升级到更高级的模型。
11. 如果担心会违反规则 8-10，也应立即升级到更高级的模型。

## Sonnet 4.5
1. 适用于需要分析问题原因和解决方法的场景。
2. 适用于编写代码或制定计划。
3. 适用于需要复杂推理的场景。
4. 适用于需要生成报告的场景。
5. 如果认为某个请求对用户非常重要，应立即升级到更高级的模型。

## Opus 4.6
1. 适用于深度研究或需要做出关键决策的场景。
2. 适用于涉及极其复杂推理或规划的场景。
3. 适用于需要提供详细解释的场景。

# 其他注意事项
1. 如果用户明确要求使用特定模型，请严格按照要求执行。
2. **所有输出结果中都必须** 显示所使用的模型名称。
3. 在使用完更高级的模型（Sonnet 4.5 或 Opus 4.6）后，应自动恢复使用 Haiku 4.5 作为默认模型。
