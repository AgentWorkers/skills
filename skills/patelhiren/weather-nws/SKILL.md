# 天气查询技能（Weather Query Skill）

该技能可获取美国国家气象局（NWS）提供的详细天气预报，并在NWS API无法使用时自动切换至全球其他天气服务（如wttr.in）作为备用。支持每小时一次的天气更新、空气质量数据以及针对冬季风暴的累积降雪量等信息的显示。

## 功能概述

该技能支持**8种模式**，以满足您的不同查询需求：

| 模式 | 触发条件 | 提供内容 |
|------|-------------------|--------------|
| 🌦️ **标准预报** | 无时间指定 | 当前/今晚/明天的12小时天气预报 |
| ⏰ **每小时预报** | 指定时间查询时 | 每小时一次的天气更新（共7天，约156条数据） |
| 🌨️ **冬季风暴** | 输入与“雪”或“风暴”相关的关键词 | 12小时天气预报及累积降雪量数据 |
| 💨 **空气质量报告** | 使用`--aqi`标志 | 当前空气质量指数（AQI）及未来预报 |
| 🌡️ **实际观测数据与预报对比** | 使用`--current`标志 | 当前气象站的观测数据与预报对比 |
| ☀️ **天文时间** | 使用`--astro`标志 | 日出/日落时间、晨昏时刻、月相信息 |
| ✈️ **航空预报** | 使用`--taf`标志 | 机场天气预报（TAF） |
| 🔥 **火灾天气** | 使用`--fire`标志 | 火灾风险信息及红色警报 |
| 🌍 **全球备用** | 非美国地区 | 使用wttr.in提供的天气数据（信息较简略） |

## AirNow API密钥（可选但推荐）

AirNow API无需密钥即可使用，但会有一定的使用限制：

| 无API密钥 | 有API密钥 |
|-----------------|--------------|
| 请求次数受限（可能导致请求失败） | 提供更高的请求次数限制 |
| 无法保证数据可用性 | 提供优先访问权限 |
| 可能返回空结果 | 提供可靠的空气质量数据 |

### 获取API密钥

1. 访问：https://docs.airnowapi.org/account/request/
2. 填写请求表格（个人使用免费）
3. 密钥将在1-2个工作日内通过电子邮件发送给您

### 设置API密钥

**方法1：环境变量（推荐）**
```bash
export AIRNOW_API_KEY="your-api-key-here"
```

**方法2：OpenClaw配置（长期有效）**
在OpenClaw的`skills.entries.weather-nws.env`文件中进行配置：
```json
{
  "skills": {
    "entries": {
      "weather-nws": {
        "env": {
          "AIRNOW_API_KEY": "your-api-key-here"
        }
      }
    }
  }
}
```

## 使用场景

✅ 当您需要查询以下内容时，可以使用此技能：
- “[美国城市]的天气如何？”
- “预计会下多少雪？”
- “[地点]的冬季风暴预报”
- “[美国城市]明天会下雨吗？”
- “[城市]今天的空气质量如何？”
- 任何与美国天气相关的查询

### 自动备用机制

- 如果查询的是非美国地区，系统会自动切换至wttr.in；
- 如果NWS API无法使用，系统会切换至wttr.in；
- 如果两种来源均无法获取数据，系统会显示错误提示。

## 快速入门指南
```bash
# Run the unified weather script
python3 ./scripts/get_weather.py "New York, NY"

# Force specific source if needed (normally auto-detected)
python3 ./scripts/get_weather.py "London, UK" --source wttr

# Get hourly forecast (auto-detected or forced)
python3 ./scripts/get_weather.py "Boston at 8 PM"
python3 ./scripts/get_weather.py "Chicago" --hourly

# Include air quality
python3 ./scripts/get_weather.py "Seattle" --aqi
```

## 每小时自动检测

该技能能自动识别用户的时间查询需求，并切换到相应的每小时天气预报模式：
```
"Boston at 8 PM"        → ⏰ Hourly mode
"Boston tonight"         → ⏰ Hourly mode
"Boston tomorrow morning" → ⏰ Hourly mode
"Boston at 5:30"         → ⏰ Hourly mode
"When will it stop raining?" → ⏰ Hourly mode
```

**常见的时间表达方式：**
- `at 8 PM`（晚上8点）
- `tonight`（今晚）
- `this afternoon`（今天下午）
- `tomorrow morning/afternoon/night`（明天早上/下午/晚上）
- `when will...`（...将在什么时候发生？）
- `how long until...`（...还有多久？）

## 空气质量（`--aqi`）

使用`--aqi`标志可获取AirNow提供的空气质量指数（AQI）数据：

```bash
python3 ./scripts/get_weather.py "Boston" --aqi
```

**输出内容包括：**
- 当前的空气质量指数（用颜色编码的emoji表示：🟢 🟡 🟠 🔴 🟣 🔵）
- 主要污染物（如PM2.5、O3等）
- 基于空气质量指数的健康建议
- 未来3天的空气质量预测

**空气质量指数（AQI）分类：**
| 范围 | 分类 | Emoji | 建议 |
|-------|----------|-------|------------------|
| 0-50 | 良好 | 🟢 | 可以进行户外活动 |
| 51-100 | 中等 | 🟡 | 敏感人群应减少户外活动 |
| 101-150 | 对敏感人群不健康 | 🟠 | 儿童和老年人应减少户外活动 |
| 151-200 | 非常不健康 | 🔴 | 所有人都应减少户外活动 |
| 201-300 | 非常不健康 | 🟣 | 应避免户外活动 |
| 301-500 | 危险 | 🔵 | 应待在室内 |

## 输出格式

无论数据来源如何，脚本都会保持一致的输出格式：
- **标题**：地点及当前天气警报状态
- **今日 → 今晚 → 明天**：按时间顺序排列的天气信息
- **降雪/降雨量**：如有相关数据，会显示具体的降雪或降雨量
- **总结**：包含 actionable 的建议和具体时间信息

## 实现细节

脚本包含以下功能：
1. 将地理位置转换为经纬度坐标
2. 判断地理位置是否位于美国
3. 对美国地区的位置调用NWS API获取详细天气数据
4. 对非美国地区使用wttr.in作为备用数据源
5. 使用emoji和统一的格式化方式展示所有数据

## 限制事项

- **NWS**：仅支持美国地区，需要互联网连接，且存在请求次数限制
- **wttr.in**：支持全球地区，但降雪量等数据的详细程度较低，且不提供官方的天气警报
- **AirNow**：仅支持美国和加拿大地区，使用API密钥才能获取可靠数据

## 示例

- **美国冬季风暴查询**：返回包含降雪量预估的NWS数据
- **国际地点查询**：自动切换至wttr.in，并提示查询地点不在美国
- **包含空气质量数据**：同时显示天气信息和空气质量指数及健康建议
- **实际观测数据与预报对比**：显示当前气象站的观测数据与预报数据的对比
- **综合信息**：整合多种数据源提供完整的天气报告

## 额外功能

### 实际观测数据与预报对比（`--current`）

使用`--current`标志可查看最近的气象观测站的实际观测数据与预报数据的对比：
```bash
python3 ./scripts/get_weather.py "Boston" --current
```

**显示内容：**
- 温度及与预报的温差
- 天气状况描述
- 风速和风向
- 湿度百分比
- 露点
- 气压（单位：英寸汞柱）
- 能见度

## 警报显示

当有天气警报时，会使用不同的颜色和图标来表示警报的严重程度和紧急性：
| 严重程度 | 显示方式 |
|--------|---------|
| 极端（4） | ⚫ |
| 严重（3） | 🔴 |
| 中等（2） | 🟠 |
| 轻微（1） | 🟡 |

**警报显示内容包括：**
- 警报名称及严重程度标识
- 紧急性标签：⏰ 立即生效 / 📅 预计发生 / 🔮 预计未来发生
- 警报的有效时间范围
- 天气状况的描述
- 建议的应对措施

### 第三阶段功能

### 天文时间（`--astro`）

使用`--astro`标志可查看日出、日落时间、晨昏时刻和月相信息：
```bash
python3 ./scripts/get_weather.py "Boston" --astro
```

**输出内容包括：**
- 🌅 **日出时间**：包含倒计时和过去的时间
- 🌇 **日落时间**：包含倒计时和过去的时间
- 💡 **晨昏时刻**：适合跑步者或骑行者的信息
- ⏱️ **白昼时长**：当天的总白昼时长
- 🌙 **月相**：当前的月相及光照百分比

**示例：**
```
☀️ **Astronomical Times — Boston**

🌅 **Sunrise:** 6:22 AM (12h ago)
🌇 **Sunset:** 5:31 PM (in 2h)
💡 **Civil Twilight:** 5:55 AM – 5:58 PM
⏱️ **Daylight:** 11h 9m
🌙 **Moon:** 🌓 First Quarter (50.0%)
```

### 航空预报（`--taf`）

使用`--taf`标志可查看最近的机场天气预报（TAF）：
```bash
python3 ./scripts/get_weather.py "SFO" --taf
```

**注意**：TAF主要用于航空用途，提供以下信息：
- 机场标识
- 风速和风向
- 能见度
- 云层高度

**提示**：此信息仅供参考，请务必查阅官方来源以获取准确的飞行计划。

### 火灾天气（`--fire`）

使用`--fire`标志可查看易发生野火的地区的火灾风险信息：
```bash
python3 ./scripts/get_weather.py "California" --fire
```

**输出内容包括：**
- 火灾风险等级（如有）
- 🔥 **红色警报**（若存在）
- 火灾天气预警
- 预警来源

**红色警报**表示火灾风险极高（低湿度 + 强风）
