# Bat

这是一个拥有翅膀的猫（cat）的克隆体。 ## 命令

- `help` - 帮助
- `run` - 运行
- `info` - 信息
- `status` - 状态

## 功能

- 搭载了 sharkdp/bat 的核心功能

## 使用方法

运行任何命令：`bat <命令> [参数]`
