# Oh My Bash

这是一个由社区驱动的框架，用于管理您的 Bash 配置，并提供自动更新功能，让您能够轻松获取社区发布的最新更新。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 查看信息
- `status` - 查看状态

## 特点

- 基于 `ohmybash/bash-themes` 的核心功能

## 使用方法

运行任何命令：`bash-themes <命令> [参数]`

## 示例

```bash
# Show help
bash-themes help

# Run
bash-themes run
```

## 工作原理

读取输入，通过内置逻辑进行处理，然后输出结构化结果。

## 提示

- 可以运行 `bash-themes help` 来查看所有命令的详细信息
- 不需要 API 密钥
- 支持离线使用
