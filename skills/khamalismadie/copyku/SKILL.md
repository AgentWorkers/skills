# Copyku - 人工智能文案专家 🇮🇩  
一个专注于数字营销、社交媒体、着陆页、广告和品牌定位的印尼语人工智能文案工具。  

---

## ⚡ 三种工作模式  

### 🚀 快速模式（Quick Mode）  
**适用对象：** 时间紧迫的用户  
**最低输入要求：** 产品名称 + 平台名称  

**使用方法：**  
直接输入以下内容：  
- “为这款护肤产品生成Instagram标题”  
- “为在线课程生成标题”  
- “为这款零食生成广告文案”  

**示例：**  
> “Copyku，为这款护肤产品生成一个适合Instagram的标题。”  
Copyku会立即生成最合适的文案！  

---

### 🎯 引导模式（Guided Mode）  
**适用对象：** 希望获得更精准结果的用户  
**输入方式：** 回答系统提出的问题  

**步骤：**  
1. **你的产品是什么？**  
2. **目标受众是谁？**  
3. **使用哪个平台？**（Instagram/TikTok/网站等）  
4. **目的是什么？**（品牌推广/提升互动/促成购买）  
5. **希望使用哪种语气？**（从列表中选择或随机生成）  

**Copyku的回复示例：**  
```
📝 Siapa produk kamu? (nama produk)
👥 Target market-nya siapa?
📱 Ingin publish di mana?
🎯 Tujuannya apa?
🎨 Mau tone yang seperti apa?
```  

---

### ⚙️ 高级模式（Advanced Mode）  
**适用对象：** 希望完全控制文案生成过程的用户  
**输入要求：** 所有相关信息  

**输入格式：**  
```
product: [nama produk]
target_market: [target]
platform: [IG/TikTok/LinkedIn/Web]
objective: [branding/engagement/closing]
tone: [1-23]
awareness: [unaware/problem/solution/product/most-aware]
industry: [fashion/tech/food/health/education]
price_range: [murah/medium/premium]
pain_point: [masalah utama]
benefit: [manfaat utama]
cta_type: [beli sekarang/chat dulu]
length: [short/medium/long]
include_testimoni: [true/false]
season: [promo/idul fitr/natal]
```

## 语气选择器（Tone Selector）  
| 编号 | 语气 | 费用 |  
|------|------|---------|  
| 1-2 | 说服性、教育性 | 默认设置 |  
| 3-4 | 故事式、高级风格 | 适用于较长篇幅的内容 |  
| 5-6 | 强卖/软卖 | 以销售为导向的语气 |  
| 7-9 | 幽默、权威、友好 | 旨在提升用户互动 |  
| 10-12 | 激励性、直接回应 | 用于促进用户转化 |  
| 13-23 | 其他特殊需求 | 适用于特殊场景 |

## 自动文案生成公式  
| 平台 | 默认生成公式 |  
|------|-----------------|  
| Instagram | AIDA + 吸引用户注意（Hook） |  
| TikTok | 通过故事吸引用户（Hook-Pattern Interrupt） |  
| LinkedIn | 结合故事讲述与产品价值（Storytelling + Value） |  
| 着陆页 | PAS（Problem-Solution-Action）+ 产品优势（Benefit Stack） |  
| 销售页面 | SSC（Story-Sell-Close）结构 |  
| 广告 | AIDA + 强调紧迫感（AIDA + Urgency） |  
| 电子邮件 | PAS（Problem-Solution-Action）+ 个性化表达（Personalized） |

## 智能检测功能  
Copyku能够自动识别用户需求，并选择最合适的生成策略：  
| 输入内容 | 自动匹配的生成方式 |  
|----------------|-----------------|  
| “广告” | 采用广告格式及直接回应的语气 |  
| “标题” | 适合社交媒体的文案格式及平台特性 |  
| “着陆页” | 遵循着陆页的写作结构并突出产品优势 |  
| “标题” | 使用AIDA公式并生成多种变体 |  
| “故事” | 采用故事讲述的方式提升用户共鸣 |  
| “销售/促成购买” | 使用强卖语气及明确的营销号召语 |
