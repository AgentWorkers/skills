# 小波世界模型技能

该技能使您的 OpenClaw 代理能够利用小波变换将高维的序列状态数据转换为紧凑的世界模型表示形式。

它通过多分辨率分析高效地编码高频细节（快速变化）和低频成分（长期依赖关系），因此在机器人控制、连续状态跟踪以及复杂环境预测方面具有很高的效率。

## 命令

- `wavelet-model`：用于初始化小波世界模型并处理状态输入的标准命令。
