# MoltFlow——潜在客户检测与CRM管理功能  

**功能概述：**  
- 从被监控的WhatsApp群组中检测购买意向信号；  
- 跟踪潜在客户在整个销售流程中的状态变化；  
- 执行批量操作；  
- 将数据导出至您的CRM系统。  

**实际应用场景：**  
- **房地产经纪人**：监控群组中包含“寻找房源”、“3居室”或“预算”等关键词的聊天记录，符合条件的客户会自动进入潜在客户管理系统。  
- **汽车经销商**：当潜在客户状态从“已联系”变为“符合条件”时，自动将其加入VIP跟进群组以发送试驾邀请。  
- **保险经纪人**：将本季度转化的潜在客户数据导出为CSV文件，便于导入CRM系统进行佣金追踪。  
- **婚礼策划师**：查看尚未联系的潜在客户信息，并按检测时间排序。  

**常用操作：**  
- 查看潜在客户列表；  
- 更新潜在客户状态；  
- 批量更新潜在客户信息；  
- 将潜在客户添加到指定群组；  
- 将潜在客户数据导出为CSV或JSON格式；  
- 检查潜在客户是否已回复消息；  
- 按状态筛选潜在客户。  

**使用前提：**  
1. **MOLTFLOW_API_KEY**：需在[MoltFlow控制台](https://molt.waiflow.app)的“设置”>“API密钥”中生成。  
2. 至少拥有一个已启用关键词检测功能的WhatsApp群组。  
3. 基础API地址：`https://apiv2.waiflow.app/api/v2`  

**所需API权限：**  
| 权限范围 | 访问权限 |  
|---------|---------|  
| `leads` | `读取/管理` |  
| `groups` | `读取` |  

**身份验证要求：**  
每个请求都必须包含有效的API密钥。  

**潜在客户检测机制：**  
1. 通过`Groups API`配置群组监控（详见[moltflow技能文档](…)；  
2. 设置`monitor_mode`为`"keywords"`，并指定如`"looking for"`、`"price"`、`"interested"`等关键词；  
3. 检测到匹配关键词时，MoltFlow会自动创建潜在客户记录；  
4. 潜在客户记录会显示为`new`状态，且触发关键词会被高亮显示；  
5. 可通过状态变更（`new` -> `contacted` -> `qualified` -> `converted`）跟踪客户进展。  

**API详细信息：**  
| 方法 | 端点 | 功能描述 |  
|--------|----------|-------------|  
| GET | `/leads` | 查看潜在客户列表（可按状态、群组筛选） |  
| GET | `/leads/{id}` | 查看单个潜在客户详情 |  
| PATCH | `/leads/{id}/status` | 更新潜在客户状态 |  
| GET | `/leads/{id}/reciprocity` | 检查潜在客户是否先发送消息（防垃圾信息） |  
| POST | `/leads/bulk/status` | 批量更新潜在客户状态 |  
| POST | `/leads/bulk/add-to-group` | 将潜在客户批量添加到指定群组 |  
| GET | `/leads/export/csv` | 将潜在客户数据导出为CSV（Pro+计划） |  
| GET | `/leads/export/json` | 将潜在客户数据导出为JSON（Pro+计划） |

完整的工作流程包括：检测潜在客户 -> 评估客户资格 -> 发送跟进消息。
