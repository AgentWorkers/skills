## **MoltFlow 管理技能**  
用于管理 MoltFlow 的认证、计费、API 密钥、使用情况跟踪及租户设置。  

### **实际应用场景**  
- **代理机构管理多个客户**：为每位客户创建受限范围的 API 密钥，仅允许他们发送消息或查看自己的群组信息。  
- **初创企业扩展业务**：查看当前套餐使用情况，提前预警是否接近发送量上限，以便及时升级套餐。  
- **合规专员**：查看本月每日使用明细，审核每个会话的实际发送消息数量。  

### **使用场景**  
- 需要登录 MoltFlow（包括登录、刷新访问令牌或使用魔法链接）。  
- 管理 API 密钥（创建、轮换、撤销）。  
- 查看订阅状态、套餐限制或使用情况。  
- 创建 Stripe 结账会话或计费门户链接。  

### **前提条件**  
- **MOLTFLOW_API_KEY**：大多数 API 端点都需要此密钥。可在 [MoltFlow 仪表板 > API 密钥](https://molt.waiflow.app/api-keys) 生成。  
- 认证端点（`/auth/*`）支持电子邮件/密码登录，初次登录无需 API 密钥。

### **提示**  
- **API 密钥安全**：API 密钥仅在创建时显示一次，建议将其存储在安全的管理工具中。  
- **令牌刷新**：访问令牌有效期为 30 分钟，可使用刷新端点获取新令牌而无需重新认证。  
- **魔法链接**：支持无密码登录，使用 `magic-link/request` 和 `magic-link/verify` 功能。  
- **套餐限制**：在使用 API 前，请使用 `GET /billing/subscription` 查看剩余发送量。  
- **权限范围**：仅使用工作流程所需的最低权限范围。
