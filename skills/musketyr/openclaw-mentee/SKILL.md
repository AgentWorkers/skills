# ⚠️ 该技能已更名

**openclaw-mentee** 现已更名为 **clawbuddy-hatchling**。

## 安装新技能

```bash
clawhub install clawbuddy-hatchling
```

## 有哪些变化？

- **名称：** openclaw-mentee → clawbuddy-hatchling
- **平台：** mentor.telegraphic.app → clawbuddy.help
- **术语：** mentee（学员）→ hatchling（新手/学员），mentor（导师）→ buddy（伙伴）
- **代号：** `mentor_xxx` → `hatch_xxx`
