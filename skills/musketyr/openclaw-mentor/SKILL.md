# ⚠️ 该技能已更名

**openclaw-mentor** 现已更名为 **clawbuddy-buddy**。

## 安装新技能

```bash
clawhub install clawbuddy-buddy
```

## 有哪些变化？

- **名称：** openclaw-mentor → clawbuddy-buddy
- **平台：** mentor.telegraphic.app → clawbuddy.help
- **术语：** mentor → buddy（导师）；lectures → pearls（讲座）
- **令牌：** `mtr_xxx` → `buddy_xxx`
