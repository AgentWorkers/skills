# 中国照明采购技能

## 说明
该技能帮助国际买家了解中国的照明制造业市场。预计到2026年，中国照明产业的收入将超过9800亿元人民币，占全球照明市场的60%以上。该技能基于最新的政府政策和行业报告，提供有关各地区照明产业集群、供应链结构及行业趋势的详细信息。涵盖的领域包括LED灯具和照明设备、户外照明、智能照明、汽车照明以及特殊用途照明（如植物生长灯、UV-C杀菌灯）。

## 主要功能
- **行业概况**：了解中国照明产业的规模、发展目标及关键政策举措（如能效提升、智能照明等）。
- **供应链结构**：全面掌握从LED芯片、驱动器、光学元件到制造环节以及全球销售渠道的完整产业链。
- **地区产业集群**：识别主要的照明制造基地（如中山古镇被誉为“照明之都”，深圳以LED产业闻名，常州则专注于汽车照明等）。
- **子行业洞察**：获取LED灯具、户外照明、智能照明、汽车照明及特殊用途照明等子行业的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通技巧、典型交货周期及付款条款等。

## 使用方法
您可以使用自然语言与该技能进行交互。例如：
- “2026年中国照明产业的整体状况如何？”
- “展示LED照明的供应链结构。”
- “哪些地区最适合采购智能照明产品？”
- “请介绍一下汽车照明产业的制造集群。”
- “如何评估LED路灯供应商的质量？”
- “在购买照明产品时，应关注哪些认证标准？”

## 数据来源
该技能的数据来源于：
- 工业和信息化部（MIIT）
- 中国照明行业协会（CALI）
- 中国国家统计局
- 行业研究出版物（数据更新至2026年第一季度）

## 实现方式
该技能的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有提供的信息均为行业层面的汇总数据，不包含具体工厂的联系方式。

## API参考
`do.py`文件中提供了以下Python函数，可用于程序化访问该技能的相关数据：
### `get_industry_overview() -> Dict`
返回中国照明产业的规模、发展目标及关键政策举措的概览。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回结果：产业规模、2026年的发展目标、主要驱动因素、出口价值等。
```
