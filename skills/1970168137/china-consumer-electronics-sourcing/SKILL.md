# 中国消费电子采购技能

## 描述
该技能帮助国际买家了解中国的消费电子制造市场。预计到2026年，中国消费电子市场的收入将超过15.2万亿元人民币。该技能基于最新的政府政策和行业报告，提供有关区域制造集群、供应链结构及行业趋势的可靠信息。涵盖的产品包括智能手机、可穿戴设备、音频设备、智能家居产品、AR/VR头显及相关配件。

## 主要功能
- **行业概况**：了解中国消费电子产业的规模、发展目标及关键政策举措（如“数字中国”计划、5G技术、人工智能等）。
- **供应链结构**：掌握从核心零部件（芯片、显示屏、电池）到组装及全球销售渠道的完整产业链。
- **区域制造集群**：识别不同产品类别的专门制造基地（例如：智能手机在珠江三角洲地区、笔记本电脑在长江三角洲地区、AR/VR设备在北京地区）。
- **子行业洞察**：获取关于智能手机、可穿戴设备、音频设备、智能家居和AR/VR等子行业的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通最佳实践、典型交货周期及付款条款等。

## 使用方法
您可以使用自然语言与该技能进行交互。例如：
- “2026年中国消费电子产业的整体状况如何？”
- “展示智能手机的供应链结构。”
- “哪些地区最适合采购TWS耳机？”
- “介绍一下智能家居设备的制造集群。”
- “如何评估智能家居设备的供应商？”
- “在采购消费电子产品时，应关注哪些认证要求？”

## 数据来源
该技能的数据来源于：
- 工业和信息化部（MIIT）
- 中国电子信息产业发展研究院
- 中国国家统计局
- 中国音频行业协会
- 行业研究出版物（更新至2026年第一季度）

## 实现方式
该技能的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有数据均为汇总后的行业信息，不包含单个工厂的联系方式。

## API参考
`do.py`文件中提供了以下Python函数，可用于程序化访问该技能的功能：

### `get_industry_overview() -> Dict`
返回中国消费电子产业的规模、发展目标及关键政策举措的概览。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回内容：产业规模、2026年的发展目标、关键驱动因素、出口价值等。
```
