# 中国家用电器采购技能

## 描述
该技能帮助国际买家了解中国的家用电器制造业格局。预计到2026年，中国家用电器行业的收入将超过2.8万亿元人民币。该技能基于最新的政府政策和行业报告，提供有关区域生产集群、供应链结构及行业趋势的可靠信息。涵盖的产品类型包括大型家电（冰箱、洗衣机、空调）、厨房电器（抽油烟机、烤箱、洗碗机）以及小型家电（吸尘器、空气净化器、电饭煲）。

## 主要功能
- **行业概况**：了解中国家用电器行业的规模、发展目标及关键政策举措（如绿色家电、智能家居）。
- **供应链结构**：掌握从原材料和核心零部件（压缩机、电机）到制造及全球销售渠道的完整产业链。
- **区域生产集群**：识别不同类型家电的专业制造基地（珠江三角洲以空调和小型家电为主，长江三角洲以洗衣机和厨房电器为主，山东以冰箱为主）。
- **子行业洞察**：获取关于主要子行业（大型家电、厨房电器、小型家电、供暖/通风设备）的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通最佳实践、典型交货周期及付款条款。

## 使用方法
您可以使用自然语言与该技能进行交互。例如：
- “2026年中国家用电器行业的整体状况如何？”
- “显示冰箱的供应链结构。”
- “哪些地区最适合采购空调？”
- “介绍一下厨房电器的生产集群。”
- “如何评估小型家电的供应商？”
- “在选购家用电器时，应该关注哪些认证标准？”

## 数据来源
该技能的数据来源于：
- 工业和信息化部（MIIT）
- 中国家用电器协会（CHEAA）
- 中国国家统计局
- 行业研究出版物（更新至2026年第一季度）

## 实现方式
该技能的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有数据均为针对整个生产集群的汇总信息，不包含单个工厂的联系方式。

## API参考
`do.py`文件中提供了以下Python函数，可用于程序化访问该技能的相关数据：

### `get_industry_overview() -> Dict`
返回中国家用电器行业的规模、发展目标及关键政策举措的概述。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回内容：行业规模、2026年的发展目标、主要驱动因素、出口价值等。
```
