# 中国运动服装及户外用品采购技能

## 描述
该技能帮助国际买家了解中国的运动服装和户外装备制造业市场。预计到2026年，该行业的收入将超过1.2万亿元人民币。中国是全球最大的生产国，为全球知名品牌以及快速发展的国内品牌提供产品。该技能基于最新的政府政策和行业报告，提供有关区域产业集群、供应链结构及行业趋势的可靠信息。涵盖的产品包括运动服装（leggings、jerseys、运动内衣）、户外服装（夹克、羽绒服）、鞋类（跑鞋、徒步靴）、户外装备（帐篷、背包、睡袋）以及相关配件。

## 主要功能
- **行业概况**：了解中国运动服装及户外用品行业的规模、发展目标及关键政策举措（如国家健身计划、冬季运动发展计划）。
- **供应链结构**：全面掌握从功能性面料、绝缘材料、零部件到制造及全球销售渠道的完整产业链。
- **区域产业集群**：识别主要的制造业基地（福建地区以鞋类为主，长江三角洲地区以户外服装和羽绒制品为主，珠江三角洲地区以运动服装为主）。
- **子行业洞察**：获取运动服装、户外服装、鞋类、户外装备及配件等子行业的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通最佳实践、典型交货周期及付款条款等。

## 使用方法
您可以使用自然语言与该技能进行交互。例如：
- “2026年中国运动服装行业的整体状况如何？”
- “展示跑鞋的供应链结构。”
- “哪些地区最适合采购羽绒服？”
- “介绍一下帐篷制造产业集群的情况。”
- “如何评估防水夹克的供应商？”
- “在选购户外装备时，应该关注哪些认证标准？”

## 数据来源
该技能的数据来源于：
- 工业和信息化部（MIIT）
- 中国体育用品联合会
- 中国纺织工业联合会
- 中国国家统计局
- 行业研究出版物（更新至2026年第一季度）

## 实现方式
该技能的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有数据均为行业层面的汇总信息，不包含具体工厂的联系方式。

## API参考
`do.py`文件中提供了以下Python函数，可用于程序化访问该技能的相关信息：

### `get_industry_overview() -> Dict`
返回中国运动服装及户外用品行业的规模、发展目标及关键政策举措的概览。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回内容：行业规模、2026年发展目标、主要驱动因素、出口价值等。
```
