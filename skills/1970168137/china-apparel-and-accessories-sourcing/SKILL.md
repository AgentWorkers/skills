# 中国服装及配饰采购技能

## 描述
该技能帮助国际买家了解中国的服装及配饰制造业市场。预计到2026年，该行业的收入将超过5.8万亿元人民币。该技能基于最新的政府政策和行业报告，提供有关区域产业集聚、供应链结构及行业趋势的可靠数据支持。涵盖的产品包括服装、鞋类、包袋、帽子、围巾、时尚配饰等。

## 主要功能
- **行业概况**：了解中国服装及配饰行业的规模、发展目标以及关键政策举措（数字化转型、可持续发展、品牌建设等）。
- **供应链结构**：全面掌握从原材料（纤维、面料、辅料）到制造及销售渠道（国内零售、跨境电子商务）的完整产业链。
- **区域产业集聚**：识别不同产品类别的专业制造基地（女装制造集中在广州，男装制造在宁波，运动服装制造在福建，配饰制造在义乌等）。
- **子行业洞察**：获取服装、鞋类、包袋/行李箱、配饰、内衣等关键子行业的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通最佳实践、典型交货周期及付款条款等。

## 使用方法
您可以通过自然语言与该技能进行交互。例如：
- “2026年中国服装行业的整体状况如何？”
- “展示一下服装行业的供应链结构。”
- “哪些地区最适合采购鞋类？”
- “介绍一下长江三角洲地区的服装制造集聚区。”
- “如何评估包袋和行李箱的供应商？”
- “在采购可持续服装时，应该关注哪些认证标准？”

## 数据来源
该技能的数据来源于：
- 工业和信息化部（MIIT）
- 中国纺织服装协会（CNTAC）
- 中国皮革行业协会
- 中国国家统计局
- 行业研究出版物（更新至2026年第一季度）

## 实现方式
该技能的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有数据均为区域级别的汇总信息，不包含个别工厂的联系方式。

## API参考
`do.py`文件中提供了以下Python函数，可用于程序化访问该技能的相关数据：

### `get_industry_overview() -> Dict`
返回中国服装及配饰行业的规模、发展目标及关键政策举措的概览。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回内容：行业规模、2026年的发展目标、关键驱动因素、出口价值等。
```
