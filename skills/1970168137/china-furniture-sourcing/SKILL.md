# 中国家具采购技巧

## 描述
本技巧帮助国际买家了解中国的家具制造业市场。预计到2026年，中国家具行业的收入将超过1.8万亿元人民币。该技巧基于最新的政府政策和行业报告，提供有关区域产业集群、供应链结构及行业趋势的可靠数据支持。涵盖的家具类型包括住宅家具（沙发、床、衣柜）、办公家具（桌子、椅子）、酒店家具、户外家具、定制家具和床垫。

## 主要功能
- **行业概况**：了解中国家具行业的规模、发展目标及关键政策举措（如绿色制造、智能家具）。
- **供应链结构**：掌握从原材料（木材、板材、五金件、软装饰材料）到制造及全球销售渠道的完整产业链。
- **区域产业集群**：识别不同类型家具的专业制造基地（珠江三角洲以生产沙发和住宅家具为主，浙江以生产办公椅和户外家具为主，四川以生产板材家具为主，江西以生产实木家具为主）。
- **子行业洞察**：获取住宅家具、办公家具、酒店家具、户外家具、定制家具和床垫等子行业的详细信息。
- **采购建议**：提供评估和选择供应商的实际指导，包括验证方法、沟通最佳实践、典型交货周期和付款条款等。

## 使用方法
您可以使用自然语言与本技巧进行交互。例如：
- “2026年中国家具行业的整体状况如何？”
- “请展示木制家具的供应链结构。”
- “哪些地区最适合采购办公椅？”
- “请介绍一下户外家具的制造产业集群。”
- “如何评估定制家具的供应商？”
- “在选择床垫时应该关注哪些认证标准？”

## 数据来源
本技巧的数据来源于：
- 工业和信息化部（MIIT）
- 中国国家家具协会（CNFA）
- 中国国家统计局
- 行业研究出版物（更新至2026年第一季度）

## 实现方式
该技巧的逻辑实现位于`do.py`文件中，该文件从`data.json`文件中读取结构化数据。所有数据均为行业层面的汇总信息，不包含单个工厂的联系方式。

## API 参考
`do.py`文件中提供了以下Python函数，可用于程序化访问：
### `get_industry_overview() -> Dict`
返回中国家具行业的规模、发展目标及关键政策举措的概览。

**示例：**
```python
from do import get_industry_overview
result = get_industry_overview()
# 返回内容：行业规模、2026年的发展目标、主要驱动因素、出口价值等。
```
