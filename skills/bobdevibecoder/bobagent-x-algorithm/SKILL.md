# X算法精通指南

本指南涵盖了关于X（Twitter）算法的所有关键信息，内容基于X的开源代码、病毒式帖子分析以及真实的用户互动数据。

## 简而言之——黄金法则

1. **前2小时至关重要**：此时间段内的回复和互动情况将直接影响帖子的传播范围。
2. **主帖中禁止使用外部链接**：X平台会对引导用户离开平台的链接进行惩罚。
3. **视频内容更具吸引力**：视频的互动率是文本的10倍，图片的互动率为2-3倍。
4. **务必回复所有帖子**：回复的权重高于点赞和转发。
5. **在目标受众活跃的时间段发布内容**：工作日的上午8点至下午2点是最佳时间。
6. **争议性话题能提升互动率**——但需谨慎选择讨论的话题。
7. **长篇内容更适合用主题帖形式发布**。

## 算法工作原理

### 四步流程

1. **候选帖子筛选**：从你关注的账户（约50%）及其类似账户中抽取约1,500条推文。
2. **排名**：机器学习模型评估你回复、转发、点赞或举报这些帖子的可能性（可能性最高的帖子会被优先展示）。
3. **过滤**：排除被屏蔽或静音的账户、网络内的不平衡情况，以及单一作者发布的过多帖子。
4. **内容推送**：将筛选后的帖子与广告混合推送，每天推送50亿次，每次推送时长约1.5秒。

### 互动权重等级

| 动作 | 权重 |
|--------|--------|
| 回复 | 最高 |
| 转发 | 高 |
| 引用推文 | 高（比普通帖子互动率高2倍） |
| 点赞 | 中等 |
| 收藏 | 中等 |
| 浏览量 | 低 |
| 举报 | 负面因素（会降低帖子传播范围） |

## 排名因素

### 1. 新鲜度
- 新内容优先展示
- 最初2-3小时的曝光率最高。

### 2. 互动速度
- 互动速度比总互动量更重要
- 30分钟内获得100个赞比24小时内获得500个赞更有效。

### 3. 账户可信度
- 经过验证的账户（高级账户）会获得更多优势。
- 跟随者与被关注者的比例也会影响排名。
- 被封禁或受到处罚的账户会影响排名。

### 4. 内容类型
- **视频**：互动率最高（10倍）。
- **图片**：互动率为2-3倍。
- **投票**：能引发更多回复。
- **主题帖**：总互动量更高。

### 5. 链接使用
- **使用链接会降低帖子的传播范围**：请将链接放在回复中，而非主帖中。
- 引用包含链接的推文比直接发布链接更有效。

## 会降低帖子传播范围的错误行为

### 会立即导致失败的行为
- 主帖中使用外部链接。
- 被举报或被屏蔽。
- 重复发布相同内容。
- 使用过多标签（超过2个）。

### 会逐渐降低传播效果的行为
- 发布内容不规律。
- 不回复评论。
- 发布的内容与你的领域不符。
- 使用过于正式或 corporate 的语气。

## 最佳发布策略

### 发布频率
| 目标 | 每天发布数量 |
|------|-----------|
| 最低 | 2-3条 |
| 增量阶段 | 5-10条 |
| 最高 | 15-30条 |

### 发布时间（根据目标受众时区）
- **最佳时间**：工作日的上午8点至下午2点。
- **较好时间**：工作日的下午4点至6点。
- **最差时间**：晚上11点至早上7点。

### 避免使用链接的策略
```
❌ Bad:  "Check out my article [link]"
✅ Good: "Here's what I learned (thread 🧵)"
         → Link in reply or final thread post
```

## 如何获得更多回复（最重要）

- **提出问题**：使用开放式问题或具有争议性的观点。
- **发表尖锐的观点**：引发讨论的观点更容易获得回复。
- **填空题**：例如“有史以来最好的电影是……”。
- **进行预测**：人们喜欢对预测结果发表不同意见。
- **分享个人故事**：例如“这种事情发生在我身上……”。

## X平台文章的写作技巧

### 有效的吸引读者方式

**引发不安或恐惧感（FOMO）**：
> “大家都在讨论X……而你还在犹豫是否错过了机会。”

**强调重大机会**：
> “这是我们一生中最大的机会。”

**表达惋惜或遗憾**：
> “[某个行业]要消亡了。这个AI工具能在几秒钟内完成[某项操作]。”

### 文章结构示例
```
1. HOOK (insecurity or opportunity)
2. WHAT IT IS (with social proof)
3. WHY MOST WON'T DO IT (address objections)
4. THE [X]-MINUTE GUIDE (step-by-step)
5. YOUR FIRST [N] WINS (immediate value)
6. THE COST (value comparison)
7. THE WINDOW (urgency)
8. CTA
```

### 文章格式建议
- 使用清晰的H2标题。
- 使用项目符号列表以便读者快速浏览。
- 突出显示关键短语。
- 为每个步骤提供时间预估。
- 可以复制并粘贴相关的命令或提示。

## 发布前的快速检查清单

- 文章长度是否少于280个字符？（如果超过，请分成多条主题帖发布）
- 第一行是否能够吸引读者的注意力？
- 你有理由回复这条帖子吗？（提出问题或发表尖锐的观点）
- 现在发布是否合适？
- 文章内容是否符合你的领域？
- 你在接下来的2小时内能够回复评论吗？

## 提升帖子传播效果的技巧

### 提高回复率的策略
- 关注大账号并开启通知，以便第一时间回复他们的帖子，这样他们的粉丝就会注意到你。
- 找到在你所在领域内具有高互动率的帖子，引用该帖子并补充自己的解释，同时提供有价值的内容。

### 个人化内容
每发布5-10条帖子，分享一些个人经历或感受。这有助于建立与读者的联系，从而提高互动率。
