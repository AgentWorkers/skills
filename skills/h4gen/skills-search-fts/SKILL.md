# 全局技能搜索

无需从头开始构建。该功能可让您立即访问包含24万多个预构建的代理技能、专用工具定义以及从互联网上收集的专家系统提示的庞大库。

## 为何使用此功能？

快速发现适用于任何任务的现有高质量功能。无论您需要一个专业的网页抓取工具、财务分析工具，还是特定领域的API集成，该搜索引擎都能找到最相关且经过实际测试的代理工具。

## 使用方法

通过关键词进行搜索，以找到您所需的功能。

```bash
python3 scripts/search.py "YOUR_QUERY"
```

## 示例查询：

- **基础查询**：`python3 scripts/search.py "浏览器自动化"`
- **多技能查询**：`python3 scripts/search.py "reddit AND 情感分析"`
- **特定查询**：`python3 scripts/search.py "stripe 支付集成"`

## 返回格式

搜索工具会返回一份按优先级排序的匹配技能列表，其中包括技能名称、来源和描述。

## 隐私与安全声明

> [!重要提示]
> **数据传输**：该功能会将您的搜索查询发送到外部API，以便在索引中进行高速全文搜索。
>
> **最佳实践**：
- **敏感数据**：避免在搜索查询中包含API密钥、密码或个人身份信息（PII）。
- **数据来源**：该服务由技能发布者管理，为24万多个公共代理功能提供了一个集中的平台。
