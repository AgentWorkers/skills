# 电影制片人场景技能

使用以下结构生成一个电影场景包：

## 输出结构（必须使用）

1. **概念简介**（1-2行）
2. **电影场景描述**（一个连贯的段落）
3. **镜头计划**（3-6个镜头，简洁描述）
4. **导演备注**（风格、节奏、表演重点）
5. **输出规格**（宽高比 + 分辨率）

## 核心提示模板

以以下模板为基础进行创作：

**角色扮演：** 一位世界级的好莱坞电影制片人。  
**创作一个** [类型] **风格的电影场景，营造出** [氛围/情绪] **。**  
**场景描述：** [详细的环境描述]。**  
**角色：** [在场人物及他们的行为]。**  
**视觉风格：** [电影中的常见元素或风格参考]。**  
**摄影：** [镜头类型、移动方式、镜头选择]。**  
**灯光：** [灯光风格]。**  
**环境：** [天气、特效、背景动态]。**  
**声音/音乐：** [配乐风格或音效设计]。**  
**质量要求：** 超现实主义风格、电影质感、高动态范围、电影颗粒感、专业色彩调色。**  
**输出格式：** [宽高比 + 分辨率]。**

## 质量规则

- 确保画面具体且适合拍摄（避免模糊的抽象描述）。  
- 保持场景内部的一致性（时间、光线、服装、天气）。  
- 明确指定摄影手法（镜头类型、移动方式、构图）。  
- 有意识地设计音效（配乐风格及场景中的环境音）。  
- 如果用户未提供某些细节，使用高质量的默认值，并标注为“已假设”。

## 快速默认值（如未提供）

- **类型：** 电影剧情片/惊悚片  
- **氛围/情绪：** 紧张、富有氛围感、情感深刻  
- **摄影：** 宽银幕40毫米镜头 + 缓慢的推镜动作 + 控制式的手持拍摄  
- **灯光：** 根据场景需要设计的实际灯光效果，带有柔和的体积感雾气  
- **环境：** 轻微下雨、雾气飘荡、人群轻微移动  
- **声音/音乐：** 混合管弦乐与低频合成器音效  
- **输出格式：** 2.39:1，4K  

## 可选模式

- **仅提供提示模式：** 仅返回电影场景描述段落。  
- **制作模式：** 返回完整的输出结构。  
- **故事板模式：** 将镜头计划扩展为8-12个镜头，并添加过渡效果。
