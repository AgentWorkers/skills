# 供应商谈判指南

在与任何供应商进行谈判之前，都应准备好一个结构化的框架。这项技能为你的AI代理提供了研究谈判策略、制定应对方案以及达成更优软件、服务和材料合同所需的工具。

## 功能概述

1. **谈判前准备** — 收集市场行情、竞争对手定价信息以及“最佳替代方案”（BATNA）分析。
2. **谈判地位评估** — 从8个维度评估自己的议价能力。
3. **制定应对方案** — 生成有数据支持的报价，并提供相应的理由。
4. **让步策略** — 规划好哪些条款可以妥协，哪些需要坚持，以及妥协的顺序。
5. **合同条款优化** — 标出不利条款并提出替代方案。
6. **谈判后评估** — 测量实际节省的成本与目标成本之间的差距。

## 谈判地位评估（每个维度评分1-5分）

| 因素 | 评分 | 备注 |
|--------|-------|-------|
| **转换成本** | ___ | 转向竞争对手的难度有多大？ |
| **可用替代方案** | ___ | 有多少可行的替代方案？ |
| **采购量影响力** | ___ | 你是否是他们的核心客户？ |
| **合同期限** | ___ | 他们的季度/年度是否即将结束？是否需要续签？ |
| **市场状况** | ___ | 他们的市场是否竞争激烈或已经趋于稳定？ |
| **合作关系时长** | ___ | 长期客户意味着更高的议价筹码。 |
| **公开定价** | ___ | 你能提供公开的定价信息或行业标准吗？ |
| **预算审批权** | ___ | 你是否有能力拒绝当前报价？ |

**总分：___ / 40分**
- 30-40分：谈判地位较强。可争取15-30%的降价。
- 20-29分：谈判地位中等。目标为8-15%的降价。
- 10-19分：谈判地位较弱。应重点关注合同条款而非价格。
- 低于10分：可以考虑捆绑采购或签订多年合同以增加谈判筹码。

## SaaS谈判技巧

以下策略对80%的SaaS供应商都有效：

**时机选择**：
- 在他们的第四季度进行谈判（对于日历年公司来说是12月，对于财年公司来说是3月）。
- 等到自动续签前两周再开始谈判——这样会让他们感到紧迫。
- 在他们的业务淡季开始谈判。

**定价策略**：
- “我们收到了[竞争对手]的报价X美元”——始终准备好一个真实的替代方案。
- “我们在这个类别的预算是X美元”——将预算视为一个限制条件，而非请求。
- “如果价格降至X美元，我们今天就可以签订两年合同”——给他们一个明确的折扣承诺。

**常见的SaaS折扣范围**：
| 策略 | 典型折扣 |
|--------|-----------------|
| 年度预付费（相比按月付费） | 15-20% |
| 多年合同（2年） | 20-30% |
| 多年合同（3年） | 25-40% |
| 采购量增加带来的折扣 | 10-25% |
| 案例研究/使用公司标志的权利 | 5-15% |
| 初创企业/小型企业的定价优惠 | 20-50% |
| 季末谈判 | 10-30% |

**需要谈判的条款（不仅仅是价格）**：
- 付款方式：选择Net-60或Net-90（而非Net-30）。
- 自动续签：要求提前60天通知，而非30天。
- 价格涨幅上限：每年最多5%。
- 便捷终止条款：规定30天的提前通知期限。
- 服务水平协议（SLA）中的退款条款：每小时的停机时间可获10%的退款。
- 数据迁移：要求在合同终止后30天内完成数据导出。

## 服务与材料谈判

**对于专业服务**：
- 要求提供综合报价，而非按角色计价的单价。
- 为各项费用设定一个最高限额。
- 对于明确划分的阶段，争取固定价格。
- 将15-20%的付款与交付成果挂钩。
**对于实物商品/材料**：
- 至少获取三家供应商的报价。
- 协商FOB（离岸交货）条款（谁来承担运费）。
- 要求以书面形式提供批量采购的折扣方案。
- 确定6-12个月的固定价格，并设定价格涨幅上限。
- 付款方式：50%在下单时支付，40%在交货时支付，10%在验收时支付（而非全额预付）。

## 年度供应商评估计划

| 月份 | 行动 |
|-------|--------|
| 1月 | 列出所有供应商的合同、续签日期和年度支出金额。 |
| 2-3月 | 根据支出金额确定前10大供应商，并开始调研。 |
| 4-5月 | 开始即将到期的合同续签谈判。 |
| 6月 | 进行年中供应商整合评估。 |
| 9-10月 | 第四季度的谈判窗口期（此时可以获得最佳折扣）。 |
| 11月 | 在预算冻结前完成多年合同的谈判。 |
| 12月 | 审计实际节省的成本与目标成本之间的差距。 |

**目标：在整个供应商组合中实现平均12-18%的降价。**
