# 代理运维运行手册

本运行手册用于生成适用于生产环境的AI代理部署方案，涵盖了部署前的检查清单、从“影子模式”到“监督模式”再到“自主模式”的部署流程、监控仪表盘、回滚程序、成本管理以及事件响应模板等内容。

## 使用场景
- 部署AI代理到生产环境
- 为代理系统构建监控和警报机制
- 为自主工作流程创建回滚程序
- 估算和控制代理的运营成本

## 指导步骤

当用户请求代理运维运行手册或部署计划时，请按照以下步骤操作：
1. 询问用户要部署的代理功能（例如：支持、销售、文档处理等）。
2. 了解用户的风险承受能力（保守型、中等型或激进型部署策略）。
3. 生成完整的运行手册，内容包括：
   - 适用于该代理功能的部署前检查清单
   - 包含指标和关键节点的三阶段部署计划
   - 监控警报设置（临界值和警告阈值）
   - 回滚程序（分为三个级别：快速回滚、部分功能回滚、完全回滚）
   - 根据用户预期使用量估算的成本
   - 90天的实施时间表
   - 事件响应模板

4. 明确具体的指标目标：
   - 准确率：高于90%
   - 错误率：低于2%
   - 各功能任务的成本基准
   - 人工升级率：5-15%

5. 标注用户所在行业特有的风险因素（如合规性要求、个人身份信息保护（PII）要求、财务数据准确性等）。

**输出格式**：Markdown格式的文档，便于与工程团队和运维团队共享。
