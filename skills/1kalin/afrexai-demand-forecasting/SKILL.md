# 需求预测框架

通过多种方法构建准确的需求预测模型，结合统计模型和市场情报以获得可操作的预测结果。

## 使用场景
- 季度/年度需求规划
- 新产品发布预测
- 库存优化
- 生产能力规划决策
- 预算周期准备

## 预测方法

### 1. 时间序列分析
适用于：具有24个月以上历史数据的老产品。

```
Decompose into: Trend + Seasonality + Cyclical + Residual

Moving Average (3-month):
  Forecast = (Month_n + Month_n-1 + Month_n-2) / 3

Weighted Moving Average:
  Forecast = (0.5 × Month_n) + (0.3 × Month_n-1) + (0.2 × Month_n-2)

Exponential Smoothing (α = 0.3):
  Forecast_t+1 = α × Actual_t + (1-α) × Forecast_t
```

### 2. 因果/回归模型
适用于：需求受外部因素影响的产品。

需要建模的关键驱动因素：
- **价格弹性**：价格每变化1%，需求变化百分比
- **营销支出**：滞后效应（通常为2-6周）
- **季节性指数**：月度系数与年平均值的对比
- **经济指标**：GDP增长、消费者信心、行业采购经理指数（PMI）
- **竞争对手行动**：新进入者、价格变动、促销活动

```
Demand = β₀ + β₁(Price) + β₂(Marketing) + β₃(Season) + β₄(Economic) + ε
```

### 3. 判断性/定性预测
适用于：新产品、市场突发变化或数据有限的情况。

方法：
- **德尔菲法**：多轮专家评估，匿名进行，逐步趋同预测结果
- **销售团队综合预测**：基于地区代表的自下而上预测（需调整15-20%的乐观程度）
- **市场调研**：基于调查的购买意愿（转化率为30-40%）
- **类比预测**：将新产品与类似产品的销售曲线进行对比

### 4. 混合预测（推荐方法）
使用加权平均值结合多种预测方法：

| 方法 | 成熟产品权重 | 新产品权重 |
|--------|------------------------|---------------------|
| 时间序列 | 50% | 10% |
| 因果模型 | 30% | 20% |
| 判断性预测 | 20% | 70% |

## 预测准确性指标

| 指标 | 公式 | 目标值 |
|--------|---------|--------|
| 平均绝对百分比误差（MAPE） | (实际值 - 预测值) / 实际值 × 100 | <15% |
| 偏差 | (预测值 - 实际值) 的总和 / 数据量 | 接近0 |
| 跟踪信号 | 累计误差 / 平均绝对偏差（MAD） | -4至+4 |
| 加权MAPE | 考虑收入影响的MAPE | 主要产品类别的MAPE <10% |

## 需求规划流程

### 每月流程
1. **第1周**：生成统计预测（自动运行模型）
2. **第2周**：整合市场情报（销售数据、竞争对手信息）
3. **第3周**：召开共识会议，协调销售、营销、运营和财务部门
4. **第4周**：最终确定预测结果，并与供应链沟通，同时与历史预测进行对比

## 安全库存计算

```
Safety Stock = Z × σ_demand × √(Lead Time)

Where:
  Z = Service level factor (95% = 1.65, 98% = 2.05, 99% = 2.33)
  σ_demand = Standard deviation of demand
  Lead Time = In same units as demand period
```

## 场景规划

对于每个预测结果，生成三种可能的情景：

| 情景 | 发生概率 | 假设条件 |
|----------|-------------|-------------|
| 熊市 | 20% | 需求下降15%至25%（相对于基准预测）；经济衰退、市场萎缩、竞争对手干扰 |
| 基准情景 | 60% | 基于历史趋势和已知订单情况；最可能的结果 |
| 牛市 | 20% | 需求上升15%至25%（相对于基准预测）；市场扩张、产品受欢迎 |

## 预测中的警示信号
- [ ] MAPE持续超过20% — 需重新训练预测模型
- [ ] 持续的正偏差 — 可能是销售团队故意低估需求
- [ ] 持续的负偏差 — 过度乐观；检查激励机制
- [ ] 跟踪信号超出±4的范围内 — 存在系统误差，需查明原因
- [ ] 预测结果长期不变 — 可能是简单复制数据造成的
- [ ] 没有外部数据输入 — 仅依赖统计预测可能导致对市场变化的忽视

| 建筑行业 | 20-35% | 12-24个月 | 执照申请、经济周期 |

## 更精确预测的ROI

对于年收入1000万美元的公司：
- **MAPE降低5%** → 可节省20万至50万美元的库存成本
- **减少缺货情况** → 提高2-5%的收入（节省20万至50万美元）
- **降低加急配送成本** → 节省5万至15万美元
- **提高产能利用率** → 降低3-8%的运营成本

**总体效益：通过MAPE降低5个百分点，每年可节省45万至115万美元。**
