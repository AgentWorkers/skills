# 并购尽职调查 —— 基于人工智能的评估工具

您可以使用该工具对任何收购目标进行结构化的尽职调查分析，涵盖财务、运营、技术、法律、文化以及人工智能/自动化准备程度等多个方面。

## 使用场景
- 评估收购目标
- 准备卖方尽职调查材料
- 评估合并伙伴
- 在收到初步投资意向书（LOI）后进行深入调查
- 应投资者或董事会的要求进行尽职调查

## 使用方法
请告知您的代理：“对[公司名称]进行并购尽职调查”或“为收购[目标公司]准备尽职调查材料”。

代理将逐一检查上述各个维度，并生成一份包含风险提示、交易障碍及是否继续进行建议的评估报告。

## 尽职调查框架

### 1. 财务状况（权重：30%）

**收入质量评分（0-100分）**
| 评估因素 | 权重 | 检查标准 |
|--------|--------|-------|
| 收入集中度 | 25% | 最大客户占比低于15% = 绿色；高于30% = 红色 |
| 收入来源（经常性 vs 一次性） | 25% | 经常性收入占比超过70% = 绿色；低于40% = 红色 |
| 收入增长趋势 | 20% | 三年复合年增长率（CAGR）超过15% = 绿色；下降 = 红色 |
| 毛利率 | 15% | SaaS业务毛利率超过60%；服务业务毛利率超过40% = 绿色 |
| 现金转化率 | 15% | 经营现金流占EBITDA的比例超过80% = 绿色 |

**财务风险提示**
- 收入在交付前就被确认
- 关联方交易占比超过收入的5%
- 流动资本逐季度恶化
- 客户获取成本（CAC）回收期超过18个月
- 预订量增加但递延收入减少
- 资产负债表外的负债或经营性租赁隐藏债务

**估值合理性检查**
| 指标 | 合理范围 | 危险范围 |
|--------|---------------|-------------|
| 市值/收入（EV/Revenue） | SaaS业务为3-8倍；服务业务为1-3倍 | 超过15倍（除非公司有高速增长） |
| 市值/EBITDA | 10-20倍 | 超过30倍 |
| 价格/自由现金流（Price/FCF） | 15-25倍 | 超过40倍或自由现金流为负 |
| 净债务/EBITDA | 小于2倍 | 超过4倍 |

### 2. 技术与产品（权重：20%）

**技术栈评估**
- 架构类型（单体应用、微服务或无服务器架构）
- 技术债务评分（部署频率、变更失败率、平均修复时间）
- 安全性（符合SOC 2、ISO 27001标准；最近是否进行过渗透测试）
- 可扩展性（能否在无需重写代码的情况下承受10倍的负载）
- 人工智能/自动化准备程度（是否以API为核心；具备自动化潜力）
- 知识产权（IP）归属情况（所有贡献者的IP分配是否清晰）

**产品与市场匹配度**
- 客户留存率（Net Revenue Retention）超过110% = 产品与市场匹配良好
- 年度Logo更换次数低于5次 = 产品健康
- 客户净推荐值（NPS）超过40分 = 优秀；超过60分 = 非常优秀
- 自然增长占比 = 真实需求的表现

### 3. 法律与合规性（权重：15%）

**法律检查清单**
- 是否存在未决或潜在的法律诉讼
- 是否存在知识产权侵权风险
- 数据隐私合规性（GDPR、CCPA、HIPAA等法规）
- 雇佣法规合规性（例如合同中对员工分类的错误）
- 环境责任问题
- 税务合规性和审计历史
- 关键合同中的控制权变更条款
- 禁止竞争/禁止招揽条款的强制执行性

**监管风险评分**
- 是否存在特定行业的监管要求
- 合规成本占收入的百分比
- 可能影响运营的待定法规变化

### 4. 运营效率（权重：15%）

**运营指标**
| 评估项 | 绿色 | 黄色 | 红色 |
|------|-------|--------|-----|
| 员工流动率 | 低于15% | 15-25% | 高于25% |
| 关键岗位依赖性 | 员工分布均匀 | 有部分岗位依赖特定人员 | 存在单一故障点 |
| 流程文档化程度 | 完整 | 部分缺失 | 依赖口头知识 |
| 供应商集中度 | 供应商来源多样化 | 仅有2-3个关键供应商 | 依赖单一供应商 |

### 5. 文化与人才（权重：10%）

**整合风险评估**
- 领导层在愿景和价值观上的共识
- 薪酬结构的兼容性
- 远程/混合办公/传统办公室文化的适应性
- 决策方式（扁平化 vs 层级化）
- 通过Glassdoor等平台收集的员工反馈
- 关键员工的流失风险

**员工留存激励方案**
| 职位层级 | 留任期限 | 常见激励方案 |
|------------|-----------------|-----------------|
| 高层管理人员 | 12-24个月 | 基本工资的1-2倍 + 股权激励 |
| 副总经理/总监 | 12-18个月 | 基本工资的0.5-1倍 + 股权激励 |
| 关键核心员工 | 6-12个月 | 一次性奖金 + 股权激励 |

### 6. 人工智能与自动化准备程度（权重：10%）

**人工智能/自动化潜力**
- 数据的质量和可访问性（是否结构化、标注清晰、可通过API访问）
- 流程自动化潜力（哪些工作流程可以通过自动化完成）
- 团队中的人工智能/机器学习人才（如ML工程师、数据科学家等）
- 现有的AI/ML模型或知识产权

**收购后的自动化提升预期**
| 部门 | 自动化潜力 | 年度节省成本 |
|------------|---------------------|----------------|
| 客户服务 | 40-60%的工单处理效率提升 | 12万至40万美元 |
| 销售运营 | 30-50%的流程自动化 | 8万至20万美元 |
| 财务/会计 | 50-70%的对账工作自动化 | 6万至15万美元 |
| 人力资源/招聘 | 40-60%的筛选效率提升 | 5万至12万美元 |
| 法律/合规 | 30-40%的合同审核效率提升 | 4万至10万美元 |

## 评分与输出

### 总体交易评分（0-100分）
- 80-100分：收购条件优秀，可继续推进至初步投资意向书（LOI）或最终协议
- 60-79分：存在条件性风险，需解决特定问题后再继续
- 40-59分：存在显著风险，需先解决这些问题
- 0-39分：存在重大障碍，建议放弃交易

### 交易终止的触发条件（无论评分如何均自动终止）
- 未公开的重大法律诉讼
- 收入欺诈或需要重新编制财务报表的风险
- 占收入超过40%的关键客户没有签订合同
- 无法解决的知识产权争议
- 可能导致业务中断的监管行动
- 关键创始人或员工离职且没有离职补偿协议

## 2026年行业基准数据

**并购市场现状**
- 平均SaaS公司的收购估值倍数：6-10倍（2021年为15-20倍）
- 尽职调查周期：60-90天（借助人工智能工具可缩短）
- 并购失败率：70-90%的并购项目未能实现预期价值
- 失败的主要原因：文化不匹配和整合计划不周
- 具备人工智能能力的公司相比非人工智能公司溢价15-25%

**2026年的变化**
- 人工智能/自动化准备程度成为核心的估值因素
- 数据资产被单独评估（专有培训数据具有竞争优势）
- 按用户计价的SaaS收入下降，更倾向于按使用量计价的模式
- 远程办公模式的公司更容易整合（无需合并办公室）
- 监管法规日益复杂（如欧盟的AI法案、各州的隐私法规）
