# 广告运营与跨渠道广告代理

这是一个为AI代理设计的自动化广告运营框架，支持在Google Ads、Meta、LinkedIn、TikTok以及程序化广告平台上管理广告活动。

## 该技能的功能

该技能可将您的AI代理转变为一名广告运营经理，使其能够无需使用任何控制面板，即可规划、审核、优化广告活动并生成报告。

## 功能特性

### 广告活动架构
- **渠道选择矩阵**：根据6个因素（CPL范围、用户意图、受众精准度、创意复杂性、最低可行预算、响应时间）对8个渠道（Google搜索、展示广告、Meta、Instagram、LinkedIn、TikTok、程序化广告、YouTube）进行评分。
- **预算分配规则**：遵循70/20/10的分配原则：70%预算用于表现稳定的渠道，20%用于具有增长潜力的渠道，10%用于实验性渠道。每周根据CPA（每次点击成本）的趋势重新调整预算分配。
- **广告活动命名规范**：所有渠道的广告活动名称统一采用 `{品牌}_{渠道}_{目标}_{受众}_{地区}_{日期}` 的格式，以便于数据报告的清晰性。

### 绩效审核（每周进行）
1. **支出效率**：标记CPA超过目标2倍或ROAS（投资回报率）低于1.5倍的广告活动。
2. **预算执行情况**：如果某个渠道的支出超出或低于周预算计划的110%，会发出警报。
3. **创意疲劳**：如果广告的点击率在14天内下降超过20%，会发出警报。
4. **受众重叠**：识别跨渠道的受众冲突（例如Meta和Google的重定向广告相互竞争）。
5. **着陆页匹配度**：检查不同广告与着陆页的匹配情况，如果跳出率超过65%，会发出警报。

### 优化策略
| 优化信号 | 应对措施 | 时间表 |
|--------|--------|----------|
| CPA上升且CTR稳定 | 受众疲劳——更新目标受众 | 48小时内 |
| CPA上升但CTR下降 | 创意疲劳——推出新创意 | 24小时内 |
| CTR高但转化率低 | 着陆页不匹配——进行A/B测试 | 72小时内 |
| 印象份额低 | 预算上限或出价过低——增加预算或调整出价策略 | 当天 |
| 单一渠道主导ROAS | 每周将预算增加20%，直到达到CPA上限 | 每周进行一次 |

### 7种会浪费预算的广告运营错误

1. **在多个渠道使用相同的受众群体**：跨平台的受众冲突会导致CPM（每次展示成本）上升。应针对每个渠道的不同销售阶段细分受众。
2. **忽视展示频率限制**：同一广告重复展示15次以上不仅无法提升品牌知名度，反而会引发用户反感。建议针对潜在客户每周展示3-5次。
3. **只关注点击率而忽略转化效果**：点击率只是表面指标，应关注每次转化带来的成本（CPA或每次机会成本）。
4. **缺乏创意测试机制**：仅发布一个广告并“观察效果”是不够的。应每周测试3-5个创意版本，并淘汰表现不佳的广告。
5. **凭直觉分配预算**：仅凭感觉分配预算是不科学的。应根据每个渠道的CPA与交易价值的比率来分配预算。
6. **忽略广告效果的归因周期**：LinkedIn的广告效果影响周期为90天，而Google为30天。直接比较不同渠道的ROAS会误导决策。
7. **大规模广告投放时依赖手动管理出价**：如果手动管理超过20个广告活动，会浪费15-30%的效率。应采用自动化工具或让AI代理来处理。
