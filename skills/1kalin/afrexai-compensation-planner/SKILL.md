# 薪酬与薪资基准规划工具

本工具旨在帮助您构建基于数据的薪酬体系，以吸引优秀人才的同时避免过度支付。涵盖基本薪资范围、股权/奖金框架、地区差异以及总薪酬构成。

## 使用场景
- 为任何职位制定或调整薪资范围
- 为招聘活动准备市场薪资数据
- 进行年度薪酬评估
- 设计股权/奖金/佣金结构
- 与竞争对手进行对比以降低员工流失率

## 工作流程

在需要制定薪酬计划时，请按照以下步骤操作：

### 1. 职位架构
定义职位等级及薪资范围：

| 等级 | 职位名称 | 基本薪资（美元） | 股权比例 | 奖金目标 |
|-------|--------------|-----------------|----------|--------------|
| L1 | 助理/初级 | 4.5万–7万美元 | 0–0.01% | 0–5% |
| L2 | 中级 | 7万–11万美元 | 0.01–0.05% | 5–10% |
| L3 | 高级 | 11万–16万美元 | 0.05–0.15% | 10–15% |
| L4 | 主管/团队负责人 | 15万–21万美元 | 0.1–0.3% | 15–20% |
| L5 | 经理/总监 | 19万–28万美元 | 0.2–0.5% | 20–30% |
| L6 | 副总裁/高管 | 25万–40万美元以上 | 0.5–2%以上 | 30–50%以上 |

### 2. 地区差异
根据劳动力成本调整薪资：（非生活成本）

| 地区 | 薪资倍数 |  
|------|---------|  
| 第一级 | 旧金山湾区、纽约市、伦敦 | 1.0倍（基准） |
| 第二级 | 西雅图、波士顿、洛杉矶、芝加哥 | 0.90–0.95倍 |
| 第三级 | 奥斯汀、丹佛、曼彻斯特、柏林 | 0.80–0.85倍 |
| 第四级 | 美国/英国的偏远地区 | 0.70–0.80倍 |
| 第五级 | 东欧、拉丁美洲、东南亚 | 0.40–0.60倍 |

### 3. 总薪酬构成
分解总薪酬：

**现金薪酬**
- 基本薪资：总薪酬的60–80%（因职位等级而异）
- 绩效奖金：基本薪资的5–30%
- 佣金（销售岗位）：总薪酬的40–60%

**股权薪酬**
- 初创公司（Pre-Series B阶段）：根据职位等级，0.01%–2%，4年锁定期，1年解锁
- 成长期公司：限制性股票单位（RSUs），比例较低但金额较高
- 上市公司：限制性股票单位每年更新

**福利与补贴**（通常占基本薪资的20–35%）
- 健康保险：雇主每年为每位员工支付6,000–24,000美元
- 401(k)计划/养老金匹配：薪资的3–6%
- 休假时间：15–25天（美国），25–33天（英国/欧盟法定标准+公司额外福利）
- 学习预算：每年1,000–5,000美元
- 远程工作补贴：每月100–250美元
- 产假：12–26周（具有竞争力）

### 4. 薪酬审计
每季度进行以下检查：
1. **职位薪资比率**：实际薪资 ÷ 薪资范围的中位数。目标范围：0.90–1.10
2. **性别薪资差距**：比较同一级别内男女员工的薪资中位数
3. **新员工薪资是否高于老员工**：如果新员工薪资过高，需通过保留政策进行调整
4. **薪资比率超过中位数的员工比例**：如果超过30%，需引起注意

### 5. 年度评估周期
| 月份 | 工作内容 |
|-------|--------|
| 1月 | 更新市场数据（参考Levels.fyi、Glassdoor、Radford、Mercer） |
| 2月 | 经理培训会议 |
| 3月 | 预算分配（通常占薪资总额的3–5%用于绩效奖金） |
| 4月 | 公布调整方案及生效日期 |
| 7月 | 更新年中股权奖励 |
| 10月 | 准备下一年的薪酬预算提案 |

### 6. 发放薪资前的核对清单
在提供薪资前，请完成以下步骤：
- [ ] 查阅至少3个数据来源（Levels.fyi、Glassdoor、Payscale、LinkedIn Salary）
- [ ] 确认所在地区的薪资倍数
- [ ] 计算总薪酬（基本薪资 + 奖金 + 年度股权收益 + 福利）
- [ ] 与同级别内部员工的薪资进行比较（误差范围±10%）
- [ ] 如薪资高于薪资范围中位数，需提供合理解释 |
- [ ] 征求招聘经理及财务/人力资源部门的批准

### 7. 员工保留风险评分
| 评分因素 | 权重 | 分数（1–5） |
|--------|--------|-------------|
| 薪资低于市场水平（低于市场水平10%以上） | 25% |
| 自上次加薪以来时间过长（超过18个月） | 20% |
| 员工离职风险信号（LinkedIn活跃度低） | 20% |
| 职位重要/难以替代 | 20% |
| 工作年限超过3年未晋升 | 15% |

**评分 > 3.5**：需立即讨论员工保留问题
**评分 2.5–3.5**：纳入下一次评估周期，优先处理
**评分 < 2.5**：每季度进行监控

### 8. 佣金与销售薪酬
对于收入相关的职位，设计目标收入（OTE）：
- **基本薪资与佣金比例**：50:50（销售顾问），60:40（销售经理），70:30（销售主管）
- **激励机制**：超额完成目标时佣金增加1.5–3倍
- **惩罚机制**：未完成目标时佣金减少50%
- **退款政策**：对90天内流失的客户定义退款规则
- **短期激励**：针对关键行动提供短期奖励（500–5,000美元）

## 需要跟踪的关键指标
- **薪资接受率**：目标应超过85%（低于此比例可能表示薪资竞争力不足）
- **员工流失率**：目标应低于10%（高于此比例可能表明存在保留问题）
- **招聘周期**：如果招聘周期延长，可能表明薪资竞争力有问题
**招聘成本**：包括招聘费用、签约奖金和搬迁费用
- **员工收入**：与行业平均水平进行对比（SaaS行业20万–40万美元，服务行业15万–25万美元）

## 数据来源（2026年）
- Levels.fyi：适合技术岗位，数据真实可靠
- Glassdoor：覆盖范围广，数据来自员工自报
- Payscale：专注于小型企业
- Radford（Aon）：企业级数据，需付费
- Mercer：全球薪酬数据，需付费
- LinkedIn Salary Insights：提供特定职位的薪资范围
- BLS职业就业统计：政府官方数据
