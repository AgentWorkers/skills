# 人才招聘框架

这是一个端到端的招聘系统，涵盖了从员工规划、offer谈判到90天入职的全过程。该系统专为员工人数在15至500人之间的公司设计，旨在帮助公司快速招聘到合适的人才，同时避免在招聘不合适的人选上浪费资金。

## 框架涵盖的内容

### 1. 招聘流程架构
- **从需求到offer的流程**：包含7个阶段，并设有相应的转化率基准（申请→筛选25%，筛选→面试40%，面试→offer 60%，offer→接受85%）
- **不同职位的招聘周期目标**：工程岗位45天，销售岗位35天，高管岗位75天，支持岗位25天
- **招聘成本基准**：平均为4,700美元（根据SHRM 2025年的数据），但会根据职位级别有所不同（最高可达平均成本的2-10倍）
- **招聘渠道的效率**：员工推荐（招聘成功率45%，平均耗时25天）> LinkedIn（12%，42天）> 招聘网站（5%，51天）

### 2. 职位架构与等级划分

### 3. 招聘策略矩阵
| 招聘渠道 | 适合的职位 | 成本 | 首次招聘所需时间 |
|---------|----------|------|-------------------|
| 员工推荐 | 所有职位 | 1,000美元至5,000美元的奖金 | 2-3周 |
| LinkedIn招聘专员 | 中高级职位及高管 | 每个职位每年8,000美元至12,000美元 | 3-6周 |
| 招聘网站（Indeed/Glassdoor） | 初级至中级职位 | 每条招聘信息200美元至500美元 | 4-8周 |

| 大学招聘项目 | 初级职位 | 每个项目5,000美元至15,000美元 | 2-6个月 |
| 通过承包商招聘 | 降低招聘风险 | 每小时1.3-1.5美元 | 1-3个月 |

### 4. 面试评分系统
根据以下维度对候选人进行1-5分的评分：
- **技术能力**（权重：30%）——他们能否胜任工作？
- **解决问题的能力**（20%）——他们的思维方式如何？
- **沟通能力**（15%）——他们能否清晰地表达自己的想法？
- **对公司的贡献**（15%）——他们能为团队带来什么？
- **个人发展潜力**（10%）——他们的职业发展方向如何？
- **职位特定要求**（10%）——领域专业知识、工具使用能力、证书持有情况

**评分标准**：4.0分及以上 = 非常合适的人选；3.5-3.9分 = 适合的人选；3.0-3.4分 = 需进一步讨论；低于3.0分 = 不适合的人选

### 5. Offer制定流程
```
Base salary: Market 50th percentile (conservative) or 65th (competitive)
Equity/options: 0.01-0.5% for IC, 0.5-2% for directors, 1-5% for VP+
Signing bonus: 10-20% of base for competitive situations
Benefits value: Add 25-35% to base for total comp messaging

Counter-offer response framework:
- Match within 10% → counter immediately
- 10-20% gap → add equity/signing bonus, don't inflate base
- >20% gap → let them go (they'll leave in 6 months anyway)
```

### 6. 多元化与包容性招聘
- 面试时隐藏候选人信息（姓名、学校、照片）
- 采用结构化的面试流程（相同的问题、相同的顺序、相同的评分标准）
- 面试小组成员来自不同的背景（至少2个不同的背景）
- 优先从代表性不足的群体中招聘（招聘目标为30%以上）
- 在每个招聘阶段跟踪不同人群的转化率

### 7. 90天入职检查点
| 时间点 | 检查内容 | 负责人 |
|-----|-----------|-------|
| 0天 | 确保新员工在入职前准备好所需的设备和访问权限 | IT/HR部门 |
| 1天 | 迎新活动、团队介绍、分配导师 | 经理 |
| 7天 | 完成第一个小项目 | 新员工 |
| 14天 | 一对一沟通，确认职位职责 | 经理 |
| 30天 | 30天评估——双方反馈 | 经理 |
| 45天 | 完成第一个有意义的项目 | 新员工 |
| 60天 | 开始跨团队合作 | 新员工 |
| 90天 | 全面评估——决定是否继续留任/调整/离职 | 经理 + HR部门 |

### 8. 招聘指标仪表盘
每月跟踪以下指标：
- **不同职位的招聘周期**（目标：平均<45天）
- **招聘成本**（目标：初级职位<5,000美元，高管职位<15,000美元）
- **招聘质量**（90天绩效评估、180天留存率）
- **offer接受率**（目标：>85%）
- **招聘渠道的效率**（每个渠道的招聘数量、成本）
- **招聘流程的效率**（每个阶段的候选人数量、转化率）
- **招聘团队的多样性**（目标：反映市场人口结构）

### 9. 雇主品牌评估标准
- [ ] Glassdoor评分>4.0（在48小时内回复所有评价）
- [ ] 职位页面上有真实员工的故事（非库存照片）
- [ ] 薪资透明（至少在职位描述中列出薪资范围）
- [ ] 面试流程公开透明 |
- [ ] 在LinkedIn上定期发布团队动态 |
- [ ] 发放候选人满意度调查（包括被拒绝的候选人）

### 10. 常见的招聘错误
1. **只关注文化“契合度”而忽视文化“多样性”——“契合度”意味着员工背景相似，“多样性”意味着思维方式的多样性**
2. **缺乏结构化的面试**——非结构化的面试预测员工表现的准确率为14%，而结构化面试的准确率为51%
3. **不与候选人保持联系**——72%的候选人会分享负面体验，这会损害公司品牌多年
4. **过度追求速度而忽视质量**——一个不合适的员工可能会给公司带来相当于其年薪30-50%的损失
5. **忽视内部候选人**——外部招聘人员的表现可能需要3年才能达到内部晋升人员的水平
6. **薪资设定不合理**——根据职位设定薪资，而非候选人的过往经验
7 **没有明确的招聘计划**——被动招聘的效率是主动招聘的2-3倍
