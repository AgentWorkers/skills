## 框架

### 1. 实体结构分析
评估当前的结构，并与其他选项进行比较：
- **个体经营者**：简单，但需缴纳自雇税（美国为15.3%；英国为Class 2+4）
- **LLC/LLP**：具有税收穿透性且提供责任保护
- **S型公司（S-Corp）**：通过薪资与利润分配的调整来降低自雇税（美国）
- **C型公司（C-Corp）**：统一税率为21%（美国）/ 25%（英国），同时享有留存收益优惠
- **有限公司（Ltd）**：税率为19-25%，股息提取有1,000英镑的免税额度

**决策规则：** 如果净利润超过8万美元，选择S型公司通常每年可节省5,000至15,000美元的自雇税。

### 2. 扣除项最大化检查清单
对每个类别进行评分（0 = 未使用；1 = 部分使用；2 = 完全优化）：

| 类别 | 常见遗漏项 | 可节省的税款 |
|----------|--------------|-----------------|
| 办公场所 | 简化计算方法 vs 实际费用 | 每年1,500至5,000美元 |
| 车辆 | 里程记录 vs 实际支出 | 每年2,000至8,000美元 |
| 退休金 | SEP-IRA/个人401(k)计划缴费 | 每年10,000至66,000美元（税延） |
| 健康保险 | 自雇者可抵扣的费用 | 每年3,000至15,000美元 |
| 设备 | 第179条款 / 年度投资额度 | 美国最高1,160,000美元；英国最高100万英镑 |
| 研发费用 | 合格的研发支出 | 可抵扣6-20% |
| 餐饮与娱乐 | 50%可抵扣（商务用餐） | 每年1,000至5,000美元 |
| 教育 | 与业务相关的培训、会议 | 每年2,000至10,000美元 |
| 软件与SaaS服务 | 所有业务工具及订阅费用 | 每年1,000至2,000美元 |
| 专业服务 | 法律、会计、咨询费用 | 全额可抵扣 |

### 3. 季度税务日程表

**美国（IRS）：**
- 1月15日 — 第四季度预估税款支付
- 4月15日 — 第一季度预估税款 + 年度纳税申报表提交
- 6月15日 — 第二季度预估税款
- 9月15日 — 第三季度预估税款

**英国（HMRC）：**
- 1月31日 — 年终税款支付 + 第一份纳税通知
- 7月31日 — 第二份纳税通知
- 10月5日 — 新成立企业的自我评估注册截止日期

### 4. 税收效率高的薪酬策略
对于领取收入的企业主：

```
Optimal Split (US S-Corp example):
- Reasonable salary: $60K-$80K (subject to FICA)
- Distributions: Remaining profit (no SE tax)
- Retirement: Max Solo 401k ($23,500 employee + 25% employer)
- Health: Deduct 100% of premiums above the line

Annual savings vs sole prop at $200K profit: ~$12,000-$18,000
```

### 5. 年终税务处理（第四季度检查清单）
- [ ] 将可抵扣项目提前计入当年
- [ ] 如果预期适用税率较低，将收入推迟到下一年
- [ ] 在12月31日之前最大化退休金缴费（401(k)计划）或4月15日之前缴纳IRA/SEP计划
- [ ] 利用资本损失来抵消收益
- [ ] 审查资产折旧计划
- [ ] 慈善捐赠（集中捐赠策略）
- [ ] 审查预估税款支付情况，避免少缴税款
- [ ] 在年底前购买设备（适用第179条款）

### 6. 国际税务考虑
- **海外收入免税额**：美国外籍人士为126,500美元（2024年标准）
- **税收协定**：美英税收协定可避免大部分收入的重复征税
- **转让定价**：公司间交易需遵循公平交易原则
- **增值税/销售税**：欧盟数字商品免税额为0美元；英国为85,000英镑
- **常设机构**：远程工作者可能被视为在英国设有常设机构

### 7. 常见的高成本错误
1. 未按时缴纳预估税款（罚款3-5%）
2. 未实时记录里程（可能引发IRS审计）
3. 混淆个人与业务支出
4. 忽视州或地方税务义务
5. 在盈利足够时未选择S型公司
6. 忘记结转亏损
7. 选择零售健康保险而非团体或ICHRA计划
