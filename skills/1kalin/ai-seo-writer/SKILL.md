# SEO 写作指南

你的博客文章不仅能在谷歌上获得较高的排名，而且内容也具有阅读价值。这种 SEO 方法摒弃了令人反感的关键词堆砌行为。

## 写作前的准备

了解用户需求：
1. **目标关键词**：用户希望通过搜索获得排名的高频词。
2. **次要关键词**：3-5个相关的词汇。
3. **搜索意图**：是信息型、交易型、导航型还是商业型的搜索？
4. **目标受众**：谁在搜索这些内容？
5. **字数要求**：通常为 1,500-2,000 字。
6. **语气**：专业、随意、技术性等。

## SEO 写作框架

### 标题（H1）
- 包含目标关键词，最好放在标题的开头部分。
- 字数不超过 60 个字符，以免在搜索结果中被截断。
- 使标题具有吸引力——因为标题要与页面上的其他 9 个结果竞争。
- 有效的标题格式包括：“如何 [X]”、“[X] 的 [数量] 种方法”、“[X]：完整指南”。

### 元描述
- 字数在 150-160 个字符之间。
- 自然地包含目标关键词。
- 用广告文案的方式撰写元描述——这是你在搜索结果中的“卖点”。
- 提供一个让人想要点击的理由。

### URL 键词
- 简短、富含关键词、全部使用小写字母、单词之间用连字符连接。
- 例如：`/how-to-write-cold-emails` 而不是 `/how-to-write-the-best-cold-emails-that-get-replies-in-2024`。

### 内容结构

**引言（100-150 字）**
- 用第一句话吸引读者。
- 说明读者将学到什么。
- 在前 100 个字中包含目标关键词。

**正文** — 使用 H2 和 H3 标题
- 每个 H2 需对应一个次要关键词或子主题。
- 使用 H3 标题来分隔长段落。
- 每个段落不超过 300 字，然后换一个标题。
- 使用项目符号和编号列表（谷歌和读者都喜欢这些格式）。

**结论（100-150 字）**
- 总结关键要点。
- 提供下一步行动的建议（CTA）。
- 不要引入新的信息。

### 关键词放置规则
- **目标关键词应出现在**：标题、第一段、一个 H2 标题、结论和元描述中。
- **关键词密度**：最高 1-2%。如果感觉过于刻意，就说明你做得过火了。
- **使用变体**：同义词、相关短语和自然语言表达。
- **LSI 关键词**：在整篇文章中穿插使用语义相关的词汇（谷歌能理解上下文）。

### 内部链接和外部链接
- **内部链接**：链接到网站内的其他 2-3 个相关页面。
- **外部链接**：链接到 2-3 个权威来源（有助于提升在谷歌中的信任度）。
- **锚文本**：使用描述性的文字，而不是简单的“点击这里”。

### 可读性
- 使用简短的句子，段落长度适中（2-3 句）。
- Flesch 阅读难度指数应在 60 以上（大多数成年人都能理解）。
- 使用过渡词。
- 通过标题、列表和图片（并添加 alt 文本）来分段。

### 特色片段优化
- 对于“如何”或“什么是”类型的查询：
- 在相关的 H2 标题后直接提供简洁的定义或步骤列表。
- 使用编号列表来展示流程。
- 使用表格进行对比。
- 保持片段中的目标答案不超过 50 字。

## 输出格式

提供以下内容：
1. **元标题**（包括字数）。
2. **元描述**（包括字数）。
3. **建议的 URL 键词**。
4. **完整的 markdown 格式文章**，包含正确的标题层次结构。
5. **图片建议**，附带 alt 文本。

## 编写规则
- 首先为读者写作，其次考虑搜索引擎。
- 绝不要使用关键词堆砌。谷歌比你更聪明。
- 每个部分都应提供价值，不要为了达到字数要求而添加无意义的段落。
- 引用数据和声明。如果需要，可以使用“[来源]”作为标注。
- 建议在哪里添加图片、信息图或嵌入式内容。
