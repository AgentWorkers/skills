# 加盟经营分析工具

通过数据驱动的框架来评估加盟机会并管理多门店运营。

## 功能概述
- **FDD（Franchise Disclosure Document）分析**：详细解读加盟披露文件中的23项内容
- **门店经济效益建模**：估算启动成本、盈亏平衡时间线及现金流预测
- **区域市场分析**：评估市场饱和度、人口统计特征及选址标准
- **特许权使用费影响分析**：计算特许权使用费及广告费用对门店盈利能力的影响
- **多门店扩张规划**：基于共享运营成本和管理层级制定扩张策略
- **合规性管理**：跟踪联邦贸易委员会（FTC）法规要求、州级注册更新及相关法律法规

## FDD第19项内容分析框架

第19项（财务绩效表现）是特许经营者展示收益数据的部分——但并非所有特许经营者都会提供。

### 如果提供了第19项内容：
1. 检查样本数量（涵盖了多少比例的门店？）
2. 收入的中位数与平均值（平均值可能受到表现最佳门店的影响而产生偏差）
3. 数据覆盖的时间段（是否包括疫情期间或扩张期的数据？）
4. 地理分布情况（全国平均值可能掩盖地区差异）
5. 门店的年龄分布（成熟门店通常比新门店表现更好）

### 如果第19项内容为空：
- 虽然不违法，但仍需询问原因——大约40%的特许经营者会省略这一项。
- 可以结合第21项（财务报表）和第20项（门店数量/关闭情况）来反向推算相关数据。

## 特许权使用费影响计算器

实际特许权使用费往往高于表面显示的百分比：
| 特许权使用费率 | 在100万美元收入时 | 在50万美元利润时 | 对实际利润的影响 |
|-------------|---------------|----------------|----------------------|
| 4% | 40,000美元 | — | 利润的8.0% |
| 6% | 60,000美元 | — | 利润的12.0% |
| 8% | 80,000美元 | — | 利润的16.0% |

还需加上广告费用（通常为1-3%），以计算特许经营者的总收益。

## FDD风险点检查清单
- [ ] 第3项：诉讼历史——是否有加盟商正在进行的诉讼？
- [ ] 第5项：初始费用——在何种情况下可以退款？
- [ ] 第6项：持续费用——是否存在隐藏的技术/供应费用？
- [ ] 第7项：投资范围——费用差异是否过大（表明不确定性？）
- [ ] 第8项：采购限制——是否必须从特许经营者指定的供应商处采购？
- [ ] 第9项：加盟商的义务——禁止竞争的范围和期限是什么？
- [ ] 第11项：特许经营者的义务——实际提供的培训和支持有哪些？
- [ ] 第12项：经营区域——是否具有排他性？保护半径是多少？
- [ ] 第17项：续签条款——续签时特许经营者是否可以调整特许权使用费率？
- [ ] 第19项：财务绩效数据——是否提供？样本是否具有代表性？
- [ ] 第20项：门店数量——过去三年内的净开业数量与关闭数量？
- [ ] 第21项：财务报表——特许经营者本身是否盈利？

## 多门店扩张模型
| 特许门店数量 | 共享运营成本节省 | 管理层级 | 平均净利润提升 |
|-------|------------------------|-------------------|----------------------|
| 1家 | 基本水平 | 业主自行经营 | — |
| 2-3家 | 管理费用减少15-20% | 需要区域经理 | 利润提升2-4% |
| 4-6家 | 管理费用减少25-30% | 需要区域经理 | 利润提升3-5% |
| 7-10家 | 管理费用减少30-40% | 需要区域管理层 | 利润提升4-7% |
| 10家以上 | 管理费用减少40%以上 | 需要公司总部支持 | 利润提升5-8% |

## 加盟经营与独立经营的比较
| 对比因素 | 加盟经营 | 独立经营 |
|--------|-----------|-------------|
| 失败率（五年内） | 约15% | 约50% |
| 品牌知名度 | 立即获得 | 需从头建立 |
| 启动成本 | 较高（包含加盟费） | 较低，但具有不确定性 |
| 运营自由度 | 受限于系统规定 | 完全自主 |
| 融资渠道 | 更容易获得（SBA支持加盟经营） | 较困难 |
| 退出/转售 | 有规范的转让流程 | 需协商 |
| 持续费用 | 包括特许权使用费和广告费用 | 无固定费用 |
