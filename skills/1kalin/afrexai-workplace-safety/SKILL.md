# 工作场所安全与OSHA合规性

生成工作场所安全计划、事故报告、危险评估及合规性文档。

## 命令

### `safety audit`  
执行工作场所安全审计。请提供以下信息：  
- 行业（建筑、制造、办公、仓库、医疗保健、餐饮服务）  
- 员工人数  
- 近12个月内发生的事故  
- 目前实施的安全措施  

生成以下内容：  
1. **危险评估矩阵** — 按严重程度和发生概率识别主要风险  
2. **OSHA合规性检查表** — 与相关标准（29 CFR 1910/1926）对齐  
3. **差距分析** — 与法规要求的差距  
4. **优先行动计划** — 按风险评分、预估成本和时间表排序  

### `incident report`  
生成结构化的事故调查报告：  
- 日期、时间、地点、涉及人员  
- 事故类别（险些发生的事故、需要急救的事故、需要记录的事故）  
- 根本原因分析（5个“为什么”+鱼骨图）  
- 纠正措施及负责人、截止日期  
- OSHA 300日志记录指南  

### `safety program`  
为特定危险制定完整的安全计划：  
- 上锁/挂牌（Lockout/Tagout, LOTO）  
- 密闭空间作业  
- 防坠落措施  
- 危险沟通（HazCom/GHS）  
- 个人防护装备（PPE）  
- 应急行动计划  
- 呼吸防护  
- 电气安全  

每个安全计划包括：目的、范围、责任、程序、培训要求及记录保存方式。  

### `training matrix`  
生成员工安全培训矩阵：  
- 基于角色的培训要求  
- 培训频率（初次培训、年度培训、复习培训）  
- 每项培训要求的法规依据  
- 培训完成情况的跟踪模板  
- 第三方培训的预估成本  

### `jha`  
针对特定任务的作业危险分析（Job Hazard Analysis, JHA）：  
- 将任务分解为步骤  
- 识别每一步中的危险  
- 评估风险等级（严重程度 × 发生概率）  
- 制定控制措施（消除危险 → 个人防护装备的使用）  
- 生成可现场使用的JHA卡片  

## 主要标准参考  
| 标准 | 范围 |  
|----------|-------|  
| OSHA 29 CFR 1910 | 通用行业 |  
| OSHA 29 CFR 1926 | 建筑行业 |  
| OSHA 29 CFR 1904 | 记录保存要求 |  
| ANSI Z10 | 职业健康与安全管理体系 |  
| ISO 45001 | 职业健康与安全管理体系 |  
| NFPA 70E | 电气安全标准 |  

## 处罚规定  
- 严重违规：每次违规罚款16,131美元（2024年数据）  
- 故意违规或重复违规：每次违规罚款161,323美元  
- 工作场所伤害的平均成本：42,000美元（直接费用）+ 126,000美元（间接费用）  
- 安全计划的投资回报率：每投入1美元可带来4至6美元的回报（OSHA数据）  

## 输出格式  
- 审计和危险评估矩阵采用Markdown表格形式  
- 检查表适用于现场使用  
- 安全计划的步骤以项目符号列表形式呈现  
- 综合仪表板供管理层审查
