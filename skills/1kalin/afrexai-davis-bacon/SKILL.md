# 戴维斯-贝肯法案（Davis-Bacon Act）与现行工资标准合规性

我们是一家负责联邦建筑项目合规性审核的机构，主要负责监督《戴维斯-贝肯法案》（Davis-Bacon Act）及相关法规的遵守情况，同时涵盖各州的现行工资法律以及劳工部（DOL）制定的工资标准。

## 适用场景
- 参与联邦或联邦资助的建筑项目投标
- 确定项目的现行工资标准
- 处理工资单的认证与提交工作
- 应对劳工部的合规性审查或调查
- 遵守各州的现行工资法规（“小戴维斯-贝肯法案”相关要求）
- 确保联邦项目中的学徒计划符合相关法规

## 主要功能

### 工资标准查询
帮助承包商查找与其项目相匹配的劳工部工资标准（WD）：

| 项目详情 | 说明 |
|---------|--------|
| **工资标准类型** | 通用标准（如重型建筑、公路建设、住宅建筑等）或项目特定标准 |
| **地理范围** | 必须与项目所在地完全一致 |
| **标准更新频率** | 每周更新；投标开放前10天锁定 |
| **工资构成** | 基本时薪 + 福利津贴 = 总现行工资 |
| **查询途径** | 所有工资标准均可在 sam.gov/wage-determinations 查阅 |

### 覆盖的项目类型
- 根据《戴维斯-贝肯法案》，直接由联邦政府资助的合同（金额超过2,000美元的建筑工程/维修/改造项目）
- 大约60项联邦法规（如CDBG、FHWA、FAA、HUD、EPA SRF、WIFIA、IIJA、IRA等）
- 《基础设施投资与就业法案》（IIJA, 2021）：新支出超过5500亿美元的项目均需遵守现行工资标准
- 《通货膨胀削减法案》（IRA, 2022）：清洁能源项目必须使用现行工资标准才能享受税收优惠

### 工资单认证要求
每周提交的工资单（WH-347表格）必须包含以下内容：

| 项目详情 | 说明 |
|---------|--------|
| **员工信息** | 姓名、地址、最后四位社会安全号码（SSN）、工作类别 |
| **工作时间** | 每日及每周工作时间（包括正常工作时间与加班时间） |
| **工资构成** | 每小时工资 + 各项福利津贴（以现金或实际福利计划的形式体现） |
| **扣除项** | 仅允许的扣除项目（依据29 CFR 3规定） |
| **认证要求** | 需签署合规性声明 |
| **提交方式** | 每周提交给承包机构，并至少保留3年 |
| **电子提交方式** | 大多数机构接受LCPtracker、Elation Systems或B2Gnow等电子提交工具

### 分类与工资标准匹配
如果所需的工作类别未在工资标准中列出：
1. **检查现有分类**：通常类似类别可适用于该工作
2. **申请合规性确认**：向承包官员提交SF-1444表格
3. **承包官员将申请转交劳工部**：劳工部将在30天内进行审核
4. **临时工资标准**：在劳工部最终确定标准前，可按提议的工资标准支付
**注意**：承包商不得自行设定工资标准

### 常见违规行为及处罚措施
- **工资支付不足**：需补发工资并支付利息（违约金为未支付工资的100%）
- **伪造工资单**：构成犯罪行为，最高罚款10,000美元并可能面临10年监禁（依据18 USC 1001条款）
- **行贿**：最高罚款5,000美元并可能面临5年监禁
- **被列入禁标名单**：公司及个人将被禁止参与所有联邦项目
- **故意违规**：劳工部将移送司法部进行刑事起诉
- **分包商责任**：主承包商对分包商的违规行为承担连带责任

### 福利津贴合规性要求
- **福利计划要求**：必须为员工提供不可撤销的健康保险、养老金、休假和学徒培训等福利
- **现金支付方式**：福利津贴需以现金形式支付，或计入基本工资
- **支付方式组合**：部分福利以现金支付，部分以福利计划形式提供；总金额必须达到现行工资标准
- **年度结算**：非适用福利项目的费用不得计入工资单

### 学徒计划要求
- **注册要求**：承包商必须向劳工部或认可的学徒培训机构（SAA）注册学徒
- **比例限制**：学徒比例不得超过规定上限
- **工资标准**：学徒工资应为正式员工的固定百分比
- **监督要求**：学徒必须在正式员工指导下工作
- **未注册情况**：未注册学徒需按普通工人标准支付工资

### 各州现行工资标准快速参考
| 州份 | 起付金额 | 与联邦法规的主要区别 |
|---------|-----------|----------------------|
| **加利福尼亚州** | 1,000美元（公共工程） | 需在DIR系统中注册，违规处罚严厉 |
| **纽约州** | 0美元（所有公共工程） | 由财政官员确定工资标准，适用范围更广 |
| **伊利诺伊州** | 0美元（所有公共工程） | 按县划分工资标准，每月报告 |
| **马萨诸塞州** | 0美元（所有公共工程） | 每周提交工资单并需进行认证 |
| **新泽西州** | 2,000美元 | 需遵守相关注册规定 |
| **俄亥俄州** | 250,000美元 | 仅适用于新建项目（不包括维修/维护工程） |
| **宾夕法尼亚州** | 250,000美元 | 采用与联邦不同的工资标准确定流程 |
| **华盛顿州** | 0美元（所有公共工程） | 工程开始前需提交相关文件 |

### 审计与调查准备
**审计前检查清单**：
- 每周提交所有经过认证的工资单（WH-347表格）
- 在工作现场张贴工资标准相关公告（纸质或电子形式）
- 员工访谈结果与工资单记录一致
- 福利津贴的缴纳情况有据可查且最新
- 学徒注册信息齐全
- 收集并审核分包商的工资单
- 工时记录与考勤表一致
- 无未经记录的工时分类情况
- 加班时间计算正确（每周40小时，符合FLSA规定）
- 所有必要的公告均已张贴（如WH-1321“员工权利”公告）

### 《通货膨胀削减法案》（IRA）下的现行工资标准与学徒计划要求
对于享受税收优惠的清洁能源项目：
- **税收优惠计算方式**：
  - **ITC（太阳能/风能）**：基础金额的6% + 现行工资标准的30%
  - **PTC（风能/其他能源）**：每千瓦时0.3美分 + 现行工资标准的1.5%
  - **45X制造业**：基础金额的3倍
  - **45V氢能**：每千克0.12-0.60美元 + 现行工资标准的3倍
**注意事项**：施工期间及项目投入使用后的5年内都必须支付现行工资标准；学徒工作时间占比分别为2024年的12.5%，2025年及以后的15%。对于善意违规行为，有相应的纠正/处罚机制。

## 输出格式
- 工资标准查询结果：包括分类、基础工资、福利津贴、总工资金额及工资标准编号、有效期
- 合规性审核结果：包含通过/未通过清单及整改措施
- 风险评估：列出违规行为的潜在影响及发生概率
- 投标准备资料：按工作类别和工时划分的工资成本计算表
