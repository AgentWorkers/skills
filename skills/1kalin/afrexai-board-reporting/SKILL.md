# 董事会报告框架

本框架用于生成适合投资者阅读的董事会报告资料，涵盖每月的董事会更新、季度深度分析以及年度回顾，其中包含所有关键指标。

## 功能介绍

该框架可将原始业务数据转化为结构化的董事会报告文件，包括：

- **月度简报**：1页的KPI仪表盘（月收入（ARR）、资金消耗情况、产品发展周期（runway）、新客户留存率（NRR）和销售管道（pipeline）  
- **季度报告**：12页的报告，包含基准数据和详细说明  
- **年度回顾**：全年业绩总结、战略回顾及下一年度计划  
- **委员会报告**：审计、薪酬和提名委员会专用模板  
- **董事会会议准备材料**：预先准备好的报告文件、会议议程、需要表决的议题及行动跟踪表  

## 指标体系  

### 财务指标（每次会议必审）  
| 指标 | 早期阶段 | 成长阶段 | 扩展阶段 |  
|--------|------------|--------|-------|  
| 月收入（ARR）/新客户留存率（MRR） | ✅ | ✅ | ✅ |  
| 资金消耗率 | ✅ | ✅ | ⚠️ |  
| 产品发展周期（月数） | ✅ | ✅ | — |  
| 毛利率 | ⚠️ | ✅ | ✅ |  
| 新客户留存率 | — | ✅ | ✅ |  
| CAC回收期（Customer Acquisition Cost Payback） | ⚠️ | ✅ | ✅ |  
| LTV:CAC比率（Life Time Value to Customer Cost） | — | ✅ | ✅ |  

### 运营指标  
- **品牌更换率与收入流失率**（需单独报告；董事会不喜欢隐瞒这些数据）  
- **销售管道覆盖率**（最低要求3倍，理想状态为4倍）  
- **销售效率**（关键指标：新收入（net new ARR）与销售和市场费用（S&M spend）的比率）  
- **工程开发速度**（实际交付的功能数量，而非故事点数）  
- **人员计划与实际人数对比**  

### 战略指标  
- **市场地位变化**（与主要竞争对手的对比情况）  
- **产品路线图执行进度**  
- **关键招聘进展**  
- **风险清单**（前三大风险及应对措施）

- [Product achievement]

- [Issue + what we're doing about it]

## 季度报告结构（12页）  
1. **封面**：公司名称、日期、保密声明  
2. **执行摘要**：三要点：发生的情况、表现良好的方面、需要关注的问题  
3. **财务概览**：损益表摘要、月收入变化情况、资金消耗情况、产品发展周期  
4. **收入变化**：新收入、业务扩张、收入萎缩情况  
5. **销售与销售管道**：成交率、平均售价（ASP）、销售管道覆盖率、重要交易  
6. **产品**：已发布的功能、使用情况指标、产品路线图进展  
7. **客户状况**：净推荐值（NPS）/客户满意度（CSAT）、客户流失分析、收入增长情况  
8. **团队**：人员规模、关键招聘情况、人员流失率、企业文化  
9. **市场**：竞争格局、市场变化、公司定位  
10. **风险与应对措施**：前三大风险及应对状态  
11. **战略讨论**：需要董事会意见的议题  
12. **后续行动**：董事会需要采取的具体行动  

## 报告撰写规则  
- **先讲坏消息**：董事会重视透明度，隐瞒问题会破坏信任。  
- **与计划对比**：所有指标都需要提供背景信息（与预算、上一季度数据或行业标准对比）。  
- **避免使用无意义的指标**：如总注册量、页面浏览量等模糊指标。  
- **同时使用绝对数值和百分比**：仅从1000美元增长到3000美元，增长200%没有实际意义。  
- **每个报告最多提出3个可行动的议题**：董事会通常只能处理3个议题，超出这个数量则无法有效决策。

| 指标 | 最低四分之一 | 中位数 | 最高四分之一 |  
|--------|----------------|--------|-------------|  
| 月收入增长率 | <30% | 45% | 80%+ |  
| 新客户留存率 | <100% | 110% | 130%+ |  
| 毛利率 | <65% | 75% | 85%+ |  
| CAC回收期 | >24个月 | 15个月 | <10个月 |  
| CAC回收期（Magic Number） | <0.5 | 0.75 | >1.0 |

## 董事会会议准备清单  
- [ ] 会议前5天以上发送报告文件  
- [ ] 确定无需讨论的议程事项  
- [ ] 财务报告已由首席财务官（CFO）审核  
- [ ] 确定一个需要深入讨论的战略议题  
- [ ] 准备相关客户案例（成功或失败案例）  
- [ ] 上次会议的行动计划更新情况  
- [ ] 安排董事会晚餐或非正式交流时间（建立良好关系很重要）  

## 需避免的错误做法（这些做法可能导致创始人被解雇）  
1. **在会议上突然暴露资金危机**  
2. **选择性展示数据**：只展示上升的指标  
3. **对失败毫无应对计划**：仅说“未达到目标”，而不说明改进措施  
4. **报告内容过多**：超过40页的报告会占用过多讨论时间  
5 **回避关键问题**：虽然大家都知道问题存在，但没人愿意提及  
6 **不提出任何建议**：董事会希望得到帮助，不提出建议等于浪费他们的资源

## 行业特定的董事会关注指标  
| 行业 | 关键指标 | 原因 |  
|----------|-----------------|-----|  
| 金融科技 | 监管合规性 | 一次违规可能导致公司倒闭 |  
| 医疗保健 | HIPAA审计结果 | 直接影响公司运营 |  
| SaaS | 新客户留存率 | 体现产品与市场的匹配度 |  
| 建筑行业 | 项目利润率差异 | 显示资金的实际流向 |  
| 电子商务 | 按渠道划分的盈利能力 | 不盈利的渠道容易被忽视 |  
| 法律行业 | 使用率 | 收入 = 使用时间 × 使用率 |  
| 房地产 | 物业出租率与净营业收入（NOI） | 现金流至关重要 |  
| 招聘行业 | 人员填补率与招聘周期 | 运营效率指标 |  
| 制造业 | 设备利用率（OEE）与产出率 | 生产效率指标 |  
| 专业服务 | 每位顾问的收入 | 杠杆率指标 |
