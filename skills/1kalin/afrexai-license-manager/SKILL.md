# 软件许可证管理器

该工具用于审计、跟踪和优化您组织的软件许可证使用情况，能够发现资源浪费、识别合规风险，并生成许可证续期日程表。

## 功能概述

1. **许可证库存审计**：全面统计组织内所有的SaaS订阅服务、永久许可证以及开源依赖项。
2. **资源浪费检测**：标记未使用的许可证、重复购买的工具以及仍在消耗费用的无效订阅服务。
3. **合规风险扫描**：识别过期的许可证、超出许可数量的许可证使用情况以及存在审计风险的漏洞。
4. **续期日程表**：生成为期12个月的续期时间表，并设置60天、30天和15天的提醒通知。
5. **谈判准备**：生成供应商评分卡，其中包含使用数据、替代方案以及谈判时的有利条件。
6. **成本优化报告**：提供降级、合并许可证或调整许可证级别的建议，同时预测相应的成本节省金额。

## 使用方法

根据您的需求向系统说明操作内容：
- “审计我们的软件许可证”：获取完整的许可证清单及资源浪费报告。
- “查找未使用的SaaS订阅服务”：检测浪费情况并估算节省成本。
- “生成许可证续期日程表”：获取包含提醒的12个月时间表。
- “我们正在续签Salesforce许可证——准备谈判”：获取供应商评分卡及谈判策略分析。
- “检查我们的开源许可证合规性”：扫描是否存在GPL/AGPL许可证相关的合规风险。

## 审计框架

### 许可证类别
| 类别          | 示例                          | 风险等级     |
|----------------|----------------------------------|-----------|
| SaaS订阅服务     | Salesforce、HubSpot、Slack、Zoom                | 中等风险——自动续订功能 |
| 永久许可证     | Microsoft Office、Adobe CS6                   | 低风险——但可能缺乏技术支持 |
| 基于使用的许可证   | AWS、Twilio、Stripe                     | 高风险——费用难以预测 |
| 开源许可证     | GPL、MIT、Apache、AGPL                     | 商业化使用存在合规风险 |
| 企业级协议     | Microsoft EA、Oracle ULA                     | 高风险——合同条款复杂 |

### 资源浪费的常见迹象
- **“幽灵用户”**：超过60天未登录的已授权用户。
- **重复购买的工具**：具有相同功能的多个工具（例如Zoom、Teams、Meet）。
- **许可证层级浪费**：实际使用需求低于企业级许可证，但仍购买高级别许可证。
- **遗留许可证**：已离职员工仍占用许可证资源。
- **闲置软件**：已购买但从未使用的软件。

### 合规风险警示
- 许可证数量超过协议规定。
- 在许可证过期后继续使用相关软件。
- 在专有产品中使用GPL/AGPL许可证代码但未进行明确标注。
- 将非商业许可证用于商业用途。
- 员工电脑上存在未经授权的软件副本。

## 谈判准备评分卡

针对每次许可证续期，系统会提供以下信息：
1. **当前费用**：年度费用及每用户费用明细。
2. **实际使用情况**：每日活跃用户（DAU）/月活跃用户（MAU）及功能采用率。
3. **替代方案**：2-3家竞争对手的产品及价格信息。
4. **谈判优势**：多年折扣、批量购买优惠、竞争对手的报价信息。
5. **最低替代方案（BATNA）**：如果供应商拒绝协商，可采取的最低成本方案。
6. **谈判时机**：最佳谈判时间为续期前90天，此时公司面临财政年度的压力。

## 成本优化策略

### 立即可实施的措施（第1周）
- 清除已离职员工的许可证资源：通常可节省8-15%的SaaS费用。
- 取消未使用的试用账户和被遗忘的订阅服务。
- 降级过度配置的许可证层级。

### 中期措施（第1-3个月）
- 合并重复购买的工具（选择其中一个并统一使用）。
- 与使用量最大的前5家供应商重新谈判费用。
- 为新订阅服务建立审批流程。

### 长期策略（第1-2季度）
- 合并企业级许可证协议。
- 优化年度与按月计费的模式（通常可节省15-20%的费用）。
- 为简单的SaaS服务开发内部替代工具。

## 行业基准数据
- 平均而言，企业会浪费25-30%的SaaS费用在未使用或使用不足的许可证上。
- 员工超过100人的企业平均拥有130多种SaaS工具（大多数企业不清楚具体数量）。
- 通过许可证审计，通常每名员工每年可节省1,000至3,000美元的费用。
- 开源许可证合规性违规的罚款金额因地区而异，可能高达10万美元至500万美元以上。

## 输出格式
- **高管摘要**：包含资源浪费情况、风险分析及节省成本的概览。
- **详细审计报告**：包含完整的许可证清单、使用情况以及优化建议。
- **续期日程表**：格式化为电子表格的形式。
- **供应商评分卡**：针对每个供应商的谈判建议。
- **合规报告**：列出风险点及相应的补救措施。
