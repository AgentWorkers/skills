# 合作协议生成器

本工具可为B2B合作伙伴关系生成全面的合作协议、合资企业框架及战略联盟文件。

## 该功能的用途

当用户需要合作协议、合资企业合同或战略联盟框架时，本工具会生成包含以下内容的完整文件：

1. **合作关系架构**：实体类型、所有权分配、资本贡献方式
2. **角色与职责**：决策权分配、管理职责、投票权
3. **财务条款**：利润/亏损分配方式、资金调用安排、资金使用流程
4. **知识产权与保密条款**：保密协议（NDA）内容、知识产权归属、竞业禁止/禁止招揽条款
5. **治理机制**：会议频率、争议解决方式、僵局处理机制
6. **退出条款**：收购触发条件、估值方法、股东权益保障机制（如“拖带权”/“跟随权”等）

## 使用方法

用户输入指令：“为[具体场景]起草合作协议”

### 第1步：收集相关信息
- 询问（或推断）合作伙伴的名称及实体类型
- 合作目的/业务范围
- 所有权分配情况
- 资本贡献金额
- 合作期限（固定期限或永久合作）
- 所适用的法律管辖区（默认为特拉华州或英格兰及威尔士）

### 第2步：生成合作协议
根据收集到的信息，生成包含上述所有6个部分的完整合作协议框架。具体内容包括：
- 各条款的具体表述（而不仅仅是标题）
- 常见的保护性条款（如“优先购买权”、“反稀释条款”、“关键人员条款”等）
- 税务相关考虑因素（资金流转方式：通过子公司还是母公司）
- 样例资金使用计划（如资本账户管理、收益分配流程）

### 第3步：审核潜在问题
检查协议中可能引发争议的条款，例如：
- 投票权与所有权分配不匹配的情况
- 知识产权归属表述模糊
- 缺少争议解决机制
- 未明确退出条款

## 支持的合作类型

| 合作类型 | 适用场景 | 主要条款 |
|------|----------|----------------|
| 普通合伙** | 小型企业、专业服务领域 | 无限连带责任、平等管理权 |
| 有限合伙** | 投资项目、房地产领域 | GP（普通合伙人）/LP（有限合伙人）结构、收益分配机制 |
| LLC运营协议** | 大多数B2B合作关系 | 灵活的结构、有限责任保护 |
| 合资企业** | 针对特定项目的合作 | 合作范围、合作期限、终止条件 |
| 战略联盟** | 非股权形式的合作关系 | 服务标准、排他性条款、终止条件 |
| 收益共享型合作** | 渠道/推荐合作 | 收益归属、支付条款、最低合作要求 |

## 退出时的估值方法

- **账面价值**：简单易算，但可能低估无形资产的价值
- **公平市场价值**：由评估机构确定，最为常见
- **基于公式的估值方法**：根据收入或EBITDA倍数进行估值（需在协议中明确）
- **双方协商确定的价值**：每年由合作伙伴协商确定，可避免争议
- **折现现金流法**：最准确的估值方法，但计算成本较高

## 需特别注意的常见错误

1. 完全不签订书面协议（口头协议可能导致法律纠纷）
2. 所有权分配为50/50但缺乏争议解决机制
- 未规定禁止竞争条款，导致合作伙伴可能复制对方的业务
- 未设置资金调用条款，影响企业扩张
- “合理努力”等模糊条款缺乏具体绩效指标
- 未明确保险要求（如董事责任险、高管责任险等）
