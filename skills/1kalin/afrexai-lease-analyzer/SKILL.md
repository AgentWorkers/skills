# 商业租赁分析工具

本工具用于分析商业租赁（办公、零售、工业、仓储类型）中的隐藏成本、不利条款以及谈判中的有利因素。适用于在新租赁协议签订、租赁续签谈判或比较多个租赁方案时使用。

## 使用场景
- 签署新的商业租赁协议
- 租赁续签谈判
- 比较多个租赁提案
- 审计现有租赁协议以降低成本
- 分租或转让租赁权的分析

## 用户需要提供的信息
1. **租赁类型**：办公、零售、工业、仓储或混合用途
2. **租赁条款**：基本租金、租赁期限、续租选项
3. **成本结构**：净租金（NNN）、调整后租金（modified gross）、全服务租金（full-service gross）
4. **使用面积**：可用面积与可出租面积（负载系数）
5. **位置/市场**：所在城市及子市场（用于成本比较）

## 分析框架

### 1. 成本分解（实际占用成本）
```
Base Rent ($/SF/yr)
+ CAM Charges (Common Area Maintenance)
+ Property Tax Pass-Through
+ Insurance Pass-Through
+ Utility Estimates
+ Parking Costs
+ Janitorial (if not included)
= Total Occupancy Cost ($/SF/yr)
÷ 12 = Monthly Cost
× Term = Total Lease Liability
```

### 2. 负载系数分析
```
Rentable SF ÷ Usable SF = Load Factor
Industry benchmarks:
- Class A Office: 1.15-1.20 (15-20% common area)
- Class B Office: 1.12-1.18
- Retail: 1.05-1.10
- Industrial: 1.02-1.05

Flag if load factor > benchmark. Every 1% = real money.
Example: 5,000 USF at 1.20 = paying for 6,000 RSF
At $30/SF = $30,000/yr for hallways and lobbies
```

### 3. 价格调整模型
预测整个租赁期间的总成本，包括：
- **固定涨幅**：每年3%为标准涨幅；若超过3.5%，请标记为异常。
- **与CPI挂钩的涨幅**：模拟2.5%、4%和6%三种情况
- **市场重估**：与预测的市场租金进行比较
- **价格调整上限**：若无上限，模拟每年5-8%的涨幅

输出年度成本表，显示基本租金、预计的调整后租金（CAM）和总成本。

### 4. 关键条款审查

**红色警示**（需立即关注）：
- 无终止期限的个人担保条款
- 拆除条款（房东可因重新开发而终止租赁）
- 续租半径限制超过3英里（适用于零售租赁）
- 无共租保护条款的持续经营条款
- 无价格调整上限且房东无审计权
- 迁址条款（房东可强制租户搬迁）
- 无转让或分租权利
- 过期滞纳金超过最终租金的150%

**黄色警示**（需协商）：
- 无因施工延误而减免租金的条款
- 无独家使用条款（适用于零售租赁）
- HVAC维护费用完全由租户承担
- 可控运营费用无上限
- 需恢复原状的修复条款
- 第3-5年后无提前终止选项
- 保险要求高于行业标准

**绿色（标准/有利）**：
- 租金补贴（办公：每平方英尺30-60美元；零售：15-30美元）
- 每年租赁期内一个月的免租期
- 对相邻空间的优先购买权
- 按公平市场价值续租的选项
- 价格调整的审计权
- 合理的转让或分租权利

### 5. 谈判中的有利因素
根据市场状况，确定以下谈判重点：
- **租户市场状况**（空置率超过15%）：争取更高的租金补贴、更多的免租期和更低的涨幅
- **房东市场状况**（空置率低于8%）：重点关注价格上限、灵活性条款和终止条款
- **市场状况平衡**（空置率在8-15%之间）：进行标准谈判，选择3-4个优先事项

按对租赁期限内的经济影响对谈判事项进行排序。

### 7. 财务影响总结
- 总租赁负债（ASC 842 / IFRS 16标准）
- 所有租赁付款的净现值（折现率为6-8%）
- 每名员工的盈亏平衡占用成本
- 每美元收入的成本（如提供收入数据）
