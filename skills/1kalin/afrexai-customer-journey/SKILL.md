# 客户旅程映射（Customer Journey Mapping）

从客户的首次点击到成为忠实推荐者的每一个接触点进行详细记录。识别整个客户生命周期中的流失点、情感高峰点以及自动化实施的潜在机会。

## 功能概述

生成一份完整的客户旅程地图，内容包括：
- **阶段划分**：认知（Awareness）→ 考虑（Consideration）→ 购买（Purchase）→ 入门（Onboarding）→ 采用（Adoption）→ 扩展（Expansion）→ 推荐（Advocacy）
- **接触点清单**：所有渠道（网站、电子邮件、聊天、电话、社交媒体、应用程序内）上的互动记录
- **情感分析**：每个阶段客户的情感状态（沮丧、中立、满意）
- **流失原因分析**：客户流失的原因及位置
- **自动化机会**：哪些接触点可以由人工智能代理处理
- **各阶段关键指标**：转化率、每个阶段的停留时间、服务成本

## 使用方法

您可以指示您的团队：
- “绘制从客户首次接触产品到续费的完整旅程地图”

## 旅程阶段框架

### 第1阶段：认知（Awareness）
- **渠道**：搜索引擎优化（SEO）、付费广告、社交媒体、推荐、活动、内容营销
- **关键指标**：每名潜在客户的成本
- **常见流失原因**：不相关的着陆页、页面加载缓慢、产品价值不明确
- **自动化机会**：基于人工智能的内容个性化、聊天机器人用于客户筛选

### 第2阶段：考虑（Consideration）
- **渠道**：网站、比较页面、用户评价、演示、免费试用
- **关键指标**：潜在客户（Lead）转化为销售线索（MQL）的转化率（基准：5-15%）
- **常见流失原因**：缺乏社交证明、价格信息隐藏、表单字段过多
- **自动化机会**：人工智能聊天机器人提供即时问答、自动安排演示

### 第3阶段：购买（Purchase）
- **渠道**：销售电话、结账流程、合同签订
- **关键指标**：销售线索转化为实际客户的转化率（基准：2-5%）
- **常见流失原因**：定价复杂、合同处理缓慢、缺乏紧迫感
- **自动化机会**：人工智能生成报价、自动审核合同、发送付款提醒

### 第4阶段：入门（Onboarding）
- **渠道**：欢迎邮件、设置向导、培训、启动会议
- **关键指标**：用户首次获得产品价值的时间（基准：SaaS产品为7天内）
- **常见流失原因**：下一步操作不明确、功能过多、缺乏集成支持
- **自动化机会**：人工智能引导的入门流程、自动检查、智能提示信息

### 第5阶段：采用（Adoption）
- **渠道**：应用程序内的帮助文档、支持工单、知识库、客户成功经理（CSM）的协助
- **关键指标**：功能采用率、日活跃用户（DAU）与月活跃用户（MAU）的比例
- **常见流失原因**：用户停留在基础功能上、支持响应缓慢
- **自动化机会**：人工智能提示使用高级功能、主动提供支持、自动化的培训路径

### 第6阶段：扩展（Expansion）
- **渠道**：定期回访（QBR）、升级提示、交叉销售活动、账户评估
- **关键指标**：净收入留存率（NRR）（B2B SaaS产品的基准：>110%）
- **常见流失原因**：缺乏明确的升级路径、无法展示投资回报率（ROI）、时机选择不当
- **自动化机会**：人工智能评估客户满意度、自动准备回访材料、触发升级建议

### 第7阶段：推荐（Advocacy）
- **渠道**：净推荐值（NPS）调查、推荐计划、案例研究、用户评价、社区互动
- **关键指标**：净推荐值（NPS）得分（基准：>50）、推荐率
- **常见流失原因**：未主动询问客户意见、缺乏激励措施、近期使用体验不佳
- **自动化机会**：人工智能触发用户评价请求、跟踪推荐行为、收集用户推荐

## 接触点评分矩阵

对每个接触点进行评分：
| 评分维度 | 评分（1-5分） | 说明 |
|-----------|-----------|-------------|
| 频率 | 客户访问该接触点的频率 |
| 影响程度 | 该接触点对购买/留存决策的影响 |
| 工作量 | 团队处理该接触点的难度（高 = 不利） |
| 客户满意度 | 客户在该阶段的满意度 |
| 自动化潜力 | 该接触点是否适合人工智能代理处理（5分表示完全可自动化） |

**优先级计算公式**：（影响程度 × 频率 × 自动化潜力） / 工作量

评分较高的接触点应优先考虑自动化。如果客户满意度低但影响程度高，应立即采取措施改进。

## 流失点诊断

当发现流失点时，请参考以下检查清单：
1. **数据分析**：销售漏斗数据显示该阶段的流失比例是多少？
2. **原因调查**：调查或访谈中是否有相关问题？支持工单中是否提到了这个问题？
3. **竞争对手做法**：竞争对手是如何处理这个阶段的？
4. **快速改进**：能否在1周内减少客户流失？
5. **自动化可行性**：人工智能代理能否完全消除该流失问题？
6 **收入影响**：如果解决了这个问题，能带来多少收入增长？

## 输出格式

您的客户旅程地图应包括：
- **可视化流程图**：各阶段之间的转化率关系
- **接触点清单**：所有互动记录、对应的渠道、负责人及自动化处理状态
- **情感曲线**：客户在整个旅程中的情感变化
- **差距分析**：当前体验与理想状态的差距
- **自动化实施计划**：按优先级排列的自动化目标及预期投资回报率（ROI）
- **90天行动计划**：短期目标（第1-2周）、中期改进（第1-2个月）、长期战略调整（第3个月）
