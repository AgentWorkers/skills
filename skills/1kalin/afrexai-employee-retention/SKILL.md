# 员工留存与流失风险分析工具

诊断员工离职的原因，并在问题发生之前采取解决方案。

## 功能概述

该工具从8个维度对员工流失风险进行系统化评估，每个维度的评分范围为0-100分，识别出高流失风险的员工和部门，计算员工替换成本，并生成为期90天的行动计划。

## 8个评估维度

1. **薪酬竞争力** — 市场对标、薪酬公平性、总薪酬分析
2. **职业发展机会** — 晋升速度、技能提升、内部晋升机会
3. **管理者素质** — 管理范围、一对一沟通频率、员工在每位管理者下的任职时长
4. **工作负担平衡** — 加班模式、休假使用情况、职业倦怠指标
5. **企业文化与归属感** — 员工净推荐值（eNPS）趋势、多元化指标、团队凝聚力
6. **认可与反馈** — 反馈的频率和质量（包括同事间的反馈和自上而下的反馈）
7. **灵活性与自主权** — 远程工作政策、工作时间安排、决策权
8. **入职体验与初期留存** — 90天内的员工流失率、新员工适应期、导师计划的效果

## 使用方法

将您的团队数据（员工人数、任职时长分布、近期离职情况、eNPS得分、薪酬等级）提供给该工具，它会：

1. 对每个维度进行0-100分的评分
2. 将评分低于60分的维度标记为关键风险点
3. 计算不同职位级别的员工替换成本：
   - 初级员工：薪资的50-60%
   - 中级员工：薪资的75-100%
   - 高级/专业员工：薪资的100-150%
   - 高管：薪资的200-400%
4. 确定最有效的3个员工留存改进措施（具有最高投资回报率）
5. 生成包含具体负责人分配的90天留存行动计划

## 员工替换成本计算公式

```
Replacement Cost = (Recruiting fees + Lost productivity + Training + Ramp time) × Role multiplier

Quick estimate:
- $65K role → $39K-$65K replacement cost
- $120K role → $90K-$180K replacement cost  
- $200K role → $200K-$800K replacement cost
```

## 2026年人工智能对员工留存的影响因素

人工智能转型带来的新风险：
- **角色焦虑**：34%的员工担心被人工智能取代（盖洛普2025年调查）
- **技能过时**：具有超过60%可自动化任务的岗位，员工流失率增加2.3倍
- **人工智能工具疲劳**：在6个月内被迫使用超过3种新人工智能工具的团队，员工倦怠率增加40%
- **技能提升不足**：在人工智能培训上投入低于工资总额2%的公司，高级人才流失速度加快1.8倍

2026年有效的员工留存策略：
- 人工智能技能提升预算（每人每年2000-5000美元）
- 定期（季度）的岗位重新设计研讨会（以协作方式而非自上而下进行）
- 将人工智能带来的效率提升转化为薪酬或休假福利
- 定期安排员工参与内部人工智能项目

## 风险警示指标

关注以下组合情况：
- 员工净推荐值下降 + 休假使用减少 = 职业倦怠加剧
- 某个团队中高级员工离职率集中 = 管理者问题
- Glassdoor上的负面评价增加 + 招聘信息浏览量上升 = 员工流失风险上升
- 晋升率低于5% + 任职超过2年的员工比例增加 = 组织发展停滞
