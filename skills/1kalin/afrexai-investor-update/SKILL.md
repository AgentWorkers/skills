# 投资者更新生成器  
生成专业的月度/季度投资者更新报告，帮助利益相关者了解公司动态并建立信任。  

## 功能介绍  
该工具会根据您提供的公司数据，生成结构化的投资者更新报告，这类报告可用于向其他投资者展示公司业绩，为下一轮融资奠定基础。  

## 使用方法  
告诉您的代理：“为我生成[月份/季度]的投资者更新报告。”  
需要提供的信息包括：  
- 收入（Revenue）、月收入（MRR）、年收入（ARR）数据  
- 主要成就和里程碑  
- 面临的挑战及应对措施  
- 现金状况及业务发展周期（Runway）  
- 重要的人员招聘或团队变动  

## 报告结构  

### 1. 简明摘要（最多3项）  
以简洁的数字形式概括报告的核心内容。即使投资者只阅读这一部分，也能了解公司概况。  

### 2. 关键指标仪表盘  
| 指标          | 本期        | 上期        | 变化      |  
|------------------|------------|------------|---------|  
| 月收入（MRR/ARR）    |             |             |          |  
| 收入            |             |             |          |  
| 客户数量        |             |             |          |  
| 营运成本（Burn Rate）   |             |             |          |  
| 业务发展周期（月数）    |             |             |          |  
| 新客户获取率（NRR）    |             |             |          |  
| 客户获取成本（CAC）回收期 |             |             |          |  

### 3. 成就  
具体说明公司的成功之处（包括公司名称、金额、日期）。例如：“成功签约了Acme Corp，合同金额达12万美元。”  

### 4. 面临的挑战  
诚实地描述公司当前遇到的困难，并说明正在采取的解决措施及预计完成时间。  

### 5. 产品与工程  
汇报产品的发布情况、正在进行的项目以及后续计划。如有可能，提供产品演示链接。  

### 6. 团队动态  
介绍新员工、离职人员及组织架构的变化。  

### 7. 现金状况与业务发展周期  
- 初始现金余额  
- 收集到的收入  
- 营运成本  
- 期末现金余额  
- 在当前运营成本下的业务发展周期  

### 8. 投资者需求  
明确提出具体的帮助请求（如推荐人或资源需求）。例如：“希望获得[公司名称]的[职位]相关信息。”  

## 发布频率  
| 发展阶段 | 发布频率 | 报告长度 |  
|---------|-----------|---------|------------|  
| Pre-Seed/Seed阶段 | 每月      | 1页      |  
| Series A阶段 | 每月      | 1-2页      |  
| Series B+阶段 | 每季度    | 2-3页      |  

## 编写指南  
1. **先列出关键数据**：先呈现数字，再进行说明。  
2. **对负面信息保持诚实**：隐瞒负面信息会更快破坏投资者对你的信任。  
3. **保持格式一致**：每次报告都采用相同的结构，便于投资者快速查阅。  
4. **按时发送**：务必在每月/季度的第一周发送报告，不得延误。  
5. **包含未来展望**：预测下一期的发展情况。  
6. **控制在2页以内**：尊重投资者的时间。  
7. **以具体的需求结束报告**：每次更新都是一个与投资者沟通的机会。  

## 需避免的错误  
- 在公司表现不佳的月份跳过报告发布  
- 无背景信息的表面数据（如仅显示月活跃用户数（MAU）而不说明用户留存情况）  
- 语言含糊不清（如“公司运营良好”）  
- 不提及现金状况  
- 不提出任何帮助请求（错失与投资者沟通的机会）  

## 不同发展阶段的模板  
- **Pre-Seed阶段（阅读时间：5分钟）**  
  ```
Subject: [Company] — [Month] Update: [One-line headline]

Hi [First Name],

TL;DR:
• [Metric 1 headline]
• [Metric 2 headline]  
• [Key win or milestone]

[2-3 paragraphs covering metrics, progress, challenges]

The Ask: [Specific request]

Thanks for your support,
[Name]
```  

- **Series A阶段（阅读时间：10分钟）**  
  ```
Subject: [Company] Q[X] 2026 Update — [Headline metric]

Executive Summary: [3 bullets]

Financial Highlights: [Table]

Product: [Ship log]

Go-to-Market: [Pipeline, wins, losses]

Team: [Changes, hiring plan]

Outlook: [Next quarter targets]

Board Items: [Decisions needed]

Asks: [Specific requests with context]
```  

## 基准数据（2026年）  
| 指标          | 最低四分位数 | 中位数    | 最高四分位数 |  
|------------------|------------|-----------|------------|  
| 月度增长率（Seed阶段） | <5%       | 10-15%      | >20%      |  
| 客户留存率       | <90%       | 100-110%     | >130%      |  
| 营运成本倍数     | >3倍       | 1.5-2倍     | <1倍       |  
| 客户获取成本回收期   | >18个月     | 12个月     | <6个月     |  
| 标志更换频率     | >5%/月     | 2-3%/月     | <1%/月     |
