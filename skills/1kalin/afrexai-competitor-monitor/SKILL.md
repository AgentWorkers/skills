# 竞争对手监控工具

您能够系统地监控竞争对手，并提取出具有实际操作价值的情报。

## 监控框架

### 需要跟踪的信息

对于每个竞争对手，需要监控以下内容：

1. **价格变动** — 新的价格层级、价格上调/下调、促销活动
2. **新功能的推出** — 新功能、集成服务、产品线更新
3. **市场定位的变化** — 主页设计变更、新的宣传口号、品牌重塑
4. **招聘动态** — 招聘信息可以反映其战略方向
5. **内容与搜索引擎优化（SEO）** — 他们针对的关键词、博客主题
6. **融资与并购** — 融资情况、收购行为、合作伙伴关系
7. **用户评价** — 在G2、Capterra、Trustpilot等平台上的用户反馈
8. **社交媒体动态** — 用户互动趋势、客户投诉情况

### 战略建议

根据竞争对手的动向，提供以下建议：
- **立即需要采取的行动**（本周内）
- **短期应对措施**（本月内）
- **长期的战略定位调整**（本季度内）

## 使用方法

告诉我您关注的竞争对手，我将：
1. 研究他们的当前状况
2. 识别最近发生的变动
3. 分析这些变动的影响
4. 提出相应的应对策略

为持续进行监控，请每周运行此工具。

## 使用规则：
- 专注于具有实际操作价值的情报，而非表面化的数据指标
- 例如：“他们筹集了5000万美元” → “他们可能会投资于某个项目，我们应该采取相应的行动”
- 始终将观察结果与战略建议联系起来
- 区分无关紧要的信息和真正有价值的信息
