## 收债流程升级顺序

### 日常操作流程

| 时间 | 操作 | 通道 | 模板 |
|-----|--------|---------|----------|
| 提前7天 | 支付提醒 | 电子邮件 | “温馨提醒——发票#[X]，金额为[$金额]，截止日期为[日期]” |
| 提前1天 | 第一次逾期通知 | 电子邮件 | “发票#[X]现已逾期，请安排付款。” |
| 提前7天 | 第二次通知 + 电话 | 电子邮件 + 电话 | “这是我们的第二次通知，请联系我们讨论。” |
| 提前14天 | 正式催款函 | 电子邮件 + 邮寄 | “正式通知：[$金额]已逾期14天，信用条款可能会受到影响。” |
| 提前21天 | 信用冻结警告 | 电话 + 电子邮件 | “如果7天内未付款，您的账户将被冻结。” |
| 提前30天 | 信用冻结生效 | 电子邮件 + 系统通知 | 账户被冻结，不再接受新订单。通知高级联系人。 |
| 提前45天 | 最终催款 | 录音邮件 | “这是最终催款通知，如未回应，我们将转交催收机构。” |
| 提前60天 | 催收机构介入 | 外部机构 | 将案件提交给催收机构或律师。 |
| 提前90天 | 认为坏账 | 内部处理 | 记录50%的坏账准备金，120天后评估是否核销。 |
| 提前120天 | 核销评估 | 内部处理 | 如果没有付款计划或法律行动，将全额核销。 |

### 逾期账单催收通话脚本

1. “您好[姓名]，我是[公司]的[您的姓名]。我打电话是想讨论编号为#[X]的发票，金额为[$金额]，日期为[日期]。”
2. “我们的记录显示，您已经超过了约定的[净30天]付款期限[X]天。您确认已经收到这张发票吗？”
3. 倾听客户的回答，并根据不同情况做出回应：
   - **“我会去核实一下”** → “请问我什么时候能得到回复？我会记下[日期]并随后跟进。”
   - **“我们遇到了现金流问题”** → “我理解。我们可以制定一个付款计划吗？我们可以分[2-3]期付款。”
   - **“存在争议”** → “让我们现在就解决这个问题。具体是什么问题？我会立即让[我们的团队]介入处理。”
   - **“支票已经在寄出了”** → “您能提供支票号码和寄出日期吗？以便我追踪？”
4. 确认下一步行动和日期，并在1小时内发送电子邮件总结。

## 付款条款优化

### 提前付款折扣计算

标准条款：净30天付款，如果10天内付款可享受2%折扣；否则30天后全额付款。

**不享受折扣的年化成本：**
```
Cost = (Discount % ÷ (100% - Discount %)) × (365 ÷ (Full Days - Discount Days))
2/10 Net 30 = (2/98) × (365/20) = 37.2% annualized
```

这意味着，如果净30天付款条款的年化成本超过2%，那么提供这种折扣是不划算的（这种情况很少见）。更好的选择包括：
- 净10天付款，年化成本为18.4%（更可持续）；
- 净30天付款，年化成本为9.2%（低风险激励）。

## 信用保险

### 何时考虑购买信用保险

- 单个客户的收入占比超过15%；
- 进入付款习惯不明的新市场或地区；
- 向建筑、酒店或零售行业提供信用；
- 客户的信用评分降至50分以下；
- 应收账款超过90天，且占比超过总应收账款的5%。

### 保险类型及费用

| 保险类型 | 保险范围 | 费用（占销售额的百分比） | 适用场景 |
|------|----------|--------------------------|----------|
| 全额保险 | 所有贸易债务人 | 0.1-0.5% | 适用于客户众多的公司 |
| 关键客户保险 | 名列前茅的大客户 | 0.3-1.0% | 客户集中型企业 |
| 单一客户保险 | 特定重要客户 | 0.5-2.0% | 依赖单一高价值客户的业务 |
| 出口信用保险 | 国际应收账款 | 0.3-1.5% | 跨境贸易 |

## 季度信用审查 checklist

- [ ] 获取所有金级/银级/铜级账户的更新信用评分 |
- [ ] 审查付款模式发生变化的客户（DSO增加超过10天的客户） |
- [ ] 关注可能影响客户的行业新闻（如破产、重组或并购） |
- [ ] 根据更新的收入数据重新计算信用额度 |
- [ ] 审查并更新信用保险覆盖范围 |
- [ ] 审查催收流程的执行情况 |
- [ ] 计算坏账准备金，并与实际核销金额进行对比 |
- [ ] 将DSO指标与行业平均水平及上一季度进行对比 |
- [ ] 向财务团队汇报应收账款老化情况 |
