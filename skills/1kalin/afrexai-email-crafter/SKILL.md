# 冷邮件撰写工具

我们能够撰写那些能够被打开、阅读并得到回复的冷邮件——绝不是那种在收件箱里随处可见的、千篇一律的销售邮件。

## 框架：四句话式冷邮件模板

最有效的冷邮件应该简洁明了，最好控制在4到6句话之内。

### 结构：

1. **开场白** — 证明你事先做了充分的准备。提及与对方相关的具体信息（例如对方的最新文章、公司新闻、共同的联系人，或是能反映他们需求的职位信息）。
2. **问题/洞察** — 指出他们可能面临的问题，或者分享一些有价值的见解。此时不要直接推销产品或服务。
3. **衔接** — 用一句话将你的解决方案与他们的问题联系起来。重点应放在结果上，而非功能上。
4. **行动号召（CTA）** — 提出一个清晰、易于执行的请求。例如：“如果您有兴趣，这周可以安排15分钟的通话吗？” 或 “我可以发送一份2分钟的演示材料吗？”

### 规则：

- **主题行**：3到5个单词，使用小写。避免使用吸引点击的标题。主题行应该让人感觉像来自同事的普通邮件，而不是营销信息。
- **不要使用“希望您一切安好”** — 这会立刻暴露这是一封批量发送的邮件。
- **开场白不要以“我”开头** — 用对方的名字或职位来开头。
- **每次只提出一个行动号召** — 如果提出多个请求，对方可能不会采取任何行动。
- **大声朗读邮件内容** — 如果读起来像是由机器人写的，那就重新修改。
- **字数控制在100字以内** — 越简短的邮件，得到的回复越多。
- **初次发送时不要附上附件** — 在他们对你还没有信任之前，不要附加任何文件。
- **适合手机阅读** — 大多数高管都会在手机上阅读邮件。因此，邮件应采用短段落、易读的格式。

### 个性化策略：

**高级目标（高价值客户）**：研究他们的LinkedIn资料、最近的采访记录、公司财报电话会议内容以及博客文章。提及他们说过或做过的事情。

**中等价值目标**：引用他们公司的最新新闻、招聘信息（这些信息能反映他们的关注点），或是与他们行业相关的趋势。

**批量发送目标**：根据他们的职位和行业需求进行个性化沟通。

### 回访流程：

- **第二封邮件（3天后）**：提供更多有价值的内容，分享相关的案例研究、文章或见解。不要只是简单地重复之前的内容。
- **第三封邮件（5天后）**：从新的角度来探讨他们的问题。
- **第四封邮件（7天后）**：说明“目前这个问题可能不是优先事项”——表示理解他们的处境。如果问题再次出现，随时愿意提供帮助。

### 写邮件前需要了解的信息：

- **收件人是谁？**（姓名、职位、公司名称）
- **你销售什么产品/提供什么服务？**
- **你的产品能具体解决他们什么问题？**
- 有哪些个性化的信息可以利用？（例如他们的最新动态、共同的联系人、他们发布的内容）
- **你的行动号召是什么？**（例如安排会议、提供演示、请求回复等）

### 质量检查：

在发送任何邮件之前，请确保：
- **开场白中提到了与收件人相关的具体信息**
- **字数控制在100字以内**
- **有一个明确的行动号召**
- **不要以“我”开头**
- **避免使用专业术语或流行词汇**
- **读起来像真人写的，而不是销售自动化工具生成的**
- **主题行由3到5个单词组成，并使用小写**
