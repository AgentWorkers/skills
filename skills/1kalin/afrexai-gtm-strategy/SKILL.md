# 市场进入策略构建器（Go-to-Market Strategy Builder）

该工具可帮助您为产品发布、市场进入或业务扩展制定全面的营销策略，涵盖市场定位、渠道策略、定价、发布时间表及成功指标等方面。

## 适用场景
- 新产品或新功能的推出
- 进入新的市场领域或地区
- 重新定位现有产品
- 规划价格调整或产品包装优化
- 准备高层管理的营销材料

**渠道选择规则：**
- 发布阶段最多选择2-3个渠道（专注核心渠道而非广撒网）
- 至少有一个渠道能在30天内产生效果
- 至少有一个渠道能在6个月内持续产生销售线索
- 60天内未产生销售线索的渠道应立即停止使用

### 3. 定价与产品包装框架（Pricing & Packaging Framework）

**价值衡量标准：**客户愿意为产品支付多少？（按用户数、使用次数、成果计价）

**定价结构：**
| 定价等级 | 名称 | 价格 | 目标客户群体 | 主要功能 |
|------|------|-------|--------|--------------|
| 免费 | | $0 | 了解产品/试用用户 | |
| 起始级 | | $/月 | 中小企业/个人用户 | |
| 专业级 | | $/月 | 快速成长型企业 | |
| 企业级 | | 定制价格 | 员工超过500人的企业 | |

**定价验证 checklist：**
- [ ] 根据客户支付意愿进行测试（如Van Westendorp模型）
- [ ] 与至少3家竞争对手的价格/功能比率进行比较
- [ ] 定价应确保12个月内收回成本（CAC）
- [ ] 价格升级路径清晰合理
- [ ] 避免设置过低的定价（以免影响转化率）

**“1%法则”：**价格提高1%通常能带来约12.5%的利润增长。价格可以适当提高，但降低价格则较难实现。

### 4. 发布时间表（12周）

**第1-2周：基础准备**
- [ ] 完成市场定位声明
- [ ] 根据客户画像和行为数据确定信息收集框架（ICP）
- [ ] 确定定价（在正式确定前先对5-10位潜在客户进行测试）
- [ ] 创建包含明确行动号召的着陆页
- [ ] 设置分析工具（用于追踪销售线索来源）

**第3-4周：内容与素材准备**
- [ ] 制作销售演示文稿（最多10页）
- [ ] 撰写3个案例研究或产品优势说明
- [ ] 编写邮件营销序列（包括5次联系和7次跟进）
- [ ] 录制产品演示视频（最长2分钟）
- [ ] 准备常见问题解答和异议处理文档

**第5-6周：小规模测试**
- [ ] 向现有客户或关系圈进行产品试用推广
- [ ] 进行10次试用邀请电话
- [ ] 对着陆页标题进行A/B测试
- [ ] 根据反馈调整市场定位
- [ ] 培训销售团队如何进行产品介绍和处理客户异议

**第7-8周：扩大营销渠道**
- [ ] 启动主要的外展销售渠道
- [ ] 开始投放付费广告（预算有限，先测试广告创意）
- [ ] 发布首批SEO内容
- [ ] 通知邮件列表和社交媒体粉丝
- [ ] 开始与合作伙伴洽谈合作

**第9-10周：优化营销策略**
- [ ] 分析每个销售环节的转化情况
- [ ] 停用效果不佳的渠道或营销内容
- [ ] 增加表现良好的渠道的投入
- [ ] 收集并发布首批客户评价
- [ ] 根据实际销售数据调整定价

**第11-12周：报告与调整**
- [ ] 进行全面销售数据分析（从访问量到潜在客户、销售机会、成交）
- [ ] 按渠道计算客户获取成本（CAC）
- [ ] 分析成交原因
- [ ] 根据数据调整第二季度的营销计划
- [ ] 编写可复用的营销文档

**终止策略：**
- 如果某个渠道的CAC超过目标值的3倍，则立即停止使用该渠道
- 如果20次机会中只有不到10次成交，则重新调整市场策略
- 如果30天内试用用户转化为付费用户的比例低于5%，则优化产品引导流程
- 如果90天后的净推荐值（NPS）低于30%，则在此之前改进产品

### 6. 常见的市场进入策略错误
1. **面向所有人发布产品**：选择一个小目标市场进行重点推广，然后再逐步扩展。
2. **同时使用过多渠道**：做好2个渠道的效果往往优于同时使用6个效果不佳的渠道。
3. **定价过低**：低价会传递出产品价值低的信号，吸引不合适的客户。
4. **缺乏销售支持**：营销产生的潜在客户如果无法转化，就是浪费资源。
5. **忽视现有客户**：拓展现有客户的收入通常是新客户的3倍。
6. **关注错误的指标**：关注流量和点击量无实际意义，应关注销售线索和收入。
7. **没有明确的终止标准**：事先明确哪些策略无效，否则会一直投入资源在无效的渠道上。
