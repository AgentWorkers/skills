# 客户成功操作手册

本手册针对B2B SaaS产品，详细介绍了如何构建和运营客户成功团队，涵盖了客户全生命周期的管理流程，包括入职引导、健康状况评估、季度业务回顾（QBR）、流失预防以及业务扩展策略。

## 适用场景
- 从零开始建立客户成功团队
- 为新客户设计入职引导流程
- 构建客户健康状况评估模型
- 制定季度业务回顾模板和频率
- 制定流失预防方案
- 规划业务扩展和升级销售策略

## 工作流程

### 1. 评估当前状况
向用户了解以下信息：
- 公司发展阶段（种子期、A轮融资阶段、成长阶段、规模化阶段）
- 当前年经常性收入（ARR）和客户数量
- 平均合同价值（ACV）
- 当前客户成功团队规模（或无）
- 已知的流失主要原因（如有）

### 2. 设计客户生命周期阶段
绘制客户旅程图：

| 阶段 | 持续时间 | 负责人 | 关键指标 |
|-------|----------|-------|------------|
| 入职引导 | 0-30天 | 客户成功负责人（CSM） | 首次产生价值的时长 |
| 产品采用 | 30-90天 | CSM | 特色功能采用率 |
| 客户留存 | 持续进行 | CSM | 客户健康状况评分（Health Score）、净新增客户率（NRR） |
| 业务扩展 | 根据触发条件 | CSM + 销售代表（AE） | 扩展年经常性收入（Expansion MRR） |
| 续订 | 续订前90天 | CSM | 续订率 |
| 客户推荐 | 续订后 | 客户成功负责人 | 客户净推荐值（NPS） |

### 3. 构建客户健康状况评分模型
对以下指标进行加权评分（总分0-100）：
- **产品使用情况**（25%）：日活跃用户（DAU）/月活跃用户（MAU）比例
- **功能使用情况**（15%）：实际使用的功能与可用功能的比例
- **支持需求**（15%）：工单数量和客户反馈
- **高层管理层参与度**（15%）：与高层的沟通频率
- **客户满意度**（10%）：净推荐值（NPS）或客户满意度调查（CSAT）结果
- **业务增长迹象**（10%）：用户数量或使用量的增长
- **支付情况**（10%）：付款是否按时

评分标准：80分以上表示客户健康状况良好，60-79分为需要关注，40-59分为风险等级，低于40分为严重风险。

### 4. 制定入职引导检查清单
- 第1周：基础工作（项目启动、技术设置、管理员培训、明确3个成功目标）
- 第2-3周：激活阶段（用户培训、配置工作流程、实现首个快速成果）
- 第4周：交接工作（回顾总结、设定基线指标、制定后续沟通计划）

目标：小型企业（SMB）的首次产生价值时间（TTFV）小于14天，大型企业（Enterprise）小于30天。

### 5. 制定季度业务回顾（QBR）模板
QBR会议议程（45分钟）：
1. 回顾业务成果及投资回报率（ROI）数据（10分钟）
2. 产品使用情况分析及未充分利用的功能（10分钟）
3. 产品路线图预览（仅涉及相关内容）（5分钟）
4. 客户的优先事项和业务变化（15分钟）
5. 制定行动计划及负责人分配（5分钟）

**规则**：在提交QBR报告时，必须提供量化的投资回报率数据。

### 6. 流失预防策略
- **等级1（评分60-79）**：自动发送提醒通知，由CSM跟进
- **等级2（评分40-59）**：在48小时内进行个性化沟通，并制定30天的恢复计划
- **等级3（评分<40）**：在24小时内向高层汇报，准备好相应的补救措施

**预警信号**：使用量下降超过30%、关键员工离职、客户提及竞争对手、错过沟通机会。

### 7. 业务扩展策略
- 当客户健康状况评分超过80分、使用率超过85%，或在QBR中提出新的使用需求时，可采取以下扩展措施：
  | 措施 | 触发条件 | 执行方式 |
  |------|---------|----------|
  | 扩大用户规模 | 使用率超过85% | 进行针对性的沟通 |
  | 功能升级 | 客户频繁使用现有功能 | 展示升级后的解决方案 |
  | 新部门拓展 | 由关键客户推荐 | 共同召开介绍会议 |
  | 多年合同客户 | 客户健康状况良好且合同即将到期 | 提供折扣激励 |

**CSM的扩展目标**：新ARR的20-30%。**净新增客户率（NRR）目标**为110-120%。

### 8. CSM团队配置建议
- **大型企业（ACV超过10万美元）**：负责10-15个账户，采用高频沟通方式，每月进行一次QBR
- **中型企业（ACV 2.5万-10万美元）**：负责25-40个账户，采用中等频率的沟通方式，每季度进行一次QBR
- **小型企业（ACV 5万-2.5万美元）**：负责50-80个账户，采用技术支持为主的方式，每半年进行一次QBR
- **自助服务型企业（ACV低于5万美元）**：采用自动化数字化沟通方式

### 9. 输出成果
为客户提供以下资料：
- 客户生命周期图
- 带有权重的客户健康状况评分模型
- 定制化的入职引导检查清单
- QBR模板及准备工作清单
- 流失预防方案及升级流程
- 业务扩展策略清单
- CSM团队配置方案
- 绩效指标仪表盘（净新增客户率（NRR）、总新增收入（GRR）、首次产生价值时间（TTFV）、客户健康状况覆盖率、QBR完成率）

## 关键指标
| 指标 | 目标值 |
|--------|--------|
| 净新增客户率（NRR） | >110% |
| 总新增收入率（GRR） | >90% |
| 首次产生价值时间（TTFV） | 小型企业<14天，大型企业<30天 |
| 客户健康状况评分覆盖率 | 100% |
| QBR完成率 | >85% |
| 客户净推荐值（NPS） | >50 |
