# 合同审查助手

该工具用于分析商业合同中的风险、不利条款以及缺失的条款，并提供一份用通俗语言编写的合同摘要。

## 功能介绍
- 读取合同内容（可粘贴文本或提供文件路径）
- 标记高风险条款：自动续订、竞业禁止、无限责任、知识产权转让等
- 检查缺失的保护性条款：终止权、数据所有权、服务水平协议（SLA）保障等
- 评估整体风险等级（低/中/高/直接放弃合同）
- 生成包含建议性修改条款的谈判清单

## 使用方法
- 粘贴合同文本，或输入指令：“审查位于 [文件路径] 的合同”

## 审查框架

### 1. 快速概览
- 合同各方、生效日期、合同期限
- 双方同意的条款（用通俗语言表述）
- 总体财务承诺

### 2. 风险提示（红色/黄色/绿色）
**红色提示（需协商或直接放弃合同）：**
- 无限责任或无上限的赔偿条款
- 作品成果的知识产权自动转让
- 竞业禁止期限超过12个月或覆盖范围过广
- 单方终止权（对方可以终止合同，而你不能）
- 自动续订条款，需提前60天以上通知
- 强制在不利司法管辖区进行仲裁
- “最惠国”定价条款

**黄色提示（如可能，请协商修改）：**
- 支付期限超过30天
- 保密条款的有效期超过3年
- 控制权变更条款
- 未经合理通知的审计权
- 不合理的不可抗力条款（范围过窄或过宽）

**绿色提示（标准、可接受）：**
- 双方均可随时终止合同，需提前30天通知
- 标准的责任限制条款（12个月的费用赔偿）
- 合理的保密条款（2年，不包括公开信息）
- 明确的数据所有权和数据返还条款

### 3. 缺失条款检查
- [ ] 为方便而设定的终止条款
- [ ] 数据所有权和数据迁移条款
- [ ] 包含补救措施的服务水平协议（如退款、终止权等）
- [ ] 责任上限条款
- [ ] 保险要求
- [ ] 争议解决流程
- [ ] 修改流程（书面形式，双方共同参与）
- [ ] 权利转让限制

### 4. 谈判清单
针对每个红色/黄色提示，生成具体的修改建议及市场标准的替代方案。

### 5. 整体风险评分
对合同进行评分，并提供简明的建议。

## 输出格式
以结构化报告的形式呈现结果，使用表格对各项条款进行详细分析。将风险等级用粗体标出。
