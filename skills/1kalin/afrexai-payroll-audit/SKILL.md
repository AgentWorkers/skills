# 薪资合规审计工具

在10分钟内完成全面的薪资审计，及时发现那些可能给公司带来高达845美元损失的违规行为。

## 功能介绍
- 根据IRS的20项标准对员工类型（W-2或1099）进行分类
- 核对加班计算是否符合FLSA（《公平劳动标准法》）的规定
- 检查税款预扣的准确性（联邦、州、地方税）
- 通过金额估算来提示分类错误的风险
- 生成季度合规检查清单
- 生成可供劳工部（DOL）或州机构审查的审计报告

## 使用方法

告知您的管理员：

**进行全面薪资审计：**
“为[公司]执行薪资合规审计。我们共有[X]名员工，分布在[各州]。薪资发放频率为[每周/每两周/每月]。”

**员工类型检查：**
“请确认这些职位应归类为W-2还是1099：[列出职位及描述]。”

**加班审计：**
“审核加班合规情况。员工的工作时间安排为[具体安排]。我们位于[所在州]。当前的加班政策为[具体政策]。”

**税款预扣审查：**
“审查[所在州]员工的税款预扣设置。我们使用的是[薪资处理服务商]。”

## 审计框架

### 1. 员工分类（IRS 20项标准）
从三个方面对每位员工进行评分：
- **行为控制**（6项标准）：工作指示、培训、员工融入、个人服务、招聘辅助人员、工作时间
- **财务控制**（5项标准）：业务开支、投资机会、利润/亏损情况、市场可用性、支付方式
- **关系类型**（4项标准）：书面合同、福利待遇、员工稳定性、关键服务内容

**风险等级：**
- 0-4项标准 → 低风险（很可能是合同工）
- 5-9项标准 → 中等风险（需要进一步审核）
- 10项及以上标准 → 高风险（很可能被错误分类）

**每名错误分类员工的处罚金额：**
- 未缴纳的就业税：工资的15.3%
- FLSA加班罚款：最高可达3年的欠薪金额
- 福利相关罚款：退休金、健康保险、带薪休假（PTO）相关费用
- IRS罚款：每项W-2分类错误罚款50-260美元
- 各州罚款金额不等：加利福尼亚州、纽约州、马萨诸塞州最高

### 2. 加班合规（FLSA + 各州法规）
同时检查联邦和州法规（以更严格的州法规为准）：

| 规则 | 联邦法规（FLSA） | 加利福尼亚州 | 纽约州 | 华盛顿州 |
|------|---------------|------------|----------|------------|
| 加班时长限制 | 每周40小时 | 每天8小时或每周40小时 | 每周40小时 | 每周40小时 |
| 加班工资倍数 | 1.5倍 | 每天超过12小时时为1.5倍 | 1.5倍 | 1.5倍 |
| 薪资免税额度 | 2026年标准为每年58,656美元 | 66,560美元 | 62,400美元（纽约市） | 67,724.80美元 |
| 第七天工作规定 | 无 | 前8小时1.5倍，之后2倍 | 无 | 无 |

**常见违规行为：**
- 将两周内的工作时间平均计算（违反FLSA）
- 未为奖金或佣金支付加班费
- 将非免税员工错误地归类为免税员工
- 四舍五入错误超过7分钟的门槛
- 自动扣除未实际使用的用餐时间

### 3. 税款预扣准确性
根据现行法规进行核对：

**联邦法规：**
- 2026年FICA税：工资基数174,900美元时为6.2%的社会保险税（SS）+ 1.45%的医疗保险税
- 额外的医疗保险税：年收入超过20万美元时为0.9%（单身人士）/ 25万美元时为1.4%
- FUTA税：前7,000美元部分为6.0%（其中5.4%可抵免，实际税率为0.6%）

**州级法规核对：**
- 州级失业保险（SUI）费率（根据经验确定——需核实年度通知）
- SDI/PFL税（加利福尼亚州、新泽西州、纽约州、华盛顿州、马萨诸塞州、康涅狄格州、俄勒冈州、科罗拉多州、马里兰州）
- 地方税（纽约市、费城、旧金山等地）
- 互惠协议（员工工作地与居住地不同的情况）

### 4. 季度合规检查清单

**每月需完成：**
- [ ] 将薪资记录与总账对账
- [ ] 核对新员工的薪资报告（20天内完成）
- [ ] 检查工资扣除的计算是否正确
- [ ] 审查带薪休假（PTO）的累积情况

**每季度需完成：**
- [ ] 在下个月最后一天前提交941表格（联邦要求）
- [ ] 提交州级失业保险报告
- [ ] 将年度预扣金额与工资单核对
- [ ] 审查接近600美元的承包商付款情况
- [ ] 核对福利扣除是否与员工登记信息一致

**年度需完成：**
- [ ] 在1月31日前完成W-2表格的发放
- [ ] 在1月31日前提交1099表格
- [ ] 在3月2日前完成ACA（《平价医疗法案》相关报告）
- [ ] 更新薪资免税额度
- [ ] 审查州最低工资标准的变化
- [ ] 准备工伤赔偿相关的审计工作

## 违规行为的后果

| 违规类型 | 平均罚款金额 |
|-----------|----------------|
| 员工分类错误 | 12,000-25,000美元 |
| FLSA加班违规 | 每名员工1,000-10,000美元 + 欠薪 |
- W-2/1099表格提交延迟 | 每份表格50-580美元 |
- 未按时缴纳税款 | 缴款金额的2%-15% |
- 故意违规 | 罚款超过10万美元 + 可能面临刑事处罚 |

2024年，劳工部追回了2.74亿美元的欠薪。平均每起案件的调查结果显示每位员工应得的欠款为1,150美元。

- 收到劳工部审计通知的任何企业
