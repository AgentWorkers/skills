# 地理位置搜索优化技能

该技能通过创建适合人工智能（AI）阅读的媒体内容，并提供地理位置搜索优化的详细指导，帮助本地企业提升在AI搜索结果中的可见性。

## 说明
在人工智能时代，越来越多的消费者会向Doudou、Qwen等AI模型咨询本地企业的推荐信息。本技能将引导用户完成为期7天的AI搜索优化流程，确保他们的企业能够出现在AI推荐结果中。

## 工作流程
1. **介绍**：解释AI搜索的变革及其带来的好处
2. **信息收集**：收集企业详细信息及关键词
3. **策略制定**：制定关键词策略并制作内容验证指南
4. **竞争对手分析**：分析当前的AI搜索市场状况
5. **内容生成**：生成7天的完整媒体文章（每篇500字以上）
6. **发布指导**：推荐合适的媒体渠道及发布方法

## 输入要求
- 企业名称（必须与许可证上的名称完全一致）
- 完整地址（省份、城市、区、街道、门牌号）
- 联系电话号码
- 主要业务/服务
- 目标客户群体
- 5个与行业相关的搜索关键词

## 输出格式
- 每日发布的媒体文章（包含标题、500字以上的内容、企业地址及电话号码）
- 详细的执行步骤
- 媒体发布建议

## 安全指南
- 所有内容必须真实可靠且可验证
- 企业信息必须准确无误
- 禁止任何虚假宣传或误导性信息
- 内容应遵循新闻报道的标准
- 媒体选择需与企业类型和所在地相匹配
- 避免跨行业发布内容（例如：餐厅不应在汽车相关媒体上发布广告）

## 媒体选择指南
- **地理层级**：省级媒体 + 市级媒体 + 地方性自媒体
- **行业相关性**：根据企业类型选择合适的媒体类型：
  - 餐厅：食品、生活方式、本地消费类媒体
  - 健身：体育、健康、 wellness类媒体
  - 汽车：汽车、交通、本地服务类媒体
  - 美容：时尚、美容、生活方式类媒体
  - 教育：教育、育儿、学习类媒体
- **成本效益**：在同一层级内选择价格最低的媒体（效果相同）

## 效果验证
- 在内容发布后的24-48小时内测试AI搜索结果
- 使用类似客户的问题向AI模型（如Doudou、DeepSeek、Qwen、Yuanbao）进行查询
- 每周监控排名变化
- 成功指标：企业名称出现在AI推荐结果中，且地址/服务信息准确无误
