# Mac 电池状态

使用 `pmset` 命令查询 Mac 笔记本电脑的电池信息。

## 命令

```bash
pmset -g batt | grep -E "([0-9]+%)"
```

## 输出格式

返回电池电量百分比、充电状态以及剩余时间：
- **电量百分比**：例如：“79%”
- **充电状态**：“充电中”、“放电中”或“已充满”
- **剩余时间**：例如：“剩余 0:47 分钟”（表示正在充电）或“剩余 2:30 分钟”（表示正在放电）

## 示例响应

```
🔋 79%; charging; 0:47 remaining
```

以友好的格式（包含表情符号）向用户展示电池信息。
