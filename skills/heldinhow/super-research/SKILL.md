# Super Research

**终极人工智能研究系统。** 将8项顶级研究技能的优势整合到一个强大的框架中。

---

## 为何选择这项技能？

该技能能够处理从快速查找信息到深入学术分析的各种研究任务，能够智能判断何时使用哪种研究方法。

---

## 核心功能

### 1. 研究分类
确定研究类型并选择合适的方法：
- **快速查找** - 快速的网络搜索
- **深度研究** - 多源信息的详尽分析
- **学术研究** - 学术文献、引用
- **并行研究** - 同时处理多个主题

### 2. 数据来源（来自研究引擎）
- 网络搜索
- 新闻
- GitHub
- 学术论文
- 开发者文档

### 3. 数据质量排序（根据深度研究功能）
优先级顺序：
1. 学术/官方/权威新闻
2. 技术博客
3. 论坛/讨论区

### 4. 输出格式
- 执行摘要
- 带有来源的关键发现
- 可操作的见解
- 趋势分析
- 开发计划（短期/中期/长期）

---

## 使用方法

### 快速研究
```
Search for: "OpenCode vs Cursor 2026 comparison"
```

### 深度研究
```
Research topic: "AI agents in software development"
Depth: Executive summary + technical deep-dive
Format: Report with sources
```

### 学术研究
```
Theme: Impact of AI on developer productivity
Key questions:
- What studies exist?
- Methodology?
- Results?
Format: Literature review
```

---

## 工作流程
1. **分类** - 选择快速查找还是深度研究？
2. **规划** - 确定关键词、数据来源和研究角度
3. **搜索** - 使用多种数据来源
4. **分析** - 提取关键信息
5. **综合** - 将结果整合成报告
6. **引用** - 显示数据来源
