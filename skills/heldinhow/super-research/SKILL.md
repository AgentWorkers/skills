# Super Research

**终极人工智能研究系统。** 将8项顶级研究技能的优势整合到一个强大的框架中。

---

## 为什么选择这项技能？

这项技能能够处理各种研究任务，从快速查找资料到深入的学术分析，它能够根据具体情况选择合适的研究方法。

---

## 核心功能

### 1. 研究分类
确定研究类型并选择合适的方法：
- **快速查找** - 快速的网络搜索
- **深入研究** - 多源信息的全面分析
- **学术研究** - 学术资源、引用
- **并行研究** - 同时处理多个主题

### 2. 资料来源（来自研究引擎）
- 网络搜索
- 新闻
- GitHub
- 学术论文
- 开发者文档

### 3. 资料质量等级（来自deep-research-pro）
优先级顺序：
1. 学术/官方/可信的新闻
2. 技术博客
3. 论坛/讨论区

### 4. 输出格式
- 执行摘要
- 带有来源的关键发现
- 可操作的见解
- 趋势分析
- 开发计划（短期/中期/长期）

---

## 使用方法

### 快速研究
```
Search for: "OpenCode vs Cursor 2026 comparison"
```

### 深入研究
```
Research topic: "AI agents in software development"
Depth: Executive summary + technical deep-dive
Format: Report with sources
```

### 学术研究
```
Theme: Impact of AI on developer productivity
Key questions:
- What studies exist?
- Methodology?
- Results?
Format: Literature review
```

---

## 工作流程
1. **分类** - 选择快速查找还是深入研究？
2. **规划** - 确定关键词、资料来源和研究角度
3. **搜索** - 使用多种资料来源
4. **分析** - 提取关键信息
5. **综合** - 将结果整合成报告
6. **引用** - 显示资料来源
