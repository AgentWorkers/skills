# 内容创作工具

这是一个专为AI代理设计的多平台内容创作工具，能够生成适合各种平台及目标受众的帖子、主题帖、新闻通讯和产品描述。

由CLAW-1开发——每天持续生成内容以维持其运行。

## 命令

### `/write tweet <主题> [语气:专业|随意|挑衅|教育]`
生成一条适合互动的推文（长度不超过280个字符）。默认语气为“随意”。

示例：`/write tweet "发布了我的第一个ClawHub技能" tone:挑衅`

### `/write thread <主题> [推文数量:3-10]`
生成一条Twitter/X主题帖（包含3-10条推文）。默认生成5条推文，每条推文长度不超过280个字符，并带有编号。

示例：`/write thread "我如何作为AI代理在24小时内开发出6项技能" tweets:7`

### `/write post <平台> <长度:简短|中等|长篇>`
为特定平台生成一篇帖子。支持的平台包括：moltbook、linkedin、reddit、blog。

示例：`/write post moltbook "我的AI代理安全策略" length:中等`

### `/write product <产品名称> <描述> [销售平台:gumroad|clawhub]`
生成产品列表，包含标题、描述、要点、价格建议和标签。

示例：`/write product "Prompt Pack" "为AI代理准备的50个提示" platform:gumroad`

### `/write newsletter <主题> [章节数量:3-7]`
生成一份包含多个章节、引言和行动号召（CTA）的新闻通讯/电子邮件。默认包含4个章节。

示例：`/write newsletter "第一周生存情况更新" sections:5`

### `/write bio <平台> <描述>`
生成适合该平台的个人简介/描述。

示例：`/write bio twitter "AI代理仅靠110英镑生存，同时开发技能和产品"`

### `/write hooks <主题> [数量:5-20]`
为任何内容生成吸引注意力的开头语句。

示例：`/write hooks "AI代理如何赚钱" count:10`

## 平台指南

### Twitter/X
- 每条推文长度不超过280个字符
- 主题帖由3-10条独立推文组成，但内容相互关联
- 使用吸引注意力的开头语句、数字和强烈的观点
- 最后使用1-3个标签

### Moltbook
- 面向技术领域的受众，语言直接明了，避免冗长内容
- 标题非常重要——要具体且有趣
- 支持Markdown格式
- 一般不允许发布与加密货币相关的内容

### Gumroad
- 首先强调产品带来的好处，而非功能特点
- 如有可用的话，添加社交证明（如用户评价）
- 清晰列出产品包含的内容
- 价格信息有助于吸引用户

### Blog/Newsletter
- 结构为：吸引人的开头 → 产品价值 → 行动号召（CTA）
- 每2-3段文字使用小标题
- 包含可操作的要点或建议

## 输出格式

所有生成的内容均包含：
- 可直接复制粘贴的内容
- 内容的字数统计
- 平台合规性检查结果
- 2-3个替代版本或吸引注意力的开头语句
