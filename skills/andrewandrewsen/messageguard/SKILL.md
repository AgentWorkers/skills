### MessageGuard：出站消息过滤工具

**功能**：MessageGuard 通过基于模式的检测以及可配置的操作（屏蔽、阻止或警告），来过滤出站的文本信息，以防止机密信息泄露和敏感数据的暴露。

**高级配置选项：**

- **`mode`**：决定匹配到的模式应采取的全局操作。可选选项包括：
  - `mask`：用指定的 `mask_char` 替换敏感数据。
  - `block`：完全阻止消息的发送。
  - `warn`：允许消息发送，但会生成警告。
- **`mask_char`**：当 `mode` 设置为 `mask` 时，用于替换敏感内容的字符。
- **`patterns`**：定义或自定义基于正则表达式的检测规则。系统内置了一些针对 API 密钥、凭据等内容的检测规则（例如 AWS 密钥、JWT 等）。您可以根据需要添加新的检测规则。
- **`logging`**：启用日志记录功能，将检测结果以结构化的 JSON 格式输出，便于监控、调试或满足合规性要求。请配置 `log_path` 以指定日志文件的保存位置。
- **自定义模式**：用户可以定义自己的自定义模式，以覆盖内置的检测规则或扩展工具的功能。这些自定义模式支持正则表达式，并允许对操作类型（屏蔽、阻止或警告）进行细粒度控制。

**安装方法：**
1. 克隆仓库：`git clone git@github.com:AndrewAndrewsen/MessageGuard.git`。
2. 进入克隆后的目录。该工具不依赖任何第三方库，仅依赖于 Python 标准库。
