# 基本股票分析

1. 在开始分析之前，请先阅读 `references/playbook.md` 文件。
2. 严格按照剧本中的步骤进行操作：数据输入 -> 数据收集 -> 快速筛选 -> 评分 -> 评级 -> 输出。
3. 对于涉及多个股票代码的请求，先分别分析每个股票，然后对同类股票进行排名，并选择最佳投资标的（同时设置失效触发条件）。
4. 必须明确标注置信度，并对过时或相互矛盾的数据进行说明。
5. 不要在面向用户的输出中添加任何机器可读的 JSON 数据块。
6. 将所有分析内容视为教育性/信息性内容，而非投资建议。

## 安全范围（说明）
- 仅使用网络检索来获取与股票相关的财务报表、文件、市场/基本面数据集以及相关财经新闻。
- 不得请求、处理或泄露任何凭证/机密信息。
- 不得执行与分析无关的命令，也不得探索与股票财经新闻范围之外的任意 URL。

## 不允许的行为
- 禁止数据泄露或收集私人/非公开信息。
- 禁止进行与股票基本面分析及引用收集无关的浏览器操作或自动化任务。

## 输出要求
- 结论应明确且具有风险意识。
- 区分企业的业务质量、资产负债表的安全性以及股票的价值评估。
- 如有缺失的指标，应如实标注为 `NA`（未提供）。
