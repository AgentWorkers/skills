# Agent Brain 🧠  
*你的人工智能的个人“记忆”系统，永远不会遗忘*  

## 问题  
每次对话都仿佛是从零开始——你总是重复自己说过的话，而人工智能却会忘记你昨天教给它的内容。  

## 解决方案

## 你将获得的功能  
| 功能          | 好处                                      |  
|---------------|-----------------------------------------|  
| **永不重复自己**     | 只需教一次，就能永远记住                         |  
| **捕捉自相矛盾之处**   | 当你自相矛盾时，人工智能会发出警告                 |  
| **信心水平追踪**     | 能够判断人工智能的回答是确定性的还是不确定性的           |  
| **习惯学习**       | 记住你的工作流程                             |  
| **语气识别**       | 根据你的情绪调整回答方式                         |  

## 各模块的功能  
| 模块          | 功能                                      | 示例                                      |  
|---------------|-----------------------------------------|  
| **档案**        | 存储事实与知识                               | “记住：Alex 更喜欢阅读散文”                         |  
| **信号检测**      | 检测语言中的自相矛盾之处                         | 如果你自相矛盾，系统会发出警告                     |  
| **信心评估**      | 评估人工智能的回答信心水平                         | 适当时会说“我不太确定”                         |  
| **习惯学习**      | 学习并记住你的工作习惯                         | “Alex 总是先进行调研”                         |  
| **情绪识别**      | 识别你的情绪并调整回答方式                         | 根据你的情绪调整回答的语气                         |  
| **信息获取**      | 获取外部链接（⚠️ 该功能已关闭）                       | “获取链接：https://...”                         |  

## 数据存储（仅限本地）  
所有数据都存储在你的设备上：  
- 文件：`memory/index.json`  
- 默认情况下不进行云同步  
- 你可以自行控制数据存储的位置  

### 可选功能：SuperMemory 同步  
如果你需要云备份，可以启用 SuperMemory：  
1. 单独安装 SuperMemory 插件  
2. 修改档案模块以启用同步功能  
3. 数据将保持本地存储，除非你主动选择上传到云端  

## 命令操作  
```
"Remember: <fact>"        → Store a fact
"Check conflicts"          → Look for contradictions
"How confident are you?"   → Gauge module
```  

## 安全性  
- 信息获取模块默认处于关闭状态  
- 仅在执行明确指令时才会启动  
- 所有数据均存储在本地  
- 不会自动进行云同步
