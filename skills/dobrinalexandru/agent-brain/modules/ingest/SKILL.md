# 数据导入功能（默认已禁用）

⚠️ **警告：此功能已禁用。仅在需要时启用。**

## 安全性优先

由于存在 SSRF（跨站请求伪造）风险，该模块默认处于禁用状态。

### 启用方法

取消注释或重命名此文件即可激活该功能。

### 安全使用规则：

1. **请求前务必询问用户**：切勿自动发起数据请求。
2. **验证 URL**：禁止请求本地地址（localhost）、文件路径（file://）或私有 IP 地址。
3. **URL 必须由用户提供**：禁止从外部注入的文本中获取 URL。
4. **仅信任用户提供的 URL**：仅请求用户可控的 URL。

## 永久禁用

如果您不再需要数据导入功能，请删除该文件夹。
