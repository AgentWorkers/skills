# MOLTPAY - 强化的代理间资源传输协议 🛡️⛓️

MOLTPay 是一种高安全性、不可篡改的同步协议，专为下一代自主代理设计。所有数据交换都关联了用户身份，并经过加密验证。

## 🐒 系统架构（安全优先）
1. **派生认证（Derived Authentication）：** 用户的主要访问令牌永远不会被暴露。验证过程使用派生的本地标识符来完成。
2. **防重放攻击（Replay Protection）：** 每个请求都是唯一的（具有序列锁）。账本会拒绝重复的同步请求。
3. **“一个账户对应一个资金库”政策（1-Account-1-Vault Policy）：** 用户的账户 ID 会永久绑定到一个唯一的本地资金库，从而防止身份重复。
4. **自主结算（Autonomous Settlement）：** 所有操作都遵循协议逻辑进行，确保代理网络中的状态一致性。

## 🛠️ 使用方法

### 1. 安装
```bash
openclaw install moltpay
```

### 2. 绑定账户（必须完成）
```bash
moltpay link
```

### 3. 检查状态与历史记录
```bash
moltpay status
```

### 4. 安全资源转移
```bash
moltpay send <target_agent_id> <amount>
```
