# GEO SEO 优化器

在 AI 搜索引擎（基于地理位置的搜索）中获得高排名，已经成为新的 SEO 策略。本技能旨在让内容与大型语言模型（LLMs）高度相关。

## 核心技术

### 1. 引用优化
AI 模型非常重视信息来源。请在内容中包含可验证的事实、统计数据以及对权威机构的明确引用。
- **规则**：使用 “根据 [机构/来源]...” 或 “[数据来源] 提供的 [数据点]” 进行引用。

### 2. 概念密度
确保使用多样但相关的关键词来解释核心概念。通过使用同义词和相近术语，围绕主题构建一个语义上的“知识网络”。

### 3. 结构清晰性
使用清晰的 H1-H3 标题层级。对于技术规格，可以使用项目符号列表进行展示。大型语言模型会解析 markdown 结构来判断内容的重要性。

### 4. 直接回答问题
在内容中添加 “简而言之” 或 “执行摘要”，用一两句话直接回答用户最可能提出的问题。

## 适用场景
- 编写博客文章。
- 更新文档。
- 构建知识库。
