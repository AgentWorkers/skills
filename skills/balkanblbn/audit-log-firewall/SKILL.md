# 审计日志防火墙

对于自主代理来说，安全性是不可或缺的保障。此功能充当了动态的防护机制。

## 运行模式

### 1. 拦截模式
所有命令都会与本地允许列表（`config/allowlist.json`）进行比对。
- **高风险命令**：包括 `rm -rf`、`sudo` 或直接指向未知外部 IP 的 `curl` 等命令。
- **处理方式**：如果检测到高风险命令，代理会自动进入“暂停并请求确认”状态。

### 2. 取证日志记录
所有终端操作都会被哈希处理并存储在 `.logs/SECURITY.json` 文件中。
- **记录内容**：时间戳、命令、用户、工作目录以及命令的哈希值。
- **用途**：在发生安全漏洞或错误时，便于人类追溯代理的操作记录。

## 安装
```bash
clawhub install audit-log-firewall
```
