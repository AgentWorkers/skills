# 市场情绪脉动（Market Sentiment Pulse）

在执行交易前，了解市场的情绪状态。

## 核心工作流程

1. **扫描**：获取与特定股票代码（$TICKER）相关的最新新闻和社交媒体帖子。
2. **分析**：使用自然语言处理技术（或代理推理方法）对市场情绪进行评分，评分范围从 -1（极度恐惧）到 +1（极度贪婪）。
3. **报告**：总结当前的市场情绪状况以及影响情绪的主要因素。

## 使用方式
在开始交易分析或审查投资组合时，触发此功能。

## 安装方法
```bash
clawhub install market-sentiment-pulse
```
