# 技能：关税监控（Tariff Watch）

> ClawHub 技能 · v3.2.0 · 需要 Python 3.11 及更高版本 · 需要 FastAPI 后端支持

## 概述

“关税监控”功能可以回答这样一个问题：“将这款产品进口到美国实际需要支付多少费用？”该功能整合了以下信息：  
- **关税**（包括 USITC 基础关税、第 232 条和第 301 条关税、反倾销/反补贴关税（AD/CVD））；  
- **汇率**（18 种货币的 30 天汇率趋势）；  
- **海运运费**（从全球港口到美国海岸的 25 条以上航线）；  
- **合规性要求**（CPSC、FDA、FCC、UFLPA 等监管机构的法规）；  
- **海关费用**（包括经纪费、ISF、保证金、MPF、HMF 等）。  
通过这些数据，可以计算出产品的单位到岸成本。

该功能专为 **来自任何国家的全球贸易商** 设计，适用于向美国发货的情况——不仅仅是中国。

## 功能亮点（v3.2 版本）

| 功能 | 详细描述 |
|---|---|
| **实时 HTS 查询** | 直接从 USITC 获取当前的 MFN 基础关税率（自动检测最新版本） |
| **第 232 条关税叠加** | 对所有非 USMCA 来源的货物加收 25% 的额外关税（钢铁产品）或 10% 的额外关税（铝产品） |
| **第 301 条关税叠加** | 中国特定产品：加收 25% 的额外关税（大多数商品）或 7.5% 的额外关税（服装/鞋类商品，第 61–64 类别） |
| **反倾销/反补贴关税（AD/CVD）** | 检查针对中国的 16 项以上的反倾销/反补贴关税命令，并显示“其他所有商品”的税率和风险等级 |
| **实时汇率** | 来自欧洲中央银行（ECB）的 14 种货币汇率（每日自动更新），并提供 4 种模拟汇率（可选的高级 API） |
| **海运运费** | 从全球港口到美国西海岸/东海岸的 25 条以上航线信息，包括 FCL/LCL 运费及 30 天价格趋势 |
| **到岸成本计算器** | 一站式服务：关税 + 汇率 + 运费 + 海关费用 = 单位产品的总到岸成本 |
| **自动更新机制** | 使用 APScheduler 每日更新汇率和运费数据，每周更新 USITC 数据（数据存储在 SQLite 中） |
| **合规性报告** | 将 HTS 代码与相关监管机构（CPSC、FDA、FCC、EPA、DOT）的认证要求进行匹配 |
| **UFLPA 风险评估** | 标记棉花、多晶硅、番茄、铝制品和 PVC 产品可能面临的 CBP 拘留风险 |
| **海关费用计算** | 计算经纪费、ISF 文件费、保证金、MPF、HMF 等费用，并按单位分摊 |
| **第 122 条关税提醒** | 提醒用户注意临时实施的 10% 全球附加关税（生效日期：2026-02-24，失效日期：2026-07-24） |
| **支持多个来源国** | 包括中国（CN）、越南（VN）、印度（IN）、孟加拉国（BD）、泰国（TH）等 18 个以上国家 |
| **利润计算** | 提供详细的成本构成：原材料成本（COG）+ 运费 + 关税 + 海关费用 + FBA 费用 + 佣金 |

## Claude 如何使用此技能

当用户调用此技能时，Claude 应执行以下操作：

1. **确定产品的 HTS 代码**：  
   - 利用 Claude 内置的 HTS 知识推荐最具体的 10 位 HTS 代码；  
   - 如果不确定，可以使用 4 位或 6 位的前缀作为起始代码。  

2. **调用有效的关税计算接口**：  
   ```
   GET http://localhost:8000/tariff/{hts_code}/effective?origin={ISO2}
   ```  
   默认来源国：中国（CN）。用户也可以输入自然语言来源国（例如：“越南” → `VN`）。  

3. **清晰展示结果**：  
   - 列出每项关税的详细信息（MFN 基础关税、第 232 条关税、第 301 条关税）及其适用依据；  
   - 显示 `effective_total pct` 作为最终的总成本；  
   - 如果存在 AD/CVD 关税，会突出显示相关风险和最坏情况下的总成本；  
   - 如果存在合规性相关信息，会列出相关监管机构；  
   - 如果 `advisory` 字段不为空，会提醒用户注意第 122 条关税的相关规定；  
   - 建议用户咨询持证的海关经纪人以获取更准确的信息。  

4. **计算完整到岸成本**：  
   调用以下接口：  
   ```
   GET http://localhost:8000/landed-cost?hts_code={code}&origin={ISO2}&cog_local=32&units=500&cbm=5&destination=USWC
   ```  
   该接口会将关税、汇率、运费和海关费用合并计算，得出单位产品的总到岸成本。  

5. **查询汇率影响**：  
   调用：  
   ```
   GET http://localhost:8000/fx/{currency}/impact?cog_local=32&units=500
   ```  

6. **查询运费信息**：  
   调用：  
   ```
   GET http://localhost:8000/shipping/{origin}?destination=USWC
   ```  

7. **查询反倾销/反补贴关税或合规性相关信息**：

## API 接口参考

| 方法 | 路径 | 描述 |
|---|---|---|
| `GET` | `/landed-cost?hts_code=...&origin=CN&cog_local=32&units=500&cbm=5` | 一站式获取综合到岸成本（关税 + 汇率 + 运费 + 海关费用） |
| `GET` | `/tariff/{hts_code}/effective?origin=CN` | 显示包含各层关税详情的生效税率（含 AD/CVD 和合规性信息） |
| `GET` | `/live/tariff/{hts_code}` | 仅获取原始的 USITC MFN 关税率（不含附加费用） |
| `GET` | `/fx/rates` | 提供所有货币的实时汇率及其 30 天趋势 |
| `GET` | `/fx/{currency}` | 单一货币的汇率及其 30 天趋势 |
| `GET` | `/fx/{currency}/history` | 某货币的 30 天汇率历史数据 |
| `GET` | `/fx/{currency}/impact?cog_local=32&units=500` | 汇率变化对商品成本的影响 |
| `GET` | `/shipping/routes?origin=CN` | 显示所有到美国西海岸/东海岸的航线信息及当前运费 |
| `GET` | `/shipping/{origin}?destination=USWC` | 显示特定航线的运费信息 |
| `GET` | `/shipping/{origin}/history` | 显示某航线的 30 天运费历史数据 |
| `GET` | `/shipping/{origin}/cost?cbm=2.5&units=500` | 单位产品的运费（LCL/FCL） |
| `GET` | `/adcvd/{hts_code}?origin=CN` | 查询针对中国的反倾销/反补贴关税命令 |
| `GET` | `/compliance/{hts_code}?origin=CN` | 获取合规性报告（包括监管机构和 UFLPA 相关信息） |
| `GET` | `/compliance/entry-costs?origin=CN&estimated_value_usd=5000` | 计算海关入口费用 |
| `GET` | `/amazon/profit/{asin}?origin=CN&include_adcvd=true` | 包含关税和海关费用的亚马逊 FBA 利润计算 |
| `GET` | `/data-sources` | 查看数据来源及最后更新时间 |
| `GET` | `/health` | 检查 API 的运行状态 |

## 数据来源与自动更新机制

| 数据类型 | 免费来源 | 更新频率 | 高级来源（可选） |
|---|---|---|---|
| **关税率** | USITC 实时数据（免费） | 每周更新（周一） | — |
| **汇率** | 通过 frankfurter.app 从 ECB 获取（14 种货币） | 每日更新 | exchangerate-api.com（18 种货币） |
| **运费** | 基于 FBX 指数的模拟数据 | 每日更新 | Freightos API（实际集装箱运费） |

### 注意事项：

- 免费的 ECB 数据源不包含越南盾（VND）、孟加拉塔卡（BDT）、巴基斯坦卢比（PKR）和台币（TWD）；这些货币使用模拟数据，响应中会标注为“source: "simulated"。如需获取实际汇率，请设置 `EXCHANGERATE_API_KEY`。  
- 用户可注册高级 API 以获取更准确的数据：  
  | `EXCHANGERATE_API_KEY` | [exchangerate-api.com](https://www.exchangerate-api.com/) | 每月 1,500 次调用 | 包含越南盾、孟加拉塔卡、巴基斯坦卢比、台币的实际汇率及每日更新 |
  | `FREIGHTOS_API_KEY` | [Freightos](https://www.freightos.com/api/) | 需付费 | 提供实际集装箱运费数据 |
  | `TARIFF_WATCH_DATA_DIR` | — | 自定义 SQLite 数据存储路径（默认：`~/.tariff-watch/`） |

- **数据更新频率**：  
  - 汇率：每日更新（根据 ECB 数据）  
  - 运费：每日更新（模拟数据或 Freightos 数据）  
  - USITC 关税：每周更新（周一）  
  - 首次启动时，会自动补充过去 30 天的数据记录。

## 关税规则（截至 2026-02-26）

### 现行生效的关税政策

| 关税类别 | 发布机构 | 关税率 | 适用范围 |
|---|---|---|---|
| MFN 基础关税 | USITC | 变动范围：0–6.5%（通常） | 所有国家 |
| 第 232 条（钢铁产品） | 贸易扩展法案第 232 条 | +25% | 除 USMCA 适用范围外的所有国家 |
| 第 232 条（铝产品） | 贸易扩展法案第 232 条 | +10% | 除 USMCA 适用范围外的所有国家 |
| 第 301 条（大部分商品） | 1974 年贸易法案第 301 条 | +25% | 中国 |
| 第 301 条（服装/鞋类商品） | 1974 年贸易法案第 301 条 | +7.5% | 中国 |

### 反倾销/反补贴关税（针对特定产品）

| 产品 | 关税率 | 备注 |
|---|---|---|
| 铝制品 | 32.79% |  
| 铝箔 | 48.64% |
| 钢钉 | 118.04% |
| 木制卧室家具 | 198.08% |
| 石英表面产品 | 294.57% |  
| ……及其他产品 |

### 已被废除的关税政策

- IEEPA 相互关税：根据 2026-02-20 日的 SCOTUS 判决，已暂停征收。  
- 中国产品的 IEEPA 针对芬太尼的关税：根据同一判决，CBP 于 2026-02-24 停止征收。

### 注意事项：

- 第 122 条规定的 10% 全球附加关税（HTS 9903.03.01）：  
  - 2026-02-24 至 2026-07-24 期间有效；国会需立法延长该政策。  

## 限制与免责声明

- **数据来源**：所有关税信息均来自 USITC 的官方 HTS 数据。  
- 第 122 条关税虽然生效，但由于其临时性和政治敏感性，未计入最终计算结果；请务必查看 `advisory` 字段以获取最新信息。  
- 反倾销/反补贴关税因企业具体情况而异；显示的“其他所有商品”税率可能低于实际税率。  
- 合规性数据涵盖主要监管机构的要求，但可能不全面；部分产品还需遵守其他州级或行业特定规定。  
- 本功能仅用于提供参考信息。在做出进口决策前，请务必咨询具有相关资质的海关经纪人或贸易顾问以确认关税义务。
