---

## 什么是供应商风险指数（SRI）  

供应商风险指数（SRI）是一个结构化的评分系统，它根据五个维度为每个供应商生成0-100的风险评分，并将供应商分为绿色（Green）、黄色（Yellow）或红色（Red）三个等级，并为每个等级指定具体的应对措施。在供应商入职时、每年以及供应商情况发生重大变化时，都应运行该系统。  

**解决的问题：** 没有统一的框架，运营团队难以有效管理供应商风险。SRI消除了基于直觉的风险评估，为采购团队提供了客观、可辩护的方法来优先处理供应商监督和做出采购决策。  

**输出：** 每个供应商的风险评分（0-100分）、等级分类（绿色/黄色/红色）以及相应的行动计划。

## 第一维度：财务稳定性（25分）  

**重要性：** 财务不稳定的供应商可能无法履行合同、保持产品质量或持续经营。财务不稳定是供应链中断的主要原因。  

**评估内容：**  
| 指标 | 评估方法 |  
|-----------|----------------|  
| 公司成立时间 | 运营年限 |  
| 收入稳定性 | 收入是否稳定增长 |  
| 资金来源/所有权 | 自筹资金、风险投资支持、公开上市 |  
| 信用风险信号 | 是否拖欠供应商款项、是否有法律诉讼 |  
| 依赖集中度 | 是否严重依赖单一客户？ |  

**评分标准：**  
| 条件 | 分数 |  
|-----------|--------|  
| 公司成立5年以上，收入稳定增长，无财务问题 | 25分 |  
| 公司成立3-5年，收入稳定 | 18-22分 |  
| 公司成立1-3年（初创企业），获得风险投资 | 10-17分 |  
| 公司存在财务压力（拖欠付款、重组、亏损传闻） | 3-9分 |  
| 公司宣布破产、进入破产程序或资不抵债 | 0-2分 |  

**数据来源：**  
- Dun & Bradstreet Paydex评分（商业信用）  
- Dun & Bradstreet或Experian商业信用报告  
- LinkedIn或公开新闻中的财务困境线索  
- 美国证券交易委员会（SEC）文件（上市公司）  
- 小型供应商的自报财务报表（年支出2.5万美元以上：要求提供过去两年的财务报表）  
- 其他主要客户的推荐信  

**评分行动：**  
- 如果供应商在D1维度上的评分低于15分，在授予新合同前，应提交给财务部门审核。

## 第二维度：单一来源依赖性（20分）  

**重要性：** 如果您依赖某个供应商提供关键产品或服务且没有替代方案，那么您将面临风险。任何财务、运营或关系上的问题都可能立即带来业务风险。  

**评估内容：**  
| 因素 | 评估问题 |  
|--------|----------|  
| 可替代性 | 如果该供应商消失，您能多快找到替代品？ |  
| 替代方案 | 市场上有多少合格的替代供应商？ |  
| 收入集中度 | 您在该供应商上的支出占比是多少？ |  
| 依赖性后果 | 如果该供应商停止供货，您的运营会受何影响？ |  
| 替换成本 | 过渡到替代供应商所需的时间和成本 |  

**评分标准：**  
| 条件 | 分数 |  
|-----------|--------|  
| 有多个合格替代供应商，可在30天内替换 | 20分 |  
| 有一些替代供应商，替代时间为30-90天，替换成本适中 | 13-19分 |  
| 替代供应商很少，替代时间为90-180天，替换成本较高 | 6-12分 |  
| 无替代供应商，依赖程度高，替代时间为180天以上 | 0-5分 |  

**依赖性乘数（如果同时满足以下两个条件适用）：**  
- 该供应商是关键产品/服务的唯一来源  
- 该供应商在该类别的支出占比超过30%  
→ **将D2维度得分降低5分**（最低分为0分）  

**评分行动：**  
- 任何在D2维度上得分低于5分的供应商都应制定书面应急计划。  
- 适用依赖性乘数的供应商应启动备用供应商识别项目。

## 第三维度：合规历史（20分）  

**重要性：** 合规性问题是潜在问题的早期信号，表明流程薄弱、管理不善或存在冒险文化。有过合规问题的供应商未来再次出现问题的可能性更大。  

**评估内容：**  
| 领域 | 需要检查的内容 |  
|------|--------------|  
| 保险合规 | 保险合同缺失、过期、续费延迟 |  
| 监管合规 | 违反行业规定、罚款、监管处罚 |  
| 法律记录 | 诉讼、判决、和解 |  
| 数据/安全事件 | 数据泄露、审计失败、安全违规 |  
| 合同合规 | 与供应商的过往合同关系、合同终止原因 |  
| 许可证 | 在所有相关司法管辖区持有有效许可证 |  

**评分标准：**  
| 条件 | 分数 |  
|-----------|--------|  
| 3年以上无已知合规问题 | 20分 |  
| 有轻微问题，已完全解决（3年内1-2次） | 14-19分 |  
| 中等程度的问题（1-2次监管警告、轻微诉讼）——已解决 | 8-13分 |  
| 严重问题（重大诉讼、监管处罚、保险合同过期）——已解决 | 3-7分 |  
| 持续存在的合规问题、正在进行的诉讼或近期严重违规 | 0-2分 |  

**数据来源：**  
- 内部供应商记录（合规性跟踪、过往问题）  
- 法院记录查询（PACER网站）  
- Better Business Bureau  
- 州级许可证管理局  
- Google新闻搜索（例如：“[供应商名称] 诉讼/违规/罚款/数据泄露”）  
- 行业特定数据库（如FDA针对食品/制药行业，OSHA针对承包商等）  

**评分行动：**  
- 任何在D3维度上得分低于7分的供应商在合同续签前都需要进行法律审查。

## 第四维度：绩效记录（20分）  

**重要性：** 过往的绩效是预测未来表现的最可靠指标。质量稳定、交付准时且问题解决迅速的供应商风险较低。  

**评估内容：**  
| 指标 | 评估方法 |  
|--------|---------------|  
| 准时交付率 | 按时交付的交付物/发票比例 |  
| 质量缺陷率 | 过去12个月内报告的质量问题数量 |  
| 问题解决时间 | 解决问题的平均天数 |  
| 沟通响应速度 | 对查询和升级的响应时间 |  
| 合同遵守情况 | 是否按合同约定交付？ |  
| 客户满意度 | 内部利益相关者的评价 |  

**评分标准（针对有绩效记录的现有供应商）：**  
| 条件 | 分数 |  
|-----------|--------|  
| 持续超出预期，每年问题少于2次，解决迅速 | 20分 |  
| 达到预期，每年2-5次小问题，及时解决 | 14-19分 |  
| 大部分符合预期，偶尔有问题，解决时间中等 | 8-13分 |  
| 表现不稳定，问题频繁，解决缓慢，内部团队有投诉 | 3-7分 |  
| 存在重大持续性问题，关系处于风险中 | 0-2分 |  

**对于新供应商（无内部记录）：**  
- 默认得分为12分（中性）  
- 根据推荐信调整得分：推荐信良好则加3分，推荐信不佳则减3分  
- 在入职后的90天内进行绩效评估并更新得分  

**评分行动：**  
- 任何在D4维度上得分低于7分的供应商都应制定绩效改进计划（参见“供应商绩效审计”技能）。  

---

## 第五维度：地理与监管风险（15分）  

**重要性：** 供应商的运营地点和注册地可能带来风险——政治不稳定、监管变化、自然灾害风险、数据主权要求以及贸易合规复杂性。  

**评估内容：**  
| 因素 | 风险指标 |  
|--------|----------------|  
| 运营国家 | 政治稳定性、制裁风险、贸易限制 |  
| 数据主权 | 数据是否出境？是否受GDPR、CCPA、HIPAA等法规约束？ |  
| 自然灾害风险 | 是否位于高风险地区（飓风、地震、洪水） |  
| 监管环境 | 所在行业是否受到严格监管？ |  
| 货币/外汇风险 | 支付是否使用波动性货币？ |  
| 出口控制 | 是否涉及ITAR、EAR等出口管制？ |  

**地理风险参考：**  
| 供应商所在地 | 风险等级 | 起始得分 |  
|----------------|------------|----------------|  
| 美国、加拿大、英国、欧盟（稳定） | 低 | 12-15 |  
| 澳大利亚、新西兰、日本、韩国 | 低 | 12-15 |  
| 墨西哥、巴西、印度 | 中等 | 8-11 |  
| 东欧、中东（稳定国家） | 中等偏高 | 5-9 |  
| 中国（数据处理问题、监管风险） | 高 | 3-6 |  
| 受美国制裁或政治不稳定的国家 | 非常高 | 0-2 |  

**监管复杂性调整因素：**  
| 条件 | 调整分数 |  
|-----------|-----------|  
| 供应商从事高度监管的行业（医疗、金融、国防） | -2分 |  
| 供应商处理跨境个人数据 | -2分 |  
| 供应商有严格的出口控制要求 | -3分 |  
| 供应商有完善的监管合规计划 | +2分 |  

**评分行动：**  
- 任何在D5维度上得分低于5分的供应商在合同执行前都应接受法律或合规部门的审查。

## 各等级的推荐行动  

### 🟢 绿色（75-100分）：低风险  
- **审查频率：** 每年  
- **监督级别：** 标准合同管理  
- **行动：**  
  - 包含在标准的季度绩效评估中  
  - 监控D1/D2/D3维度是否触发风险事件  
  - 将评分记录在供应商档案中  
  - 有资格获得优先供应商地位、延长合同期限、增加采购量  

### 🟡 黄色（50-74分）：中等风险  
- **审查频率：** 每半年  
- **监督级别：** 积极监控  
- **行动：**  
  - 识别得分最低的维度并重点改进  
  - 要求与供应商会面讨论风险问题  
  - 对于D2维度的问题：开始寻找备用供应商  
  - 对于D3维度的问题：要求提供合规文件  
  - 对于D4维度的问题：启动绩效改进讨论  
  - 为特定维度设定90天的改进目标  
  - 在得分改善前，不得增加采购量或授予新合同  

### 🔴 红色（低于50分）：高风险  
- **审查频率：** 每月  
- **监督级别：** 积极风险管理  
- **行动：**  
  - 立即上报给上级经理  
  | 通知依赖该供应商的内部利益相关者  
  | 启动应急计划（寻找备用供应商）  
  | 暂停新的采购订单，等待改进计划  
  | 向供应商发送正式的风险通知  
  | 设定60天的改进期限  
  | 如果90天内得分未提升至黄色，建议更换供应商

## 触发事件（立即重新评分）  

在定期审查之外，当出现以下情况时，应立即重新评估供应商：  
- 有财务困难的消息（裁员、资金削减、破产传闻）  
- 保险合同到期或合规问题  
- 供应商的主要客户宣布更换供应商  
- 供应商高层发生重大变动  
- 发生监管行动或公开诉讼  
- 发生数据泄露或安全事件  
- 发生合并、收购或所有权变更  
- 供应商的运营受到自然灾害影响  
- 团队报告严重的质量或交付问题

## 投资组合级风险分析  

评估所有供应商后，进行投资组合审查：  

### 风险分布目标  
| 等级 | 目标 | 超过目标的处理方式 |  
|------|--------|--------------------|  
| 🟢 绿色 | 投资组合中占比超过70% | — |  
| 🟡 黄色 | 投资组合中占比低于25% | 首先处理风险最高的黄色等级供应商 |  
| 🔴 红色 | 投资组合中占比低于5% | 立即采取补救措施或更换供应商 |  

### 依赖性分析  
- 根据年度支出金额，确定前5大供应商  
- 如果前5大供应商中有多个属于红色等级 → 提高优先级  
- 如果超过50%的支出集中在SRI评分低于75分的供应商身上 → 发出投资组合风险警报  

### 单一来源审计  
- 列出所有在D2维度上得分低于5分的供应商  
- 这些是您的关键单一来源依赖供应商  
- 每个供应商都应在90天内制定书面应急计划
