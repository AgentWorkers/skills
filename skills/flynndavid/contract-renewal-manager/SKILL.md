# 合同续签与到期管理器  
**框架：续签准备协议**  
**输出：** 一个主动的续签日历 + 决策框架 + 重新谈判准备包  

错过合同续签会带来实际损失——自动续签可能面临不利的费率、紧急寻找替代供应商的情况，以及失去谈判优势。本系统为您提供了一套结构化的流程，以主动管理所有供应商合同的续签工作：包括提醒触发、续签决策、重新谈判准备以及相关文档记录。  

---

## 第一阶段：续签清单与日历设置  

**步骤1——建立续签日历**  
对于每份有效的供应商合同，记录以下信息：  
| 字段 | 需要记录的内容 |  
|-------|---------------|  
| 供应商名称 | 法定名称 + 商业名称（DBA） |  
| 合同开始日期 | |  
| 合同结束日期 | |  
| 自动续签条款 | 是/否；如果是，需要通知期限 |  
| 通知截止日期 | 合同结束日期减去通知期限 |  
| 年度价值（ACV） | 或合同总价值 |  
| 类别 | 软件 / 服务 / 供应商 / 专业服务 |  
| 负责人 | 负责该供应商的内部利益相关者 |  

**优先级标记：**  
- 年度价值（ACV）超过1万美元的合同，或包含自动续签条款的合同，属于优先级（提前90天审核）；其他合同属于标准级（提前60天审核）。

## 第二阶段：90/60/30天提醒协议  

**续签前90天：**  
- 负责人收到自动日历提醒  
- 触发条件：进行供应商绩效审计（参见“供应商绩效审计”相关内容）  
- 决策：该供应商是否仍是最佳选择？标记为需要审核的对象。  
- 行动：安排续签决策会议。  

**续签前60天：**  
- 完成续签决策（是续签、重新谈判还是更换供应商）  
- 如果选择续签原合同：确认条款并起草续签通知  
- 如果选择重新谈判：开始准备工作（进入第三阶段）  
- 如果选择更换供应商：开始寻找替代供应商，并同步启动新供应商的入职流程。  

**续签前30天：**  
- 所有文件需完成并签署  
- 如果合同包含自动续签条款：在此日期前发送书面通知  
- 确认新的供应商已准备好（如果需要更换）  
- 更新下一周期的续签日历。  

**紧急情况处理协议（在续签前30天内发现问题）：**  
如果在续签前30天内发现合同问题：  
1. 立即通知负责人  
2. 评估：能否暂时维持按月续签的安排？自动续签是否可行？  
3. 如果需要更换供应商：立即启动紧急寻源流程  
4. 将该供应商标记为优先级处理对象（后续优先处理）。

## 第三阶段：续签决策框架  
对每份合同从四个维度进行评分（每个维度1-5分）：  
| 维度 | 1分 | 3分 | 5分 |  
|-----------|---------|---------|---------|  
| **绩效** | 经常低于服务水平协议（SLA） | 大多数时间符合SLA | 始终超出SLA |  
| **价值** | 价格高于市场水平 | 市场平均水平 | 价格低于市场水平但具有较高性价比 |  
| **战略契合度** | 不符合我们的业务方向 | 中立/基本符合需求 | 对我们的运营至关重要 |  
| **合作关系** | 合作困难 | 专业、基于交易的关系 | 拥有良好的合作关系 |  

**决策矩阵：**  
- 16-20分：续签，考虑扩大合作范围  
- 11-15分：续签，评估是否需要重新谈判  
- 6-10分：重新谈判或评估其他替代方案  
- 1-5分：更换供应商——开始寻找替代供应商。

## 第四阶段：重新谈判准备  
如果决定重新谈判，请在与供应商沟通之前完成以下准备工作：  

**市场分析：**  
- 找到2-3个类似的替代方案并获取报价。  
- 当前市场与该合同签订时的市场价格对比情况如何？  
- 你的最佳替代方案（BATNA）是什么？  

**价值评估：**  
- 在合同期间从该供应商那里获得了哪些价值？（尽可能量化）  
- 出现了哪些问题？记录问题发生的时间及对业务的影响。  
- 希望对哪些方面进行修改？按优先级排序：价格 → 条款 → 服务水平协议（SLA） → 合作范围。  

**谈判开场策略：**  
- 在首次沟通前明确表达你的要求  
- 首先强调从该供应商那里获得的价值，然后说明当前条款与期望条款之间的差距  
- 以市场数据为依据进行谈判，而不仅仅是基于个人偏好。  

**让步策略：**  
- 你愿意做出哪些让步？（例如：长期合作以换取更低价格、承诺增加采购量以获得折扣等）  
- 哪些条款是不可谈判的？（提前明确，避免在谈判中陷入僵局）。

## 第五阶段：文档模板  

**续签通知函（适用于需要书面通知的合同）：**  
```
[Date]
[Vendor Name]
[Address]

RE: Contract Renewal Notice — [Contract ID / Service Description]

Dear [Vendor Contact],

This letter serves as formal notice of our intent to [renew / not renew] the agreement dated [Contract Start Date] for [Service Description].

[If renewing:] We look forward to continuing our relationship under the same terms for an additional [term]. Please confirm receipt of this notice.

[If not renewing:] Please consider this our formal notice of non-renewal per Section [X] of our agreement. We will coordinate transition details separately.

Regards,
[Name, Title]
[Company]
```  

**内部续签总结（供利益相关者参考）：**  
- 供应商名称、合同ID、合同期限、年度价值（ACV）  
- 续签决策及理由  
- 谈判后的变更内容（如有）  
- 下一次续签日期  
- 负责人的签字确认
