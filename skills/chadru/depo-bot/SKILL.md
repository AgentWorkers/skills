# Depo Bot — 证词报告起草助手

该工具帮助诉讼团队更快地制作庭审准备报告。您只需提供证词记录并说明自己在案件中的角色，Depo Bot便会为您起草一套完整的分析报告以供您审阅。

## 功能介绍

- **战略备忘录**：按重要性、案件主题和推荐的诉讼策略对关键发现进行排序。
- **证人分析**：审查证词内容，指出其中的矛盾之处及语言使用模式。
- **事实摘要**：整理事件的时间线、相关方关系以及信息缺失的部分。
- **法律研究摘要**：分析证据问题、适用的法律标准及相关判例。
- **审讯大纲**：按主题领域分类的问题清单。
- **执行摘要（.docx格式）**：适用于向庭审团队分发的Word文档。

## 使用对象

需要处理证词记录的诉讼律师和法律团队。该工具负责处理繁琐的证词记录审查工作，您仍可完全掌控案件策略和专业判断。

## 使用方法

1. 将证词记录文件放入相应的案件文件夹中。
2. Depo Bot会自动审查该文件并提取相关信息。
3. 您会收到按主题分类的结构化报告。
4. 您可仔细审阅每份报告，并将其中的信息纳入庭审准备工作中。

## 入门指南

完整的设置说明和工作流程文档位于项目仓库中。

## 数据处理规则

- 您的证词记录文件将保存在您的本地项目文件夹中。
- 项目仓库中不会存储任何案件数据。
- 所有报告输出文件都将保存在您的本地项目文件夹中。
