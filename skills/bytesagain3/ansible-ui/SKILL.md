# 信号量（Semaphore）

这是一个专为 Ansible、Terraform/OpenTofu/Terragrunt、PowerShell 等 DevOps 工具设计的现代用户界面（UI）和强大 API。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 显示命令信息
- `status` - 查看命令状态

## 特点

- 该工具的核心功能源自 ansible-uiui/ansible-ui 库。

## 使用方法

运行任意命令：`ansible-ui <命令> [参数]`
