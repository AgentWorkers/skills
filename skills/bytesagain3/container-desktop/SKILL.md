# Podman Desktop

Podman Desktop 是一款专为开发者设计的免费开源工具，用于处理容器（Containers）和 Kubernetes。它提供了直观、友好的用户界面，让您能够轻松地在桌面环境下构建、管理和部署容器及 Kubernetes 应用程序。

## 命令

- `help` - 显示帮助信息
- `run` - 运行容器
- `info` - 查看容器信息
- `status` - 查看容器状态

## 主要功能

- 兼容 container-desktop 和 container-desktop 的核心功能

## 使用方法

运行任意命令：`container-desktop <命令> [参数]`
