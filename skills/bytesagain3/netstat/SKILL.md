---# NetStat

NetStat 是一个用于查看网络统计信息和连接状态的工具。它可以显示当前活跃的连接、监听端口、连接状态、带宽使用情况以及网络接口的详细信息。

## 快速入门

运行 `netstat help` 可以查看可用的命令及其使用示例。

## 主要特性

- 快速且轻量级：基于纯 Bash 语言开发，并内置了 Python 功能；
- 无需任何外部依赖；
- 数据存储在用户主目录下的 `~/.netstat/` 文件夹中；
- 支持 Linux 和 macOS 系统。

## 使用方法

```bash
netstat help
```
