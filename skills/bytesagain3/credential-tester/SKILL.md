# Mimikatz

这是一个用于测试Windows系统安全性的小工具

## 命令

- `help` - 显示帮助信息
- `run` - 运行指定命令
- `info` - 显示工具信息
- `status` - 显示工具状态

## 功能

- 其核心功能源自gentilkiwi/credential-tester项目

## 使用方法

运行任意命令的格式为：`credential-tester <命令> [参数]`
