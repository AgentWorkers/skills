# ohmytoken-tracker

**将您的人工智能使用情况以像素艺术的形式实时呈现出来。**

您是否曾经好奇自己的代币都去哪儿了？ohmytoken 将您在人工智能模型中使用代币的情况转化为一个迷人的像素艺术界面，每当您进行聊天、编写代码或使用 AI 功能时，界面上的代币数量就会增加。

![ohmytoken 控制面板](https://ohmytoken.dev/preview.png)

## 您会喜欢它的原因

**它具有独特的吸引力。** 每个模型都有自己独特的颜色——Claude 是珊瑚色，GPT 是绿色，Gemini 是蓝色。您可以观察界面上的代币如何以螺旋状、雨滴状或从中心向外扩散的方式逐渐填充。您还可以选择自己喜欢的界面形状，比如猫形、心形、马里奥蘑菇形或星形。您的代币艺术作品是独一无二的。

**它具有社交功能。** 您可以将界面生成为像素艺术卡片，并通过二维码分享给他人。查看自己在开发者中的排名，解锁诸如“夜猫子”（凌晨 2 点编程）或“百万富翁”（消耗 100 万代币）等成就。还可以浏览其他用户的界面作品。

## 设置步骤：3 个步骤，2 分钟完成

### 第 1 步：获取 API 密钥

1. 访问 [ohmytoken.dev](https://ohmytoken.dev)
2. 点击 **“GITHUB”** 或 **“GOOGLE”** 进行登录（一键登录，无需密码）
3. 系统会显示一个欢迎页面，其中包含您的 API 密钥（格式类似 `omt_f9c399...`）
4. 点击 **COPY** 以复制密钥

### 第 2 步：安装该工具

在终端中运行以下命令：

```bash
openclaw skill install @0x5446/ohmytoken-tracker
```

### 第 3 步：添加 API 密钥

打开您的 `openclaw.json` 文件，并添加以下代码块：

```json
{
  "skills": {
    "ohmytoken-tracker": {
      "config": {
        "api_key": "omt_paste_your_key_here"
      }
    }
  }
}
```

**完成！** 现在您可以访问 [ohmytoken.dev](https://ohmytoken.dev)，并在使用 AI 功能时实时查看代币数量的增减情况。

## 隐私与安全

**我们仅收集代币使用情况的数据，不会收集其他任何信息。**

| 我们收集的信息 | 我们不收集的信息 |
|----------------|----------------------|
| 模型名称（例如 “gpt-4o”） | 您的输入内容 |
| 代币数量（输入内容 + 输出结果） | 您的回答内容 |
| 时间戳 | 您提供给服务的 API 密钥 |
| | 您的文件、代码或数据 |
| | 您的 IP 地址 |
| | 任何对话内容 |

该工具每次请求仅发送 **4 个数据点**：模型名称、输入内容的代币数量、输出结果的代币数量以及时间戳。您的输入内容、回答、文件和代码 **永远不会离开您的设备**。

源代码完全公开：[github.com/0x5446/ohmytoken-oss](https://github.com/0x5446/ohmytoken-oss) — 您可以自行审核代码。

## 您将获得什么

- **实时代币显示界面**：每当您使用代币时，它们会以彩色像素的形式显示在界面上
- **7 种界面形状**：正方形、猫形、心形、星形、圆形、菱形、蘑菇形
- **7 种填充动画效果**：顺序填充、螺旋填充、从中心向外填充、随机填充、蛇形填充、对角线填充、雨滴状填充
- **4 种时间视图**：今日、本月、本年、历史总计
- **10 项成就**：从“首次使用 AI”到“百万富翁”
- **排行榜**：总代币数量、使用效率、模型多样性、夜间使用频率
- **分享卡片**：包含像素艺术图像、二维码和用户个人资料的卡片
- **可嵌入的徽章**：适用于 GitHub README 的 SVG 徽章
- **年度代币使用报告**：为您生成的个性化代币使用报告
