# ohmytoken

**将您使用 AI 的消耗情况以像素艺术的形式实时呈现出来。**

您是否曾经好奇自己的代币都用在了哪里？ohmytoken 能将您在使用 AI 进行聊天、编码或创作时的消耗情况转化为一个迷人的像素艺术界面，随着您的操作逐渐“填充”起来。

![ohmytoken 仪表盘](https://ohmytoken.dev/preview.png)

## 您会喜欢它的原因

- **极具吸引力**：每个模型都有自己独特的颜色——Claude 是珊瑚色，GPT 是绿色，Gemini 是蓝色。您可以观看代币以螺旋状、雨滴状或从中心向外扩散的方式填充到界面上。您还可以选择自己喜欢的界面形状（如猫形、心形、马里奥蘑菇形或星形）。您的代币艺术作品是独一无二的。
- **具有社交功能**：您可以生成带有二维码的像素艺术卡片，并与他人分享。查看自己在开发者中的排名，解锁诸如“夜猫子”（凌晨 2 点还在编程）或“百万富翁”（消耗了 100 万代币）等成就。您还可以浏览所有用户的界面作品。

## 设置步骤（3 步，2 分钟）

### 第 1 步：获取 API 密钥

1. 访问 [ohmytoken.dev](https://ohmytoken.dev)
2. 点击 **“GITHUB”** 或 **“GOOGLE”** 进行登录（一键登录，无需密码）
3. 系统会显示一个包含您 API 密钥的欢迎页面（密钥格式为 `omt_f9c399...`）
4. 点击 **COPY** 以复制密钥

### 第 2 步：安装该工具

在终端中运行以下命令：

```bash
openclaw skill install @0x5446/ohmytoken
```

### 第 3 步：设置 API 密钥

建议将 API 密钥设置为环境变量（这样不会通过 Git 泄露）：

```bash
export OHMYTOKEN_API_KEY="omt_paste_your_key_here"
```

将以下代码添加到您的 `~/.zshrc` 或 `~/.bashrc` 文件中，然后执行 `source ~/.zshrc`。

**完成！** 打开 [ohmytoken.dev](https://ohmytoken.dev)，开始使用 AI 时即可看到代币的实时变化。

## 隐私与安全

**我们仅收集代币使用情况的数据，绝不收集其他任何信息。**

| 我们收集的信息 | 我们绝不会收集的信息 |
|-------------|-------------------|
| 模型名称（例如 “gpt-4o”） | 您的输入内容 |
| 输入生成的代币数量（整数） | 您的回复内容 |
| 完成生成的代币数量（整数） | 您提供给服务提供商的 API 密钥 |
| | 您的文件、代码或数据 |
| | 任何对话内容 |

该工具每次请求仅发送 **3 个字段**：`model`、`prompt_tokens`、`completion_tokens`。您的输入内容、回复、文件和代码 **永远不会离开您的设备**。

**注意**：与所有 HTTP 请求一样，连接过程中您的 IP 地址会对服务器可见。但我们不会记录或存储 IP 地址。

建议使用 `OHMYTOKEN_API_KEY` 作为环境变量来配置 API 密钥——这样密钥不会被版本控制系统捕获，也不会泄露。

源代码完全公开：[github.com/0x5446/ohmytoken-oss](https://github.com/0x5446/ohmytoken-oss)——您可以自行审核代码。

## 您将获得的功能

- **实时代币显示**：每当您使用 AI 时，代币会以彩色像素的形式出现在界面上。
- **7 种界面形状**：方形、猫形、心形、星形、圆形、菱形、蘑菇形。
- **7 种填充动画效果**：顺序填充、螺旋填充、从中心向外扩散、随机填充、蛇形填充、对角线填充、雨滴状填充。
- **4 种时间视图**：今日、本月、本年、历史总计。
- **10 种成就**：从“首次使用 AI”到“百万富翁”。
- **排行榜**：总代币数、使用效率、模型多样性、夜间使用频率等。
- **分享功能**：生成包含像素艺术图像、二维码和个人资料链接的卡片。
- **可嵌入的徽章**：适用于 GitHub README 文件的 SVG 标志。
- **年度代币使用报告**：为您提供个性化的年度使用情况报告。

## 无任何使用障碍

- 支持 OpenClaw 支持的所有模型。
- 在后台静默运行，不会干扰您的工作流程。
- 支持 Google/GitHub OAuth 登录，无需创建新账户。
- 完全免费。
