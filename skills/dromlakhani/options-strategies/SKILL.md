# 期权策略技巧

## 概述

本技能提供了关于最流行的期权策略的系统性知识及逐步执行指南，涵盖了从基本的单腿交易到复杂的多腿期权组合。

---

## 策略分类

### 1. 向上趋势策略

#### 多头看涨策略
- **建立头寸**：买入1个看涨期权
- **最大利润**：无限
- **最大损失**：支付的权利金
- **盈亏平衡点**：执行价格 + 权金
- **适用场景**：对市场有强烈看涨预期且风险可控
- **希腊字母**：+Delta, +Vega, -Theta

#### 多头看涨价差（借差策略）
- **建立头寸**：买入较低执行价格的看涨期权，卖出较高执行价格的看涨期权（到期日相同）
- **最大利润**：价差宽度 - 净借差
- **最大损失**：支付的净借差
- **盈亏平衡点**：较低执行价格 + 净借差
- **适用场景**：中等程度的看涨预期，成本低于单独买入看涨期权
- **希腊字母**：+Delta（降低），-Vega（降低），-Theta（降低）

#### 多头看跌价差（贷差策略）
- **建立头寸**：卖出较高执行价格的看跌期权，买入较低执行价格的看跌期权（到期日相同）
- **最大利润**：收到的净贷差
- **最大损失**：价差宽度 - 净贷差
- **盈亏平衡点**：较高执行价格 - 净贷差
- **适用场景**：中等程度的看涨预期，希望获得收入
- **希腊字母**：+Delta, -Vega, +Theta

#### 现金担保看跌期权
- **建立头寸**：卖出看跌期权，持有现金以应对行权需求
- **最大利润**：收到的权利金
- **最大损失**：执行价格 - 权金（股票价格跌至零）
- **盈亏平衡点**：执行价格 - 权金
- **适用场景**：愿意以折价买入股票，希望获得收入

#### 合成多头股票
- **建立头寸**：买入平价看涨期权，卖出平价看跌期权（执行价格和到期日相同）
- **最大利润**：无限
- **最大损失**：可能损失较大（类似于直接持有股票）

---

### 2. 向下趋势策略

#### 多头看跌策略
- **建立头寸**：买入1个看跌期权
- **最大利润**：执行价格 - 权金（股票价格跌至零）
- **最大损失**：支付的权利金
- **盈亏平衡点**：执行价格 - 权金
- **适用场景**：对市场有强烈看跌预期，用于投资组合对冲
- **希腊字母**：-Delta, +Vega, -Theta

#### 多头看跌价差（借差策略）
- **建立头寸**：买入较高执行价格的看跌期权，卖出较低执行价格的看跌期权（到期日相同）
- **最大利润**：价差宽度 - 净借差
- **最大损失**：支付的净借差
- **盈亏平衡点**：较高执行价格 - 净借差
- **适用场景**：中等程度的看跌预期，希望获得收入

---

### 3. 中性/区间波动策略

#### 铁鹰期权组合
- **建立头寸**：
  - 卖出虚值看涨期权 + 买入更虚值的看涨期权（看涨价差）
  - 卖出虚值看跌期权 + 买入更虚值的看跌期权（看跌价差）
  - 所有期权具有相同的到期日
- **最大利润**：收到的净贷差
- **最大损失**：较宽翼的价差宽度
- **适用场景**：预期波动性较低，市场处于区间波动
- **希腊字母**：-Delta（接近零），-Vega, +Theta

#### 铁蝶期权组合
- **建立头寸**：
  - 卖出平价看涨期权 + 买入虚值看涨期权
  - 卖出平价看跌期权 + 买入虚值看跌期权
  - 平价执行价格相同
- **最大利润**：到期时在平价执行价格获得的净贷差
- **最大损失**：翼的宽度 - 净贷差
- **适用场景**：预期价格波动范围较小，相比铁鹰期权组合获得更高的净贷差

#### 短跨式期权组合
- **建立头寸**：卖出平价看涨期权 + 卖出平价看跌期权（执行价格相同，到期日相同）
- **最大利润**：收到的总权利金
- **最大损失**：无限（看涨期权侧）
- **适用场景**：预期波动性极低，风险较高，需要保证金
- **希腊字母**：-Vega（较高），+Theta（较高），Delta接近零

#### 短挤压式期权组合
- **建立头寸**：卖出虚值看涨期权 + 卖出虚值看跌期权（执行价格不同，到期日相同）
- **最大利润**：收到的总权利金
- **最大损失**：无限（看涨期权侧）
- **盈亏平衡点**：看涨期权执行价格 + 权金 / 看跌期权执行价格 - 权金
- **适用场景**：预期波动性较低，虽然风险较高，但潜在利润空间较大

#### 遮盖看涨期权
- **建立头寸**：持有100股股票 + 卖出1个虚值看涨期权
- **最大利润**：（看涨期权执行价格 - 股票成本）+ 权金
- **最大损失**：股票价格跌至零 - 支付的权利金
- **适用场景**：在现有股票基础上希望获得收入，对市场持中性或略偏看涨态度
- **希腊字母**：-Delta（有限），+Theta

---

### 4. 波动性策略（波动性投机）

#### 多头跨式期权组合
- **建立头寸**：买入平价看涨期权 + 买入平价看跌期权（执行价格相同，到期日相同）
- **最大利润**：无限
- **最大损失**：支付的总权利金
- **盈亏平衡点**：执行价格 ± 总权利金
- **适用场景**：预期股票价格会有大幅波动但方向不明确（如公司盈利或突发事件）
- **希腊字母**：Delta接近零，+Vega（较高），-Theta（较高）

#### 多头挤压式期权组合
- **建立头寸**：买入虚值看涨期权 + 买入虚值看跌期权（执行价格不同，到期日相同）
- **最大利润**：无限
- **最大损失**：支付的总权利金（低于跨式期权组合）
- **盈亏平衡点**：比跨式期权组合更宽的波动范围
- **适用场景**：预期股票价格会有大幅波动，但所需波动幅度更大才能获利

#### 多头Guts期权组合
- **建立头寸**：买入实值看涨期权 + 买入实值看跌期权
- **适用场景**：较为罕见；类似于跨式期权组合，但权利金较高，损失范围较窄

---

### 5. 高级/多腿策略

#### 日历价差（时间价差）
- **建立头寸**：卖出近期期权，买入相同执行价格的远期期权
- **最大利润**：当股票价格在近期到期日达到执行价格时
- **最大损失**：支付的净借差
- **适用场景**：近期波动性较低，远期期权的隐含波动率较高

#### 对角价差
- **建立头寸**：卖出近期期权，买入不同执行价格的远期期权
- **最大利润**：利润因情况而异
- **适用场景**：有明确的方向性预期，并且能够从隐含波动率的衰减中获利

#### 比例价差（看涨比率/看跌比率）
- **建立头寸**：买入1个期权，卖出2个执行价格不同的期权（到期日相同）
- **最大利润**：卖出期权所覆盖的执行价格区间
- **最大损失**：未覆盖的期权侧可能损失无限
- **适用场景**：有明确的方向性预期，且预期价格波动有限；仅适合经验丰富的交易者

#### 蝴蝶期权组合
- **建立头寸**：
  - 买入1个低执行价格的期权，卖出2个中等执行价格的期权，买入1个高执行价格的期权
  - 所有期权具有相同的到期日，执行价格等距
- **最大利润**：到期时在中间执行价格获得的净贷差
- **最大损失**：净借差
- **适用场景**：对目标价格有精确预期，风险较低

#### Jade Lizard期权组合
- **建立头寸**：卖出虚值看跌期权 + 卖出虚值看涨价差（看涨价差）
- **最大利润**：收到的总贷差（如果贷差大于价差宽度，则无上行风险）
- **最大损失**：看跌期权执行价格 - 总贷差（下行风险）
- **适用场景**：对市场持中性或略偏看涨态度，可消除上行风险

#### Broken Wing Butterfly期权组合
- **建立头寸**：标准蝴蝶期权组合，但翼的宽度不等
- **适用场景**：有明确的方向性预期，一侧风险可控，有可能获得净贷差

#### PMCC（穷人覆盖看涨期权）
- **建立头寸**：买入深度实值长期看涨期权（LEAPS），卖出近期虚值看涨期权
- **适用场景**：以较低的资本成本实现类似覆盖看涨期权的效果

## 决策框架：选择哪种策略？

```
Market Outlook
├── Strongly Bullish
│   ├── High conviction → Long Call
│   ├── Defined risk/reward → Bull Call Spread
│   └── Own stock → Covered Call (slight upside only)
│
├── Moderately Bullish
│   ├── Income focus → Bull Put Spread (credit)
│   └── Capital efficient → Bull Call Spread (debit)
│
├── Neutral / Sideways
│   ├── Low volatility expected
│   │   ├── Wide range → Iron Condor
│   │   ├── Tight range → Iron Butterfly / Short Straddle
│   │   └── Income on stock → Covered Call
│   └── Elevated IV → Sell premium (straddle, condor)
│
├── Moderately Bearish
│   ├── Income focus → Bear Call Spread (credit)
│   └── Capital efficient → Bear Put Spread (debit)
│
├── Strongly Bearish
│   ├── High conviction → Long Put
│   └── Hedging portfolio → Long Put / Bear Put Spread
│
└── Big Move Expected (No Direction)
    ├── High conviction → Long Straddle
    └── Lower cost → Long Strangle
```

## 希腊字母快速参考

| 希腊字母 | 含义 | 多头期权 | 短头期权 |
|-------|---------|-------------|--------------|
| **Delta** | 对标的资产价格的敏感度 | +（看涨期权）/ -（看跌期权） | 相反 |
| **Gamma** | Delta的变化率 | + | - |
| **Theta** | 每日的时间衰减 | 负值（不利） | 正值（有利） |
| **Vega** | 对隐含波动率（IV）变化的敏感度 | +（IV上升有利） | -（IV上升不利） |
| **Rho** | 对利率的敏感度 | 对大多数散户交易影响较小 | 较小 |

## 评估任何策略的关键指标

1. **最大利润**：最佳情况下的收益
2. **最大损失**：最坏情况下的亏损
3. **盈亏平衡点**：股票价格需达到的水平以避免亏损
4. **盈利概率**：盈利的统计可能性
5. **风险/回报比率**：最大利润 ÷ 最大损失
6. **到期日（DTE）**：每种策略的最佳到期日
7. **隐含波动率（IV）排名**：IV较高时适合卖出期权（收取权利金），IV较低时适合买入期权（支付权利金）

---

## IV排名指南

| IV排名 | 策略偏好 |
|---------|-------------------|
| < 20 | 买入期权（多头跨式、多头看涨/看跌期权） |
| 20–40 | 中性策略，方向性借差策略 |
| 40–60 | 贷差策略，铁鹰期权组合 |
| > 60 | 卖出期权（短跨式、挤压式期权、铁鹰期权组合） |

---

## 不同策略的理想到期日（DTE）

| 策略 | 典型到期日 |
|----------|------------|
| 卖出期权（铁鹰、跨式） | 30–45天 |
| 买入期权（跨式、看涨/看跌） | 60–90天 |
| 日历价差 | 近期：7–14天 / 远期：30–60天 |
| LEAPS策略 | 6–24个月 |
| 盈利相关策略 | 1–7天 |

## 输出格式

在呈现策略分析时，务必包括以下内容：

1. **策略名称及建立方式**（具体的期权组合、执行价格、到期日）
2. **成本/收益**（净借差或净贷差）
3. **最大利润/最大损失/盈亏平衡点**
4. **盈利概率**（如果可计算）
5. **理想市场环境**
6. **风险考量**
7. **调整建议**（如果交易出现问题）

---

## 常见调整措施

| 交易情况 | 调整方法 |
|---------------------|-----------|
| 多头看涨期权亏损 | 向下滚动或平仓，转换为价差组合 |
| 短头看跌期权表现不佳 | 向下滚动或平仓以获取更多净贷差 |
| 铁鹰期权组合中一侧表现不佳 | 将未测试的一侧调整至市场价格；或平仓 |
| 多头跨式期权无波动 | 通过平仓一侧期权转换为方向性策略 |
| 遮盖看涨期权处于实值状态 | 向上滚动看涨期权以获取更多净贷差 |
