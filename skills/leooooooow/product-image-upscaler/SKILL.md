# 产品图片放大器

## 技能卡片

- **类别：** 图像与媒体工具  
- **核心问题：** 产品图片质量较低或尺寸过小，不适合用于电子商务展示。  
- **适用场景：** 适用于产品列表、广告以及产品详情页（PDP）图片的优化处理。  
- **所需输入信息：** 原始图片的状态、目标平台、所需的图片尺寸、品牌相关要求。  
- **输出内容：** 图片质量诊断结果、优化方案以及输出格式的详细说明。  

## 工作流程：  
1. 对当前图片质量进行诊断。  
2. 根据实际使用场景选择合适的放大策略。  
3. 确保放大后的图片仍能真实反映产品特征，避免出现失真或伪影。  
4. 提供推荐的输出图片尺寸及相关检查建议。  

## 输出格式：  
1. 图片质量诊断报告  
2. 放大方案  
3. 输出格式规范  
4. 风险提示  

## 质量与安全要求：  
- 放大后的图片应更加清晰，同时保持真实感。  
- 不得改变产品的核心外观特征。  
- 优化图片以适应电子商务平台的显示需求，而非追求视觉效果上的噱头。
