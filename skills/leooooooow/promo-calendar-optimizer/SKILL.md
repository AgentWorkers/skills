    # 促销日历优化器

    ## 技能卡片

    - **类别：** 运营管理
    - **核心问题：** 如何在不增加运营负担的情况下，合理安排促销活动以最大化转化率？
    - **适用场景：** 跨渠道促销计划制定
    - **所需输入：** 活动目标、库存限制、创作者可用时间、付费预算窗口
    - **预期输出：** 包含渠道组合、负责人任务及风险提示的每周促销日历
    - **交接方式：** 将输出结果用于批量内容冲刺（batch-content-sprint-os）和交易运营（deal ops）执行流程

    ## 工作流程

    1. 收集促销活动的各项限制条件（库存、预算、创作者可用时间、活动安排等）。
    2. 将促销计划与最适合的渠道/时间窗口进行匹配。
    3. 检测执行计划中的冲突和超负荷风险。
    4. 输出一份均衡的促销日历及备用方案。

    ## 输出格式

    输出内容需按照以下顺序排列：
    1. 执行摘要（最多5行）
    2. 优先级行动事项（P0/P1/P2）
    3. 证据表格（包含信号指标、置信度及风险等级）
    4. 7天执行计划

    ## 质量与安全规则

    - 确保促销活动的安排不会超出团队的处理能力。
    - 保护库存紧张的商品不受过度推广的影响。
    - 为创作者或内容制作的延误提供备用方案。
