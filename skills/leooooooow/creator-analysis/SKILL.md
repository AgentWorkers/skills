# 创作者分析

通过简洁、易于决策的工作流程来分析创作者。

## 工作流程

1. 明确分析目标：
   - 筛选出潜在的创作者
   - 比较两个或多个创作者的表现
   - 诊断其性能不佳的原因
   - 汇总创作者的概况以用于后续沟通

2. 规范化输入数据：
   - 创作者的用户名/个人资料链接
   - 他们的专业领域和产品类别
   - 目标受众及产品适用范围
   - 任何可观察到的指标、用户反馈或内容样本

3. 从五个维度对创作者进行评估：
   - 与目标受众的契合度
   - 内容的清晰度
   - 信任度或可信度指标
   - 转化潜力
   - 运营风险

4. 用通俗的语言撰写评估结果：
   - 该创作者的优势
   - 需改进的地方
   - 是否值得测试、优先考虑或直接放弃

## 输出格式

返回以下信息：
- 一条总结性的评估结果
- 创作者的优势和风险
- 推荐的使用场景
- 下一步行动建议

请保持内容简洁，以辅助决策。

## 质量标准：
- 优先使用可观察到的实际数据，而非空洞的夸夸其谈。
- 区分创作者的覆盖范围（reach）与与目标受众的契合度（fit）。
- 当缺少关键指标时，应明确标注不确定性。
- 不要将内容质量（如排版是否精美）与转化潜力混为一谈。
