# 将音频转换为文本字幕

## 技能卡片

- **类别：** 图像与媒体工具  
- **核心问题：** 团队在将创作者的音频转换为可用的字幕或文本草稿时浪费了大量时间。  
- **适用场景：** 短视频、实时片段以及内容再利用的工作流程。  
- **输入要求：** 音频源、语言、输出格式、样式偏好。  
- **输出结果：** 清晰的文字记录、可用于字幕制作的文本以及审核备注。  

## 工作流程：  
1. 将音频内容转录成文本。  
2. 去除冗余信息及格式化噪声。  
3. 格式化文本，以便后续用于字幕或剧本制作。  
4. 标记需要人工审核的模糊或难以理解的片段。  

## 输出格式：  
1. 文本记录摘要  
2. 清晰的转录文本  
3. 可用于字幕制作的最终文本  
4. 需要人工审核的片段列表  

## 质量与安全规范：  
- 优先考虑文本的可读性和可重用性。  
- 不要随意解释或创造不清晰的语音内容。  
- 根据字幕或剧本的需求调整文本格式。
