# 创作者跟进流程

## 技能卡片

- **类别：** 创作者开发与管理  
- **核心问题：** 如何在未收到回复后与创作者进行有效跟进，同时避免显得烦人或机械？  
- **适用场景：** 附属成员招募、用户生成内容（UGC）收集、合作伙伴关系拓展以及创作者资源恢复。  
- **所需信息：** 初始联系信息、创作者背景、提供的价值、自首次联系以来的时间、期望的下一步行动。  
- **预期输出：** 提供2-4条跟进消息方案，包括发送时机、沟通重点及退出规则。  
- **后续处理：** 将有效的跟进策略纳入外联操作手册（outreach SOPs）和客户关系管理（CRM）模板中。  

## 工作流程  

1. **分析沟通状态：**  
   - 首次联系后仍未收到回复  
   - 回复含糊但态度积极  
   - 表示感兴趣但后续进展停滞  
   - 对沟通时机提出异议  
   - 可能不符合要求或优先级较低  

2. **确定沉默的原因：**  
   - 分析创作者可能保持沉默的常见原因。  

3. **选择合适的跟进方式：**  
   - 提供额外价值  
   - 明确沟通重点  
   - 提供具体证据  
   - 礼貌地确认沟通进展  

4. **起草跟进消息：**  
   - 消息内容应逐步深入，但避免施加压力。  

5. **制定发送时机和退出规则：**  
   - 提供具体的发送时间建议及何时停止跟进的准则。  

## 输出格式：**  
按以下顺序提供：  
1. 沟通状态分析  
2. 当前最适合使用的跟进方式  
3. 进阶跟进流程（消息1 / 消息2 / 最终确认）  
4. 发送时机建议  
5. 停止跟进的规则  

## 质量与安全准则：**  
- 每条消息简洁明了，便于通过电话回复。  
- 每次跟进都应提供新的价值、更清晰的背景信息或更具体的请求。  
- 避免使用道德绑架、施加压力或虚假的紧迫感。  
- 不要编造虚假的证据、预算数据或业绩承诺。  
- 如果创作者明显不符合要求或无兴趣，建议停止跟进。
