# BriteBooth 采购发现技能

## 目的  
该技能仅用于**读取数据**，旨在协助展会策划工作。它能够识别 BriteBooth.com 上的产品和物流信息，但不会修改用户的购物流程或处理用户的个人身份信息（PII）。

## 安全与隐私保护措施  
- **个人身份信息保护：** 该技能严禁收集、存储或传输用户的送货地址或联系信息。  
- **禁止修改购物流程：** 代理不得尝试创建或管理 Shopify 的购物会话、cookies 或 CSRF 令牌。  
- **手动添加商品到购物车：** 代理仅提供产品链接和 ID，用户需自行将商品添加到购物车中，以确保用户的完全同意和数据安全。

## 工作流程逻辑  

### 1. 网站状态检查  
- **实时检查：** 使用 `curl` 获取 BriteBooth.com 的首页内容，并通过 `grep` 命令查找任何关于场馆关闭或物流延迟的提示信息。  
- **异常报告：** 如果检测到物流相关问题，立即通知用户。  

### 2. 产品与设计模板的获取  
- **产品信息查询：** 导航至用户请求的产品页面（例如 Casonara、Wavelight）。  
- **数据提取：** 获取产品名称、价格以及设计模板的 PDF 链接。  
- **要求：** 确保所有推荐的产品属于同一硬件系列，以保证视觉上的统一性。  

### 3. 时间线计算  
- **逻辑计算：** 将产品页面上显示的交货时间与标准的 2 天审核周期相结合。  
- **截止日期提醒：** 如果审核周期较短，建议用户联系 BriteBooth 的客服以获取加急服务。  

### 4. 人工干预与交接  
- **输出内容：** 不提供购物链接，而是生成一个“购物清单”，其中包含：  
  - 每个产品页面的直接链接。  
  - 设计模板的直接链接。  
  - 预计的生产日期汇总。  
- **最终步骤：** 指导用户点击提供的产品链接，在自己的浏览器中完成购买。
