# Douyin Creator Assistant 🎬

这是一个专为抖音（TikTok中国版）内容创作者设计的AI辅助工具。

## 功能

### 1. 标签推荐 🏷️
获取个性化的标签建议。

```bash
./scripts/tags.sh "美食"
```

### 2. 最佳发布时间 ⏰
了解何时发布内容以获得最大的互动效果。

```bash
./scripts/time.sh "美食博主"
```

### 3. 标题优化 ✍️
帮助你优化视频标题。

```bash
./scripts/title.sh "今天做了红烧肉"
```

### 4. 竞争对手分析 📊
研究那些成功的账号。

```bash
./scripts/analyze.sh "@美食作家王刚"
```

## 安装

```bash
clawhub install douyin-creator-cn
```
