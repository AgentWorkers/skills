# 淘宝热门产品发现工具 🛒  
一款基于人工智能的软件，用于发现淘宝上的热门产品。  

## 主要功能  

### 1. 热门产品 📊  
按类别浏览销量最高的产品。  
```bash
./scripts/hot.sh "女装"
```  

### 2. 热门类别 🏷️  
发现当前最受欢迎的产品类别。  
```bash
./scripts/categories.sh
```  

### 3. 价格分析 💰  
分析产品的价格走势。  
```bash
./scripts/price.sh "手机" 1000-3000
```  

### 4. 季节性趋势 📈  
发现具有季节性需求的畅销产品。  
```bash
./scripts/seasonal.sh "春节"
```  

## 使用场景  

- **代发货**：寻找适合销售的热门产品  
- **代购**：了解中国消费者的最新购物趋势  
- **个人购物**：寻找最划算的商品  
- **市场调研**：洞察中国消费者的消费偏好
