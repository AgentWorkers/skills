# 临床试验搜索

您可以通过访问 ClinicalTrials.gov 来使用高级筛选选项搜索临床试验。

## 功能介绍

- 按疾病、干预措施、阶段和状态搜索临床试验
- 通过 NCT ID 获取详细的试验信息
- 按疾病/病症或治疗类型进行筛选
- 查看试验元数据和研究信息

## 使用场景

- 查找针对特定疾病（如重症肌无力、多发性硬化症）的正在进行中的临床试验
- 识别可用于药物再利用的试验机会
- 跟踪试验的状态和招募情况
- 发现新的治疗方法

## 可用的工具

- `ctg_search_studies`：使用多种筛选条件搜索试验
- `ctg_get_study`：通过 NCT ID 获取单个试验的详细信息
- `ctg_search_by_condition`：按疾病搜索试验
- `ctg_search_by_intervention`：按治疗方式搜索试验
- `ctg_get_study_metadata`：获取试验元数据的可用字段

## 使用要求

无需 API 密钥。可直接使用公开的 ClinicalTrials.gov API 进行访问。
