# 商业实体注册文件生成器 — 一站式生成所有注册企业所需的法定文件

本工具可根据用户提供的配置信息，自动生成注册企业所需的所有法定文件。

## 支持的实体类型及所在州

| 实体类型 | 所在州 | 参考文件 |
|--------|--------|---------------|
| C型公司（C-Corp）| 内华达州 | `references/nevada-corp.md` |
| C型公司（C-Corp）| 特拉华州 | `references/delaware-corp.md` |
| 有限责任公司（LLC）| 内华达州 | `references/nevada-llc.md` |
| 有限责任公司（LLC）| 特拉华州 | `references/delaware-llc.md` |

## 快速入门

1. 向用户询问公司详细信息（或让用户填写配置信息）。
2. 阅读 `references/config-template.md` — 这是输入格式的示例文件。
3. 根据用户选择的州和实体类型，查阅相应的参考文件。
4. 从 `assets/templates/` 目录中获取相应的文档模板（例如，特拉华州的文件前缀为 `de-*`，LLC 的文件前缀为 `llc-*`）。
5. 使用配置信息生成所有文件。
6. 将生成的文件上传至 Google Drive，或保存为 `.docx` 格式。

## 工作流程

### 第一步：收集配置信息

向用户询问以下信息（或接受预先填写的配置信息）：

**必填项：**
- 公司名称
- 实体类型（C型公司或有限责任公司）
- 注册州
- 注册代理人姓名及地址
- 董事/经理姓名及职务
- 股东/成员姓名及持股比例
- 股票结构（股票类别、授权股数、面值、投票权）

**可选项（提供默认值）：**
- 股票面值（默认：$0.00001）
- 财务年度结束日期（默认：12月31日）
- 对价类型（默认：“服务”）
- 注册办事处地址（默认：注册代理人地址）

**推荐填写的选项：**
- 期权池规模（为未来员工或顾问预留的股份）
- 标准初创企业股份比例：授权股份的10-20%

**配置验证：**
- 所有股东持股比例之和必须为100%；
- 授权股数必须涵盖所有已发行的股份（包括期权池预留的股份）。

### 第二步：选择实体类型及所在州的参考文件

根据用户选择的实体类型和所在州，查阅相应的参考文件：
- 内华达州 C型公司 → `references/nevada-corp.md`
- 特拉华州 C型公司 → `references/delaware-corp.md`
- 内华达州 LLC → `references/nevada-llc.md`
- 特拉华州 LLC → `references/delaware-llc.md`

**用户决策指南：**
- **C型公司 vs LLC：**
  - C型公司适合吸引风险投资（VC），支持多种股票类别，适合未来上市；
  - LLC 适用于结构较为简单的业务，采用穿透税制，利润分配更加灵活。
- **特拉华州 vs 内华达州：**
  - 特拉华州更符合投资者的期望，有专门的法院（Chancery Court），文件格式符合 VC 的标准；
  - 内华达州无需缴纳特许经营税，年度成本较低，且更注重隐私保护。
- **特拉华州 C型公司的注意事项：**
  - 如果授权股数较多，特许经营税可能很高，务必使用“假设面值法”（Assumed Par Value Method，详见 `delaware-corp.md`）。

### 第三步：生成文件

从 `assets/templates/` 目录中读取每个模板，并将其中的 `{{VARIABLES}}` 替换为实际的配置值。所有模板均为 Markdown 格式，生成后需转换为 `.docx` 文件。

**C型公司生成的文件包括：**
1. **公司章程** — 提交给州务卿（Secretary of State）
2. **公司章程细则** — 内部治理规则
3. **公司成立决议** — 任命初始董事会成员
4. **组织决议** — 董事会通过公司章程细则，选举管理人员
5. **股票登记册** — 记录所有股东持股情况及投票权
6. **文件提交指南** — 详细步骤说明，包含链接、83(b) 条款提交截止日期、第一年合规性要求及银行账户信息
7. **83(b) 条款选举表** （如通过服务方式发行股票则自动生成）—— 用于向 IRS 申报股票面值，确保税务处理正确。此文件非常重要，必须在股票发行后30天内提交。
8. **限制性股票购买协议（RSPA）** — 为每位股东生成的文件，涵盖股票归属、回购选项、转让限制等内容。⚠️ 复杂的协议安排需咨询律师。

**LLC 生成的文件包括：**
1. **组织章程** — 提交给州务卿（模板：`llc-01-articles-of-organization.md`）
2. **运营协议** — 规定公司治理结构和成员权利（模板：`llc-02-operating-agreement.md`）
3. **成员登记表** — 记录股东持股情况（模板：`llc-03-membership-ledger.md`）
4. **文件提交指南** — 详细提交步骤说明（模板：`llc-04-filing-checklist.md`）

**特拉华州 C型公司的特殊说明：**
- 使用 `de-01-certificate-of-incorporation.md` 代替 `01-articles-of-organization.md` 作为公司章程文件；
- 特拉华州将此类文件称为“Certificate of Incorporation”而非“Articles of Incorporation”；
- 文件中包含免责条款（DGCL §102(b)(7)）及空白支票授权条款；
- 其他 C型公司的文件（公司章程细则、成立决议、股票登记册、提交指南）与普通 C型公司相同，但需根据特拉华州的规定进行相应调整。

### 第四步：格式化及文件交付

- 所有文本使用 Times New Roman 字体，标题为黑色、12号字体，正文为11号字体；
- 表格需加边框，行间有阴影，标题为深蓝色；
- 需手动填写的字段用黄色高亮显示；
- 将所有文件上传至 Google Drive 或保存为本地 `.docx` 文件；
- 在文件提交指南中提供每个文件的链接。

### 第五步：指导用户后续操作

生成文件后，向用户说明文件提交流程：
1. 将公司章程提交给州务卿（如支持在线提交，则通过在线渠道提交）；
2. （在内华达州）将初始董事名单文件与公司章程一起提交；
3. 在文件提交确认后签署公司成立决议；
4. 召开组织会议并签署相关决议；
5. 发行股票并更新股票登记册；
6. **在股票发行后30天内提交83(b) 条款相关文件** — 适用于通过服务方式发行股票的情况（具体要求请参考所在州的规定）；
7. 从 IRS 获取企业识别号（EIN）（非美国创始人需指定具有美国社会安全号码（SSN）的代理人在线申请）；
8. 与所有股东签署限制性股票购买协议（RSPA）；
9. 开设公司银行账户（所需材料：EIN、已提交的文件、公司章程细则、决议文件及身份证明）。

## 文件提交后的注意事项

文件生成并提交后，请阅读 `references/post-filing-learnings.md` 以获取实际操作指南：
- 实际文件提交费用与预估费用之间的差异；
- 如遇到 SilverFlume 服务器错误如何处理；
- 非美国创始人申请 EIN 时需要注意的事项；
- 83(b) 条款提交的截止日期管理；
- 常见错误及避免方法；
- 自行处理与聘请律师的费用对比。

本工具的生成规则基于2026年3月在内华达州注册一家 C型公司的实际案例，其中包含了一些模板中未涵盖的实用建议。

## 重要提示：

- 本工具提供的文件仅为标准模板，不构成法律建议；
- 对于复杂的结构（如可转换债券、股票归属安排、多系列优先股等），建议咨询律师；
- 各州的具体要求可能有所不同，请务必查阅相应的参考文件；
- 各州的年度维护要求各不相同，具体内容详见文件提交指南；
- 特拉华州的特许经营税可能较高，请参阅 `references/delaware-corp.md` 了解“假设面值法”的相关内容；
- LLC 的运营协议具有高度可定制性，虽然模板涵盖了标准条款，但复杂的安排（如瀑布式利润分配、股票归属权等）仍需律师审核。
