# 🟡 KakaoTalk AI 代理（KakaoTalk Agent）

通过 KakaoTalk 联频道，与 **专属 AI 秘书** 进行对话吧！  
无需担心服务器问题，即可 **免费且永久地** 使用 Vercel 进行部署。

支持两种模式：

| 模式 | 难度 | 特点 | 推荐对象 |
|------|--------|------|-----------|
| **🅰️ Basic** | ⭐ | 只需要 Vercel 即可快速搭建。简单易用。 | 适用于简单聊天机器人、问答机器人 |
| **🅱️ Relay** | ⭐⭐⭐ | 与您的电脑（OpenClaw）连接，具备 **记忆功能**。 | 适用于个人助理、自动化办公场景 |

## 🚀 共同准备材料

1. **Kakao 商业账户**：[business.kakao.com](https://business.kakao.com)（用于创建频道）  
2. **Kakao i OpenBuilder**：[i.kakao.com](https://i.kakao.com)（用于创建机器人）  
3. **Vercel 账户**：[vercel.com](https://vercel.com)（用于部署）

## 🅰️ Option A: Basic 模式（5分钟快速入门）

如果 AI 只需要提供简单回答，可以选择此模式。

### 1. 部署
在终端中执行以下命令（需要 Gemini API 密钥）：

```bash
# 1. 스킬 폴더로 이동
cd skills/kakaotalk

# 2. Vercel 배포 (환경변수 설정)
# GEMINI_API_KEY는 본인 키로 변경하세요.
vercel deploy --prod --env GEMINI_API_KEY=AIzaSy...
```

部署完成后，会得到一个 `https://...vercel.app` 的地址。  
**技能 URL**：`https://[您的地址].vercel.app/kakao`

### 2. 连接 OpenBuilder
1. **注册技能**：进入 OpenBuilder > 技能 > 添加新技能 > 输入技能 URL  
2. **设置回退逻辑**：选择合适的回退方案  
3. **部署**：点击右上角的 [部署] 按钮  

完成！现在就可以在 KakaoTalk 联频道中与机器人对话了。

## 🅱️ Option B: Relay 模式（高级功能）

如果您希望让机器人执行更复杂的任务（例如：从电脑中读取文件、继续之前的对话等），请使用此模式。  
Vercel 作为“中间件”，由您的电脑直接处理请求并回复。

### 1. 准备 Supabase 数据库（作为“中间件”）
1. 在 [Supabase](https://supabase.com) 上创建项目。  
2. 在 **SQL Editor** 中执行以下查询：  
   ```sql
   create table kakaotalk_queue (
     id bigint generated by default as identity primary key,
     user_id text not null, utterance text not null, callback_url text,
     status text default 'pending', response text, created_at timestamptz default now()
   );
   ```

### 2. 使用 Vercel 部署（作为“中间件”）
在部署时，请使用 **Supabase 密钥** 替代 Gemini 密钥。  
```bash
# SUPABASE_URL, SERVICE_KEY는 본인 프로젝트 설정 > API 메뉴에서 확인
vercel deploy --prod \
  --env SUPABASE_URL=https://... \
  --env SUPABASE_SERVICE_KEY=eyJ...
```

### 3. 在 OpenBuilder 中设置 AI 聊天机器人
1. **设置 > 更改为 AI 聊天机器人**（必选！用于实现回调功能）  
2. 注册并部署技能 URL（步骤与 Basic 模式相同）

### 4. 运行本地脚本（处理聊天请求）
在您的电脑上运行以下脚本，以便处理 KakaoTalk 的消息：

```bash
# ~/.openclaw/.env 파일에 SUPABASE_URL, SERVICE_KEY 저장 후:
python3 skills/kakaotalk/scripts/watcher.py
```

只要电脑保持开启状态，聊天机器人就能具备与您相同的智能水平。

## 🔧 高级设置（自定义机器人角色）

如果您想更改机器人的语气或角色，可以在部署时添加环境变量。  
```bash
vercel deploy --prod \
  --env KAKAOTALK_SYSTEM_PROMPT="너는 친절한 고양이 로봇이야. 말 끝마다 '냥'을 붙여."
```
