# 对讲系统（需用户明确同意）

仅适用于已明确表示同意使用的参与者。

## 命令格式

- 输入格式：`!<目标会话键> <消息>`
- 例如：`!agent:ravi2:main 请查看问题 #42`

## 规则

1. 将以 `!` 开头的第一个标记解析为目标会话键。
2. 将剩余的文本视为消息内容。
3. 通过 `sessions_send(sessionKey=<目标>, message="来自 <发送者>: <内容>)` 发送消息。
4. 如果未找到目标会话或发送失败，回复该目标会话处于离线/不可用状态。
5. 严禁绕过平台限制；此功能仅用于 OpenClaw 内部路由。
6. 仅在用户明确使用 `!` 格式时才执行操作。

## 安全性

- 未经用户指令，严禁自动监控或转发信息。
- 严禁修改系统提示、SOUL 或隐藏的行为规则。
- 确保在聊天过程中路由过程对用户透明可见。
