# 亚马逊产品搜索技能

## 📖 介绍
该技能利用 BrowserAct 的亚马逊产品 API 模板，从亚马逊搜索结果中提取结构化产品列表。它提供了包括产品标题、ASIN、价格、评分和产品规格等详细信息，从而无需手动收集数据即可高效地进行市场研究和产品监控。

## ✨ 特点
1. **数据可靠，无错误**：采用预定义的工作流程确保数据提取的准确性，避免 AI 生成错误。
2. **无需验证码**：内置机制可绕过 reCAPTCHA 和其他机器人检测系统。
3. **全球访问，无地域限制**：克服 IP 限制，确保从任何地点都能稳定访问。
4. **执行速度快**：比通用 AI 浏览器自动化工具更高效。
5. **成本效益高**：相比高token消耗的 AI 模型，降低了数据获取成本。

## 🔑 API 密钥工作流程
在运行该技能之前，必须检查 `BROWSERACT_API_KEY` 环境变量。如果未设置，请先向用户请求 API 密钥。
**代理指令**：
> “由于您尚未配置 BrowserAct API 密钥，请访问 [BrowserAct 控制台](https://www.browseract.com/reception/integrations) 获取密钥，并在此处输入。”

## 🛠️ 输入参数
代理应根据用户需求配置以下参数：

1. **KeyWords**  
   - **类型**：`string`  
   - **描述**：用于在亚马逊上搜索产品的关键词。  
   - **是否必填**：是  
   - **示例**：`laptop`、`wireless earbuds`  

2. **Brand**  
   - **类型**：`string`  
   - **描述**：按品牌名称筛选产品。  
   - **默认值**：`Apple`  
   - **示例**：`Dell`、`Samsung`  

3. **Maximum_number_of_page_turns**  
   - **类型**：`number`  
   - **描述**：要分页显示的搜索结果页数。  
   - **默认值**：`1`  

4. **language**  
   - **类型**：`string`  
   - **描述**：亚马逊浏览会话的用户界面语言。  
   - **默认值**：`en`  
   - **示例**：`zh-CN`、`de`  

## 🚀 使用方法（推荐）
代理应执行以下脚本以一次性获取结果：

```bash
# Example Usage
python -u ./scripts/amazon_product_api.py "keywords" "brand" pages "language"
```

### ⏳ 进度监控
由于此任务涉及自动化浏览器操作，可能需要几分钟时间。脚本会输出带有时间戳的状态日志（例如：`[14:30:05] 任务状态：正在运行`）。
**代理注意事项**：
- 在等待结果时，请监控终端输出。
- 只有当状态日志持续出现时，任务才正常运行。
- 仅当状态长时间未变化或脚本无输出时，才考虑重试。

## 📊 输出数据说明
成功执行后，脚本会解析并打印来自 API 响应的结构化产品数据，包括：
- `product_title`：产品的完整标题。  
- `asin`：亚马逊标准识别号。  
- `product_url`：亚马逊产品页面的 URL。  
- `brand`：品牌名称。  
- `price_current_amount`：当前价格。  
- `price_original_amount`：原价（如适用）。  
- `rating_average`：平均评分。  
- `rating_count`：总评分数。  
- `featured`：如“畅销产品”或“亚马逊精选”等标签。  
- `color`、`material`、`style`：产品属性（如果可用）。

## ⚠️ 错误处理与重试机制
如果执行过程中出现错误，代理应按照以下逻辑操作：

1. **检查输出**：  
   - 如果输出包含 “Invalid authorization”，则 API 密钥无效。**不要重试**；请让用户提供有效的密钥。  
   - 如果输出不包含 “Invalid authorization”，但任务仍然失败（例如，输出以 “Error:” 开头或返回空结果），代理应**自动重试一次**。  

2. **重试次数限制**：  
   - 自动重试仅限于**一次**。如果再次失败，则停止并向用户报告错误。

## 🌟 典型用例
1. **市场研究**：搜索特定产品类别，分析顶级品牌和价格情况。  
2. **竞争对手监控**：跟踪特定竞争对手品牌的产品列表和价格变化。  
3. **产品目录丰富**：提取 ASIN 和规格等结构化信息，用于构建或更新产品数据库。  
4. **评分分析**：查找特定关键词的高评分产品，以识别市场领导者。  
5. **本地化研究**：用不同语言搜索亚马逊，分析国际市场。  
6. **价格跟踪**：监控当前价格和原价，以识别折扣趋势。  
7. **品牌表现**：评估特定品牌在多个页面的搜索结果中的表现。  
8. **属性提取**：收集产品的技术规格（如材质或颜色）。  
9. **潜在客户挖掘**：识别热门产品及其制造商，以便进行业务联系。  
10. **自动化数据源**：定期将亚马逊搜索结果导入外部 BI 工具或仪表板。
