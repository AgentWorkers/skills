# Meegle API — 组（Groups）

这些API用于组织和管理工作项组（work item groups）。

**先决条件：** 从 **meegle-api-users** 获取域名（domain）和访问令牌（access token）。

## 功能范围（Scope）：

- 创建、列出、更新工作项组
- 管理组成员关系及组层级结构
- 相关的组操作端点（group-related endpoints）

*详细的API规范将在相关文档完成编写后陆续添加。*
