# Meegle API — 设置

用于设置和配置相关的 OpenAPI（例如工作项类型、字段、流程模板）。当您需要读取或更改空间或工作项的设置时，请使用此 API。

**先决条件：** 首先获取域名和访问令牌；有关域名、plugin_access_token/user_access_token 以及请求头的信息，请参阅技能文档 **meegle-api-users**。

## 待文档化的 API

- 获取工作项创建元数据、字段定义、流程设置等相关端点的信息将在这里进行文档化。
