# Meegle API — 附件

提供用于在工作项上上传、列出和管理附件的相关 API。

**前提条件：** 从 **meegle-api-users** 获取域名和访问令牌。

## 功能范围

- 向工作项上传附件
- 列出附件
- 下载或删除附件
- 相关的附件操作端点

*详细的 API 规范将在文档完善后陆续提供。*
