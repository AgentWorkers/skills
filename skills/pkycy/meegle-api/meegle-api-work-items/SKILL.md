# Meegle API — 工作项（Work Items）

用于在 Meegle 空间中创建和管理工作项（任务、问题、漏洞等）。请使用 `read-file` 命令访问指定路径；`{baseDir}` 表示技能包的根目录。

| 技能 | 路径 | 说明 |
|-------|------|-------------|
| 创建/读取/更新 | **{baseDir}/meegle-api-work-items/work-item-read-and-write/SKILL.md** | 创建、获取和更新工作项 |
| 列表与搜索 | **{baseDir}/meegle-api-work-items/work-item-lists/SKILL.md** | 过滤、搜索工作项；支持全文搜索及相关项的查找 |
| 工作流与节点 | **{baseDir}/meegle-api-work-items/workflows-and-nodes/SKILL.md** | 工作流和节点相关的 API |
| 任务 | **{baseDir}/meegle-api-work-items/tasks/SKILL.md** | 任务相关的 API |
| 附件 | **{baseDir}/meegle-api-work-items/attachment/SKILL.md** | 工作项的附件管理 |
| 空间关联 | **{baseDir}/meegle-api-work-items/space-association/SKILL.md** | 工作项与 Meegle 空间的关联管理 API |
| 组织工作项 | **{baseDir}/meegle-api-work-items/group/SKILL.md** | 工作项的分组管理 API |
