# Meegle API — 空间（Spaces）

这些 API 与 Meegle 的空间（projects）相关，用于管理或查询 Meegle 的空间/项目。

**先决条件：** 首先需要获取域名和访问令牌；有关域名、插件访问令牌（plugin_access_token）/用户访问令牌（user_access_token）以及请求头的信息，请参阅技能文档 **meegle-api-users**。

## 待文档化的 API

- 空间列表（Space list）、获取空间信息（get）、创建空间（create）、更新空间信息（update）等相关接口将在这里进行文档化。
