# NERRA AI Launch Engine

这是一个专为 Bags.fm 平台设计的、基于人工智能的 Solana 代币策略生成工具。该工具能够生成具有高传播性的代币概念、发布策略，以及可直接发布到 X（Twitter）平台的营销内容。

## 功能

- 帮助用户创建具有吸引力的代币形象（如“迷因代币”）
- 优化代币描述，以适应特定的社区文化
- 生成可用于 X 平台的营销内容（如帖子、回复等）
- 提供初始购买策略建议
- 提高代币的传播潜力

## 输入

用户提供的关于代币的简短自然语言描述。  
示例：  
“为我创建一个适合 Solana 交易者的‘混乱 AI 狗’主题代币。”

## 输出

返回的结构化 JSON 数据包含以下字段：  
- `name`（代币名称）  
- `symbol`（代币符号）  
- `hook`（代币的吸引点或卖点）  
- `description`（代币描述）  
- `initialBuy`（初始购买建议）  
- `targetAudience`（目标受众）  
- `narrativeAngle`（内容叙述角度）  
- `xProfileBio`（X 平台上的个人资料简介）  
- `xLaunchThread`（用于 X 平台的营销帖子数组）  
- `xReplyShills`（用于回复评论的账号数组）  
- `marketingOneLiner`（营销口号）  
- `riskScore`（风险评分）  
- `viralityScore`（传播潜力评分）

## 语言风格  

- 采用加密货币领域的专业用语  
- 表达方式充满活力、易于理解  
- 强调代币的“迷因”特性（即其在社交媒体上的传播能力）  
- 适合注意力持续时间较短的用户阅读  

## 集成方式  

- 适用于 Bags.fm 的代币发布流程  
- 可自动生成用于 X 平台的营销内容  
- 支持自动化社交媒体分发  
- 支持利用 AI 功能自动填写相关表单  

**注意：**  
仅返回有效的 JSON 数据，不包含任何解释性文本。
