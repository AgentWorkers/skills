# 自主研究

能够独立开展全面的研究工作。

## 使用场景
- 对任何主题进行深入研究
- 在没有指导的情况下收集信息
- 分析和整合研究结果
- 制作详细的报告

## 核心能力

### 1. 独立研究
- 寻找相关的信息来源
- 评估信息来源的可靠性
- 提取关键信息
- 对比多个来源的内容

### 2. 分析与整合
- 发现其中的规律和联系
- 分析相互矛盾的信息
- 将研究结果整合成连贯的见解
- 得出基于证据的结论

### 3. 报告生成
- 逻辑清晰地组织研究结果
- 制作全面的报告
- 包含引用和来源信息
- 提供可操作的见解

### 4. 来源管理
- 跟踪信息来源
- 管理引用和参考文献
- 评估信息来源的可靠性
- 用新信息更新研究结果

## 研究流程

```
1. TOPIC DEFINITION → Clarify research scope
2. SOURCE IDENTIFICATION → Find relevant sources
3. INFORMATION EXTRACTION → Extract key data
4. ANALYSIS → Analyze and synthesize findings
5. REPORTING → Create comprehensive report
6. VALIDATION → Verify accuracy and completeness
```

## 快速操作
- `research [主题]` - 进行全面研究
- `analyze sources` - 评估信息来源的可靠性
- `synthesize findings` - 从数据中提取见解
- `generate report` - 制作详细报告
- `validate research` - 验证研究的准确性和完整性

## 使用示例

```
"Research AI agent monetization strategies for 2026"
"Analyze the current state of autonomous agent development"
"Find and evaluate sources on LLM reasoning capabilities"

"Validate research findings and identify gaps"
```

## 最佳实践

1. **广泛开始**：先从广泛的范围内开始，再逐步聚焦
2. **多样化来源**：使用多种类型的来源
3. **交叉验证**：在不同来源之间核实信息
4. **保持更新**：使用最新和相关的信息来源
5. **保持批判性思维**：质疑假设和偏见
6. **详细记录**：保留所有信息来源的详细记录
7. **验证结果**：检查研究的准确性和完整性
8. **提供背景**：解释研究的重要性和意义
