# Anthropic Frontend Design

本技能指导您创建具有独特性的、符合专业标准的前端界面，避免使用那些千篇一律的、缺乏创意的“AI风格”设计。它将结构化的设计理念（如可访问性、用户体验规则以及技术栈指南）与鲜明、有目的性的美学风格相结合。

## 核心理念：反对“AI风格的平庸设计”

Claude（以及所有AI助手）虽然具备出色的创造力，但往往仍倾向于使用安全、通用的设计模式。本技能要求您打破这些模式：

- **应避免的元素**：Inter字体、Roboto字体、Arial字体、纯白色背景上的紫色渐变效果、千篇一律的SaaS界面布局、以及将表情符号作为图标使用。
- **必须采用的元素**：独特的字体、与上下文相匹配的颜色方案、有意图的动态效果、出人意料的视觉布局，以及符合专业标准的功能代码。

## 设计思维流程

在开始编码之前，先了解设计背景并明确一个鲜明的美学方向：

1. **设计目的**：这个界面要解决什么问题？它是为谁设计的？
2. **设计风格**：选择一个极端的方向——极简主义、极繁主义、复古未来主义、自然风格、奢华风格、趣味风格、编辑风格等。
3. **设计参考**：使用内部设计工具收集相关设计资源（详见下文）。
4. **差异化**：是什么让这个界面令人难忘？

## 实施标准

### 1. 专业UI设计规则

| 规则 | 应遵循 | 应避免 |
|------|----|-----|
| **图标** | 使用SVG格式的图标（如Heroicons、Lucide、Simple Icons） | 不要将表情符号（如🎨、🚀、⚙️）作为UI图标 |
| **字体** | 使用美观且独特的Google字体或自定义字体 | 避免使用Inter、Roboto、Arial等系统字体 |
| **悬停效果** | 实现稳定且清晰的视觉过渡效果（如颜色变化、透明度调整或阴影效果） | 避免使用会改变界面布局的缩放效果 |
| **光标** | 为所有交互元素添加`cursor-pointer` | 按钮和卡片上保持默认的光标样式 |
| **对比度** | 保持至少4.5:1的对比度以确保可访问性 | 低对比度的设计会降低界面的可读性 |

### 2. 动画与交互效果
- 尽可能优先使用纯CSS实现的动画效果。
- 重点处理页面加载时的高交互性瞬间（例如元素的渐进式显示）。
- 微交互效果的持续时间应控制在150-300毫秒之间。

### 3. 视觉布局
- 使用不对称布局、元素重叠或对角线布局来打破传统的网格结构。
- 在保持足够负空间的同时，也可以选择密集的视觉元素布局。

## 交付前的检查清单

在提交代码之前，请确认以下各项内容：

### 视觉质量
- [ ] 不得使用表情符号作为图标（仅使用SVG格式的图标）。
- [ ] 字体设计具有独特性，不符合“AI风格”的通用标准。
- [ ] 颜色方案与设计背景相匹配（避免使用通用的渐变效果）。
- [ ] 悬停状态能提供清晰、稳定的视觉反馈。

### 用户体验与可访问性
- [ ] 所有交互元素都应显示`cursor-pointer`。
- [ ] 表单输入框应配有标签；图片应带有alt文本。
- [ ] 文本对比度至少达到4.5:1（在亮色/暗色模式下均需满足此要求）。
- [ ] 界面在所有屏幕分辨率（375px、768px、1024px、1440px）下均能正常显示。
- [ ] 移动设备上不得出现水平滚动条。

## 可选的设计风格参考

- **极简主义**：单色背景、极少的视觉元素、简洁的字体设计。
- **极繁主义**：元素重叠、信息密集、多种设计元素的混合使用。
- **复古未来主义**：采用Chrome浏览器特有的视觉效果、霓虹色调、80年代风格的元素。
- **奢华/精致风格**：使用金色或深色调作为装饰元素、使用衬线字体、保持适当的间距。
- **趣味/儿童化风格**：圆角设计、明亮的浅色调、活泼的动画效果。
- **编辑风格/杂志风格**：基于网格的布局、醒目的标题、清晰的层次结构。
- **原始主义/工业风格**：使用等宽字体、强烈的对比度、具有工业风格的视觉元素。
- **艺术装饰风格**：尖锐的几何形状、金属质感的装饰元素、复杂的边框设计。

*请坚持选择一种设计风格，并将其完整地贯彻到整个项目中——切勿采用折中的方案。*
