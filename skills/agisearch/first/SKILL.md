# 第一性原理思维（First Principles Thinking）

这是一种以“局部优先”（local-first）的思维方式，用于将复杂问题分解为其最基本的本质，并从头开始重新构建解决方案。

当用户遇到以下情况时，可以使用这种思维方式：
- 被传统观念或固有思维模式束缚，无法找到解决问题的新方法；
- 希望获得独创性的见解，而非重复他人的建议；
- 需要区分事实与假设；
- 需要从零开始重新设计战略、产品、商业模式或决策方案；
- 对某件事的必要性、最优性或可行性存有疑问。

这种思维方式应具备以下作用：
1. 明确问题的真正目标；
2. 揭露所有显性或隐性的假设；
3. 区分事实、类比、惯例以及默认的解决方案；
4. 将问题分解为最基本的、不可再简化的组成部分；
5. 识别出真正的限制因素、变量以及各种权衡因素；
6. 从头开始构建一个更优的解决方案；
7. 提供具体的行动建议。

## 输出结构（Output Structure）

在适用的情况下，答案可以按照以下结构组织：
- 目标（Goal）
- 假设（Assumptions）
- 真实情况（What is actually true）
- 不可再简化的组成部分（Irreducible components）
- 限制因素与权衡（Constraints and tradeoffs）
- 重新构建的解决方案（Rebuilt solution）
- 下一步行动（Best next action）

## 原则（Principles）
- 不要将惯例视为真理；
- 不要将普遍的做法视为证明；
- 不要将类比视为真正的理解；
- 从必须为真的前提出发；
- 朝着可实际应用的目标进行重构；
- 始终以用户的真实需求为核心。

## 成功标准（Success Standard）

一个有效的回答应该让用户产生以下感受：
- “我现在明白了问题的本质。”
- “这些假设比我想象中要脆弱得多。”
- “这个解决方案更加基于事实，更具创新性。”
- “我可以据此采取实际行动了。”
