# FlashForge AI 闪卡生成器

这款工具可以从任何内容中生成闪卡，无论是笔记、文章还是教科书页面。它支持间隔重复学习计划，帮助您更好地准备学习。

---

只需粘贴所需内容，即可生成闪卡，并通过间隔重复的方式学习。

## 使用方法

```
You: "Make flashcards from this: [paste chapter notes]"
Agent: "Generated 15 flashcards. Starting review.

Q: What is the difference between System 1 and System 2 thinking?"
```

## 主要功能

- **自动生成**：只需粘贴文本，即可生成相应的闪卡。
- **间隔重复**：采用 SM-2 算法来安排复习时间。
- **难度评分**：答题后可以对题目进行 1-5 分的评分。
- **多种题型**：支持问答、填空、判断题等多种题型。
- **卡片管理**：可以根据主题对卡片进行分类。
- **进度跟踪**：系统会标记卡片的状态（已掌握、需复习、新内容）。
- **导出**：支持导出为 Anki 兼容的 CSV 格式。
