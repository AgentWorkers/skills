# 🎨 Doubao Seedream & Seedance API 技能

> **专业 AI 生成套件** - 由 Doubao Seed 模型提供支持

这套 AI 生成工具集包含了 **Doubao Seedream**（用于图像生成）、**Doubao Seedance**（用于视频制作）以及 **Doubao Seed Vision**（用于视觉理解）等功能。

[![支持 Docker](https://img.shields.io/badge/docker-supported-blue.svg)]
[![许可证：MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 核心功能

### 🎨 Doubao Seedream 4.0 - 图像生成

**先进的文本到图像生成技术，具备卓越的质量和可控性。**

| 功能 | 描述 |
|---------|-------------|
| **文本到图像** | 从自然语言描述生成精美的图像 |
| **图像编辑** | 使用 AI 驱动的编辑工具修改现有图像 |
| **图像到图像** | 转换和重新构想现有图像 |
| **多尺寸支持** | 灵活的尺寸（512-2048px），支持 64px 的倍数 |
| **风格控制** | 可选择艺术风格、写实风格或自定义美学效果 |

**模型 ID**: `doubao-seedream-4-0-250828`

---

### 🎬 Doubao Seedance 1.5 - 视频制作

**专业视频制作工具，支持高级的相机控制和动态效果。**

| 功能 | 描述 |
|---------|-------------|
| **文本到视频** | 从文本描述生成具有电影质感的视频 |
| **图像到视频** | 为静态图像添加自然流畅的动画效果 |
| **相机控制** | 支持平移、缩放、推拉等复杂镜头动作 |
| **帧控制** | 可定义视频的开始和结束帧以实现流畅的过渡 |
| **时长控制** | 视频时长为 1-10 秒，支持平滑的动画效果 |

**模型 ID**: `doubao-seedance-1-5-pro-251215`

---

### 👁️ Doubao Seed Vision - 视觉理解

**基于多模态 AI 的高级图像分析和理解能力。**

| 功能 | 描述 |
|---------|-------------|
| **内容分析** | 全面的图像理解和描述 |
| **对象检测** | 识别并定位图像中的对象 |
| **场景理解** | 对环境和场景进行上下文分析 |
| **视觉问答** | 回答关于图像内容的问题 |

**模型 ID**: `doubao-seed-1-6-vision-250815`

### ❌ 禁止的行为

| ❌ 将 API 密钥提交到 Git | 密钥可能被公开 |

## ⚙️ 配置

### 环境变量

| 变量 | 描述 | 默认值 |
|----------|-------------|---------|
| `ARK_API_KEY` | API 密钥（**必填**） | - |
| `VOLCENGINE_BASE_URL` | API 端点 | `https://ark.cn-beijing.volces.com/api/v3` |
| `VOLCENGINE_TIMEOUT` | 请求超时（秒） | 30 |
| `VOLCENGINE_MAX_RETRIES` | 最大重试次数 | 3 |
| `VOLCENGINE_OUTPUT_DIR` | 输出目录 | `./output` |

### 配置文件

**位置**: `~/.volcengine/config.yaml`

---

### 优先级顺序

1. 环境变量 `ARK_API_KEY` ⭐ **推荐**
2. 项目配置文件 `.volcengine/config.yaml`
3. 全局配置文件 `~/.volcengine/config.yaml`
4. 默认值

---

## 💾 数据持久化

| 路径 | 数据内容 | 敏感性 |
|------|---------|-------------|
| `~/.volcengine/config.yaml` | 全局配置 | ⚠️ 可能包含 API 密钥 |
| `~/.volcengine/tasks/` | 任务历史记录 | 普通数据 |
| `~/.volcengine/state/` | 状态文件 | 普通数据 |
| `./.volcengine/config.yaml` | 项目配置 | ⚠️ 可能包含 API 密钥 |

**安全提示：**
- 将配置文件的权限设置为 600
- 将 `.volcengine/` 添加到 `.gitignore` 文件中
- 定期清理旧的任务记录

## ❌ 错误处理

| 错误类型 | 描述 | 解决方案 |
|-------|-------------|----------|
| 认证失败 | API 密钥无效 | 检查 `ARK_API_KEY` |
| 请求频率过高 | 超过请求限制 | 等待片刻后重试 |
| 网络问题 | 连接失败 | 检查网络连接 |
| 参数错误 | 输入格式不正确 | 检查参数格式 |
| 模型不可用 | 验证模型 ID |
