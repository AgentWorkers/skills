# AI Genie 3 游戏提示

## 概述

本技能提供了专门用于使用 Google 的 Genie 3 创建 AI 视频游戏的提示模式和示例。通过有效的提示设计，这些提示能够将自然语言描述转化为可玩的游戏体验。

## 提示模式

### 核心结构

有效的 Genie 3 提示遵循以下结构：

```
[Character description] has to [action] through [environment that defies logic], [visual elements/effects]
```

**关键要素：**
1. **角色**：主角是谁？（例如：“法国女性”、“机器人”、“太空探索者”）
2. **动作**：核心游戏玩法是什么？（例如：“穿越”、“导航”、“生存”）
3. **环境**：故事发生在哪里？强调超现实或违反物理定律的设定
4. **视觉效果**：什么让游戏在视觉上具有吸引力？

### 成功的示例

#### 示例 1：超现实冒险
**提示**：“法国女性必须穿越一个违背逻辑的‘词’，周围到处都是飞舞的物体”

**数据统计**：
- 原始推文链接：https://x.com/TrueSlazac/status/2016959063699906740
- 点赞数：6,450
- 转发数：252
- 收藏数：1,124

**为什么有效**：
- 角色明确：“法国女性”
- 动作具体：“穿越”
- 环境超现实：“违背逻辑的‘词’”
- 视觉效果动态：“周围到处都是飞舞的物体”

#### 示例 2：挑战重力的平台游戏
**提示**：“机器人必须跳过在天空中不断移动的岛屿，瀑布向上流淌”

**为什么有效**：
- 角色具体：“机器人”
- 动作明确：“跳过”
- 物理定律被打破：“岛屿在移动”
- 视觉效果新颖：“瀑布向上流淌”

## 提示模板

### 模板 1：冒险探索
```
[Character] has to [traverse/exploration verb] through [surreal environment], [unexpected phenomena] everywhere
```

**示例**：
- “赛博朋克黑客必须潜入一个不断变化的数字迷宫，霓虹灯随处出现”
- “中世纪骑士必须穿越由水晶构成的森林，动物们倒挂在空中”

### 模板 2：生存挑战
```
[Character] needs to survive in [impossible setting], [dangerous elements] appearing randomly
```

**示例**：
- “宇航员必须在一个由熔岩构成的星球上生存，冰暴会随机出现”
- “潜艇船长必须穿越一片由玻璃构成的海洋，水母由光构成”

### 模板 3：解谜游戏
```
[Character] solves [type of puzzles] by [unusual method], in a world where [impossible rules]
```

**示例**：
- “侦探通过进入画作来破案，在这个世界里艺术变成了现实”
- “巫师通过倒着唱歌来打开门，在这座城堡里物理定律随音乐而变化”

## 最佳实践

### 1. 角色要具体
✅ **好的示例**：“法国女性”、“赛博朋克侦探”、“赛博朋克武士”
❌ **不好的示例**：“某人”、“一个人”、“角色”

### 2. 强调违反物理定律的设定
✅ **好的示例**：“违背逻辑的‘词’”、“瀑布向上流淌”、“移动的岛屿”
❌ **不好的示例**：“奇怪的地方”、“奇特的世界”、“不寻常的环境”

### 3. 包含动态视觉效果
✅ **好的示例**：“周围到处都是飞舞的物体”、“霓虹灯随处出现”、“由光构成的水母”
❌ **不好的示例**：“一些视觉效果”、“有趣的视觉效果”、“酷炫的元素”

### 4. 保持提示简洁
- 总长度在 15-30 个词之间
- 专注于 3-4 个关键要素
- 避免过于复杂的描述

## 常见主题

尝试使用这些既定的主题以获得一致的游戏效果：
- **重力操控**：物体向上、向侧面或以特定模式下落
- **现实扭曲**：纹理变化、维度扭曲、物理定律被打破
- **元素混合**：水下有火、冰火山、水晶森林
- **时间/空间扭曲**：循环、传送门、不可能的几何形状
- **超现实生物**：会移动的植物、由光构成的动物、混合生物

## 测试你的提示

在最终确定提示之前，请问自己：
1. **角色是否明确？** 我能立即想象出这个角色的样子吗？
2. **动作是否具体？** 我能理解游戏的具体玩法吗？
3. **环境是否违反物理定律？** 它是否违背了常规的物理规律？
4. **是否有动态视觉效果？** 玩家会看到什么在移动或变化？
5. **提示是否简洁？** 我能在 10 秒内读完它吗？
