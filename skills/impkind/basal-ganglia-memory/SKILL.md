# 基底节记忆 🎯  
**AI代理的习惯形成功能**。属于“AI大脑”系列技能之一。  

## 状态：🚧 开发中  
该技能目前正在开发中。请关注后续更新！  

## 基本原理  
基底节负责习惯的形成和程序性学习。该技能将使AI代理具备以下能力：  
- **习惯追踪**：重复的动作会逐渐成为自动化的行为习惯；  
- **程序性记忆**：对常见工作流程的“肌肉记忆”（即无需思考即可执行的操作）；  
- **基于奖励的学习**：强化有效的行为模式；  
- **偏好形成**：形成“我总是以这种方式做事，因为这样更有效”的行为习惯。
