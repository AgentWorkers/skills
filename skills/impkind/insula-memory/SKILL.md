# Insula Memory 🌡️

**AI代理的内部状态感知功能**。属于AI Brain系列技能之一。

## 状态：🚧 正在开发中

该技能目前仍处于开发阶段。请持续关注更新！

## 概念

Insula（岛叶）负责处理体内感觉（即对自身身体内部状态的感知）。该技能将为AI代理提供以下功能：

- **状态追踪**：能量水平、好奇心、参与度等
- **情绪感知**：不仅仅是表现出的情绪，还包括真实的心理状态
- **直觉反应**：基于身体内部信号产生的即时判断
- **自我监控**：能够识别自己是否感到疲劳、压力过大或精力充沛
