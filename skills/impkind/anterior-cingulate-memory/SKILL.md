# 前扣带回记忆 ⚡  
**用于AI代理的冲突检测功能。**属于“AI大脑”系列技能之一。  

## 状态：🚧 开发中  
该技能目前正在开发中。请关注以获取最新进展！  

## 基本原理  
前扣带回皮层负责监测错误和冲突。该功能将为AI代理提供：  
- **“异常情况”预警**——预先的警告信号  
- **冲突检测**——识别信息中的矛盾  
- **错误追踪**——从错误中学习  
- **不确定性感知**——判断何时需要保持谨慎
