# 开立企业账户

帮助用户使用 `vivid-mcp` 开立 Vivid 企业账户。

## 触发条件

用户希望开设企业账户或开始企业入职流程。

## 流程

1. 如果用户未提供相关信息，询问其**法人类型**（例如：GmbH、UG、自由职业者等）。
2. 如果用户未提供相关信息，询问其所在国家。默认为国家：德国。
3. 调用 MCP（Vivid 的管理控制面板）以获取所需的字段和认可的文档类型。
4. 提供两种方式供用户选择：
   - **上传文档**：系统会自动提取并预先填充相关信息。
   - **手动输入**：通过聊天界面收集用户提供的信息。
5. 使用确认后的数据调用 `build_onboarding_link` 函数，并返回相应的链接。

## 规则

- 必须首先从 MCP 获取所有必要的信息，切勿直接使用硬编码的值。
- 将用户上传的文档视为敏感信息，仅对其进行摘要处理，切勿显示文档的实际内容。
- 绝不要询问用户的密码或 API 密钥。
- 如果出现错误，应提示用户补充缺失的字段或建议更换所需的文档。
