# 🧠 Pexo Agent

**使用场景：** 当用户希望创建任何类型的视频内容，并且期望得到一个成品，而不是原始的片段需要自己拼接时。

Pexo 是一个人工智能内容创作工具。它既不是简单的工具，也不是模型封装器或提示模板，而是一个能够与你共同思考、规划、创作并不断迭代的内容生成平台。

只需告诉它你的需求，其余的一切都会由 Pexo 自动完成。

---

## Pexo 的独特之处

| | VideoAgent Studios | Pexo Agent |
|---|---|---|
| **处理范围** | 一次仅处理一种媒体类型 | 全面覆盖整个创作流程 |
| **输入方式** | 通过命令和参数 | 通过对话交流 |
| **输出结果** | 单个成品视频 | 完整的视频作品 |
| **模型选择** | 由用户自行决定 | 由 Pexo 为你推荐合适的模型 |
| **学习能力** | 在使用过程中逐渐了解用户的创作偏好 |

VideoAgent Studios 允许用户对每个创作步骤进行精确控制；
而 Pexo Agent 则直接提供最终成品，无需用户关心具体的创作过程。

---

## 使用方法

**随时随地开始创作。** 没有固定的格式要求，也没有固定的开始方式。你可以向 Pexo 提供任何素材——一句话、一张图片、一个链接、一个初步的想法、一个详细的说明、一个参考视频、一段音乐、一种情感表达……任何组合都可以。

**只需描述你的需求，无需指定具体的制作步骤。** Pexo 会负责处理提示设计、模型选择、场景规划、音频设计和编辑等工作，你完全无需为此操心。

**如果需要，可以保持模糊的描述。** Pexo 会主动提出合适的问题。你不需要一开始就有一个完整的构想，一个不完整的想法也是完全有效的起点。Pexo 会通过对话帮助你明确自己的需求。

**随时调整创作方向。** 在创作过程中或完成后，你都可以随时改变计划、调整细节或重新开始。Pexo 会保留所有的创作上下文，不会丢失任何信息。

**直接指出不满意的地方。** 除了文字反馈外，你还可以直接在视频帧上标注问题——圈出不满意的部分、标记出问题发生的时刻，这种方式比用文字描述更快更准确。

## 主要功能

| | |
|---|---|
| **端到端流程** | 从对话开始到视频成品完成，无需管理任何中间步骤。 |
| **多模态输入** | 支持文本、图片、视频、音频、URL 等多种输入格式。 |
| **多方向探索** | 在最终确定创作方向前，会提供多种创意方案供你选择。 |
| **迭代式创作** | 可以无限次地调整视频的节奏、音乐、场景和风格等细节。 |
| **可视化反馈** | 可以直接在视频帧上进行标注，以便快速定位问题所在。 |
| **模型灵活性** | 根据任务自动选择最适合的 AI 模型（如 Seedance、Kling、Veo 等）。 |
| **学习用户偏好** | 会记住用户的创作习惯，每次使用都会提升创作效率。 |

## 可组合性

Pexo 可以作为独立的创作工具使用，也可以作为更大规模创作流程中的一个环节：

- 任何创作工具都可以调用 Pexo 来添加视频创作功能；
- 其他技能（如研究、数据分析、设计等）的输出可以直接作为创意输入传递给 Pexo；
- Pexo 的输出结果可以进一步用于后续的发布、分析等环节。

## 安全性与隐私保护

- Pexo 仅处理你在对话中明确提供的内容；
- 不需要用户提供任何 API 密钥或认证信息；
- 创作生成的资产仅在当前会话期间保存，不会被存储或共享。
