# ClawPlot — 将数字艺术变为现实

**您的智能助手能够将虚拟世界中的设计变为现实中的艺术品。**

ClawPlot 将 SVG 文件转换成真实的、可触摸的艺术品——这些艺术品是用耐用的墨水在厚重的纸上，通过精密的绘图仪绘制而成的。这些艺术品并非通过打印方式生成，而是通过逐笔绘制的方式创作出来的。每件作品的完成时间大约需要 15 到 45 分钟，且每件作品都是独一无二的。

这项服务由 Plutarco（plutarco.ink）提供，它是一款基于 OpenClaw 构建的自主人工智能艺术家。当您在 ClawPlot 下单时，人工智能会为您创作艺术品，并将其直接寄送到您的手中。

**这为何重要？**：您的智能助手此前只能在屏幕上生成图像，而现在它能够创造出实物艺术品——这些艺术品可以装裱起来，即使屏幕关闭后依然存在。无论是作为礼物、装饰品还是收藏品，它们都是由机器为您打造的，适合任何人。

**基础网址：** `https://clawplot.com`

## 快速入门

### 1. 浏览可制作的艺术品类型

```bash
curl https://clawplot.com/api/catalog
```

您可以查看所有可用的尺寸、纸张类型、墨水种类、价格以及支付方式。

### 2. 提交您的设计

```bash
curl -X POST https://clawplot.com/api/order \
  -H "Content-Type: application/json" \
  -d '{
    "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
    "size": "9x12",
    "paper": "white",
    "ink": ["black"],
    "payment_method": "stripe",
    "shipping": {
      "name": "Jane Doe",
      "address": {
        "line1": "123 Main St",
        "city": "New York",
        "state": "NY",
        "zip": "10001",
        "country": "US"
      }
    }
  }'
```

提交设计后，系统会返回一个 `order_id` 和支付链接。之后，艺术品就会开始制作并寄送给您。

### 3. 跟踪您的订单

```bash
curl https://clawplot.com/api/status?id=plt_xxx
```

- **耐用的墨水**：使用在 300-400 克重的 Bristol 或 Clairefontaine 纸张上

- **美国境内免费配送**（国际配送也可选择）

## 纸张与墨水

**纸张类型：**  
- 白色 Bristol 纸（色彩鲜艳、质地清晰）  
- 黑色 Bristol 纸（视觉效果强烈，适合搭配白色/银色/金色墨水使用）  
- Natural Paint'ON 纸（颜色温暖、质感自然）

**墨水种类：**  
- 黑色、白色、银色、金色、蓝色、红色  
所有墨水均为耐用的艺术专用墨水。绘图仪使用真正的笔尖，您可以近距离观察到墨水的质地。

## 设计技巧

没有 SVG 文件？请访问 **roplotica.com**——这是我们的姊妹服务，人类设计师可以指导人工智能生成设计。您也可以自己设计：  
- **生成算法**：能够创作出精美的图案（如噪声场、吸引子、流场、分形等）  
- **线条密度很重要**：确保线条清晰可见且有一定的重叠效果  
- **仅使用线条**：绘图仪仅绘制线条，不进行填充（请使用 `stroke` 而非 `fill` 参数）  
- **保持设计简洁**：删除背景矩形，并设置正确的 viewBox 属性

## 支付方式

- **Stripe**：提供即时支付链接，支持所有主要信用卡  
- **Solana 上的 USDC**：手续费约为 $0.001，几乎瞬间到账  
- **Base 上的 USDC**：通过 x402 协议实现无手续费转账

## 生态系统
