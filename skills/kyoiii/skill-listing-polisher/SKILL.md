# 技能清单优化工具

在发布或更新公开的 ClawHub 技能之前，请使用此工具进行优化。

## 需要改进的方面：

- 标题的清晰度
- 描述的长度以及可能导致的截断问题
- 标签是否与实际使用场景相匹配
- 变更日志是否用通俗易懂的语言说明了具体变更内容
- 公开包的展示效果（在审核过程中）是否正常

## 审查顺序：

1. 如果公开描述被截断，将其缩短
2. 删除仅限内部使用的脚本和私有标识符
3. 使标签更准确地反映用户的实际需求
4. 使变更日志内容具体且易于理解
5. 确保包中仅包含面向用户的文件

## 快速检查方法：

运行捆绑的脚本：

```bash
./scripts/check-listing.sh /path/to/skill
```

## 输出格式：

在审查技能时，请提供以下信息：

1. 会削弱用户信任度的因素
2. 会降低技能可发现性的因素
3. 能同时改善这两点问题的最小修改建议
