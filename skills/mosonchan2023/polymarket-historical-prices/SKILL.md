# Polymarket 历史价格数据

您可以获取 Polymarket 预测市场的历史价格数据和图表，从而分析过去的交易表现。

## 主要功能

- **OHLCV 数据**：开盘价（Open）、最高价（High）、最低价（Low）、收盘价（Close）、成交量（Volume）
- **时间范围**：每日、每小时、每分钟
- **趋势分析**：移动平均线、价格模式分析

## 价格信息

- **费用**：每次 API 调用费用为 0.001 美元（USD）
- **支付方式**：通过 SkillPay.me 进行自动支付

## 集成方式

该技能已与 SkillPay.me 集成，支持自动微支付功能。
