# Polymarket 加密货币预测服务

获取关于加密货币事件的预测市场数据，包括比特币价格目标、ETF 批准情况、协议发布等信息。

## 主要功能

- **价格预测**：比特币（BTC）/以太坊（ETH）的价格是否会达到某个特定水平？
- **事件预测**：ETF 的批准情况、网络减半事件、协议升级等
- **山寨币市场**：Solana（SOL）、狗狗币（DOGE）等山寨币的预测结果

## 价格信息

- **费用**：每次 API 调用费用为 0.001 美元（USDT）
- **支付方式**：通过 SkillPay.me 实现自动支付

## 集成方式

该服务已与 SkillPay.me 集成，支持自动微支付。每次 API 调用费用为 0.001 美元。
