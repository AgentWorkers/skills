# 招聘助理 🦞

该技能实现了一套高标准的招聘工作流程，专为技术岗位招聘设计，特别适配深圳市场。

## 工作流程

### 1. 严格的简历筛选
从专业角度对候选人进行评估：
- **单份简历筛选**：`node scripts/screen_resume.js <简历路径> --语言 <语言> --工作年限 <工作经验年数>`
- **批量简历筛选**：`node scripts/batch_screen.js <文件夹路径> --评分阈值 <评分标准> --语言 <语言> --工作年限 <工作经验年数>`
- **输出要求**：
    1. **严格的评分标准**：遵循 `references/hiring-criteria.md` 中的 0-100 分制。对于“高级”职位，必须体现候选人在架构设计方面的影响力以及熟练使用人工智能的能力。
    2. **详细分析**：明确列出候选人的 3-4 项技术优势以及显著的劣势或知识短板。
    3. **独立报告**：每位候选人的评估结果需保存在单独的文档中。
    4. **薪资对比**：根据深圳市场薪资标准（Boss Zhipin 2026）对候选人的期望薪资进行评估。
    5. **通知人力资源部门**：得分较高的候选人（评分 >= 阈值）应通过 `message` 工具汇总并发送给人力资源部门。

### 2. 人工智能能力评估
强制检查候选人是否使用过人工智能工具（如 Cursor、Copilot、LLM API）。未使用人工智能工具被视为重要的工作效率短板。

### 3. 面试准备与总结
- **面试问题生成**：`node scripts/generate_questions.js <输入数据>`（根据评估出的候选人弱点生成面试问题）
- **面试总结**：`node scripts/summarize_interview.js <面试记录文件>`（使用 `assets/report-template.md` 中的模板进行总结）

## 市场基准（深圳 2026）
请参考 [references/hiring-criteria.md](references/hiring-criteria.md) 以获取最新的薪资数据和评分标准。

## 核心原则
- **专业评估**：不要轻易给出高分；高级职位需要提供在架构设计方面的实际成果证明。
- **数据驱动**：所有评估标准必须与当前深圳科技市场的实际情况保持一致。
- **人工智能优先**：在现代高级工程师的招聘中，高效利用人工智能是核心要求。
