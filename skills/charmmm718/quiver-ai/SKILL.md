# QuiverAI 技能

QuiverAI 是一个基于人工智能的矢量图形生成平台，能够根据文本提示和参考图像生成高质量的 SVG 图形。

## 入门指南

### 网页界面
1. 访问 https://app.quiver.ai/creations
2. 登录或创建账户
3. 输入描述所需图形的文本提示
4. （可选）上传参考图像
5. 生成并下载 SVG 文件

### API 使用
如需通过编程方式使用 QuiverAI 的功能，请参阅 [API 参考文档](references/api.md)。
