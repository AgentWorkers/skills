# 高级质量经理（QMR）职责

根据ISO 13485第5.5.2条款的要求，负责质量体系的问责制、管理评审的领导工作以及监管合规性的监督。

## QMR职责

### ISO 13485第5.5.2条款要求

| 职责 | 范围 | 依据 |
|----------------|-------|----------|
| 质量管理体系（QMS）的有效性 | 监控体系性能和适用性 | 管理评审记录 |
| 向管理层报告 | 向高层管理人员通报QMS的绩效 | 质量报告、仪表板 |
| 提高质量意识 | 推广监管和质量要求 | 培训记录、沟通 |
| 与外部方联络 | 与监管机构和认证机构沟通 | 会议记录、往来邮件 |

### QMR问责矩阵

| 领域 | 负责人 | 报告对象 | 频率 |
|--------|-----------------|------------|-----------|
| 质量政策 | 政策的充分性和沟通 | 首席执行官/董事会 | 年度审查 |
| 质量目标 | 目标的达成情况 | 执行团队 | 季度 |
| QMS绩效 | 体系有效性指标 | 管理层 | 月度 |
| 监管合规性 | 各司法管辖区的合规状态 | 首席执行官 | 季度 |
| 审计计划 | 审计计划的完成情况、问题处理 | 管理层 | 每次审计后 |
| 不符合项的纠正措施（CAPA） | CAPA的有效性和及时性 | 执行团队 | 月度 |

### 权限范围

| 决策类型 | QMR的权限 | 需要升级的情况 |
|---------------|---------------|---------------------|
| QMS内的流程变更 | 与负责人共同批准 | 重大流程重新设计 |
| 文件审批 | 最终的质量保证（QA）审批 | 政策级别的变更 |
| 不符合项的处理 | 与质量评审委员会（MRB）共同决定 | 产品发布决策 |
| 供应商质量行动 | 质量暂停、审计 | 供应商终止合作 |
| 审计安排 | 调整内部审计计划 | 外部审计时间安排 |
| 培训需求 | 确定培训需求 | 全组织范围内的培训预算 |

## 管理评审工作流程

根据ISO 13485第5.6条款的要求，进行管理评审。

### 工作流程：准备和执行管理评审

1. 安排管理评审（至少每年一次，通常为季度或半年一次）
2. 至少提前两周通知所有需要参加的人员
3. 从流程负责人处收集所需的信息：
   - 审计结果（内部和外部）
   - 客户反馈（投诉、满意度、退货）
   - 流程绩效和产品符合性
   - CAPA的状态和有效性
   - 上次评审的待办事项
   | 影响QMS的变更（监管、组织方面的）
   | 改进建议
4. 编制包含趋势分析的输入摘要报告
5. 准备包含支持数据的演示材料
6. 在评审前一周分发议程和输入资料包
7. 按照议程进行评审会议
8. **验证：**所有必要的输入都已审查；决策与负责人及截止日期一起记录下来

### 必需参加的人员

| 角色 | 要求 | 输入责任 |
|------|-------------|---------------------|
| 首席执行官/总经理 | 必需 | 战略性决策 |
| QMR | 主持人 | 整体QMS状况 |
| 部门负责人 | 必需 | 流程绩效 |
| 质量审核经理 | 必需 | 监管变更 |
| 生产经理 | 必需 | 产品符合性 |
| 客户质量部门 | 必需 | 投诉数据 |

## 质量KPI管理工作流程

建立、监控并报告质量绩效指标。

### 工作流程：建立质量KPI框架

1. 确定需要衡量的质量目标
2. 根据SMART标准为每个目标选择KPI：
   - 具体：明确的定义和计算方法
   - 可衡量：可用数据可以量化
   - 可操作：团队能够影响结果
   - 相关：与质量目标一致
   - 有时间限制：定义了测量频率
3. 根据基准数据和标杆确定目标值
4. 分配数据来源和收集责任
5. 为每个KPI类别确定报告频率
6. 配置仪表板显示和趋势分析
7. 设定升级阈值和警报触发条件
8. **验证：**每个KPI都有负责人、目标、数据来源和升级标准

### 核心质量KPI

| 类别 | KPI | 目标 | 计算方法 |
|----------|-----|--------|-------------|
| 流程 | 首次通过率 | >95% | （首次通过的单位数 / 总单位数）× 100 |
| 流程 | 不符合项率 | <1% | （不符合项数量 / 总单位数）× 100 |
| CAPA | CAPA关闭率 | >90% | （按时关闭的CAPA数量 / 应关闭的CAPA数量）× 100 |
| CAPA | CAPA有效性 | >85% | （有效的CAPA数量 / 经核实的CAPA数量）× 100 |
| 审计 | 发现关闭率 | >90% | （按时关闭的发现数量 / 应关闭的发现数量）× 100 |
| 客户 | 投诉率 | <0.1% | （投诉数量 / 销售单位数）× 100 |
| 客户 | 满意度得分 | >4.0/5.0 | 调查得分的平均值 |

### KPI审查频率

| KPI类型 | 审查频率 | 趋势周期 | 目标受众 |
|----------|------------------|--------------|----------|
| 安全/合规 | 每日监控 | 每周 | 运营部门 |
| 生产质量 | 每周 | 每月 | 部门负责人 |
| 客户质量 | 每月 | 每季度 | 执行团队 |
| 战略质量 | 每季度 | 每年 | 董事会/高层管理层 |

### 绩效响应矩阵

| 绩效水平 | 状态 | 需要采取的行动 |
|-------------------|--------|-----------------|
| 超过目标的110% | 超出目标 | 考虑提高目标 |
| 达到目标的100-110% | 达到目标 | 维持当前方法 |
| 接近目标的90-100% | 接近目标 | 密切监控 |
| 低于目标的80% | 低于目标 | 需要改进计划 |
| 低于目标的80% | 低于目标 | 立即干预 |

详见：[references/quality-kpi-framework.md](references/quality-kpi-framework.md)

## 质量目标工作流程

根据ISO 13485第5.4.1条款的要求，建立并维护可衡量的质量目标。

### 工作流程：年度质量目标设定

1. 审查上一年的目标达成情况
2. 分析质量绩效趋势和差距
3. 与组织战略计划保持一致
4. 起草具有可衡量目标的目标
5. 确认实现目标的资源可用性
6. 获得高层管理的批准
7. 在整个组织内传达目标
8. **验证：**每个目标都是可衡量的，有负责人、目标和时间表

### 质量目标结构

```
QUALITY OBJECTIVE [Number]

Objective Statement: [Clear, measurable statement]
Aligned to Policy Element: [Quality policy section]
Target: [Specific measurable target]
Baseline: [Current performance]
Owner: [Name and title]
Due Date: [Target achievement date]

Success Criteria:
- [Criterion 1]
- [Criterion 2]

Measurement Method: [How progress is tracked]
Reporting Frequency: [Monthly/Quarterly]

Supporting Initiatives:
- [Initiative 1]
- [Initiative 2]

Resource Requirements:
- [Resource 1]
- [Resource 2]
```

### 目标类别

| 类别 | 示例目标 | 典型目标 |
|----------|-------------------|-----------------|
| 客户质量 | 降低投诉率 | <0.1%的投诉单位数 |
| 过程质量 | 提高首次通过率 | >96% |
| 合规性 | 维持认证 | 零重大不符合项 |
| 效率 | 降低质量成本 | <4%的收入 |
| 文化 | 提高培训完成率 | >98%的培训按时完成 |

### 季度目标审查

| 审查要素 | 评估 | 行动 |
|----------------|------------|--------|
| 进展与目标对比 | 符合目标 / 落后 / 超前 | 如果落后，调整资源 |
| 相关性 | 仍然有效 / 需要更新 | 如果情况变化，进行修改 |
| 资源 | 资源充足 / 不足 | 如有需要，请求额外资源 |

## 质量文化评估工作流程

评估并改进组织的质量文化。

### 工作流程：年度质量文化评估

1. 设计或选择质量文化调查工具
2. 确定调查对象（所有员工或样本）
3. 传达调查目的和保密性
4. 在两周内进行调查并收集反馈
5. 按部门、角色和任职时间分析结果
6. 识别优势和改进领域
7. 制定针对文化差距的行动计划
8. **验证：**响应率>60%；行动计划针对得分最低的30%的员工

### 质量文化维度

| 维度 | 指标 | 评估方法 |
|-----------|------------|-------------------|
| 领导力承诺 | 管理层对质量的可见支持 | 调查、观察 |
| 质量责任 | 员工感到对质量负责 | 调查 |
| 沟通 | 质量信息有效传递 | 调查、审计 |
| 持续改进 | 建议的提出和实施 | 指标 |
| 培训和能力 | 员工感到接受过充分培训 | 调查、记录 |
| 问题解决 | 问题从根本原因解决 | CAPA分析 |

### 文化调查类别

| 类别 | 示例问题 |
|----------|------------------|
| 领导力 | “管理层表现出对质量的承诺” |
| 资源 | “我有工具和培训来从事质量工作” |
| 沟通 | “质量期望得到清晰传达” |
| 赋权 | “我被鼓励报告质量问题” |
| 认可 | “质量成就得到认可” |

### 文化改进行动

| 识别出的差距 | 可能的改进措施 |
|----------------|-------------------|
| 领导力可见度低 | 开展质量现场巡查、全员质量会议 |
| 培训不足 | 基于能力的培训计划 |
| 沟通不畅 | 发布质量通讯、部门会议 |
| 报告不足 | 匿名报告系统、无责备文化 |
| 认可不足 | 设立质量奖励计划、团队庆祝活动 |

## 监管合规性监督

监控并维护各司法管辖区的监管合规性。

### 多司法管辖区合规性矩阵

| 司法管辖区 | 监管法规 | 要求 | 状态跟踪 |
|--------------|------------|-------------|-----------------|
| 欧盟 | MDR 2017/745 | CE标志、认证机构 | 技术文件、年度审查 |
| 美国 | 21 CFR 820 | FDA注册、QSR合规 | 年度注册、检查 |
| 国际 | ISO 13485 | QMS认证 | 监控审计 |
| 德国 | MPG/MPDG | 国家实施 | 向相关机构提交文件 |

### 合规性监控工作流程

1. 维护监管要求登记册
2. 订阅监管更新服务
3. 每月评估监管变化的影响
4. 在法规生效后的90天内更新受影响的流程
5. 验证针对监管变化的培训完成情况
6. 在管理评审中记录合规状态
7. 维护检查准备清单
8. **验证：**所有适用要求都已覆盖；无过期的注册记录

### 监管机构接口

| 活动 | QMR的角色 | 需要的准备 |
|----------|----------|---------------------|
| 认证机构审计 | 主要联系人 | 审计包、人员安排 |
| FDA检查 | 主办人、协调员 | 检查准备审查 |
| 相关机构查询 | 回应协调员 | 技术文件访问 |
| 监管会议 | 与会者或代理人 | 简报材料 |

### 检查准备清单

| 领域 | 准备情况 | 需要采取的行动 |
|------|-------|---------------|
| 文件控制系统 | 是 | |
| 培训记录完整 | 是 | |
| CAPA系统 | 是，无逾期项目 | |
| 投诉文件完整 | 是 | |
| 设备校准 | 是 | |
| 供应商资格文件 | 是 | |
| 管理评审记录 | 是 | |
| 内部审计计划 | 是 | |

## 决策框架

### 升级决策树

```
Issue Identified
      │
      ▼
Is it a regulatory violation?
      │
  Yes─┴─No
  │      │
  ▼      ▼
Escalate to    Is it a safety issue?
Executive          │
immediately    Yes─┴─No
               │      │
               ▼      ▼
          Escalate to   Does it affect
          Safety Team   multiple departments?
                             │
                         Yes─┴─No
                         │      │
                         ▼      ▼
                    Escalate to  Handle at
                    Executive    department level
```

### 质量投资优先级

| 标准 | 权重 | 评分方法 |
|----------|--------|--------------|
| 监管要求 | 30% | 必需=10分，推荐=5分，可选=2分 |
| 客户影响 | 25% | 直接影响=10分，间接影响=5分，无影响=0分 |
| 成本节省潜力 | 20% | >10万美元=10分，5万-10万美元=7分，<5万美元=3分 |
| 实施复杂性 | 15% | 简单=10分，中等=5分，复杂=2分 |
| 战略一致性 | 10% | 核心要求=10分，支持性要求=5分，次要要求=2分 |

### 资源分配矩阵

| 资源类型 | 分配权限 | 升级阈值 |
|---------------|---------------------|---------------------|
| 质量人员 | QMR | 需要增加超过1个全职员工 |
| 质量设备 | QMR | 超过2.5万美元 |
| 外部顾问 | QMR | 超过5万美元或超过30天 |
| 质量系统 | 需要高层管理批准 | 超过10万美元 |
