# 技能重定向：**内容创建者**

> **此技能已被拆分为两个更专业的技能。** 请选择符合您需求的技能：

| 您想要... | 选择相应的技能 |
|----------------|-----------------|
| **撰写** 博文、文章或指南 | [内容制作](../content-production/) |
| **规划** 要创建的内容、主题分类、内容日程表 | [内容策略](../content-strategy/) |
| **分析品牌语言风格** | [内容制作](../content-production/) （包含 `brand_voice_analyzer.py`） |
| **优化现有内容的 SEO** | [内容制作](../content-production/) （包含 `seo_optimizer.py`） |
| **创建社交媒体内容** | [社交媒体内容](../social-content/) |

## 为何进行拆分

原来的“内容创建者”技能试图涵盖所有方面：规划、写作、SEO 优化以及社交媒体内容管理。这种做法导致该技能变得“样样都会但样样不精”。现在，这两个专业技能能够更专注于各自的领域：

- **内容制作**：提供完整的内容制作流程：研究 → 制定写作大纲 → 草稿编写 → 优化 → 发布。该技能继承了原“内容创建者”工具包中的所有 Python 工具。
- **内容策略**：负责内容规划，包括主题分类、关键词研究、内容日程表制定以及优先级排序。

## 智能路由规则

- **用户询问“内容创建者”** → 重定向至 [内容制作](../content-production/)（用户很可能希望撰写内容）。
- **用户询问“内容规划”或“我应该写什么”** → 重定向至 [内容策略](../content-strategy/)。

## 输出结果

| 用户请求... | 被路由至... |
|---------------------|-------------|
| “撰写一篇博客文章” | [内容制作](../content-production/) |
| “内容日程表” | [内容策略](../content-strategy/) |
| “品牌语言风格分析” | [内容制作](../content-production/)（使用 `brand_voice_analyzer.py`） |
| “SEO 优化” | [内容制作](../content-production/)（使用 `seo_optimizer.py`） |

## 说明

这是一个技能重定向功能：请将用户引导至相应的专业团队或工具。请勿尝试在此直接处理用户的请求。

## 相关技能

- **内容制作**：负责完整的内容制作流程。
- **内容策略**：负责内容规划和主题选择。
- **内容优化**：用于提升内容的真实感和吸引力。
- **营销背景知识**：这两个专业技能都需要了解的基础知识。
