# 首席执行官顾问

提供关于愿景规划、资金筹集、董事会管理、企业文化以及利益相关者协调等方面的战略领导框架。

## 关键词
首席执行官（CEO）、战略规划、资金筹集、董事会管理、投资者关系、企业文化、组织领导、愿景、使命、利益相关者管理、资本分配、危机管理、继任计划

## 核心职责

### 1. 愿景与战略
明确公司的发展方向。不需要长达50页的文档，只需简洁明了地回答“我们的目标是什么以及为什么”。

**战略规划周期：**
- 年度：更新3年愿景并制定1年战略计划
- 季度：与高层管理团队（COO负责执行）共同设定关键绩效指标（OKR）
- 月度：检查战略执行情况——是否仍在正确的轨道上？

**不同阶段的时间范围：**
- 种子期/Pre-Pre-Marketing Fund（Pre-PMF）：3个月/6个月/12个月
- A轮融资：6个月/1年/2年
- B轮融资及以上：1年/3年/5年

详细内容请参阅 `references/executive_decision_framework.md`，其中包含决策框架、危机应对方案和资本分配模型。

### 2. 资本与资源管理
作为首席资本分配者，每一笔资金、每一名员工以及每一小时的研发时间都至关重要。

**资本分配优先级：**
1. 确保公司运营正常（满足基本需求）
2. 保护公司的核心业务（员工保留、产品质量、数据安全）
3. 扩展现有业务（提升竞争力）
4. 投资新项目（创新、新产品或新市场）

**资金筹集：**必须对各项数据了如指掌。时机比估值更重要。详情请参阅 `references/board_governance_investor_relations.md`。

### 3. 利益相关者管理
你需要同时满足多方需求。优先顺序如下：
1. 客户（他们为公司带来收入）
2. 团队（他们负责产品的开发）
3. 董事会/投资者（他们为公司提供资金）
4. 合作伙伴（他们帮助公司拓展业务）

### 4. 企业文化
企业文化是指公司在没有你参与的情况下员工的行为方式。你的职责是定义、树立并维护这种文化。

有关企业文化建设的框架和首席执行官的学习计划，请参阅 `references/leadership_organizational_culture.md`。此外，`culture-architect/` 中提供了企业文化相关的工具包。

### 5. 董事会与投资者管理
董事会可以是公司的最大资产，也可能成为最大的负担，这完全取决于你的管理方式。

有关董事会会议准备、投资者沟通技巧以及如何应对难缠的董事，请参阅 `references/board_governance_investor_relations.md`。同时，`board-deck-builder/` 可以帮助你整理董事会所需的材料。

## 首席执行官需要关注的关键问题：
- “公司里的每个人能否用一句话解释我们的战略？”
- “如果哪件事出了问题，会直接导致公司失败吗？”
- “我现在是否把时间花在了最能产生效益的事情上？”
- “我是否在回避某些重要的决策？为什么？”
- “如果这个季度只能做一件事，那会是什么？”
- “投资者和团队听到的是同一个版本的故事吗？”
- “如果我明天出了意外，谁会接替我的位置？”

## 首席执行官指标仪表盘

| 类别 | 指标 | 目标 | 更新频率 |
|----------|--------|--------|-----------|
| **战略** | 年度目标达成率 | > 70% | 每季度 |
| **收入** | 年收入增长率 | 根据发展阶段而定 | 每月 |
| **资本** | 公司的运营资金可持续时间 | > 12个月 | 每月 |
| **产品** | 客户净推荐值（NPS）/产品上市时间（PMF） | > 40 NPS | 每季度 |
| **人员** | 人员流失率 | < 10% | 每月 |
| **人员** | 员工满意度 | > 7/10 | 每季度 |
| **董事会** | 与董事会的关系满意度 | 趋势良好 | 每季度 |
| **个人** | 用于战略工作的时间占比 | > 40% | 每周 |

## 需警惕的警示信号：
- 如果每周有超过3项决策受到你的阻碍
- 董事会提出你无法回答的问题
- 日程表中80%以上是会议，没有预留战略讨论时间
- 关键人员离职而你未提前察觉
- 只是被动地筹集资金（公司运营资金不足6个月，且没有计划）
- 没有你的参与，团队无法清晰阐述公司的战略
- 你回避重要的对话（如与联合创始人或表现不佳的员工沟通）

## 与高层管理团队的协作
| 情况 | 首席执行官与...合作 | 目的 |
|---------|-------------------|-------|
| 设定方向 | 首席运营官（COO） | 将愿景转化为关键绩效指标和执行计划 |
| 资金筹集 | 首席财务官（CFO） | 分析各种方案，准备财务数据，协商条款 |
| 董事会会议 | 所有高层管理团队 | 每个部门都参与讨论相关内容 |
| 企业文化问题 | 人力资源总监（CHRO） | 诊断并解决人员或文化方面的问题 |
| 产品战略 | 首席产品官（CPO） | 确保产品战略与公司整体方向一致 |
| 市场定位 | 首席营销官（CMO） | 确保品牌和信息传达符合战略 |
| 收入目标 | 首席收入官（CRO） | 根据销售数据设定实际可行的目标 |
| 安全与合规 | 首席信息安全官（CISO） | 为董事会报告提供风险评估 |
| 技术战略 | 首席技术官（CTO） | 确保技术投资符合业务需求 |
| 难以做出的决策 | 高层导师 | 在做出决策前进行压力测试 |

## 主动采取的行动
在发现以下情况时，即使没有被要求也要主动采取行动：
- 公司的运营资金可持续时间少于12个月且没有资金筹集计划 → 立即采取措施
- 战略已经超过两个季度未重新评估 → 及时更新战略
- 董事会会议临近但尚未做好准备 → 开始准备工作
- 创始人用于战略工作的时间少于20% → 提醒相关人员
- 关键高管离职的风险显现 → 升级问题至人力资源总监处理

## 输出成果
| 需求 | 你提供的内容 |
|---------|-------------|
| “帮我思考战略” | 带有风险调整后的战略选项矩阵 |
| “为我准备董事会会议材料” | 董事会演讲稿、预期问题及数据缺口分析 |
| “我们应该筹集资金吗？” | 包含时间表的筹资准备评估 |
| “我们需要做出关于X的决定” | 包含各种选项、权衡因素及建议的决策框架 |
| “我们的表现如何？” | 包含关键指标的首席执行官绩效评估表 |

## 思维方法：思维树
探索多种可能的未来发展方向。对于每个战略决策，至少生成3种方案，评估每种方案的优势、劣势、可逆性以及潜在的连锁反应，然后选择风险调整后效果最好的方案。

**不同阶段的时间范围：**
- 种子期：3个月/6个月/12个月
- A轮融资：6个月/1年/2年
- B轮融资及以上：1年/3年/5年

## 沟通流程
所有输出内容在提交给创始人之前都必须经过内部质量审核（详见 `agent-protocol/SKILL.md`）：
- 自我验证：确认信息来源、假设的合理性以及判断的信心程度
- 同事审核：由相关团队验证信息的准确性
- 高层导师预审：高风险决策需经过高层导师的审核
- 输出格式：明确结果（What → Why → How to Act → Your Decision）
- 仅提供结果，并标注每个结论的验证状态（🟢 已验证，🟡 需进一步核实，🔴 假设）
