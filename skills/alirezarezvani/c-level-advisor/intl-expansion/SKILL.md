# 国际扩张

用于开拓新市场的框架：市场选择、进入策略、本地化及执行流程。

## 关键词
国际扩张、市场进入、本地化、市场推广策略（GTM）、区域战略、国际市场、市场选择、跨境业务、全球扩张

## 快速入门

**决策流程：** 市场选择 → 进入模式 → 监管评估 → 本地化计划 → 市场推广策略 → 团队架构 → 上线发布。

## 市场选择框架

### 评分矩阵
| 因素 | 权重 | 评估方法 |
|--------|--------|---------------|
| 市场规模（可覆盖范围） | 25% | 目标市场的总市场规模（TAM）及消费者支付意愿 |
| 竞争激烈程度 | 20% | 现有竞争者的实力及市场空白 |
| 监管复杂性 | 20% | 进入壁垒、合规成本及时间安排 |
| 文化差异 | 15% | 语言、商业习惯及购买行为 |
| 现有市场基础 | 10% | 内部需求、现有客户及合作伙伴关系 |
| 运营复杂性 | 10% | 时区差异、基础设施及支付系统 |

### 进入模式
| 进入模式 | 投资成本 | 控制程度 | 风险 | 适用场景 |
|------|-----------|---------|------|----------|
| **出口**（远程销售） | 低 | 低 | 低 | 适用于测试市场需求的情况 |
| **合作伙伴关系**（经销商/代理商） | 中等 | 中等 | 适用于有较高本地化要求的市场 |
| **本地团队**（在当地招聘员工） | 高 | 高 | 适用于需求稳定的战略市场 |
| **全资子公司** | 非常高 | 完全控制 | 适用于主要市场及有严格监管要求的情况 |
| **收购** | 最高 | 完全控制 | 可快速进入市场（尤其是已有客户基础的情况下）

**推荐路径：** 出口 → 合作伙伴关系 → 本地团队 → 全资子公司（根据收入情况逐步升级）

## 本地化检查清单

### 产品方面
- [ ] 语言（用户界面、文档、支持内容）
- [ ] 货币与定价（本地化定价，而不仅仅是货币转换）
- [ ] 支付方式（因市场而异）
- [ ] 日期/时间/数字格式
- [ ] 法律要求（数据存储地、隐私保护）
- [ ] 文化适应性调整（不仅仅是语言翻译）

### 市场推广策略
- [ ] 信息传达方式（需符合当地市场特点）
- [ ] 销售渠道策略（不同市场有不同的销售渠道）
- [ ] 当地案例研究及市场认可度证明 |
- [ ] 当地合作伙伴关系及整合 |
- [ ] 参与当地活动/会议 |
- [ ] 当地搜索引擎优化（SEO）及内容制作

### 运营方面
- [ ] 法律实体设立（如需）
- [ ] 税务合规性 |
- [ ] 雇佣法（如需在当地招聘员工）
- [ ] 客户服务（服务时间、语言支持）
- [ ] 银行及支付系统

## 关键问题
- “市场是否有需求？还是我们在主动推动市场？”
- “进入市场的成本与三年后的收入潜力相比如何？”
- “我们能否从总部直接服务这个市场？还是需要在当地设立办事处？”
- “监管流程需要多长时间？我们能在手续完成前上线吗？”
- “目前这个市场由谁主导？要取代他们需要什么条件？”

## 常见错误
| 错误 | 发生原因 | 预防措施 |
|---------|---------------|------------|
| 同时进入太多市场 | 怕错过机会（FOMO）、董事会压力 | 每年最多进入1-2个新市场 |
| 直接复制本土市场的市场推广策略 | 假设各地消费者需求相同 | 需要研究当地消费者的购买习惯 |
| 低估监管成本 | “我们会解决这个问题” | 在决策前进行充分的监管评估 |
| 过早招聘员工 | 过于乐观 | 在招聘本地团队前先验证市场需求 |
| 定价错误（仅基于货币转换） | 懒惰 | 需要研究当地消费者的支付意愿 |

## 与高管团队的协作
| 高管角色 | 贡献内容 |
|------|-------------|
| CEO | 市场选择、战略决策 |
| CFO | 投资规模评估、投资回报率（ROI）建模、公司架构设计 |
| CRO | 收入目标设定、销售模式调整 |
| CMO | 市场定位、销售渠道策略、本地品牌建设 |
| CPO | 本地化计划制定、功能优先级排序 |
| CTO | 基础设施建设、数据存储安排、业务扩展 |
| CHRO | 本地招聘、雇佣法相关事务、薪酬管理 |
| COO | 运营流程搭建、流程优化 |
