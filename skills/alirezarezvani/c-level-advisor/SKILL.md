# C级顾问生态系统  
（C-Level Advisory Ecosystem）  
为创始人及高管提供的完整虚拟董事会服务。  

## 快速入门  
（Quick Start）  
```
1. Run /cs:setup → creates company-context.md (all agents read this)
2. Ask any strategic question → Chief of Staff routes to the right role
3. For big decisions → /cs:board triggers a multi-role board meeting
```  

## 包含内容  
（What’s Included）  
### 10个核心高管职位  
CEO（首席执行官）、CTO（技术总监）、COO（首席运营官）、CPO（首席产品官）、CMO（首席营销官）、CFO（首席财务官）、CRO（首席运营官）、CISO（首席信息安全官）、CHRO（首席人力资源官）、执行导师（Executive Mentor）  

### 6项核心协调能力  
创始人入职支持（Founder Onboard）、首席参谋（Chief of Staff）、董事会会议管理（Board Meeting Management）、决策记录系统（Decision Logger）、代理协议（Agent Protocol）、上下文引擎（Context Engine）  

### 6项跨领域能力  
董事会资料库构建（Board Deck Builder）、情景分析工具（Scenario Analysis Tool）、竞争情报收集（Competitive Intelligence）、组织健康诊断（Organizational Health Diagnostic）、并购策略（M&A Strategy）、国际扩张支持（International Expansion）  

### 6项文化与协作工具  
企业文化构建（Culture Development）、公司运营管理系统（Company Operations Management）、创始人辅导（Founder Coaching）、战略一致性（Strategic Alignment）、变革管理（Change Management）、内部沟通机制（Internal Communication）  

## 主要特点  
（Key Features）  
- **内部质量审核流程**：自我验证 → 同行审核 → 专家预审 → 最终提交  
- **双层信息存储**：原始会议记录 + 经批准的决策内容（防止虚假共识）  
- **董事会会议隔离机制**：交叉审查前的独立分析阶段  
- **主动预警机制**：基于上下文的自动预警  
- **结构化输出**：结果 → 原因 → 行动方案 → 最终决策  
- **25个Python工具**：全部使用标准库，优先支持命令行接口（CLI），输出格式为JSON，无外部依赖
