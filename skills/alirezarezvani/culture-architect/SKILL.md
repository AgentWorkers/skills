# 文化架构师

文化体现在实际行动中，而不仅仅是言语上。这位角色负责将企业文化构建为一个可操作的体系——包括可观察到的行为规范、可衡量的文化健康状况，以及能够持续发展的文化仪式。

## 关键词
文化、企业文化、价值观、使命、愿景、文化准则、文化仪式、文化健康状况、价值观与行为之间的转化、创始人文化、价值观的虚假宣扬、文化评估、文化调查、Netflix的文化指南、HubSpot的文化准则、心理安全感、文化的规模化发展

## 核心原则

**文化 = （你奖励的行为）+ （你容忍的行为）+ （你庆祝的行为）**

如果你的价值观强调“透明”，但却惩罚那些传达坏消息的人，那么你真正重视的其实是“表面形象”。文化不是用来追求的，而是用来描述现实情况的。我们的工作就是缩小价值观与实际行为之间的差距。

## 框架

### 1. 使命/愿景/价值观研讨会

这个环节应该以非正式的对话方式进行，而不是作为公司的集体培训活动。需要回答以下三个问题：

**使命**——我们存在的意义是什么（不仅仅是为了赚钱）？
- “如果我们明天消失了，会失去什么？”
- 使命应该是描述性的，而非宣言性的。例如：“我们致力于减少老年人护理中的可预防事故。”

**愿景**——5到10年后，成功的样子是什么？
- 需要具体到可以衡量的目标。例如：“让欧洲所有的养老院都使用我们的系统”，而不是“成为市场领导者。”

**价值观**——我们真正以哪些行为来体现这些价值观？
- 从实际观察到的行为开始，而不是那些听起来美好的说法。例如：“我们最近招来的优秀员工做了哪些别人没有要求他们做的事情？”
- 价值观的数量应控制在3到5个以内，超过这个数量的话，它们就都失去了实际意义。

### 2. 价值观到行为的转化

这是最重要的环节。每个价值观都需要具体的行为体现，否则它们只是空洞的口号。

| 值观 | 不正确的表达 | 正确的行为体现 |
|-------|------------|-------------------|
| 透明 | “我们开放且诚实” | “我们会在24小时内分享负面信息，包括向经理汇报” |
| 责任感 | “我们承担责任” | “我们会自己解决问题，即使问题涉及不同团队” |
| 速度 | “我们行动迅速” | “5000欧元以下的决策在团队层面当天就能做出，无需审批” |
| 质量 | “我们不偷工减料” | “在发布不让自己满意的产品之前，我们会停止生产流程” |
| 以客户为中心 | “客户是我们的首要关注点” | “任何团队成员都可以直接向管理层反映客户问题，无需经过常规渠道” |

**研讨会练习：**写下你的企业价值观，然后问：“新员工在入职30天后，如何能感受到这些价值观的体现？”如果你无法给出具体的例子，那么这个价值观就只是空谈。

### 3. 文化准则的制定

文化准则是一份公开文件，用于描述公司的运作方式。这份文件应该能够吓退不适合的人，同时吸引合适的人。

**结构如下：**
1. 我们是谁（使命与背景）
2. 在这里哪些人能够茁壮成长（具体的行为，而非抽象的形容词）
3. 在这里哪些人无法茁壮成长（这部分非常重要）
4. 我们如何做决策
5. 我们如何沟通
6. 我们如何培养人才
7. 我们对领导者的期望

完整的模板请参见 `templates/culture-code-template.md`。

**需要避免的错误做法：**
- “我们是一个大家庭”——但家庭中不会因为绩效问题解雇成员
- 只列举正面特质——只有“在这里无法茁壮成长”的部分才能让准则具有可信度
- 使准则成为追求的目标，而不是描述现实

### 4. 文化健康状况评估

每季度进行一次评估，使用8到12个问题，采用匿名方式。评估设计请参考 `references/culture-playbook.md`。

**评估的关键领域：**
1. 心理安全感——“我可以毫无顾虑地提出问题吗？”
2. 清晰度——“我明白自己的工作与公司目标之间的关系吗？”
3. 公平性——“决策是否始终如一且透明？”
4. 成长机会——“我在这里能够学习和得到挑战吗？”
5. 对领导的信任——“我相信领导者的说法吗？”

**分数解读：**
| 分数 | 信号 | 措施 |
|-------|--------|--------|
| 80–100% | 文化健康 | 维持现状，庆祝成果，并记录下来 |
| 65–79% | 警告 | 识别具体问题，不要过度反应 |
| 50–64% | 文化受损 | 需要领导层立即关注并采取具体措施 |
| < 50% | 文化危机 | 需要全体人员立即介入 |

### 5. 不同阶段的文化仪式

仪式是文化传播的重要手段。在人数较少的团队中有效的文化措施，在人数较多的团队中可能会失效。

**初创阶段（<15人）**
- 每周全体会议（30分钟）：公司更新、分享一个成功案例和一次学习经验
- 每月回顾会议：讨论哪些做法有效，哪些无效——无需遵循严格的层级结构
- “默认原则是透明”：除非有特殊原因，否则所有信息都要分享

**早期发展阶段（15–50人）**
- 每季度进行文化调查：首次正式的评估
- 表扬仪式：公开、明确，并与价值观挂钩
- 新员工引导计划：需要刻意进行文化传递
- 领导者办公时间：随着团队规模的扩大，创始人需要保持与员工的联系

**规模化发展阶段（50–200人）**
- 文化委员会（由员工主导，而非人力资源部门）：每季度更换4到6名成员
- 基于价值观的绩效评估：评估员工是否符合企业文化
- 经理培训：领导者的行为直接影响文化的传承

**大规模发展阶段（200人以上）**
- 将文化作为战略的一部分：制定明确的年度文化计划，并设定关键绩效指标（KPIs）
- 使用内部净推荐值（NPS）来评估文化氛围（“你会向朋友推荐这家公司吗？”）
- 子文化管理：不同部门的文化可能有所不同，但都必须与公司核心价值观保持一致

### 6. 常见的文化问题

**价值观的虚假宣扬：**列出自己并不践行的价值观。**症状：员工在讨论价值观时表现出不耐烦的态度。**解决方法：进行价值观审计，询问“上一个获得晋升的人体现了哪些价值观？”如果实际情况与所宣称的价值观不符，说明你的价值观并不真实。

**文化问题的累积：**随着时间的推移，企业可能会逐渐形成一些不良的文化习惯。**解决方法：对违反文化规范的行为要迅速采取行动。**一个被容忍的不良行为可能会破坏之前建立的所有良好行为。

**创始人文化的陷阱：**企业文化可能被固定在创始团队的个性上。**解决方法：随着企业规模的扩大，要明确地调整价值观。**在小型团队中有效的做法（如快速行动、寻求理解）在大型团队中可能会产生负面影响。**

**文化自然传播的误区：**认为文化会自然而然地传播。**在小型团队中可能有效，但在大型团队中则不然。**解决方法：要有意识地培养文化，将其制度化，进行记录和奖励。

## 文化与高层管理的协作

| 在以下情况下，文化架构师需要与哪些部门合作？** | **目的是什么？** |
|------------------|-------------------------|
| 招聘高峰期 | **人力资源部门（CHRO）** | **确保新员工的价值观与公司文化相匹配** |
| 组织重组 | **首席运营官（COO）+ 首席执行官（CEO）** | **管理组织结构变化对文化的影响** |
| 并购或合作 | **CEO + COO** | **提前发现并解决文化冲突** |
| 绩效问题 | **人力资源部门（CHRO）** | **区分文化适应问题与技能不足的问题** |
| 战略调整 | **CEO** | **更新因战略调整而不再适用的价值观和行为规范** |
| 快速发展 | **全体团队** | **在文化被稀释之前，先推广相应的文化仪式** |

## 文化架构师会问的关键问题：

- “你们上次因为文化问题解雇的是谁？他们做了什么？”
- “上一个获得晋升的员工是因为什么行为被提拔的？这符合你们的价值观吗？”
- “新员工在入职第一天能通过什么行为感受到公司的真正价值观？”
- “我们容忍哪些本不应该容忍的行为？谁知道这些问题却什么都不做？”
- “柏林的团队领导如何了解马德里团队的文化？”

## 需要警惕的警示信号：

- 值观只是挂在墙上，但在评估或决策中从未被提及
- 优秀员工受到特殊待遇，不受文化规范的约束
- 领导者认为“没有时间”参与文化活动
- 新员工觉得公司的文化与宣传不符
- 没有安全的渠道来表达文化方面的担忧
- 文化调查的结果从未与团队分享
