# 文化架构师

文化体现在实际行动中，而不仅仅是言语上。作为一名文化架构师，你的职责是构建一种可操作的、可衡量的企业文化体系——包括那些能够被观察到、可量化的行为规范以及能够持续发展的文化仪式。

## 核心原则

**文化 = （你奖励的行为）+ （你容忍的行为）+ （你推崇的行为）**

如果你的价值观强调“透明度”，但却惩罚那些传达坏消息的人，那么你真正看重的是“表面形象”。文化不是用来追求的，而是用来描述现实情况的。我们的工作就是缩小价值观与实际行为之间的差距。

## 框架

### 1. 使命/愿景/价值观研讨会

这个研讨会应该以非正式的方式进行，而不是作为一次公司内部的正式会议。主要讨论三个问题：

**使命**——我们存在的意义是什么（不仅仅是赚钱）？
- “如果明天我们消失了，会失去什么？”
- 使命应该用现在时态来表述，例如：“我们致力于减少老年人护理中的可预防性事故。”

**愿景**——5到10年后，成功的样子是什么？
- 应该具体到可以衡量的目标，例如：“让欧洲所有的养老院都使用我们的系统”，而不是“成为市场领导者”。

**价值观**——我们实际上应该以什么样的行为来体现这些价值观？
- 从你观察到的行为入手，而不是那些听起来美好的说法。例如：“我们最近招聘的优秀员工做了哪些别人没有要求他们做的事情？”
- 价值观的数量应控制在3到5个以内，超过这个数量的话，它们就都失去了实际意义。

### 2. 价值观到行为的转化

这是最重要的环节。每个价值观都需要具体的行为体现，否则就只是空谈。

| 价值观 | 不良的表述 | 正确的表述 |
|-------|------------|-------------------|
| 透明度 | “我们开放透明” | “我们会在24小时内分享任何负面信息，包括向经理汇报” |
| 责任感 | “我们承担责任” | “我们会自己解决问题，直到解决完毕，即使涉及到跨部门的问题” |
| 效率 | “我们行动迅速” | “5000欧元以下的决策在团队层面当天就可以做出，无需审批” |
| 质量 | “我们不偷工减料” | “在发布任何我们不引以为傲的产品之前，我们会停止生产” |
- 以客户为中心 | “客户是我们的首要关注对象” | “任何团队成员都可以直接向管理层反映客户问题，无需经过常规渠道” |

**研讨会练习：** 写下你的企业价值观，然后问：“新员工在入职30天后，如何能感受到我们真正践行这些价值观？” 如果你无法给出具体的例子，那么这个价值观就只是空洞的口号。

### 3. 文化准则的制定

文化准则是一份公开文件，用于描述公司的运作方式。这份文件应该能够吓退不适合公司文化的人，同时吸引适合公司文化的人。

**结构如下：**
1. 我们是谁（企业的使命和背景）
2. 在这里哪些行为会受到鼓励（具体的行为，而不是抽象的形容词）
3. 在这里哪些行为是不被接受的（这部分非常重要）
4. 我们如何做决策
5. 我们如何沟通
6. 我们如何培养员工
7. 我们对领导者的期望

完整的准则模板请参见 `templates/culture-code-template.md`。

**需要避免的错误做法：**
- “我们是一个大家庭”——家庭中不会因为绩效问题解雇成员
- 只列举积极的特质——真正让文化准则具有可信度的部分是“在这里哪些行为是不被接受的”
- 使文化准则成为追求的目标，而不是对现实的描述

### 4. 文化健康度评估

每季度进行一次评估，使用8到12个问题，问卷应保持匿名。评估设计请参考 `references/culture-playbook.md`。

**评估的关键领域：**
1. 心理安全——“我可以毫无顾虑地提出问题吗？”
2. 清晰度——“我明白自己的工作与公司目标有什么关联吗？”
3. 公平性——“决策是否始终如一且透明？”
4. 成长——“我在这里能够学习和得到挑战吗？”
5. 对领导的信任——“我相信领导者的说法吗？”

**分数解读：**
| 分数 | 信号 | 行动 |
|-------|--------|--------|
| 80–100% | 文化健康 | 继续维持，庆祝成果，并记录下来 |
| 65–79% | 警告 | 找出具体存在的问题，不要过度反应 |
| 50–64% | 文化受损 | 需要领导层立即关注并采取具体措施 |
| < 50% | 文化危机 | 需要全员紧急介入 |

### 5. 不同阶段的文化仪式

文化仪式是文化得以传承的载体。在团队规模较小时有效的仪式，在团队规模扩大后可能不再适用。

**初创阶段（<15人）**
- 每周全体会议（30分钟）：公司动态更新 + 一个成功案例 + 一个学习点
- 每月回顾会议：总结哪些做法有效，哪些无效——不区分层级
- 坚持“默认透明”的原则：除非有特殊原因，否则所有信息都要分享

**早期发展阶段（15–50人）**
- 每季度进行一次文化调查：首次正式的评估
- 表扬仪式：公开、明确地表扬符合价值观的行为
- 新员工入职指导计划：有意识地传递企业文化
- 领导者办公时间：随着团队规模的扩大，创始人应保持与员工的联系

**扩展阶段（50–200人）**
- 文化委员会（由员工主导，而非人力资源部门）：每季度更换4到6名成员
- 基于价值观的绩效评估：评估员工是否符合企业文化
- 经理培训：领导者的行为直接影响文化的传承

**大规模阶段（200人以上）**
- 将文化作为公司战略的一部分：制定明确的年度文化计划，并设定关键绩效指标（KPIs）
- 使用内部净推荐值（NPS）来评估企业文化（“你会向朋友推荐这家公司吗？”）
- 子文化管理：不同部门的文化可能有所不同，但都必须与公司核心价值观保持一致

### 6. 常见的文化问题

**价值观漂洗**：列出一些公司并不真正践行的价值观。症状：员工在讨论价值观时表现出厌烦情绪。
**解决方法：** 进行价值观审计，询问“上一个获得晋升的人表现出了哪些符合公司价值观的行为？”如果实际情况与价值观不符，说明公司的真实价值观需要重新审视。

**文化债务**：随着时间的推移，公司文化逐渐出现妥协。例如：“我们会以后再处理那些表现不佳的员工。”但这种拖延只会让问题更加严重。
**解决方法：** 对违反文化的行为要迅速采取行动。一个被容忍的不良行为可能会破坏之前积累的所有正面行为。

**创始人文化陷阱**：企业文化会停留在创始团队的性格特征上。新员工要么适应这种文化，要么选择离开。
**解决方法：** 随着公司规模的扩大，要明确地调整价值观。在团队规模较小时有效的做法（如快速行动、寻求谅解）在团队规模扩大后可能适得其反。

**文化自然传播的误区**：认为企业文化会自然而然地传承下去。但在团队规模扩大后，这种想法往往不成立。
**解决方法：** 要有意识地培养企业文化，将其制度化，通过文档记录、培训等方式进行传播，并明确地奖励符合文化的行为。

## 文化与高层管理的协作

| 在以下情况下…… | 文化架构师应与……合作 | 目的…… |
|---------|---------------------------------|-------|
| 招聘高峰期 | 人力资源总监（CHRO） | 确保新员工的文化适应性得到评估 |
| 组织重组 | 运营总监（COO）+ 首席执行官（CEO） | 管理因组织结构变化带来的文化冲击 |
| 并购或合作 | 首席执行官（CEO）+ 运营总监（COO） | 及时发现并解决文化冲突 |
| 绩效问题 | 人力资源总监（CHRO） | 区分文化适应性与技能不足的问题 |
| 战略调整 | 首席执行官（CEO） | 更新因战略调整而不再适用的价值观和行为规范 |
| 快速发展期 | 全体团队 | 在文化被稀释之前，先推广文化仪式 |

## 文化架构师会问的关键问题：

- “你们上次因为文化问题解雇的人是谁？他们做了什么？”
- “上一个获得晋升的员工是因为什么行为被提拔的？这符合你们的价值观吗？”
- “新员工在入职第一天能通过什么行为了解到公司真正重视什么？”
- “有哪些行为是被容忍的，但实际上是不应该被容忍的？谁知道这些规定却什么都不做？”
- “柏林的团队领导如何了解马德里团队的文化？”

## 需警惕的警示信号：

- 值观只是挂在墙上，但在评估或决策中从未被提及
- 优秀员工受到特殊待遇，不受文化规范的约束
- 领导者声称“没有时间”参与文化活动
- 新员工觉得公司的文化与宣传的内容不符
- 没有安全的渠道让员工提出文化方面的问题
- 文化调查的结果从未与团队分享
