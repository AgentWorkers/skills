该工具用于分析社交媒体活动的表现，包括用户参与度指标、投资回报率（ROI）的计算以及平台基准数据。

## 分析工作流程

分析社交媒体活动的表现步骤如下：

1. 验证输入数据的完整性（覆盖范围 > 0，日期有效）；
2. 计算每条帖子的参与度指标；
3. 汇总活动级别的各项指标；
4. 如果提供了广告支出，计算 ROI；
5. 与平台基准数据进行比较；
6. 确定表现最佳和最差的帖子或账户；
7. 提出优化建议；
8. **验证要求**：参与度不低于 100%，ROI 与支出数据一致。

### 输入要求

| 字段 | 是否必需 | 说明 |
|-------|----------|-------------|
| 平台 | 是 | 支持的平台包括 Instagram、Facebook、Twitter、LinkedIn、TikTok |
| 帖子数组 | 是 | 包含帖子详细信息的数组 |
| 帖子的点赞数 | 是 | 点赞/互动次数 |
| 帖子的评论数 | 是 | 评论数量 |
| 帖子的覆盖范围 | 是 | 接触到的唯一用户数量 |
| 帖子的展示次数 | 否 | 总观看次数 |
| 帖子的分享次数 | 否 | 分享/转发的次数 |
| 帖子的保存/书签次数 | 否 | 保存/书签的次数 |
| 帖子的点击次数 | 否 | 链接点击次数 |
| 总支出 | 否 | 用于计算 ROI 的广告支出 |

### 数据验证

在分析之前，请检查以下内容：
- 所有帖子的覆盖范围均大于 0（避免除以零的情况）；
- 参与度数据为非负数；
- 时间范围有效（开始时间 < 结束时间）；
- 平台识别正确；
- 如果需要计算 ROI，则支出金额大于 0。

## 用户参与度指标

### 参与度计算

```
Engagement Rate = (Likes + Comments + Shares + Saves) / Reach × 100
```

### 指标定义

| 指标 | 公式 | 解释 |
|--------|---------|----------------|
| 参与度 | 参与次数 / 覆盖范围 × 100 | 用户互动程度 |
| 点击率（CTR） | 点击次数 / 展示次数 × 100 | 内容的吸引力 |
| 覆盖率 | 覆盖范围 / 关注者数量 × 100 | 内容的传播效果 |
| 病毒式传播率 | 分享次数 / 展示次数 × 100 | 内容的分享潜力 |
| 保存率 | 保存次数 / 覆盖范围 × 100 | 内容的价值 |

### 表现评估标准

| 评估等级 | 参与度 | 推荐措施 |
|--------|-----------------|--------|
| 优秀 | > 6% | 扩大规模并复制成功策略 |
| 良好 | 3-6% | 优化并扩大影响力 |
| 一般 | 1-3% | 测试改进方案 |
| 较差 | < 1% | 分析原因并调整策略 |

## ROI 计算

计算广告支出的回报：

1. 求所有帖子的总参与次数；
2. 计算每次参与的成本（CPE）；
3. 如果有点击数据，计算每次点击的成本（CPC）；
4. 使用基准数据估算参与价值；
5. 计算 ROI 百分比；
6. **验证要求**：ROI = (参与价值 - 广告支出) / 广告支出 × 100

### ROI 计算公式

| 指标 | 公式 |
|--------|---------|
| 每次参与的成本（CPE） | 总支出 / 总参与次数 |
| 每次点击的成本（CPC） | 总支出 / 总点击次数 |
| 每千次展示的成本（CPM） | (支出 / 展示次数) × 1000 |
| 广告支出回报率（ROAS） | 收入 / 广告支出 |

### 参与价值估算

| 行为 | 价值 | 原因 |
|--------|-------|-----------|
| 点赞 | 0.50 美元 | 提升品牌知名度 |
| 评论 | 2.00 美元 | 活跃的用户互动 |
| 分享 | 5.00 美元 | 内容的传播效果 |
| 保存 | 3.00 美元 | 用户的保存行为 |
| 点击 | 1.50 美元 | 带来的流量 |

### ROI 解释

| ROI 百分比 | 评估等级 | 建议 |
|-------|--------|----------------|
| > 500% | 优秀 | 大幅增加预算 |
| 200-500% | 良好 | 适度增加预算 |
| 100-200% | 可接受 | 在扩大规模前进行优化 |
| 0-100% | 收支平衡 | 重新评估目标受众和创意策略 |
| < 0% | 负值 | 暂停活动并重新调整策略 |

## 示例

### 示例输入

详见 `assets/sample_input.json`：

```json
{
  "platform": "instagram",
  "total_spend": 500,
  "posts": [
    {
      "post_id": "post_001",
      "content_type": "image",
      "likes": 342,
      "comments": 28,
      "shares": 15,
      "saves": 45,
      "reach": 5200,
      "impressions": 8500,
      "clicks": 120
    }
  ]
}
```

### 示例输出

详见 `assets/expected_output.json`：

```json
{
  "campaign_metrics": {
    "total_engagements": 1521,
    "avg_engagement_rate": 8.36,
    "ctr": 1.55
  },
  "roi_metrics": {
    "total_spend": 500.0,
    "cost_per_engagement": 0.33,
    "roi_percentage": 660.5
  },
  "insights": {
    "overall_health": "excellent",
    "benchmark_comparison": {
      "engagement_status": "excellent",
      "engagement_benchmark": "1.22%",
      "engagement_actual": "8.36%"
    }
  }
}
```

### 解释

示例活动显示：
- **参与度为 8.36%**，而基准值为 1.22%，属于**优秀**（是平均水平的 6.8 倍）；
- **点击率为 1.55%**，而基准值为 0.22%，属于**优秀**（是平均水平的 7 倍）；
- **ROI 为 660%**，意味着每投入 500 美元的广告获得了显著的回报；
**建议**：扩大预算，并复制成功的策略。

## 自动触发条件

- **参与度低于平台平均水平** → 内容可能没有引起受众共鸣。分析表现最佳的帖子以找出原因；
- **粉丝增长停滞** → 可能是内容传播或发布频率的问题。检查发布策略；
- **展示次数高但参与度低** → 可能是内容质量不佳；
- **竞争对手表现显著优于自己** → 需要分析他们的成功策略。

## 输出结果

- 请求“社交媒体审计”时，会得到包含平台基准数据的全面分析报告；
- 请求“哪些内容表现良好？”时，会得到包含分析结果和优化建议的报告；
- 请求“竞争对手的社交媒体分析”时，会得到对比分析报告。

## 通信规范

所有输出结果均经过质量验证：
- 可自我验证：包含来源信息、假设验证和信心评分；
- 输出格式清晰：首先说明结果，然后解释原因，最后提供行动建议；
- 仅提供分析结果。所有结果都会标注是否经过验证（🟢 已验证，🟡 可能有误，🔴 需进一步验证）。
