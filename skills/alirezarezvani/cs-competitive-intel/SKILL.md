# 竞争情报

系统化的竞争对手跟踪。这不仅仅是一种痴迷，而是一种能够推动实际决策的情报工具。

## 关键词
竞争情报、竞争对手分析、竞争态势图、胜负分析、竞争定位、竞争跟踪、市场情报、竞争对手研究、SWOT分析、竞争地图、功能差距分析、竞争策略

## 快速入门

```
/ci:landscape         — Map your competitive space (direct, indirect, future)
/ci:battlecard [name] — Build a sales battlecard for a specific competitor
/ci:winloss           — Analyze recent wins and losses by reason
/ci:update [name]     — Track what a competitor did recently
/ci:map               — Build competitive positioning map
```

## 框架：五层情报系统

### 第一层：竞争对手识别

- **直接竞争对手**：具有相同的业务模式（ICP，即 Industry Context and Problem）、面临相同的问题、提供类似的解决方案、价格区间相近。
- **间接竞争对手**：预算相同但采用不同的解决方案（包括选择“不采取任何行动”或“自行开发解决方案”）。
- **潜在竞争对手**：在相关领域拥有充足资金的初创企业；或那些有明确产品路线图的大型现有企业。

**2x2 威胁矩阵：**

| | 相同的ICP | 不同的ICP |
|---|---|---|
| **相同的问题** | 直接威胁 | 相关领域的竞争对手 |
| **不同的问题** | 可能导致替代的风险 | 目前可忽略 |

请每季度更新此矩阵，看看有哪些竞争对手发生了变化。

### 第二层：跟踪维度

针对每个竞争对手，跟踪以下8个维度：

| 维度 | 数据来源 | 更新频率 |
|-----------|---------|---------|
| **产品动态** | 产品更新日志、G2/Capterra用户评价、Twitter/LinkedIn | 每月 |
| **价格变动** | 价格页面信息、销售电话中的信息、客户反馈 | 需要时更新 |
| **融资情况** | Crunchbase、TechCrunch、LinkedIn | 需要时更新 |
| **招聘动态** | LinkedIn上的职位发布、Indeed | 每月 |
| **合作伙伴关系** | 新闻稿、联合营销活动 | 需要时更新 |
| **客户获取情况** | 客户案例研究、评价网站信息、LinkedIn | 每月 |
| **客户流失情况** | 客户访谈、流失账户数据 | 持续跟踪 |
| **市场宣传策略** | 官网内容、广告（Facebook/Google Ad Library） | 每季度 |

### 第三层：分析框架

**每个竞争对手的SWOT分析：**
- **优势**：他们在哪些方面表现优异？他们在哪些方面具有竞争力？
- **劣势**：他们在哪些方面存在不足？客户对他们有哪些抱怨？
- **机会**：他们可能会采取哪些行动来对你构成威胁？
- **威胁**：他们可能面临的致命威胁是什么？

**竞争定位图（两个轴）：**
选择对买家来说重要的轴：
- **价格 vs 功能深度**；**适用于大型企业 vs 适用于小型企业**；**易于实施 vs 可配置**  
- 选择能够清晰展示你产品差异化的轴。

**功能差距分析：**
| 功能 | 你的产品 | 竞争对手A | 竞争对手B | 差距情况 |
|---------|-----|-------------|-------------|------------|
| [功能] | ✅ | ✅ | ❌ | 你的优势 |
| [功能] | ❌ | ✅ | ✅ | 存在差距——他们的产品路线图中是否有该功能？ |
| [功能] | ✅ | ❌ | ❌ | 仅竞争对手B拥有该功能 |

### 第四层：输出格式

- **针对销售团队（CRO）：** 竞争态势图——每个竞争对手一页，用于销售前准备。
  参见 `templates/battlecard-template.md` 文件。

- **针对市场团队（CMO）：** 定位更新——市场宣传策略的变化、新的差异化优势、需要停止或开始推广的功能。

- **针对产品团队（CPO）：** 功能差距总结——客户需要的功能但我们没有的、竞争对手已经提供的功能、需要优先处理的功能。

- **针对CEO/董事会：** 每月竞争情报总结——谁发生了变化、这些变化意味着什么、建议的应对措施。

### 第五层：情报更新频率

- **每月（定期更新）：**
  - 审查所有一级竞争对手（直接威胁者、排名前三的竞争对手）。
  - 用新情报更新竞争态势图。
  - 向管理层发布一页的总结报告。

- **基于事件触发更新：**
  - 竞争对手获得融资 → 在48小时内评估影响。
  - 竞争对手发布重要功能 → 在1周内制定产品策略和销售应对措施。
  - 竞争对手挖走关键客户 → 在2周内进行客户访谈。
  - 竞争对手调整价格 → 在1周内进行分析并作出回应。

- **每季度：**
  - 全面评估竞争格局。
  - 更新竞争定位图。
  - 重新评估竞争对手的威胁等级。
  - 从跟踪名单中添加或删除公司。

## 胜负分析

这是最具有参考价值的竞争数据。然而，大多数公司很少进行此类分析。

**何时进行访谈：**
- 每笔损失超过5万美元的订单。
- 每个流失的客户的使用时长超过6个月。
- 每次赢得的客户。

**访谈人员：**
- 不应由负责该交易的销售代表进行（他们可能过于了解情况，客户不会坦诚回答）。
- 可以由客户成功团队、产品团队或外部研究人员来进行访谈。

**访谈问题结构：**
1. “请详细介绍一下你们的评估流程。”
2. “你们还考虑过哪些竞争对手？”
3. “做出决策时最重要的三个标准是什么？”
4. “我们的产品在哪些方面做得不够好？”
5. “是什么决定了客户的决策？”
6. “如果情况不同，你们的决定会如何改变？”

**每月汇总分析结果：**
- 获胜的原因（按频率排序）。
- 失败的原因（按频率排序）。
- 各竞争对手的获胜率（按竞争对手、客户群体分类）。
- 随时间变化的趋势。

## 平衡：理性利用情报，避免过度关注

**过度关注竞争对手的迹象：**
- 决策主要基于“竞争对手刚刚发布了某个功能”。
- 当竞争对手融资时，团队士气下降。
- 为了迎合竞争对手的列表而推出自己并不相信的功能。
- 价格讨论总是从“他们的价格是多少”开始。

**关注不足的迹象：**
- 销售代表在客户访谈中措手不及。
- 前景客户对竞争对手的了解比你的团队更多。
- 由于客户的通知，你才得知竞争对手的重大产品发布。
- 尽管市场情况发生变化，但你的产品定位12个月以上都没有调整。

**正确的态度：**
- 充分了解竞争对手，以便在竞争中取得优势。
- 不要让竞争对手主导你的决策。
- 你的产品路线图应基于客户的需求，并结合竞争对手的差距来制定。

## 智报的传播

| 目标受众 | 输出格式 | 更新频率 | 负责人 |
|----------|--------|---------|-------|
| 销售代表和销售开发人员（AEs + SDRs） | CRM中的更新后的竞争态势图 | 每月 + 根据需要触发更新 | 销售团队（CRO） |
| 产品团队 | 功能差距分析报告 | 每季度 | 产品负责人（CPO） |
| 市场团队 | 定位更新报告 | 每季度 | 市场团队负责人（CMO） |
| 领导层 | 一页的竞争情报总结 | 每月 | CEO/首席运营官（CEO/COO） |
| 董事会 | 竞争格局幻灯片 | 每季度 | CEO |

**统一的信息来源：** 所有的竞争情报都存储在一个地方（如Notion、Confluence、Salesforce）。避免仅通过Slack传播信息，因为这样信息容易丢失。

## 竞争情报中的警示信号

| 信号 | 含义 |
|--------|---------------|
| 竞争对手在你的主要客户群体中的获胜率超过50% | 这是定位上的根本问题，而非销售问题。 |
| 有5笔以上交易中客户都提到相同的反对理由：“竞争对手有XX功能” | 这是真实存在的功能差距，而不仅仅是表面现象。 |
| 竞争对手在你所在的领域招聘了10名工程师 | 他们即将进行重大产品投资。 |
| 竞争对手筹集了超过2000万美元的资金，并瞄准你的目标客户群体 | 他们有12个月的准备时间来展开激烈竞争。 |
| 前景客户在评估你时以竞争对手的产品为依据 | 你在客户眼中只是满足他们需求的“选项”——需要改变这种印象或调整产品定位。 |

## 与高层管理团队的协作

| 智报类型 | 接收方 | 输出格式 |
|------------------|----------|---------------|
| 产品动态 | 产品负责人（CPO） | 用于制定产品路线图、功能差距分析。 |
| 价格变动 | 销售团队（CRO）、财务总监（CFO） | 价格策略建议。 |
| 融资情况 | CEO、财务总监（CFO） | 战略定位更新。 |
| 招聘动态 | 人力资源总监（CHRO）、技术总监（CTO） | 人才市场情报。 |
| 客户获取/流失情况 | 销售团队（CRO）、市场团队（CMO） | 竞争态势图更新、市场策略调整。 |
| 营销活动 | 市场团队（CMO） | 对抗性市场策略、渠道情报。 |
