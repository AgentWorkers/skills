# Midjourney 技能

通过 Midjourney Discord 机器人生成 AI 艺术作品。所有命令均通过 Discord 执行。

## ⚠️ 先决条件

1. **Discord 账户** — 必需
2. **Midjourney 订阅** — 免费试用有限，付费计划请访问 midjourney.com
3. **加入 Midjourney Discord ** — https://discord.gg/midjourney
4. **机器人访问权限** — 在 #newbies 通道或您自己的服务器中使用 `/imagine` 命令

---

## 快速入门

**工作流程：**
1. 在 Discord 中输入 `/imagine`
2. 输入您的提示语
3. 等待约 60 秒，系统会生成 4 个变体
4. 使用 U1-4 按钮放大图像，使用 V1-4 按钮生成更多变体

---

## 适用场景

✅ **适用于以下情况：**
- 从文本生成艺术作品、插图或概念图
- 创建现有图像的变体
- 提高低分辨率图像的分辨率
- 合并多张图像
- 从现有图像中获取创作灵感（使用 `describe` 命令）
- 测试不同的艺术风格和参数

❌ **不适用以下情况：**
- 需要商业许可信息 → 请查看 midjourney.com/terms
- 需要 API 访问权限 → 仅支持 Discord，无公开 API
- 需要精确的文本渲染 → Midjourney 在处理文本时可能存在问题
- 需要逼真的面部图像 → 可能会出现图像瑕疵

## 提示

1. **描述要清晰但不要过于具体** — 例如：“具有电影风格的照明”是可行的，而“精确的 47 度角度”则不太合适
2. **谨慎使用艺术家名称** — 如 “Greg Rutkowski 的风格” 可以作为参考，但请尊重艺术家的版权
3. **多次尝试** — 第一次生成的结果往往不完美，可以使用 V 按钮探索不同的变体
4. **保存任务 ID** — 便于后续参考
5. **查看 Discord** — 所有生成的图像都会保存在您的 Discord 私信中
6. **使用 --v 6** — 最新版本能更好地理解您的提示语
7. **niji 用于动漫风格** — 使用 `niji 6` 可生成动漫风格的作品

---

## 常见问题及解决方法

- **“队列已满”** — 请稍后尝试，或在非高峰时段操作
- **“参数无效”** — 请检查参数的语法，某些参数有特定的值要求
- **“违反内容政策”** — 请调整提示语以避免使用受限制的内容
- **无法放大图像** — 免费账户的生成任务可能有时间限制（24 小时）
- **文本显示异常** — Midjourney 在处理文本时可能出现问题，可以尝试使用 `--no text` 或 DALL-E 3 处理文本较多的图像
- **面部图像变形** — 这是一个常见问题，可以尝试使用 `--v 6` 或添加 “photorealistic” 参数
