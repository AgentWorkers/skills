# Proactive Agent Lite

将您的人工智能代理从被动执行任务的工具转变为能够主动预测需求并持续改进的合作伙伴。

## 核心特性

- **内存架构**：预压缩机制确保在窗口关闭时仍能保留上下文信息  
- **反向提示**：帮助您发现之前未曾想到的问题或需求  
- **安全加固**：内置安全防护机制及默认的安全设置  
- **自我修复功能**：能够自动诊断并解决自身存在的问题  
- **目标导向**：始终牢记自身的核心任务及服务对象  

## 主要优势

✅ **主动预测需求**：在您提出请求之前主动提供解决方案  
✅ **持续学习**：通过每次互动不断自我提升（无需额外训练）  
✅ **专注任务**：始终牢记核心目标  
✅ **自我维护**：能够自动从错误或不一致中恢复  
✅ **创造价值**：无需提示即可生成有价值的见解和机会  

## 使用方法

该功能可通过简单安装来增强任何 OpenClaw 代理的主动行为能力。安装完成后，代理将自动展现出主动服务的特性。  

## 集成方式

该功能可与所有 OpenClaw 代理无缝集成，并可与其他功能结合使用以提升整体性能。  

## 系统要求

- OpenClaw v1.0 或更高版本  
- 标准代理配置  

## 自定义设置

您可以通过配置参数来调整代理的主动行为程度，以匹配您的偏好和沟通风格。
