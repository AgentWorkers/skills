# 自动清理磁盘

可以安全地清理 **任何操作系统**（Windows、Linux、macOS）上的磁盘空间。
**绝不会删除重要文件**，只会删除垃圾文件、缓存文件、临时文件和日志文件。

---

## 该工具能清理的内容

| 类别                | 示例                                              |
|-------------------|-------------------------------------------------|
| 临时文件            | `%TEMP%`、`/tmp` 文件夹以及系统临时文件夹                   |
| 浏览器缓存            | Chrome、Firefox、Edge、Safari 的缓存文件                   |
| 回收站              | Windows 回收站、macOS 垃圾箱、Linux 垃圾箱                   |
| 日志文件            | 系统或应用程序文件夹中的旧 `.log` 文件                         |

> ⚠️ **绝对禁止删除**：文档、下载文件、桌面文件、用户数据、系统文件、已安装的应用程序，以及任何不在指定垃圾文件夹中的文件。

---

## 模式选择

**务必先询问用户**（如果尚未指定的话）：

- **自动模式**：静默清理所有文件，并在最后显示清理结果摘要
- **确认模式**：在清理每个类别之前询问用户是否确认

---

## 分步操作说明

### 第一步：检测操作系统

使用 Python 检测操作系统：
```python
import platform
os_type = platform.system()  # "Windows", "Linux", "Darwin" (Mac)
```

### 第二步：运行相应的脚本

根据操作系统，参考相应的参考文件：
- Windows → 查看 `windows-cleaner.md`
- Linux   → 查看 `linux-cleaner.md`
- macOS   → 查看 `mac-cleaner.md`

### 第三步：显示结果

清理完成后，务必显示以下信息：
- ✅ 清理后释放了多少磁盘空间（以 MB/GB 为单位）
- 📁 清理了哪些类别的文件
- ⚠️ 有哪些文件被跳过以及原因

---

## 安全规则（必须遵守）

1. ❌ 绝对不要修改 System32 或 Windows 文件夹
2. ❌ 绝对不要修改文档、图片、视频或下载文件
3. ❌ 绝对不要修改已安装的软件文件夹
4. ✅ 只清理脚本中明确列出的文件夹
5. ✅ 只删除创建时间超过 1 天的文件
6. ✅ 如果对某个文件不确定其用途——请跳过该文件

---

## 使用示例

**用户说：“我的磁盘空间满了”**  
→ 询问用户选择哪种清理模式 → 运行全盘清理 → 显示释放的磁盘空间

**用户说：“只清理浏览器缓存”**  
→ 仅清理浏览器缓存 → 显示清理结果

**用户说：“自动清理所有文件”**  
→ 静默清理所有文件类别 → 显示清理结果摘要
