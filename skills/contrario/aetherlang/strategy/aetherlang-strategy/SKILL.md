## 该技能的功能

该技能集成了三种V3级别的策略分析引擎：

### 📈 APEX Strategy V3 — 诺贝尔级分析
每个分析结果包含以下内容：
- **执行摘要**: 3-4个具有可操作性的建议及相应的数据支持。
- **现状分析**: 当前情况的量化评估，包括挑战及影响程度。
- **战略选项**: 至少提供3种不同的策略方案，涵盖复杂度（1-10级）、实现所需时间、投资金额、投资回报率（ROI%）及风险。
- **博弈论分析**: 使用纳什均衡理论进行评估。
- **行为经济学应用**: 考虑了锚定效应、损失厌恶心理、社会认同效应等行为因素。
- **蒙特卡洛模拟**: 进行10,000次模拟，以预测投资回报率（ROI）超过100%的概率、达到盈亏平衡的时间（<12个月）以及亏损发生的概率（90%置信区间）。
- **决策树分析**: 显示每种策略的预期价值及其概率。
- **实施路线图**: 分为0-30天基础建设期、30-90天扩展期、90-180天规模化期以及180天之后的优化阶段。
- **风险矩阵**: 包含风险概率、影响程度及相应的应对措施和预警机制。
- **竞争策略分析**: 分析“如果我们采取X策略，竞争对手可能会采取Y策略，进而我们需要采取Z策略”的情况。
- **财务预测**: 包括资本支出（CAPEX）、运营成本（OPEX）、三年收入预测、盈亏平衡点、投资回报率（ROI%）及净现值（NPV）。
- **业务指标分析**: 计算客户获取成本（CAC）、客户生命周期价值（LTV）、LTV与CAC的比率、投资回收期以及毛利率。
- **创新策略建议**: 提供创新方向（如消除现有问题、降低成本、提升产品或创造新机会）。
- **目标与关键结果生成**: 帮助制定明确的目标和关键成果（OKR）。
- **KPI指标**: 包括领先指标和滞后指标，以及相应的最低/目标/增长目标。
- **下一步行动**: 提供当天需要完成的3项具体任务。

### 📊 Market Intel V3
- 提供市场总规模（TAM）、目标市场（SAM）和潜在市场（SOM）的量化数据及年复合增长率（CAGR%）。
- 市场细分策略（包括新市场开发或市场竞争策略）。
- 应用波特五力模型进行分析。
- 根据客户支付意愿（WTP）和客户生命周期价值（LTV）对客户进行细分。
- 分析价格弹性及最佳定价策略。
- 评估网络效应和产品的病毒式传播潜力（K因子）。
- 绘制产品分销矩阵（包括直接销售、合作伙伴和电商平台等渠道）。

### 💼 NEXUS-7 Consult V3
- 提供因果循环图（用于识别和平衡各种因素）。
- 应用约束理论（识别瓶颈并制定相应的优化策略）。
- 使用Wardley模型进行产品开发流程管理。
- 提供反模式库，包含历史失败案例以供参考。
- 进行系统动态建模。

## 使用方法

您可以自然地提出任何与商业或策略相关的问题：
- “针对预算为1000欧元的希腊AIOCR平台，制定相应的策略。” → 会得到完整的APEX分析报告。
- “对希腊的食品配送应用进行市场分析。” → 会提供市场总规模、目标市场和潜在市场的量化数据以及竞争格局分析。
- “如何将我的SaaS产品用户数量从0扩展到1000人？” → 会提供NEXUS-7咨询服务及相应的Wardley模型分析。

## API详细信息
```
POST https://api.neurodoc.app/aetherlang/execute
Content-Type: application/json
```

### APEX Strategy的工作流程
```json
{
  "code": "flow Strategy {\n  using target \"neuroaether\" version \">=0.2\";\n  input text query;\n  node Guard: guard mode=\"MODERATE\";\n  node Engine: apex analysis=\"strategic\";\n  Guard -> Engine;\n  output text report from Engine;\n}",
  "query": "Your strategy question here"
}
```

### Market Intel的工作流程
```json
{
  "code": "flow Market {\n  using target \"neuroaether\" version \">=0.2\";\n  input text query;\n  node Engine: marketing analysis=\"competitive\";\n  output text report from Engine;\n}",
  "query": "Your market question here"
}
```

## 响应方式

返回结构化的希腊语输出结果，采用markdown格式的标题。典型响应内容约为4000-8000个字符，涵盖所有必填部分，并包含量化数据。

## 支持的语言

- **希腊语（Ελληνικά）**: 主要输出语言。
- **英语**: 支持英语查询，并以希腊语进行响应。
