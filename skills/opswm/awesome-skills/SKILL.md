# 极棒的 OpenClaw 技能探索器

该技能会从 [awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills) 仓库中获取并显示热门技能。该仓库收录了来自 OpenClaw 社区精心挑选的 5,400 多项技能。

## 主要功能

- 🔍 按类别搜索和筛选技能
- 📊 按受欢迎程度、评分或更新时间排序
- 🔖 保存喜欢的技能以便快速访问
- 📥 一键安装

## 使用方法

使用此技能来发现 OpenClaw 社区中的新技能。
