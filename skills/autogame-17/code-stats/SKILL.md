# 代码统计

该功能用于分析当前工作区，以提供开发相关的指标。

## 使用方法

```bash
node skills/code-stats/index.js [path]
```

如果未指定路径，系统将默认使用当前工作目录。

## 输出结果

输出为一个 JSON 对象，包含以下内容：
- `files`：文件总数。
- `lines`：总行数（近似值）。
- `byExt`：按文件扩展名分类的文件数量统计。
