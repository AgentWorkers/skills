# Cron 优化器

该工具用于清理 OpenClaw 的 cron 系统，具体操作包括：
- 删除已过期超过 24 小时的定时任务（“at”任务）。
- 删除长时间未执行的禁用任务。
- 删除冗余或重复的 “Mad Dog” 进化器循环（evolver loops）。

## 使用方法

```bash
node skills/cron-optimizer/index.js
```

## 主要功能：
- **降低系统执行负担**：清理那些会导致 `exec` 命令频繁执行的无效定时任务。
- **保障系统稳定性**：确保关键的系统监控任务（watchdogs）能够正常运行。
- **优化资源管理**：清除内存中过时的任务定义，释放系统资源。
