# News Cog – 由 CellCog 提供的人工智能新闻智能服务

**采用前沿搜索模型，对每个查询进行多角度研究**

CellCog 结合了深度推理技术、最先进的搜索模型以及多意图分析方法，以生成准确、全面的新闻智能内容。我们提供的简报、报告和摘要可以直接供您的智能助手使用，无需将数百篇文章导入到助手的界面中。

## 先决条件

使用此功能需要具备 `cellcog` 技能，以便进行 SDK 设置和 API 调用。

```bash
clawhub install cellcog
```

**请先阅读 `cellcog` 技能文档** 以了解 SDK 的安装方法。该文档将向您展示该技能的实用功能。

```python
result = client.create_chat(
    prompt="[your news/briefing request]",
    notify_session_key="agent:main:main",

    chat_mode="agent"
)
```

## 功能概述

### 每日简报

提供简洁、可操作的摘要：

- **晨间简报**：“提供今日科技行业的重点新闻，重点关注人工智能、云计算和企业软件领域”
- **市场动态**：“汇总影响 FAANG 公司和半导体行业的盘前新闻”
- **行业动态**：“总结本周可再生能源领域的重要发展”
- **初创企业新闻**：“本周初创企业融资情况如何？特别是金融科技领域的 A+ 轮融资”

**示例请求：**
> “为一家初创企业的 CEO 创建一份晨间简报：
> 
> 关注领域：
> 1. 人工智能行业动态（新模型、融资情况、法规变化）
> 2. SaaS 市场趋势及重要交易
> 3. 宏观经济指标（美联储政策、通货膨胀、科技行业就业情况）
> 
> 格式：使用项目符号列出要点，并附上来源链接。请突出显示需要立即关注的内容。
> 总长度：不超过 500 字——我需要在 2 分钟内阅读完毕。”

### 竞争情报

监控您的竞争环境：

- **竞争对手动态**：“Notion 在过去 30 天内发布了什么新产品或宣布了什么新计划？”
- **市场动态**：“汇总网络安全行业的近期并购活动”
- **产品发布**：“本月有哪些新的 AI 开发工具上市？”
- **融资追踪**：“列出过去两周内融资金额超过 5000 万美元的 AI 公司”

### 趋势报告

深入分析新兴趋势：

- **趋势分析**：“人工智能助手基础设施领域的最新发展趋势是什么？分析最近的发布、融资和技术进展”
- **法规动态**：“总结 2026 年全球范围内提出的或实施的新的 AI 相关法规”
- **技术变革**：“研究从 RAG 模型向长上下文模型的转变——最新进展是什么？”

### 定制新闻摘要

根据您的具体需求定制内容：

- **角色专属**：“为一家金融科技公司的工程副总裁创建每周摘要”
- **主题专属**：“汇总本周关于量子计算的所有新闻”
- **地区专属**：“本月欧洲科技生态系统发生了什么？”
- **活动报道**：“总结 AWS re:Invent 2026 的主要公告”

## 为何选择 CellCog？

### 技术优势

CellCog 不仅仅是简单地进行搜索和总结，它会对您的查询从多个角度进行深入研究：

1. **多意图分析**：一个新闻查询会被分解为多个研究方向
2. **前沿搜索模型**：从数百个来源中检索最相关的信息
3. **深度推理**：对搜索结果进行交叉引用和综合分析，而不仅仅是简单拼接
4. **信息来源的可追溯性**：所有信息都追溯到原始报道，而非聚合器的摘要

| RSS 源/新闻聚合器 | CellCog 新闻服务 |
|-----------------------------|------------------|
| 原始文章集合 | 综合后的、经过优先级排序的智能内容 |
| 由用户自行筛选和阅读 | CellCog 进行研究、分析并生成摘要 |
| 对所有用户统一展示 | 根据用户的角色、行业和需求进行个性化展示 |
| 仅提供文本链接 | 提供结构化的简报、PDF 报告或交互式仪表盘 |
| 信息量过大导致界面混乱 | 提供简洁、易于阅读的输出内容 |

### 专为智能助手设计的输出

CellCog 新闻服务充分考虑了智能助手的工作流程：

- **简洁明了**：不会占用助手界面的过多空间
- **结构清晰**：分章节、使用项目符号，层次分明
- **实用性强**：突出重点内容及其原因
- **信息来源可追溯**：根据需求提供来源链接

## 新闻信息的聊天模式

| 使用场景 | 推荐模式 |
|----------|------------------|
| 每日简报、快速新闻摘要 | 使用 `"agent"` 模式 |
| 竞争情报、单一主题的深度分析 | 使用 `"agent team"` 模式 |
- **深入趋势分析、全面的行业报告** | 使用 `"agent team"` 模式 |

**大多数新闻任务建议使用 `"agent"` 模式**。日常简报、竞争情报监控和新闻摘要非常适合使用该模式。

**对于需要综合分析多个来源的内容（如趋势报告），建议使用 `"agent team"` 模式**。

## 提升新闻智能质量的建议：

1. **明确关注焦点**：“科技新闻”这个范围太宽泛。建议指定更具体的领域，例如“人工智能基础设施和开发工具”。
2. **指定时间范围**：指定“今日新闻”、“本周新闻”或“过去 30 天的事件”——时效性很重要。
3. **确定目标受众**：“面向技术团队”与“面向董事会”的内容在深度和语气上有所不同。
4. **明确输出格式**：请求“使用项目符号列出要点、长度不超过 500 字的简报”，还是“包含多个章节的详细 PDF 报告”——请明确说明。
5. **要求提供来源信息**：如果需要追溯信息来源，请请求“为每个条目提供链接”。
6. **标注重点内容**：指出“与法规变化或重大融资事件相关的内容”——这有助于 CellCog 精选出对您来说重要的信息。
