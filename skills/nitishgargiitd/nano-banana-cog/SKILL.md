# Nano Banana Cog

**Nano Banana × CellCog**：通过地球上最强大的AI工具之一，体验Nano Banana的图像生成功能。

Google DeepMind开发的这款广受欢迎的图像生成模型——具备角色一致性、多图像合成以及高度逼真的编辑效果——现在可以通过CellCog的任意到任意执行层被任何OpenClaw工具所使用。

---

## 先决条件

使用此功能需要具备`cellcog`技能，以便进行SDK设置和API调用。

```bash
clawhub install cellcog
```

**请先阅读`cellcog`技能的相关内容**，以了解SDK的设置方法。该技能会向您展示所有可用的功能。

**快速使用指南（v1.0及以上版本）：**
```python
result = client.create_chat(
    prompt="[your image request]",
    notify_session_key="agent:main:main",
    task_label="image-task",
    chat_mode="agent"
)
```

## 您可以创建的内容

### 高度逼真的图像生成

根据文本描述创建令人惊叹的图像：
- **肖像画**：“制作一张在温暖工作室灯光下的专业头像”
- **产品图片**：“在暗色背景下生成一款高端智能手表的展示图片”
- **场景图片**：“创作一张充满晨光的温馨咖啡馆内景”
- **食物摄影**：“生成一张彩色佛碗的俯拍照片”

### 角色一致性

Nano Banana在保持角色形象的一致性方面表现出色：
- **角色系列**：“创建一个科技创业家的角色，并在4个不同的场景中展示他们”
- **品牌吉祥物**：“设计一个吉祥物，并在多种姿势和场景中生成它的图像”
- **故事情节**：“创建一个角色，并在5个故事片段中对其进行描绘”

### 多图像合成

将多张参考图像中的元素融合在一起：
- **风格融合**：“将图像A的色彩搭配与图像B的构图结合起来”
- **角色放置**：“将这个角色放置在新环境中，同时保持其外貌特征”
- **产品模型图**：“将这个产品放入一个符合其生活方式的场景中”

### 图像编辑

对现有图像进行修改和优化：
- **风格转换**：“将这张照片转换成宫崎骏风格的插画”
- **背景替换**：“将这个产品放在光滑的大理石表面上”
- **图像增强**：“添加戏剧性的光线效果和电影级的色彩分级”
- **场景调整**：“将这幅风景画的季节从夏季改为冬季”

## 图像规格

| 参数 | 可选值 |
|--------|---------|
| **宽高比** | 1:1, 16:9, 9:16, 4:3, 3:4, 3:2, 2:3, 21:9 |
| **分辨率** | 1K (~1024px), 2K (~2048px), 4K (~4096px) |
| **风格** | 高度逼真, 插画, 水彩, 油画, 动画, 数字艺术, 矢量图 |

## 聊天模式

| 使用场景 | 推荐模式 |
|----------|------------------|
| 单张图片或简单编辑 | `"agent"` |
| 需要角色一致性的系列图像或复杂构图 | `"agent"` |
| 需要遵循品牌指导原则的大规模图像项目 | `"agent team"` |

**大多数图像处理任务建议使用`"agent"`模式。**

## 提高图像质量的技巧

1. **详细描述**：例如，“办公室里的女性”可以描述为“一位40多岁的自信女性，穿着银色西装外套，在现代玻璃幕墙的办公室里，周围环绕着温暖的午后光线”。
2. **指定图像风格**：选择“高度逼真的”、“数字插画”、“水彩”、“动画”等。
3. **描述光线效果**：如“柔和的自然光”、“戏剧性的侧光”、“黄金时刻的光线”。
4. **为了保持角色一致性**：首先详细描述角色特征，然后在后续请求中提及“同一个角色”。
5. **说明构图要求**：例如“使用三分法则”、“特写肖像”或“宽景开场镜头”。
