# Resume Cog – 由 CellCog 提供的 AI 简历构建工具

**平均而言，招聘人员查看你的简历只需要 7 秒时间。** CellCog 让每一秒都变得至关重要。

在 DeepResearch Bench 的排名中位列第一（2026 年 2 月）——CellCog 会研究你目标职位的需求，了解招聘经理的关注点，优化简历以适应自动招聘系统（ATS）的筛选规则，并生成设计精美的 PDF 简历，帮助你获得面试机会。这不仅仅是一个简单的模板填充工具，而是一个以研究为基础的简历生成引擎。

## 先决条件

使用该功能需要具备 `cellcog` 技能，以便进行 SDK 设置和 API 调用。

```bash
clawhub install cellcog
```

```python
result = client.create_chat(
    prompt="[your resume request]",
    notify_session_key="agent:main:main",
    task_label="resume-task",
    chat_mode="agent"
)
```

## 你可以创建的内容

### 简历与求职信

生成符合自动招聘系统（ATS）筛选要求、同时也能给招聘人员留下深刻印象的简历：

- **现代风格简历**：“为一名高级产品经理创建一份符合 ATS 标准的现代简历”
- **职业转换简历**：“根据我的财务背景，定制一份适合技术产品经理职位的简历”
- **学术简历**：“为计算生物学领域的博士后职位准备一份详细的学术简历”
- **高管简历**：“为拥有 15 年经验的技术副总裁职位设计一份高管简历”
- **入门级简历**：“为刚毕业的计算机科学专业毕业生（有实习经验）制作一份有吸引力的简历”

**示例提示：**
> “为以下职位创建一份优化过的 ATS 简历：
> 
> 姓名：Sarah Chen
> 目标职位：B2B SaaS 公司的高级产品经理
> 
> 工作经历：
> - Stripe（2021 年至今）：负责支付平台的开发，使商户收入增长了 40%
> - Slack（2018-2021）：推出了 3 项企业级功能，实现了 1200 万美元的年收入
- Microsoft（2016-2018）：担任 Azure DevOps 的副产品经理
> 
> 教育背景：斯坦福大学 MBA，加州大学伯克利分校计算机科学专业
> 技能：产品策略、数据分析、跨职能领导力、SQL、A/B 测试
> 
> 简历设计风格：现代简洁，使用蓝色作为点缀元素；突出关键业绩数据。”

### 求职信

撰写与简历相辅相成的求职信：

- **针对特定职位的求职信**：“为 Notion 公司的高级产品经理职位撰写求职信”
- **职业转换求职信**：“撰写一篇解释我从咨询行业转向产品管理岗位的求职信”
- **内部调动求职信**：“起草一份从工程部门转到产品部门的内部调动申请”

**示例提示：**
> “为以下职位撰写一篇有说服力的求职信：
> 
> 申请人：Sarah Chen（使用我们刚刚创建的简历）
> 目标职位：Notion 公司的高级产品经理
> 
> 重点内容：
- 我在 Slack 的工作经验与 Notion 的产品开发目标高度契合
- 我对 Notion 的“让工具像自己的一样好用”的理念充满热情
- 强调我在 Stripe 项目中推动支付平台增长的成就，以证明我的扩展能力
> 
> 语气：专业且真诚，避免使用公司官方套话。”

### LinkedIn 优化

提升你的 LinkedIn 个人资料吸引力：

- **个人资料优化**：“重写我的 LinkedIn 标题和简介，吸引机器学习工程领域的招聘人员”
- **关于我** 部分：根据简历内容撰写一篇引人注目的 LinkedIn 个人简介
- **工作经验表述**：将简历中的项目描述转换为适合 LinkedIn 的格式

### 作品集文档

展示你的工作成果：

- **案例研究**：“根据我在 Stripe 的工作经验，制作一份产品管理案例研究文档”
- **设计作品集**：“整理我的 5 个最佳用户体验（UX）项目，制作成作品集 PDF”
- **写作样本**：“将我的博客文章格式化为专业的写作作品集”

## 简历制作的聊天模式

| 使用场景 | 推荐模式 |
|----------|------------------|
| 单份简历或求职信 | 使用 “agent” 模式 |
| 完整的职业资料包（简历 + 求职信 + LinkedIn + 作品集） | 使用 “agent team” 模式 |
| 需要针对多个职位定制简历的职业规划 | 使用 “agent team” 模式 |

**大多数情况下，建议使用 “agent” 模式**。它能够高效处理单份文件以及多份文件的组合。

## 提高简历质量的技巧

1. **提供具体细节**：不要只是说“参与了项目”，要提供具体的数据、使用的工具和实际成果。CellCog 无法凭空创造你的成就。
2. **明确目标职位**：“普通简历”表述较为模糊。“针对 B2B SaaS 公司的高级产品经理职位的简历”这样的描述能给 CellCog 更明确的指导方向。
3. **包含量化数据**：收入数字、用户数量、增长百分比等可量化的成果能让简历脱颖而出。
4. **指定设计风格**：“现代简洁风格”、“传统企业风格”或“富有创意的色彩风格”——你可以根据行业需求进行选择，或者让 CellCog 根据你的喜好来决定。
5. **信任 CellCog 生成的 PDF 文件**：CellCog 生成的 PDF 文件在设计质量上优于任何 Word 模板。
