# 本地内存搜索

这是一个轻量级的语义搜索工具，用于 OpenClaw 的内存文件，无需任何外部 API。

## 使用方法

```bash
python3 search.py "your query"
```

构建索引：
```bash
python3 search.py --build
```

## 主要特性

- 支持搜索 `MEMORY.md` 以及所有以 `memory/` 开头的 `.md` 文件
- 基于 TF-IDF 的语义匹配算法
- 完全不依赖任何外部库
- 执行速度快，支持在本地运行
- 返回包含文件路径和行号的搜索结果

## 工作原理

1. 为所有内存文件构建倒排索引
2. 使用 TF-IDF 评分算法来评估搜索结果的相关性
3. 返回按相关性排序的搜索结果，并提供上下文信息

## 系统要求

- Python 3.8 及以上版本
- 无需安装任何额外的 pip 包（仅使用标准库）
