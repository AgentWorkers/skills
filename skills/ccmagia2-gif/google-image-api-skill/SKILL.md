# Google 图片 API 自动化技能

## 📖 介绍
该技能允许用户通过使用 BrowserAct 的 Google 图片 API 模板，一键从 Google 图片中提取图片数据。用户可以输入搜索关键词，设置搜索国家/地区和语言，控制滚动深度和结果数量，从而通过 API 直接获取结构化、清晰的图片元数据。

## ✨ 特点
1. **无生成式错误（无虚假图像）**：预设的工作流程避免了生成式 AI 产生的错误图像。
2. **无需处理验证码**：无需应对 reCAPTCHA 或其他验证挑战。
3. **无 IP 限制或地域屏蔽**：不受地区 IP 限制的影响。
4. **执行速度快**：相比纯 AI 驱动的浏览器自动化解决方案，执行速度更快。
5. **成本效益高**：与消耗大量 API 令牌的 AI 解决方案相比，显著降低了数据获取成本。

## 🔑 API 密钥指南
在运行之前，请检查 `BROWSERACT_API_KEY` 环境变量。如果该变量未设置，请勿继续操作；应请求用户提供正确的 API 密钥。
**此时，代理应通知用户：**
> “由于您尚未配置 BrowserAct API 密钥，请先访问 [BrowserAct 控制台](https://www.browseract.com/reception/integrations) 获取密钥。”

## 🛠️ 输入参数
代理在调用脚本时应根据用户需求灵活配置以下参数：

1. **KeyWords（搜索关键词）**
   - **类型**：`string`
   - **描述**：用于在 Google 图片中搜索的关键词。
   - **示例**：`flower`、`ai agent`、`tesla`

2. **Country（国家/地区偏好）**
   - **类型**：`string`
   - **描述**：搜索结果的国家或地区偏好。
   - **支持的值**：`us`、`gb`、`ca`、`au`、`de`、`fr`、`es`、`jp`、`kr`
   - **默认值**：`us`

3. **Language（用户界面语言）**
   - **类型**：`string`
   - **描述**：Google 图片界面和返回文本的语言。
   - **支持的值**：`en`、`zh-CN`、`zh-TW`、`ja`、`ko`、`fr`、`de`、`es`
   - **默认值**：`en`

4. **Scroll_count（滚动次数）**
   - **类型**：`number`
   - **描述**：用于加载更多图片结果的滚动次数。
   - **默认值**：`5`

5. **Datelimit（最大结果数量）**
   - **类型**：`number`
   - **描述**：从结果列表中提取的最大结果数量。
   - **默认值**：`50`

## 🚀 调用方式（推荐）
代理应执行以下命令来实现“一键获取结果”的功能：

```bash
# 示例调用
python -u ./scripts/google_image_api.py "KeyWords" "Country" "Language" Scroll_count Datelimit
```

### ⏳ 执行状态监控
由于该任务涉及自动化浏览器操作，可能需要较长时间（几分钟）。脚本在运行过程中会持续输出带有时间戳的状态日志（例如：`[14:30:05] 任务状态：正在运行`）。
**代理注意事项：**
- 在等待脚本返回结果时，请密切关注终端输出。
- 只要终端仍在输出新的状态日志，说明任务正在正常运行；不要误认为是死锁或无响应。
- 如果状态长时间不变或脚本停止输出且未返回结果，请考虑触发重试机制。

## 📊 数据输出
脚本成功执行后，会直接从 API 响应中解析并打印结果。输出内容包括：
- `is_product`：结果是否被识别为产品列表。
- `link`：与结果关联的点击链接。
- `title`：图片结果的标题或说明文字。
- `source_logo`：来源网站的标志 URL。
- `source`：结果中显示的来源网站名称。
- `related_content_id`：Google 图片中的相关内容标识符。
- `thumbnail`：缩略图图片 URL。
- `index`：结果在列表中的索引。

## ⚠️ 错误处理与重试
在脚本执行过程中，如果出现错误（如网络波动或任务失败），代理应按照以下步骤处理：
1. **检查输出内容**：
   - 如果输出中包含 “Invalid authorization”，则表示 API 密钥无效或已过期。此时，请指导用户检查并提供正确的 API 密钥。
   - 如果输出中不包含 “Invalid authorization”，但任务仍然失败（例如，输出以 “Error:” 开头或结果为空），代理应 **自动尝试再次执行脚本**。
2. **重试次数限制**：
   - 自动重试仅允许 **一次**。如果第二次尝试仍然失败，请停止重试并向用户报告具体的错误信息。

## 🌟 常见应用场景
1. **视觉内容采集**：为创意研究和设计工作寻找特定的图片素材。
2. **竞争对手资产监控**：扫描 Google 图片以获取竞争对手的产品样式和标志。
3. **市场视觉研究**：收集不同国家的产品列表数据。
4. **本地化图像趋势分析**：追踪特定关键词在日本（`jp`）或法国（`fr`）等地区的搜索结果。
5. **电子商务发现**：提取点击链接以了解产品的销售渠道。
6. **数据丰富化**：获取与关键词相关联的缩略图和标题信息。
7. **品牌追踪**：查找作为图片结果出现的特定品牌。
8. **SEO 关键词可视化**：查看排名靠前的 SEO 关键词的视觉结果。
9. **自动化内容聚合**：每日提供特定主题的可视化元数据列表。
10. **全球图像搜索**：以目标语言查找与全球事件或人物相关的图片。
