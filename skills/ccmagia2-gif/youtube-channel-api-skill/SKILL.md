# YouTube频道API技能

## 📖 介绍  
本技能通过BrowserAct的YouTube频道API模板，为用户提供一站式频道数据提取服务。用户只需输入搜索关键词和可选的上传日期筛选条件，即可直接获取结构化、可用的频道数据。

## ✨ 特点  
1. **数据稳定可靠**：采用预设的工作流程，避免AI生成错误数据。  
2. **无需处理验证码**：无需应对reCAPTCHA等验证挑战。  
3. **无IP限制或地域屏蔽**：不受地区IP限制的影响。  
4. **执行速度快**：相比纯AI驱动的浏览器自动化方案，执行速度更快。  
5. **高性价比**：相比消耗大量API令牌的AI方案，显著降低了数据获取成本。  

## 🔑 API密钥指南  
在运行脚本之前，请先检查`BROWSERACT_API_KEY`环境变量。如果该变量未设置，请勿继续操作；您需要用户提供API密钥。  
**此时，代理应告知用户**：  
> “由于您尚未配置BrowserAct API密钥，请前往[BrowserAct控制台](https://www.browseract.com/reception/integrations)获取密钥。”  

## 🛠️ 输入参数  
代理在调用脚本时应根据用户需求灵活配置以下参数：  

1. **KeyWords**  
   - **类型**：`string`  
   - **说明**：用于搜索YouTube频道的关键词。  
   - **示例**：`AI`、`automation`、`tech reviews`  

2. **Upload_date**  
   - **类型**：`string`  
   - **说明**：筛选在指定时间范围内发布内容的频道。  
   - **可选值**：`This week`（本周）、`Any time`（任意时间）等  
   - **默认值**：`Any time`（任意时间）  

## 🚀 使用方法  
代理应执行以下独立脚本以实现“一次命令获取结果”的目标：  

```bash
# Example call
python -u ./scripts/youtube_channel_api.py "Search Keywords" "Upload date"
```  

### ⏳ 运行状态监控  
由于该任务涉及自动化浏览器操作，可能需要较长时间（几分钟）。脚本在运行过程中会**持续输出带有时间戳的状态日志**（例如：`[14:30:05] 任务状态：正在运行`）。  
**代理注意事项**：  
- 在等待脚本返回结果期间，请密切关注终端输出。  
- 只要终端持续输出新的状态日志，说明任务正在正常运行，请勿误判为死锁或无响应。  
- 如果状态长时间未变化或脚本停止输出且未返回结果，请考虑触发重试机制。  

## 📊 数据输出  
脚本成功执行后，会直接解析并打印API响应中的数据，包括：  
- `channel_title`：YouTube上显示的频道名称  
- `channel_link`：频道页面URL  
- `verified`：频道是否已认证  
- `handle`：频道标识符  
- `subscribers`：订阅者数量  
- `description`：频道描述文本  
- `thumbnail`：频道头像或缩略图URL  

## ⚠️ 错误处理与重试  
在脚本执行过程中，如果出现错误（如网络波动或任务失败），代理应按照以下逻辑处理：  

1. **检查输出内容**：  
   - 如果输出中包含“Invalid authorization”，说明API密钥无效或已过期。此时**不要重试**，应引导用户重新检查并提供正确的API密钥。  
   - 如果输出中不包含“Invalid authorization”，但任务执行失败（例如输出以“Error:”开头或返回空结果），代理应**自动尝试再次执行脚本**。  

2. **重试限制**：  
   - 自动重试次数限制为**一次**。如果第二次尝试仍然失败，请停止重试，并向用户报告具体的错误信息。  

## 🌟 典型应用场景  
1. **创作者发现**：查找与特定主题相关的频道以进行联系。  
2. **影响者挖掘**：识别特定行业中的潜在影响者。  
3. **竞争格局分析**：跟踪竞争对手频道的增长和更新情况。  
4. **频道数据库构建**：提取数据以构建创作者数据库。  
5. **内容趋势监测**：监控围绕特定关键词新出现的频道。  
6. **市场研究**：收集目标细分市场中领先YouTube频道的数据。  
7. **品牌监控**：追踪讨论您品牌的非官方或相关频道。  
8. **受众分析**：通过订阅者数量评估特定领域的受众规模。  
9. **合作伙伴推广**：查找最近上传视频的活跃频道。  
10. **自动化数据整合**：将YouTube频道信息导入客户关系管理（CRM）系统或内部工具。
