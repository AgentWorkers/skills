# QuarkPan 备份套件

使用此技能来部署一个生产环境安全的备份工作流程，具体目标如下：
- 将自动化的工作区备份数据存储到 QuarkPan 中。
- 强制实施账户绑定机制，以防止数据被错误地上传到其他账户。
- 系统回滚快照仅允许手动执行（避免自动覆盖数据的风险）。
- 提供可复制的恢复步骤以及可分发的技能包。

## 工作流程
1. 使用 `scripts/check_env.sh` 运行预检查。
2. 根据 `references/commands.md` 配置备份策略。
3. 在数据上传前执行账户绑定、检查及快照轮换等操作。
4. 先进行模拟恢复测试，以确保恢复路径的正确性。
5. Lighthouse 快照仅允许手动创建，并设置每周自动提示的策略。
6. 使用 `package_skill.py` 将此技能包进行打包和共享。

## 安全限制
- 禁止自动执行具有破坏性的恢复操作。
- 禁止通过 cron 任务自动创建系统快照。
- 快照的创建和回滚操作必须经过明确确认。
- 即使云端上传失败，也不会影响本地备份的完成。
