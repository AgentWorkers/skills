# 钱包 👛

统一化的数字钱包管理工具。

## 主要功能

- 支持多种货币
- 账户余额查询
- 交易记录查看
- 资金发送与接收
- 钱包备份与恢复

## 支持的类型

- 加密货币钱包
- 数字支付钱包
- 硬件钱包集成

## 使用示例

```
"Show all my wallets"
"What's my total balance?"
"Transfer funds between wallets"
```
