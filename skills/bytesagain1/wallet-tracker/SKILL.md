# 钱包追踪器 👛

您可以追踪以太坊（EVM）链和Solana链上的多个钱包，汇总您的投资组合，监控资产持有情况，并在发生重大交易时收到警报。

## 使用场景

### 场景1：个人投资组合概览

您在以太坊、Polygon和Solana上拥有资产，分布在5个不同的钱包中。您希望有一个统一的仪表板来查看这些资产。

**输出：** 一份HTML报告，显示总价值、各链上的资产分布以及每个钱包的持有情况。

---

### 场景2：追踪大额交易

监控某个大型钱包的交易情况（例如，当交易金额超过10万美元时触发警报）：

**输出：** 控制台会显示警报信息。

---

### 场景3：空投活动追踪

您正在使用多个钱包参与空投活动。该工具会记录您的参与情况：

**输出：** 每个钱包的交易次数、交互的合约数量、活跃天数以及交易量。

---

### 场景4：团队资金库监控

监控DAO（去中心化自治组织）的资金库，防止未经授权的提款：

**输出：** 监控资金库的变动情况。

---

### 场景5：跨链余额快照

生成所有链上账户的实时余额快照，用于财务核算：

**输出：** 所有链上账户的余额信息。

## 命令

| 命令        | 功能                        |
|-------------|---------------------------|
| `portfolio`   | 汇总所有钱包及链上的资产信息           |
| `watch`     | 实时监控并设置交易阈值警报             |
| `activity`  | 显示交易活动摘要                 |
| `monitor`    | 检测账户余额变化                 |
| `snapshot`   | 生成账户余额的快照                   |
| `compare`    | 比较两个时间点的账户余额变化           |

## 隐私声明

本工具仅查询**公开区块链数据**，不会要求或使用任何私钥。所有数据均在本地进行处理。
