# 基准测试工具

这是一个命令行基准测试工具 ## 命令

- `help` - 显示帮助信息
- `run` - 运行基准测试
- `info` - 显示工具信息
- `status` - 显示工具状态

## 特点

- 该工具的核心功能源自 sharkdp/hyperfine

## 使用方法

运行任意命令：`benchmark-tool <命令> [参数]`
