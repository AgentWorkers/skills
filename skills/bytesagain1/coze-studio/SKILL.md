# Coze Studio

这是一个用于开发AI代理的平台，集成了全方位的可视化工具，极大地简化了代理的创建、调试和部署流程。通过Coze Studio，您可以更轻松地构建AI代理。

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 查看信息
- `status` - 查看当前状态

- 兼容 coze-dev/coze-studio 的核心功能
