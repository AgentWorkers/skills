---# PerfTest

这是一个用于性能测试和基准测试的工具。它可以测量命令的执行时间、评估磁盘I/O性能、测试网络吞吐量、比较不同命令的执行效率，并生成相应的报告。

## 快速入门

运行 `perftest help` 可以查看所有可用的命令及其使用示例。

## 主要特性

- 快速且轻量级：基于纯Bash脚本，内置了Python功能
- 无需任何外部依赖
- 数据存储在本地目录 `~/.perftest/` 中
- 支持Linux和macOS操作系统

## 使用方法

```bash
perftest help
```
