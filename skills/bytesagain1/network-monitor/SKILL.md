# 网络监控工具

轻松监控您的互联网流量 🕵️‍♂️

本工具的灵感来源于 [GyulyVGC/sniffnet](https://github.com/GyulyVGC/sniffnet)（在 GitHub 上获得了 32,966 个星标）。

## 命令

- `help` - 显示帮助信息
- `run` - 运行工具
- `info` - 查看信息
- `status` - 查看状态

## 功能

- 兼容 GyulyVGC/sniffnet 的核心功能

## 使用方法

运行任意命令：`network-monitor <命令> [参数]`
