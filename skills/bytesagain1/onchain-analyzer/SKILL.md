# Onchain Analyzer

深入分析任何钱包的链上活动。

## 快速参考

| 命令 | 描述 |
|---------|-------------|
| `profile <addr>` | 钱包概览和统计数据 |
| `tokens <addr>` | 代币持有情况 |
| `txns <addr> [limit]` | 最近的交易记录 |
| `defi <addr>` | DeFi（去中心化金融）持仓及收益 |
| `patterns <addr>` | 交易模式分析 |
| `compare <addr1> <addr2>` | 比较两个钱包 |

## 支持的链

Ethereum、BSC、Polygon、Arbitrum、Base、Optimism、Avalanche、Solana

## 使用场景

- **研究**：分析大型钱包和智能资金的活动 |
- **安全**：验证交易对手方的钱包历史记录 |
- **监控**：监控钱包是否存在复制交易行为 |
- **合规性**：获取交易历史用于税务/审计目的 |
