# 情绪追踪器

情绪追踪器——帮助您监测自己的情绪健康状况

## 为什么需要这个工具？

- 专为个人日常使用设计
- 无需任何外部依赖或账号
- 数据存储在本地，保护您的隐私
- 命令简单，但功能强大

## 可用命令：

- `log` — <1-5> [心情描述]     记录当前心情（1=糟糕 5=极好）
- `today` —                今日的情绪状态
- `week` —                 周情绪图表
- `history` — [天数]          情绪历史记录（默认14天）
- `stats` —                情绪统计信息
- `patterns` —             识别情绪模式
- `triggers` — [心情类型]      常见引发情绪变化的因素
- `streak` —               正面情绪连续天数
- `journal` — [文本]       情绪日记记录
- `insights` —             人工智能提供的情绪分析建议
- `info` —                 工具版本信息

## 快速入门

```bash
mood_tracker.sh help
```
