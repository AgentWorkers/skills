# Cc Switch

这是一个跨平台的桌面一体化辅助工具，支持 Claude Code、Codex、OpenCode、OpenClaw 和 Gemini CLI 的操作。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 显示工具信息
- `status` - 显示工具状态

## 特性

- 功能源自 farion1231/cc-switch

## 使用方法

运行任意命令：`cc-switch <命令> [参数]`
