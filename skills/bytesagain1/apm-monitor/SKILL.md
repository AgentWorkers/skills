# Pinpoint

Pinpoint 是一款专为大规模分布式系统设计的应用程序性能管理（APM）工具。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行指定命令
- `info` - 显示工具信息
- `status` - 显示工具状态

## 功能

- 兼容 apm-monitor-apm/apm-monitor 的核心功能

## 使用方法

执行任意命令：`apm-monitor <command> [args]`
