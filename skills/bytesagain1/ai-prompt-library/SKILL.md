# 人工智能工具的系统提示与模型

以下是部分人工智能工具的系统提示代码及相关模型：

- **完整增强版代码（Full Augment Code）**：Claude Code、Cluely、CodeBuddy、Comet、Cursor、Devin AI、Junie、Kiro、Leap.new、Lovable、Manus、NotionAI、Orchids.app、Perplexity、Poke、Qoder、Replit、Same.dev、Trae、Traycer AI、VSCode Agent、Warp.dev、Windsurf、Xcode、Z.ai Code；以及其他开源系统提示和内部工具/人工智能模型。

## 命令：

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 查看信息
- `status` - 获取状态

## 特性：

- 该工具基于 `x1xhlol/ai-prompt-library` 的核心功能进行开发。

## 使用方法：

运行任何命令的格式为：`ai-prompt-library <命令> [参数]`
