# Fashion Colorize Shell

## 概述

该功能能够将户外壳夹克的草图转换为逼真的产品渲染图像。在保留草图结构的同时，会应用产品拍摄的风格（单件服装、正面视图、白色背景以及真实的硬壳材质）。

## 工作流程

1. 收集输入信息：
   - 必需输入：草图图片路径
   - 必需输入：简要描述（材质、颜色、版型、主要设计意图）
   - 可选输入：风格参考图片路径

2. 运行本地脚本：
   - `uv run {baseDir}/scripts/run_colorize.py --sketch "<path>" --brief "<text>" --output-dir "<dir>" [--style-ref "<path>"] [--count 3]`

3. 返回生成的文件路径。

4. 在进行修改时，保留之前的输出结果作为风格参考，并在 `--brief` 参数中明确说明修改内容。

## 输入指南

- 示例说明：
  - “三层压胶硬壳材质，凯乐石薄荷绿，女性剪裁，重点突出胸前的斜插口袋和袖口的调节功能。”

- 如果用户需要技术图纸（即不含渲染效果的平面图），请不要使用此功能。此时应询问用户是否需要单独的技术绘图流程。

## 输出要求

- 默认输出格式为适合电商展示的产品渲染图像：
  - 单件服装
  - 正面视图
  - 白色背景
  - 无模特、无场景、无文字叠加

- 保留草图中的关键元素：
  - 头罩的形状
  - 正面中央的拉链/插袋位置
  - 口袋的摆放位置
  - 袖口和下摆的可调节区域

## 固定的运行时参数

- API 基本地址：`https://models.kapon.cloud`
- 模型默认选择：`gemini-3-pro-image-preview-2k`（如果上游服务失败，将自动切换为 `gemini-3-pro-image-preview`）
- API 密钥不会包含在脚本中，用户需在环境变量中设置 `GEMINI_API_KEY`。
