# Acmesh

这是一个用于自动化处理 SSL/TLS 证书的纯 Unix shell 脚本（ACME 客户端）

## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 显示相关信息
- `status` - 显示当前状态

## 特点

- 采用 `acmesh-official/acme.sh` 的核心功能

## 使用方法

运行任意命令：`acmesh <命令> [参数]`
