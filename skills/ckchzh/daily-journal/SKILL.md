# 每日日记

每日日记 — 记录生活，反思自我，铭记点滴

## 为什么需要这项技能？

- 专为个人日常使用设计
- 无需依赖任何外部服务或账户
- 数据存储在本地，保护您的隐私
- 命令简单，功能强大

## 常用命令：

- `write` — [文本]       输入今天的日记内容
- `today` —          查看今天的日记
- `yesterday` —          查看昨天的日记
- `list` — [数量]       列出最近的日记条目（默认显示10条）
- `search` — <查询>      搜索所有日记条目
- `mood` — <1-5> [备注]    记录当天的情绪（1=糟糕，5=非常好）
- `streak` —         维持每日写日记的习惯
- `gratitude` — [文本]     记录感恩之事
- `prompt` —         提供随机写作提示
- `stats` —         查看日记统计信息
- `export` — [格式]       导出日记（md/json/html）
- `info` —         查看软件版本信息

## 快速入门

```bash
daily_journal.sh help
```
