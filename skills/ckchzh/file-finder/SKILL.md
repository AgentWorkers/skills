# 文件查找器（File Finder）

这是一个简单、快速且用户友好的文件查找工具，可作为 `find` 命令的替代品。

灵感来源于 [sharkdp/fd](https://github.com/sharkdp/fd)（在 GitHub 上获得了 42,080 多个星标）。

## 命令

- `help` - 显示帮助信息
- `run` - 运行文件查找器
- `info` - 查看文件查找器的信息
- `status` - 查看文件查找器的状态

## 功能特点

- 兼具了 sharkdp/fd 的核心功能

## 使用方法

运行任意命令格式如下：`file-finder <命令> [参数]`
