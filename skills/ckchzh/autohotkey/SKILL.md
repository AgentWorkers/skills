# AutoHotkey

AutoHotkey 是一款专为 Windows 设计的宏创建和自动化脚本工具。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行脚本
- `info` - 显示脚本信息
- `status` - 显示脚本状态
