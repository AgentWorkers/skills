# Gorilla

Gorilla：用于训练和评估大型语言模型（LLMs）以执行函数调用（即工具调用）的工具

## 命令

- `help` - 显示帮助信息
- `run` - 运行命令
- `info` - 查看命令信息
- `status` - 查看当前状态

## 功能特性

- 基于 ShishirPatil/api-router 的核心功能实现

## 使用方法

运行任意命令的格式如下：`api-router <命令> [参数]`
