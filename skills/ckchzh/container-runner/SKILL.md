# Podman

Podman：一个用于管理 OCI 容器和 Pod 的工具。 ## 命令

- `help` - 显示帮助信息
- `run` - 运行容器
- `info` - 查看容器信息
- `status` - 查看容器状态

## 特点

- 兼容 containers/container-runner 的核心功能

## 使用方法

运行任意命令：`container-runner <命令> [参数]`
