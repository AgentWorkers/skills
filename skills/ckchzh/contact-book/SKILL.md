# 联系人管理工具

**联系人管理工具** — 用于管理个人联系人信息

## 为什么需要这个工具？

- 专为日常个人使用设计
- 无需依赖任何外部服务或账户
- 数据存储在本地，保护您的隐私
- 命令简单，功能强大

## 命令列表：

- `add` — <姓名> <电话> [电子邮件]  添加联系人
- `list` — [数量]                列出所有联系人（默认显示全部）
- `search` — <查询条件>            搜索联系人
- `view` — <姓名>                查看联系人详情
- `edit` — <姓名> <字段> <值>    修改联系人信息
- `group` — <姓名> <组名>           将联系人添加到组中
- `groups` —                显示所有联系人组
- `birthday` — <姓名> <日期>          设置联系人生日
- `birthdays` —             显示即将到期的联系人生日
- `delete` — <姓名>               删除联系人
- `export` — [格式]                导出联系人信息（csv/vcf/json）
- `stats` —                显示联系人统计信息

## 快速入门

```bash
contact_book.sh help
```
