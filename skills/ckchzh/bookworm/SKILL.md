# 阅读日志

阅读日志 — 用于记录阅读的书籍和阅读习惯

## 为什么需要这个工具？

- 专为个人日常使用设计
- 无需任何外部依赖或账户
  — 保护您的隐私和数据安全
- 命令简单，功能强大

## 命令说明

- `add` — `<书名> [作者]`    将书籍添加到阅读列表中
- `start` — `<书名>`           开始阅读某本书
- `progress` — `<书名> <完成百分比>`  更新阅读进度（0-100%）
- `finish` — `<书名> [评分]`  标记书籍已读完（1-5星）
- `list` — `[筛选条件]`       列出所有书籍（全部/正在阅读/已读/待读）
- `search` — `<查询内容>`       搜索书籍
- `stats` —            阅读统计信息
- `streak` —            阅读连贯性统计
- `recommend` — `[类型]`       推荐书籍
- `shelf` —            可视化书架展示
- `export` — `[格式]`         导出数据（md/csv/json）
- `info` —            工具版本信息

## 快速入门

```bash
reading_log.sh help
```
