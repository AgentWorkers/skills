---# ColorPick

这是一个颜色选择器和转换工具，支持在十六进制（hex）、RGB、HSL颜色格式之间进行转换，能够生成颜色调色板、查找互补色，并在终端中预览颜色。

## 快速入门

运行 `colorpick help` 可以查看可用的命令及其使用示例。

## 主要功能

- 快速且轻量级：基于纯 Bash 语言开发，内置了 Python 功能
- 无需任何外部依赖
- 数据存储在用户的主目录下的 `~/.colorpick/` 文件夹中
- 支持 Linux 和 macOS 系统

## 使用方法

```bash
colorpick help
```
