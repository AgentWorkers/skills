# Antlr4

ANTLR（Another Tool for Language Recognition）是一款功能强大的解析器生成器，可用于读取、处理、执行或转换结构化文本或二进制文件。

## 命令

- `help` - 显示帮助信息
- `run` - 运行指定命令
- `info` - 显示工具信息
- `status` - 显示工具状态

## 特点

- 基于 ANTLR4 的核心功能实现

## 使用方法

运行任意命令的格式为：`antlr4 <命令> [参数]`
