## 软件包包含的内容

| 技能 | 功能说明 |
|-------|-----------|
| 📄 dogovor-ru | 解释租赁合同、劳动合同、贷款合同以及抵押贷款合同的内容 |
| 🏠 zhkh-ru | 解读物业管理公司的账单，计算费用总额，并识别其中的错误 |
| 🩺 analizy-ru | 将医疗检测报告从专业术语翻译成通俗易懂的语言 |
| ✍️ pretenziya-ru | 帮助撰写针对银行、商店或政府机构的投诉信 |
| 🔤 chinovnik-ru | 将官方文件和决议翻译成简单易懂的语言 |
| 🧮 nalog-ru | 为个体经营者（IP）计算6%的统一社会税（USN）和15%的国民所得税（NPD） |

## 使用方法

只需输入您的需求，软件包会自动选择相应的技能来帮助您：

- “解释这份租赁合同”
- “解读这份物业管理公司的账单”
- “我的检测报告是什么意思？”
- “帮忙撰写针对商店的投诉信”
- “这封来自税务局的信件是什么意思？”
- “我是个体经营者，本季度收入为20万”

## 适用人群

适用于所有居住在俄罗斯和独联体国家的人士，希望无需律师或会计师的帮助，也能自己理解各种文件、物业管理费用、医疗报告以及税收相关内容的人。
