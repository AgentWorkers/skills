#### 触发条件

| 触发方式 | 适用场景 |

| **用户无操作时** | 当用户长时间无操作时重新吸引用户注意 |
| **点击特定元素时** | 用户点击特定元素时显示弹窗 |

| 定位方式 | 可选选项 | 适用场景 |

| **地理位置** | 国家、地区、城市 | 根据用户所在地理位置展示本地化内容 |
| **语言** | 根据用户浏览器语言展示多语言弹窗 |
| **流量来源** | 来源网站 | 根据用户访问的网站类型展示相应信息 |
| **引用链接** | 引用页面地址 | 根据用户访问的引用页面展示特定内容 |
