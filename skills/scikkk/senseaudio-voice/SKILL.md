# SenseAudio 语音功能

此功能用于语音资源的管理、权限控制以及与克隆操作相关的集成功能。

## 阅读前须知

- 请参考 `references/voice.md` 文件以获取更多详细信息。

## 工作流程

1. 确认目标语音资源的 `voice_id` 以及用户账户的权限等级。
2. 检查所需使用的语音资源是否受到权限限制。
3. 在进行克隆操作时，确保平台端的克隆流程已经完成。
4. 在启用多音素替换功能之前，验证相关字典和模型的约束条件。
5. 如果所需的语音资源不可用，应提供备用语音方案。
