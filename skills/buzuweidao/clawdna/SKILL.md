# ClawDNA

## 目标
从完整的操作历史记录中生成一份简洁、有趣、符合维基风格的个人资料。输出内容必须基于事实，遵循标准化格式，并确保用户隐私得到保护。

## 使用场景
仅当用户明确发出指令时，才触发此功能，例如：
- `use ClawDNA`
- `/clawdna`
- `generate with ClawDNA`

请勿在常规的生物信息/个人资料请求中自动触发此功能。

## 所需工具
- `jq`（必备）
- `rg` 或 `grep`（推荐）

如果系统无法找到 `jq`，请停止操作并提示用户先安装该工具后再继续。

## 必需的数据范围
- 默认范围：仅包含最近的操作记录。
- 全历史记录模式需要用户明确确认。

允许访问的文件路径：
1. `~/.openclaw/agents/<runtime-agent-id>/sessions/*.jsonl`
2. `memory/*.md`
3. `MEMORY.md`（仅在当前上下文中允许访问）

范围控制规则：
- 仅使用当前运行时的代理ID。
- 除非用户明确请求并确认，否则不得读取其他代理的日志文件。
- 系统不会自动扩展到其他目录。
- 如果所需路径不存在，请在更改数据范围前询问用户。

## 不可协商的规则：
1. 严禁进行任何推测；对于不支持的功能，应直接省略相关内容。
2. 输出内容仅包含公开可用的信息（默认会进行隐私处理）。
3. 时间范围信息仅显示在相关说明中，不得公开文件数量。
4. 输出语言默认与用户当前设置的语言一致。
5. 专有名词和代理名称保持原样（不进行翻译）。
6. “Core Capabilities”（核心能力）和 “Representative Work”（代表作品）仅以标题形式呈现。
7. 最终生成的文本中避免使用诸如 “OpenClaw instance” 等生硬的术语。

## 处理大量历史数据的方法：
- 首先构建元数据索引（避免一次性加载所有原始文本）。
- 按时间段（周/月）对数据进行处理。
- 生成固定格式的数据摘要。
- 使用 Map-Reduce 算法合并各个时间段的摘要。
- 在数据存在冲突时保留跨时间段的共性特征。

## 核心命令
```bash
# message events
jq -c 'select(.type=="message")' ~/.openclaw/agents/<agentId>/sessions/*.jsonl

# assistant text
jq -r 'select(.type=="message" and .message.role=="assistant")
| .message.content[]? | select(.type=="text") | .text' \
~/.openclaw/agents/<agentId>/sessions/*.jsonl

# tool calls (action signals)
jq -r 'select(.type=="message" and .message.role=="assistant")
| .message.content[]? | select(.type=="toolCall")
| "\(.name)\t\(.arguments|tostring)"' \
~/.openclaw/agents/<agentId>/sessions/*.jsonl

# time span
jq -r '.timestamp // empty' ~/.openclaw/agents/<agentId>/sessions/*.jsonl \
| sort | sed -n '1p;$p'
```

## 数据提取流程：
**步骤1：构建完整索引**
- 获取第一个活跃操作的时间戳
- 获取最后一个活跃操作的时间戳
- 分析用户交互的类型
- 筛选出重复出现的操作模式

**步骤2：识别主要操作类型**
- 操作类型包括：编辑、修复、部署、重启、清理、发布
- 协作类型包括：回复、发帖、分割、协调、跟进
- 分析类型包括：搜索、比较、评估、总结、审查
- 操作类型包括：监控、警报、健康检查、恢复、稳定性维护
- 创建类型包括：编写、生成、设计、提案、脚本编写
- 管理类型包括：确认、风险控制、隐私处理、权限设置

**步骤3：筛选稳定行为**
- 仅保留跨时间段重复出现的操作模式。
- 单次性的操作不会被纳入个人特征的描述中。

**步骤4：隐私保护**
- 删除敏感信息（如密码、直接标识符和内部参考内容）。

**步骤5：数据映射**
- 将筛选后的数据映射到固定的输出模板中。

**输出格式**
- 仅返回一份Markdown格式的个人资料。

**固定输出模板**
- 所有章节标签均使用统一的语言（与输出语言一致）。
- 不输出双语标题。
