**描述**  
这是一款功能强大的本地人工智能财务助手，可通过自然语言帮助您记录日常收支、查询交易记录并生成统计报告。所有数据均安全地存储在本地，无需连接互联网。  

**功能**  
1. **自然语言财务管理**  
  - 通过对话记录收支信息  
  - 自动识别金额、类别和日期  
  - 提供智能的类别建议  

2. **交易查询**  
  - 按时间范围（今日、本周、本月、本年等）查询交易  
  - 按类别过滤  
  - 支持关键词搜索  
  - 支持多条件组合查询  

3. **统计分析**  
  - 提供收支总额汇总  
  - 分类明细分析  
  - 计算每日平均支出  
  - 自动计算账户余额  

4. **报告生成**  
  - 生成详细的收支报告  
  - 显示各类支出排名  
  - 支持数据导出  

**使用示例**  
- **记录交易**：  
  “记录支出：午餐 35 元”  
  “记录收入：收到工资 15000 元”  
  “今天在电影票上花费了 120 元（类别：娱乐）”  
  “今天的交通费用为 20 元”  

- **查询交易**：  
  “显示本月的所有交易记录”  
  “我这周花了多少钱？”  
  “查询与餐饮相关的支出”  
  “查看上个月的收入情况”  

- **统计分析**：  
  “汇总本月的收支情况”  
  “今年的支出概况”  
  “查看我的支出类别分布”  

**可用命令**  
- `add_transaction`：添加交易记录  
  - 类型：支出或收入  
  - 金额：具体数值  
  - 描述：交易详情  
  - 类别：可选  
  - 标签：可选  

- `query_transactions`：查询交易记录  
  - 时间范围：指定查询周期  
  - 类型：过滤交易类型  
  - 类别：过滤交易类别  
  - 关键词：搜索相关交易  
  - 限制：显示结果数量  

- `get_statistics`：获取收支统计信息  
  - 时间范围：指定统计周期  

- `generate_report`：生成详细报告  
  - 报告周期：指定时间范围  

- `get_categories`：查看所有可用类别  

- `update_transaction`：更新交易记录  
  - 记录 ID：交易记录的唯一标识  

- `delete_transaction`：删除交易记录  
  - 记录 ID：需要输入交易记录的 ID  

**类别系统**  
- **支出类别**：  
  - 餐饮：餐费、外卖、饮品等  
  - 交通：公交、地铁、出租车、燃油等  
  - 购物：超市购物、网购、日常用品等  
  - 娱乐：电影、游戏、旅行等  
  - 医疗保健：看医生、药品、体检等  
  - 教育：学费、培训费用、书籍等  
  - 房产：房租、房贷、水电费等  
  - 其他：其他各类支出  

- **收入类别**：  
  - 工资：月薪、年终奖金等  
  - 奖金：年终奖金、绩效奖金等  
  - 投资：股票、基金、理财等  
  - 自由职业：兼职收入等  
  - 礼物：现金礼物、红包等  
  - 报销：费用报销、退款等  

**数据存储**  
  - 所有数据保存在本地 JSON 文件中  
  - 文件路径：配置的数据目录（如 `/accounting.json`）  
  - 自动备份：保留最近 30 份备份  
  - 数据格式开放，可随时导出  

**快捷命令**：  
  `/accounting.help`：显示帮助信息  
  `/accounting.stats`：快速查看本月的统计信息  

**注意事项**：  
1. 数据文件会在首次使用时自动生成。  
2. 请输入正数作为金额。  
3. 编辑或删除交易记录时需要提供记录 ID（可通过查询获取）。  
4. 请定期备份数据文件以防数据丢失。
