# OpenClaw AI 合作伙伴提示包  
（OpenClaw AI Coworker Prompt Pack）  

这是一个包含100多个精确提示的集合，用于使用OpenClaw构建高效的人工智能合作伙伴。  

## 快速安装  
1. 在OpenClaw中执行：`clawhub install openclaw-coworker-prompts`  
2. 使用`prompts/`目录中的某个提示创建一个新的代理（agent）。  
3. 为该代理分配相应的工具（如网络查询、代码编辑、执行等）。  

## 目录结构  
- `prompts_full.json`：包含所有100个提示及其分类的文件  
- `samples/`：包含可供直接使用的代理配置文件（格式为YAML/JSON）  
- `README.md`：详细的使用指南  

## 示例代理  
使用以下提示创建一个代理：  
```
You are a senior software engineer. Review pull requests for bugs, security issues, and best practices. Comment line-by-line with constructive feedback.
```  
（此处应插入具体的提示内容）  

**代理配置示例：**  
启用以下工具：`exec`、`web_search`、`read`、`edit`。
