# 投资备忘录生成

从收购方的角度生成结构化的投资备忘录。

## 报告视角

- 以收购方的立场撰写（使用“我们”作为主语）
- 介绍目标公司（使用公司名称或“目标公司”）
- 与收购方的投资理念和收购策略保持一致
- 提供全面客观的视角：包括机会与风险

## 备忘录结构

### 1. 执行摘要（1页）
- 交易概览：目标公司名称、行业、交易规模、交易结构
- 3-4个要点中的战略理由
- 关键财务指标（收入、息税折旧摊销前利润（EBITDA）、增长情况、估值）
- 建议：继续进行 / 有条件继续进行 / 放弃

### 2. 公司概况
- 公司业务描述及历史
- 产品/服务及收入构成
- 客户基础（数量、集中度、客户留存率）
- 管理团队评估
- 组织结构及员工人数

### 3. 市场与竞争地位
- 行业概况及增长前景（参考市场调研数据）
- 竞争格局及目标公司的市场定位
- 可持续的竞争优势
- 市场地位面临的主要风险

### 4. 财务分析
- 过去3-5年的财务数据：收入、EBITDA、利润率、自由现金流（FCF）
- 收入质量：经常性收入占比、客户集中度、客户群体分析
- 流动资金动态及现金周转情况
- 资本支出需求及资本密集度
- 主要财务趋势及转折点

### 5. 估值
- 现金流折现（DCF）分析（基准情景/乐观情景/悲观情景）
- 可比公司分析（公开上市公司数据）
- 以往交易案例分析
- 预计的估值范围及建议的收购价格
- 对关键假设的敏感性分析

### 6. 战略理由与协同效应
- 收入协同效应（交叉销售、市场扩张、定价策略）
- 成本协同效应（消除重复支出、共享资源）
- 实现协同效应的时间表（第1年/第2年/第3年）
- 整合的复杂性和风险评估
- 协同效应价值与支付溢价之间的比较

### 7. 风险评估
- 交易特有的风险（按影响程度和发生可能性排名前5项）
- 每项风险的应对策略
- 交易失败的临界条件
- 关键风险情景对回报的影响分析

### 8. 交易结构与回报
- 提议的交易结构（资产交换/股票交换、现金支付/股权支付）
- 资金来源及用途
- 预计的杠杆率和偿债比率
- 预期回报：内部收益率（IRR）、资本回报率（MOIC）、回收期
- 影响回报的关键假设

### 9. 建议与下一步行动
- 明确的建议及信心程度
- 需要解决的条件或尽职调查事项
- 下一步工作的时间安排
- 所需的审批流程及步骤

## 数据收集策略

1. 从平台获取现有的尽职调查（DD）数据（快速，<1秒）
2. 通过RAG查询补充上传文件中的缺失信息（中等速度，约3秒）
3. 通过网络研究补充市场背景信息（较慢，约5秒）
4. 利用人工智能技术将数据整合成叙述性内容（约5-10秒）

## 质量标准

- 所有陈述均需有数据支持，并注明数据来源
- 财务数据在各部分之间必须一致
- 清晰区分事实与假设
- 对预测性内容使用条件性语言（例如“我们估计”、“管理层预测”）
- 明确标注数据缺失情况，而非用通用文本填补
- 总生成时间目标：15-20秒

## 输出格式

- **Markdown**：平台显示和编辑的主要格式
- **PDF**：用于内部沟通的专业格式
- **PPTX**：用于交易委员会会议的演示格式
- **Excel**：包含财务模型和敏感性分析的辅助文件
