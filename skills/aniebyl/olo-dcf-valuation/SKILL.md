# 用于并购尽职调查的DCF估值方法

在并购场景中，使用折现现金流（DCF）模型对目标公司的价值进行评估。

## 单位系统（非常重要）

- **存储单位**：绝对金额（原始财务数据）
- **计算单位**：百万（DCF模型通常使用此单位）
- **显示格式**：根据数值大小采用智能格式化（例如，使用“百万/十亿/千亿”等）
- 在进行DCF计算之前，务必先对存储的原始数据进行单位转换

## 计算前的验证

1. 确认基础收入存在且不为零
2. 验证加权平均资本成本（WACC）在5%至25%之间（对异常值进行标记并说明原因）
3. 终值增长率必须低于WACC（符合戈登增长模型（Gordon Growth Model）的约束条件）
4. 预测期：5至10年（默认为5年）
5. 验证息税折旧及摊销前利润（EBITDA）利润率是否在行业合理范围内

## 默认假设

| 参数 | 默认值 | 原因 |
|---------|---------|-----------|
| 税率 | 21%（美国）/ 17%（新加坡） | 根据所在地区进行调整 |
| 资本支出占收入的百分比 | 5% | 根据行业情况调整（SaaS行业约为3%，制造业约为8-12%） |
| 终值增长率 | 2.5% | 不应超过长期GDP增长率 |
| WACC | 使用资本资产定价模型（CAPM）计算；若无法确定，则使用10-12%作为替代值 |
| 折旧率 | 资本支出的百分比 | 根据行业资本密集度进行调整 |
| 流动资金变动 | 收入变化的百分比 | 如有历史数据，则使用历史平均值 |

## WACC的计算（使用CAPM模型）

```
Cost of Equity = Risk-Free Rate + Beta × Equity Risk Premium
WACC = (E/V × Cost of Equity) + (D/V × Cost of Debt × (1 - Tax Rate))
```

- 无风险利率：10年期国债收益率
- 股票风险溢价：5-7%（根据Damodaran的研究）
- Beta系数：根据目标公司的资本结构，使用同类上市公司的Beta值（未杠杆化/杠杆化情况）
- 规模溢价：对于小型/中型目标公司，额外增加2-4%的Beta系数

## 预测方法

1. **收入**：从最近一次报告的数据开始，应用相应的增长率（逐渐趋于零）
2. **息税折旧及摊销前利润（EBITDA）**：根据行业平均水平调整利润率
3. **自由现金流（Free Cash Flow）**：EBITDA - 税收 - 资本支出 - 流动资金变动
4. **终值**：使用戈登增长模型或退出倍数法（Exit Multiple）计算
5. **折现率**：将WACC应用于每年的自由现金流及终值

## 终值计算

在并购评估中，建议使用**退出倍数法**（Exit Multiple method）：
- 将企业价值（Enterprise Value）除以终年EBITDA得到退出倍数
- 与戈登增长模型预测的终值增长率进行交叉验证
- 如果预测的永续增长率超过3%，则需予以标记

## 必须提供的输出结果

- **企业价值**范围（最低值/平均值/最高值）
- **股权价值**（企业价值减去净债务）
- **隐含的EV/EBITDA倍数**（与同类公司进行合理性对比）
- **敏感性分析表**：WACC与终值增长率或退出倍数的对应关系
- 如果使用多种估值方法，还需提供**对比图表**

## 并购特有的考虑因素

- 如果评估的是收购整家公司而非少数股权，应加上**控制溢价**（20-40%）
- **协同效应价值**应单独考虑，不与独立DCF估值合并
- 折现率应反映**买方的资本成本**，而非目标公司的成本
- 将**整合成本**作为协同效应价值的一部分进行扣除
- 需分别提供包含协同效应和不含协同效应的估值结果
