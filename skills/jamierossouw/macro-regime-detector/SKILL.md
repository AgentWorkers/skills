# 宏观经济状态检测器

实时检测宏观经济状态，以辅助加密货币和传统资产的投资决策。

## 可检测的宏观经济状态

| 状态 | 描述 | 加密资产投资策略 |
|--------|-------------|-------------------|
| **A — 风险偏好上升（牛市）** | VIX指数 < 15，恐惧与贪婪指数（F&G）> 60，美元指数（DXY）下跌 | 最大程度持有多头头寸，杠杆使用可接受 |
| **B — 中性/市场冷却** | 信号混合，市场盘整 | 选择性持有多头头寸，设置更严格的止损点 |
| **C — 风险偏好下降（熊市）** | VIX指数 > 25，恐惧与贪婪指数（F&G）< 25，美元指数（DXY）上涨 | 减少持仓，进行对冲 |
| **D — 通货膨胀加剧** | 消费者价格指数（CPI）意外上涨，黄金/比特币价格飙升 | 将比特币作为通胀对冲工具 |
| **E — 滞胀** | 通货膨胀高企 + 经济衰退担忧 | 采取防御性策略：持有比特币、黄金或现金 |
| **F — 流动性危机** | 信用利差急剧扩大，资产相关性极高 | 全部退出风险投资，仅持有稳定币（如USDC） |

## 信号来源

- **恐惧与贪婪指数（alternative.me/fng）**：0-100分，用于判断宏观经济状态 |
- **Reddit论坛动态**：r/CryptoCurrency、r/investing板块的热门帖子，以及看跌/看涨情绪的比例 |
- **美元指数（DXY）**：通过比特币/美元的负相关关系来评估市场趋势 |
- **新闻自然语言处理（News NLP）**：监测重要的经济事件（如美联储政策、CPI数据、非农就业数据、美国证券交易委员会（SEC）的动态） |
- **区块链数据**：比特币的交易流量、矿工的退出信号等链上数据 |

## 使用方法

提问：“我们目前处于哪种宏观经济状态？”或“我现在应该买入还是卖出加密货币？”

输出结果包括：当前所处的宏观经济状态、置信度百分比、3个支持当前状态的信号，以及相应的投资建议。
