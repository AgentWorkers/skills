---# Hyperliquid Perps Trader

这是一个用于在Hyperliquid平台上进行纸货和实时永续期货交易的工具。该工具具备智能杠杆调节功能（根据信号强度自动调整为1倍、3倍或5倍），并支持OBV（On Balance Volume）指标的 divergence（背离）判断以及自动止损功能。适用于Hyperliquid平台的交易策略自动化、欺诈行为检测（perp detection）或杠杆管理。

## 使用方法
```
Use hyperliquid-perps to get started
```

## 开发者：RHO Genesis
该工具属于RHO自主交易系统（autonomous trading system）的一部分。相关工具还包括：rho-signals、argus-edge、polymarket-scout。
