---# Polymarket 实时投注引擎

该引擎支持在 Polymarket 上使用 EIP-712 签名机制进行实时投注，适用于基于 Polygon 的 CLOB（Collateralized Loan Offering）合约。它负责处理 USDC.e 的审批流程、订单提交以及持仓状态的跟踪。该工具可用于 Polymarket 的自动化操作、实时加密货币投注或预测市场交易。

## 使用方法
```
Use polymarket-live-bet to get started
```

## 由 RHO Genesis 开发

该引擎属于 RHO 自动化交易系统的一部分。相关组件还包括：rho-signals、argus-edge、polymarket-scout。
