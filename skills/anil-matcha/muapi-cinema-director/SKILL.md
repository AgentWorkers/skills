# 🎬 AI 电影导演技能

**这是一项专门用于 AI 代理的技能，旨在指导制作高保真度的电影视频。**  
该技能能够将高层次的创意意图转化为技术性的电影指导指令，以便应用于最先进的视频模型（如 Veo3、Kling、Luma）。

## 核心能力

1. **镜头构图分析**：将情感节奏与合适的拍摄构图相匹配（例如，使用特写镜头来表现亲密感，使用广角镜头来营造孤立感）。  
2. **摄像机运动编排**：指挥复杂的物理摄像机运动（如移动摄影车、云台、起重机）以及基于镜头的特效（如焦距调整、推拉镜头）。  
3. **灯光与氛围设计**：指定特定的灯光风格和时间效果（如黄金时刻的光线、明暗对比、立体光效果）。  
4. **技术参数优化**：自动选择最佳的帧率、宽高比以及适合各模型的设置参数。

## 🏗️ 技术规格

### 1. 意图映射表

| 创意意图 | 构图 | 摄像机运动 | 灯光 |
| :--- | :--- | :--- | :--- |
| **英雄式的场景呈现** | 低角度/广角镜头 | 使用起重机向上拍摄/环绕拍摄 | 边缘照明/高对比度 |
| **紧张/不安的氛围** | 荷兰角镜头 | 手持拍摄并抖动镜头 | 低调的光线/强烈的阴影效果 |
| **内省的场景** | 特写镜头 | 慢速推近镜头 | 温和的伦勃朗式光线/窗口光线 |
| **宏伟/史诗般的场景** | 极广角镜头 | 无人机航拍 | 黄金时刻的光线/立体光效果 |
| **忧郁的氛围** | 侧面镜头/中景镜头 | 慢速拉远镜头 | 蓝调光线/饱和度较低的灯光 |

### 2. 摄像机物理运动  
- **移动摄影车（Dolly In/Out）**：摄像机沿轨道向主体移动或远离主体。  
- **横向移动摄影车（Truck Left/Right）**：摄像机横向移动。  
- **使用起重机/云台（Crane/Jib）**：从高处进行垂直移动。  
- **环绕拍摄（Orbit）**：围绕中心点进行圆形移动。  
- **升降平台（Pedestal）**：仅改变摄像机的垂直高度（不倾斜）。

### 3. 镜头与光学控制  
- **浅景深（Shallow DOF）**：背景模糊（散景效果）。  
- **变形镜头（Anamorphic）**：产生水平光晕，营造宽屏电影效果。  
- **焦距调整（Rack Focus）**：在镜头内切换不同的焦平面。

## 🧠 提示优化协议（代理指令）

**在调用脚本之前，代理必须根据以下导演规则对用户的原始提示进行扩展：**

1. **技术转化**：将“酷炫的动作”转化为具体的拍摄指令。  
   - **公式**：`[镜头类型] + [主体/动作] + [环境] + [灯光] + [摄像机运动] + [镜头效果]`  
2. **动态效果**：使用电影相关的动词来描述动作（如“Dolly In”表示亲密感，“Crane Up”表示宏伟的场景，“Orbit”表示环绕拍摄）。  
3. **灯光设置**：应用特定的灯光效果（如“立体光效果”、“蓝橙色调的光线”、“赛博朋克风格的边缘照明”、“伦勃朗式人像照明”）。  
4. **光线描述**：详细描述光线关系（例如“霓虹灯光在雨后的柏油路上闪烁”），以触发模型的自动处理。

## 🚀 使用方法

### 第一步：定义创意要求  
向代理提供拍摄主题和导演的创意意图。

### 第二步：调用脚本  
`generate-film.sh` 脚本接受一个 `--brief` 参数，并根据其内置的电影知识来扩展该提示内容。

```bash
# Directing a scene
bash scripts/generate-film.sh 
  --subject "A lone samurai in a blizzard" 
  --intent "epic reveal" 
  --model "kling-master"
```

### 第三步：处理异步响应  
视频生成是异步进行的。使用返回的 `request_id`，通过 `core/platform/check-result.sh` 命令来检查生成进度。

## ⚠️ 限制与规则  

- **时间一致性**：避免在一个镜头中同时使用复杂的场景转换（例如，角色突然变成鸟类）。  
- **运动冲突**：不要同时使用相互矛盾的摄像机运动（例如，同时使用“Dolly In”和“Dolly Out”）。  
- **物理真实性**：优先选择使用现实世界设备能够实现的运动方式，以获得更专业的电影效果。  
- **模型适用性**：  
  - `Veo3`：最适合拍摄缓慢、高质量的视觉效果。  
  - `Kling`：最适合表现复杂的角色动作和物理效果。  
  - `Luma`：最适合拍摄快节奏、动作激烈的电影场景。
