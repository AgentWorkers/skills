# 🖼️ 标志设计技能

**这是一项专为AI代理设计的专业技能，用于打造专业级别的品牌标识。**  
该技能能够将品牌愿景转化为简约、可扩展且具有标志性的矢量风格图标。

## 核心能力  

1. **基本几何元素构建**：使用圆形、正方形、三角形等基本形状来创建易于记忆的图标。  
2. **负空间运用**：通过巧妙地运用空白区域来传达额外的含义。  
3. **符号抽象**：将复杂的品牌概念简化为最直观的视觉形式。  
4. **可扩展性考量**：确保设计在16像素（如网站图标）到大型广告牌等不同尺寸下仍保持清晰可读。  

---

## 🏗️ 技术规范  

### 1. 标志分类表  

| 标志类型 | 风格 | 适用对象 | 重点 |
| :--- | :--- | :--- | :--- |
| **具象标志** | 象征性 | 成熟品牌 | 强烈的识别度 |
| **抽象标志** | 几何风格 | 科技/现代品牌 | 概念/情感表达 |
| **字母标志/缩写标志** | 文字风格 | 长名称品牌 | 首字母/字体设计 |
| **徽章** | 传统风格 | 机构/咖啡馆 | 详细设计/徽章用途 |
| **吉祥物** | 字符风格 | 社区/游戏品牌 | 个性化表达 |

### 2. 品牌设计指导原则  
- **色彩理论**：使用高冲击力的单色或双色调配色方案，以增强视觉效果。  
- **字体搭配**：将标志与简洁的几何无衬线字体（如Gotham、Helvetica）搭配使用。  
- **设计风格**：仅采用扁平设计，禁止使用渐变、3D效果或写实纹理。  

---

## 🧠 提示优化协议（代理使用说明）  

**在调用脚本之前，代理必须将品牌愿景细化为以下几何简约设计要求：**  
1. **基本元素**：将设计概念简化为圆形、正方形、三角形等基本形状。  
2. **负空间运用**：明确指示模型通过空白区域来表达额外含义（例如：“字母之间的空隙可形成箭头”。  
3. **设计限制**：仅使用纯色、单色方案以及扁平设计，禁止使用渐变或3D效果。  
4. **布局要求**：确保标志始终在白色背景上居中显示，以实现最佳的可扩展性。

## 🚀 使用流程  

### 第一步：定义品牌信息  
向代理提供品牌名称、核心价值观及所属行业。  

### 第二步：执行脚本  
`create-logo.sh`脚本将生成完整的品牌设计文档。  

```bash
# Designing a Fintech Logo
bash scripts/create-logo.sh \
  --brand "Aura" \
  --concept "geometric security shield with a spark" \
  --style minimalist \
  --color "black and gold"
```  

---

## ⚠️ 限制与注意事项  

- **设计要求**：**必须**使用纯白色或黑色背景，禁止使用任何纹理、桌面元素或样图（如T恤、名片等）。  
- **可读性**：避免使用过细的线条或复杂的重叠元素，以防止在小尺寸下出现显示问题。  
- **简约原则**：标志设计应仅包含最多3个核心几何元素。  
- **文字呈现**：使用Flux工具确保品牌名称的准确显示。
