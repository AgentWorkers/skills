# 创建 R 研究项目

此技能用于帮助用户创建和管理基于 R 的完整研究分析工作流程，适用于科学计算、统计建模、生物信息学和数据可视化等领域。

当用户需要执行以下操作时，可以使用此技能：
- 使用 R 分析数据集
- 进行统计测试或建模
- 在 R 中进行生物信息学或组学分析
- 生成图表或报告
- 创建可复现的 R 项目结构
- 安装和管理 R 包的依赖关系

---

## 该技能的功能

激活此技能后，它会执行以下操作：
1. **创建结构化的 R 项目**：
   - `data/`：存放原始数据和处理后的数据
   - `scripts/`：存放分析代码
   - `results/`：存放分析结果
   - `reports/`：存放 R Markdown 或 Quarto 格式的报告

2. **设置开发环境**：
   - 初始化 `.Rproj` 文件（如果使用 RStudio）
   - 创建 `renv` 环境以确保分析结果的可复现性
   - 安装所需的 CRAN 或 Bioconductor 包

3. **生成分析脚本**：
   - 数据加载与清洗
   - 统计分析或建模
   - 使用 `ggplot2` 进行数据可视化
   - 保存分析结果（CSV、图表、模型摘要）

4. **生成报告**：
   - 使用 R Markdown 或 Quarto 编写报告
   - 报告中包含分析方法、结果和图表
   - 可将报告渲染为 HTML 或 PDF 格式

## 常用工具及包

| 功能 | R 包            |
|--------|-------------------|
| 数据处理 | tidyverse, data.table     |
| 数据可视化 | ggplot2, patchwork     |
| 统计分析 | stats, lme4, survival     |
| 生物信息学 | Bioconductor 包（如 DESeq2, edgeR, limma） |
| 报告生成 | rmarkdown, quarto       |
| 可复现性 | renv            |

---

## 注意事项：
- 尽量使用可复现的工作流程（如 `renv`）和脚本化分析方法
- 除非特别要求，否则避免使用仅依赖交互式界面的功能
- 所有分析结果都应保存为文件，而不仅仅是输出到控制台
