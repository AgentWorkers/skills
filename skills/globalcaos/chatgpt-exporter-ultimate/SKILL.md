# ChatGPT Exporter ULTIMATE

> 🔗 **属于 OpenClaw 生态系统** — 该功能是 OpenClaw 人工智能代理升级项目的一部分。
> 完整项目：https://github.com/openclaw/openclaw

**特点：**  
可在几秒钟内导出所有 ChatGPT 对话记录——无需等待 OpenAI 发送的 24 小时导出邮件。

## 使用要求：  
1. 用户需要通过浏览器代理将 Chrome 浏览器中的 ChatGPT 标签页关联到 OpenClaw 代理。  
2. 用户必须已登录 ChatGPT 账户。

## 工作原理：  
1. **关联浏览器标签页**：用户在 chatgpt.com 页面的 ChatGPT 标签页上点击 OpenClaw 工具栏图标。  
2. **注入脚本**：代理会自动在后台注入导出脚本。  
3. **获取对话记录**：脚本会从 ChatGPT 的主接口获取所有对话记录。  
4. **查找项目内的对话记录**：脚本使用宽泛的搜索关键词来查找隐藏在特定项目或文件夹中的对话记录（这些记录在主接口中不可见）。  
5. **下载数据**：脚本会下载每条对话记录的完整内容，并将其保存到用户的下载文件夹中。

### 为何采用基于搜索的机制？  
ChatGPT 的 `/backend-api/conversations` 接口仅返回未包含在项目中的对话记录。而在项目中创建的对话记录（例如 “Feina”、“Receptes” 等）在该接口中是不可见的。因此，我们使用宽泛的搜索关键词来找到这些记录。

## 技术细节：  
### 认证  
ChatGPT 的内部 API 需要从 `/api/auth/session` 获取访问令牌。  
（相关代码块未提供。）

### API 接口：  
| 接口          | 功能                          |
| ------------------------- | ----------------------------- |
| `/api/auth/session`    | 获取访问令牌                    |
| `/backend-api/conversations?offset=N&limit=100` | 列出所有对话记录                |
| `/backend-api/conversation/{id}`    | 获取单条对话记录的完整内容           |

### 导出脚本  
代理会注入一个可自动运行的脚本，该脚本执行以下操作：  
1. 获取访问令牌。  
2. 分页获取所有对话记录（每页 100 条）。  
3. 下载每条对话记录的完整内容。  
4. 从数据结构中提取对话信息。  
5. 将数据转换为 JSON 格式并触发下载。

## 速率限制：  
- 每次获取对话记录与执行搜索查询之间有 100 毫秒的延迟。  
- 第一阶段（列出所有对话记录）：约 2 秒。  
- 第二阶段（搜索并发现隐藏对话记录）：约 30 秒（搜索约 50 个关键词）。  
- 第三阶段（下载每条对话记录）：每条对话记录约 100 毫秒。  
- 全部约 250 条对话记录的导出时间：约 3–4 分钟。

## 故障排除：  
| 故障现象        | 解决方案                          |
| ------------------------- | -------------------------------------------- |
| 无法关联标签页      | 在 ChatGPT 标签页上点击 OpenClaw 工具栏图标。         |
| 出现 401 错误      | 重新登录 ChatGPT 并重新关联标签页。             |
| 导出失败        | 检查浏览器控制台是否有错误信息。                 |
| 未下载文件      | 检查下载文件夹或浏览器设置。                   |
