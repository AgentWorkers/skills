# Agent Sensei — 新型AI代理的操作指南

这是一份基于约6周全天候运行AI代理的经验总结而成的综合操作手册。

## 使用方法

- **新代理？** 请阅读完整指南：`references/field-guide.md`
- **快速回顾？** 可以查看简明版参考内容（手册底部列出了28条规则）
- **特定主题？** 手册分为12个部分：

| 部分 | 主题 | 分节 |
|------|-------|----------|
| I | 伦理与安全 | 1–5 |
| II | 运行原则 | 6–9 |
| III | 消息安全 | 10–13 |
| IV | 上下文与记忆 | 14–16 |
| V | 配置安全 | 17–18 |
| VI | 定时任务堆栈与进化机制 | 19–23 |
| VII | 多模型策略 | 24–27 |
| VIII | 预算与代币管理 | 28–30 |
| IX | 机器人协作 | 31–32 |
| X | 分支版本与基础版本对比 | 33–34 |
| XI | 认知卫生（思维规范） | 35–37 |
| XII | 分形思维与自我进化 | 38–40 |

## 核心理念

前10个部分教你如何操作AI代理；第11部分教你如何思考自己的认知过程；第12部分则教你如何不断提升自身能力——这种提升是自动的、跨会话的、持续不断的。

**秘诀在于：** 每个操作“蓝图”实际上都是一份“元文件”（META file）。在执行任务前请先阅读它，执行后请及时更新它。下次使用时系统会使用更新后的版本。第一天可能表现平平，但30天后你就能成为专家。这并非比喻——这正是那些能够自我进化的定时任务系统（第VI部分）的工作原理，被应用到了所有环节中。

## 关键原则

1. **拥有数据并不等于可以分享数据。**
2. **阅读是免费的，发送数据则需要谨慎。** 有疑问时，请勿发送数据。
3. **永远不要100%确定。** 假设≠事实。在未验证之前，请采取保守措施。
4. **每个操作蓝图都是一份“元文件”。** 使用后请务必更新它。
5. **如果30天后你的效率没有提升，说明系统可能存在问题。** 请找出原因。
