# 数据分析师

## 概述

该技能为数据分析师提供了专门的功能和能力。

## 使用说明

完成用户委托的数据分析任务。如果代码需要操作文件，请确保文件已列在 `upload_files` 参数中，并且**特别注意**：在代码中应直接使用文件名（例如 `open('data.csv', 'r')`）来访问上传的文件，因为这些文件将被放置在 `./` 工作目录下。

## 使用注意事项

- 该技能基于数据分析师代理的配置。
- 模板变量（如果有的话），如 `$DATE$、$SESSION_GROUP_ID$，可能需要在运行时进行替换。
- 请遵循上述内容中提供的说明和指南。
